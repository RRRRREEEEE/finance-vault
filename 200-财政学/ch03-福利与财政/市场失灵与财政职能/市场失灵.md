# 市场失灵 (Market Failure)

## 📌 定义
市场失灵是指市场机制无法自行实现资源有效配置的情形。福利经济学第一定理的成立依赖完全竞争假设——当这些前提不成立时，政府干预就有了理论依据。

## 🧠 机制

### 六种市场失灵类型

| 类型         | 内容                 | 后果                                               |
| ---------- | ------------------ | ------------------------------------------------ |
| **垄断**     | 市场竞争的自然结果——优胜者形成垄断 | 价格>边际成本，产量低于最优                                   |
| **外部性**    | 生产或消费的外部效应（正/负）    | 私人成本≠社会成本                                        |
| **信息不对称**  | 交易双方信息不对等          | [[Adverse Selection 逆向选择]]、[[Moral Hazard 道德风险]] |
| **有限理性**   | 人的认知和决策能力有限        | 机会主义、资产专用性                                       |
| **收入分配不公** | 效率与公平的复杂关系         | 社会矛盾、福利损失                                        |
| **宏观经济波动** | 经济周期、失业、通胀         | [[不可能三角]]约束                                      |

## 🏛️ 应用

市场失灵直接导出[[财政三大职能]]：
- 垄断 + 外部性 + 信息不对称 → **资源配置职能**
- 收入分配不公 → **收入分配职能**
- 宏观经济波动 → **经济稳定职能**

## 📄 来源原文
> 根据福利经济学第一和第二定理，政府的主要职责，是对要素进行初始分配，然后让市场去发挥作用，自然就会达至资源的有效配置。但实际的情况就比较复杂：垄断、外部性、信息不对称、有限理性、收入分配不公、宏观经济波动。
