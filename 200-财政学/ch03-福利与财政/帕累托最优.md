# 帕累托最优 (Pareto Optimum)

## 📌 定义
帕累托最优（帕累托效率）是一种资源配置状态：在此状态下，任何重新配置都无法在不使至少一个人处境变坏的前提下使至少一个人处境变好。位于福利边界 WW' 曲线上。

> 💬 通俗理解：桌上有一块蛋糕，两个人分。帕累托最优就是——你想多拿一块，就必须从对方手里抢一块。没有任何浪费，但也无法"双赢"了。

## 🧠 机制

### 几何表述
- WW' 曲线 = 福利边界（Welfare Frontier）
- 曲线右侧：任何资源配置都无法达到
- 曲线左侧：资源配置低效率
- 曲线上的点 = 帕累托最优状态

### 实现条件
通过埃奇沃思盒状图分析，帕累托最优要求：
- **交换效率**：所有消费者的边际替代率相等（MRS_A = MRS_B）
- **生产效率**：所有生产要素的边际技术替代率相等
- **全局效率**：边际替代率 = 边际转换率（MRS = MRT）

在局部均衡中可简化为：**每一种产品的社会边际成本 = 其社会边际收益（MSC = MSB）**

## ⚖️ 易混点对照
- 帕累托最优 ≠ 公平：P3和P5都是帕累托最优点，但分配结构完全不同
- 帕累托最优只关心效率，不关心谁得多少

## 📄 来源原文
> E点，以及WW´曲线上的点，被称为帕累托最优（Pareto Optimum）状态——一部分人处境的改善必须以另一部分人处境的恶化为代价。从D点到E点的变化，被称为帕累托改进（Pareto Improvement）——资源配置的变化符合：1.至少有一个人处境变好，2.没有一个人处境变坏。在完全竞争的市场条件下，经济会"自动地"实现有效的资源配置，无须任何中央集权式指导。
