# 财政三大职能 (Fiscal Functions)

## 📌 定义
经济学认为现代财政有三项重要职能：资源配置、收入分配、经济稳定。它们分别对应市场失灵的三种类型——政府通过收支行为弥补市场缺陷。

## 🧠 机制

### 一、资源配置职能

核心问题：市场与政府的关系（十八届三中全会：市场在资源配置中起决定性作用）

| 内容     | 手段                  |
| ------ | ------------------- |
| 矫正外部性  | 正外部性补贴、负外部性征税       |
| 提供公共产品 | 政府直接供给（解决正外部性的一种模式） |
| 管制自然垄断 | 价格管制、国有化、引入竞争       |

**机制与手段**：调整资源在政府和私人部门间的配置、调整财政支出结构、运用财政收支政策、改善财政配置效率。

### 二、收入分配职能

核心问题：效率与公平的关系。通过以下手段实现：

| 手段 | 内容 |
|------|------|
| **税收机制** | 所得税调整个人收入和企业利润；财产税调整财富水平；流转税调整产品相对价格 |
| **政府资产性收入** | 介入实体经济，以股息、红利、利息、利润取得收入 |
| **工资补贴支出** | 公营事业工资、明补和暗补 |
| **社会保障制度** | 养老、医疗、生育、工伤、失业、住房等基础保障——极大地改变了市场经济的分配格局 |

### 三、经济稳定职能

包含：经济增长、充分就业、物价稳定、国际收支平衡

| 手段 | 内容 |
|------|------|
| 反周期财政货币政策 | 萧条时扩张，过热时紧缩 |
| 财政自动稳定器 | 累进税制、失业保险等自动调节 |
| 产业政策配合 | 财政引导产业结构调整 |
| 不可能三角 | 固定汇率、货币政策独立性、资本自由流动——三者不可兼得 |

### 财政职能的局限性

- 政府信息的有限性
- 政府决策的滞后性
- 政治过程的复杂性
- 政府官员的代理问题
- 财政活动对市场的**挤出效应**
- 财政活动对收入分配的复杂影响

## 📄 来源原文
> 市场运行中会出现各种无效率的情形，政府和财政的职能，有助于弥补这种无效率情形的出现。经济学认为，现代财政至少有以下三项重要职能：资源配置、收入分配、经济稳定。资源配置的核心问题，是市场和政府之间的关系。收入分配要解决的核心问题，是效率和公平之间的关系。经济稳定包含经济增长、充分就业、物价稳定和国际收支平衡等多重含义。
