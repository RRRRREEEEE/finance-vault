# 第三讲 知识串讲：市场做对了什么、做错了什么——福利经济学如何为财政奠基

## 一、一句话回答

**福利经济学画出"市场最优"的图，然后告诉你：这图很美，但现实中画不出来。政府的手就从这些画不出来的地方伸进来。**

---

## 二、这一讲在整门课的位置

财政学前面三讲：第一讲告诉你财政怎么来的（历史），第三讲告诉你财政凭什么存在（理论）。如果没有第三讲的论证——市场为什么会失灵、公平为什么需要修正——后面八讲讨论的税收、支出、预算、政府干预，全是无根之木。

第三讲 = 财政学的理论基石。

## 三、上半场：市场能做到的最好

### 3.1 两个剩余

市场交易产生两个东西——**消费者剩余**和**生产者剩余**。前者是你愿意付但没付的钱，后者是你愿意接受但多赚的钱。加起来就是**社会福利**（总剩余 = CS + PS）。

为什么关心这个？因为政府做的几乎每件事——征税、管制、补贴、贸易政策——都在动这块蛋糕。理解剩余，才能理解谁赚了、谁赔了。

### 3.2 帕累托最优：不能更好的状态

**帕累托最优**：一种状态，要让任何一个人变好，就必须让另一个人变差。

**帕累托改进**：有人变好，没人变差。

注意：帕累托最优不关心公平。一个富人有 99、穷人有 1，只要不能再调整而不伤富人，它就是帕累托最优。这个盲点正是下半场"公平"要解决的事。

### 3.3 三个效率条件

**埃奇沃思盒状图**用两个人的无差异曲线描述了三个效率条件：

| 效率类型 | 条件 | 含义 |
|------|------|------|
| 交换效率 | MRS_A = MRS_B | 两人对两种商品的边际替代率相等 |
| 生产效率 | MRTS_K = MRTS_L | 两种要素的边际技术替代率相等 |
| 混合效率 | MRT = MRS | 生产成本等于消费意愿 |

三个条件全满足时，经济处于**帕累托最优的局部均衡**：MSB = MSC（边际社会收益等于边际社会成本）。

### 3.4 两个福利定理

- **第一定理**：完全竞争市场自动达到帕累托最优。市场是有效率的。
- **第二定理**：任何帕累托最优都可以通过市场实现——只要初始禀赋分配得当。

关键含义：**效率交给市场，公平交给分配。** 政府不需要替代市场（第一定理），但可以通过再分配达到想要的结果（第二定理）。

## 四、下半场：公平为什么需要修正

### 4.1 效率是尺子，公平是选择

效率是客观的（计算剩余就知道），公平是主观的（你觉得什么算"公正"？）。

三种社会福利函数给出了三种不同答案：

| 函数 | 核心原则 | 政策含义 |
|------|------|------|
| **功利主义** | 总福利最大化 | 从富人拿一块钱给穷人，如果边际效用递减，总福利增加 |
| **罗尔斯主义** | 最差者优先 | 只看社会最底层过得好不好，不平等无所谓 |
| **平均主义** | 人人完全平等 | 差距本身就是坏的，不需要计算效率损失 |

现实中的社会政策在这三者之间摇摆——**钟摆效应**：今天偏向效率，明天偏向公平，但不会完全倒向任何一端。

### 4.2 社会无差异曲线

**效用可能性曲线**：给定一个人的效用水准，另一个人能到的最高效用。**社会无差异曲线**叠上去找到切点，那才是"又有效率又公平"的点。

## 五、市场失灵：政府登场的理由

市场不是万能的。六种失灵：

| 失灵类型 | 例子 |
|------|------|
| 垄断 | 一个卖家控制价格 → 产量不足、价格偏高 |
| 外部性 | 你污染了河、下游的人倒霉（第四讲展开） |
| 信息不对称 | 你分不出好产品和坏产品（柠檬问题） |
| 有限理性 | 人不是完全理性的计算器 |
| 机会主义 | 签完合同就变脸（道德风险） |
| 资产专用性 | 投资一旦做了就收不回来 |

### 5.1 财政三大职能

因为市场会失灵，所以政府有三件事要做：

| 职能 | 做什么 | 例子 |
|------|------|------|
| **资源配置** | 矫正外部性、提供公共产品、管制垄断 | 庇古税、国防、反垄断 |
| **收入分配** | 用税收和转移支付缩小差距 | 个人所得税累进、低保 |
| **经济稳定** | 反周期政策、自动稳定器 | 经济衰退时增加支出 |

### 5.2 职能的边界

- **挤出效应**：政府花钱多了，私人投资少了。1 块钱政府支出 ≠ 1 块钱净增长。
- **不可能三角**：固定汇率、资本自由流动、独立货币政策——三者只能取其二。
- **有机论 vs 机械论**：社会是一个有机生命体（政府是大脑）→ 政府做更多；社会是机器的集合（政府是修理工）→ 政府做更少。

## 六、概念对照表

| 概念对 | 区别 |
|------|------|
| **帕累托最优 vs 帕累托改进** | 前者是不能再改了（改就伤人），后者是还能改（改了人人获益） |
| **第一定理 vs 第二定理** | 第一定理说"市场有效率"，第二定理说"公平可以通过再分配实现" |
| **MRS vs MRT vs MRTS** | MRS=消费替代（我愿意怎么换），MRT=生产转换（技术上怎么换），MRTS=要素替代（资本换劳动） |
| **功利主义 vs 罗尔斯主义** | 功利主义追求总和最大（允许一部分人极差），罗尔斯追求最差者最好（允许所有人不差但也不富） |
| **资源配置 vs 收入分配** | 前者是做大蛋糕（效率逻辑），后者是切蛋糕（公平逻辑） |

---

## 七、最重要的故事：从帕累托最优到财政职能的完整论证链

```
完全竞争市场 → 自动达到帕累托最优（第一定理）
                          ↓
           但你想要任何分配结果都可以通过市场+再分配实现（第二定理）
                          ↓
           但市场会失灵：垄断、外部性、信息不对称……
                          ↓
       政府登场 → 三大职能：资源配置 + 收入分配 + 经济稳定
                          ↓
           但政府也会失灵：挤出效应、不可能三角……
                          ↓
           第四讲开始讨论：具体怎么应对每一种市场失灵
```
