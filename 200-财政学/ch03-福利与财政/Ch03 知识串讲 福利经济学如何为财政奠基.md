# 第三讲 知识串讲：福利经济学如何为财政奠基

## 一、一句话回答

**福利经济学画出"市场最优"的图，然后告诉你：这图很美，但现实中画不出来——因为垄断、外部性、信息不对称、公平诉求都会让市场偏离最优。政府的手就从这些画不出来的地方伸进来。财政三大职能——资源配置、收入分配、经济稳定——全部根植于福利经济学的论证。**

---

## 二、这一讲在整门课的位置

PPT 开篇就点明了现代财政学的基本思路：

> 按照社会资源最优分配的原则，分析市场机制的缺陷，解释政府经济活动的必要性和有效性，并在此基础上，界定政府经济活动的范围和职能。

第一讲告诉你财政怎么来的（历史），第三讲告诉你财政凭什么存在（理论）。如果没有这一讲的论证，后面八讲讨论的税收、支出、预算、政府干预——全是无根之木。

PPT 提出了一个核心问题：**政府存在的合理性或政府的边界在哪里？** 三个可能的答案：
1. 政府的经济活动有助于实现**公平和效率之间的平衡**
2. 政府的经济活动有助于解决**市场失灵**的问题
3. 政府的经济活动有助于增进**整体社会经济福利**

这三个答案，对应了这一讲的三大板块。

## 三、上半场：市场能做到的最好——社会福利与经济效率

### 3.1 两个剩余：福利的基本单元

供需交易的经典图——**社会总剩余（福利）= 消费者剩余（CS）+ 生产者剩余（PS）= 总支付意愿 - 总卖方成本**。

从这张图出发，两个关键结论：
1. **市场交易是社会福利的前提**——没有市场交易就没有社会福利
2. 在市场均衡状态下，计划者没有办法通过改变买卖双方的资源配置来改善社会福利

**模型假定**：完全竞争 + 没有外部性。后续所有"市场失灵"的讨论，都是因为这两个假定在现实中不成立。PPT 提醒思考：垄断、管制、税收、寻租、贸易、产权会怎样影响社会福利？


### 3.2 福利边界与帕累托最优

假定社会有两个参与者 A 和 B，WW' 曲线代表他们加总的福利总量（福利边界）：
- WW' 右侧：任何资源配置都无法达到
- WW' 左侧：资源配置低效率
- **WW' 上的点**：帕累托最优——一部分人处境的改善必须以另一部分人处境的恶化为代价
- 从 D 点向 E 点的移动：**帕累托改进**——至少一个人变好，没有人变差


### 3.3 埃奇沃思盒状图：效率如何实现

PPT 用了 5 页来详细展开这个过程，这是整个福利经济学的微观基础：

**设定**：两种物品（苹果和无花果叶），两个人（亚当和夏娃）。框图的长度 = 苹果总量，高度 = 无花果叶总量。


**初始状态**：在 v 点，亚当消费 Ou 无花果叶和 Ox 苹果，夏娃消费 O'y 苹果和 O'w 无花果叶。

**寻找帕累托最优**：假定夏娃的效用最初在 g 点。将夏娃的效用从 g → h → p，夏娃的效用未变而亚当的效用增加了。在 p 点，不可能使其中一个人境况变好而不伤害另一个人——这就是帕累托效率配置。从 g 到 h、从 h 到 p，都属于帕累托改进。

**边际替代率相等条件**：在无差异曲线的切点，满足 MRS_亚当 = MRS_夏娃。找到所有切点的轨迹——**契约曲线（Contract Curve）**——这就是所有帕累托效率点的集合。


### 3.4 从消费效率到生产效率

PPT 把分析从消费者推进到生产者：

**生产可能性曲线（PPC）**：在苹果产量既定条件下，社会能生产的无花果叶最大数量。其斜率 = **边际转换率（MRT）**——技术上用一种产品换另一种的成本。

**等产量线**：能生产一定产量的各种要素组合的轨迹。等产量线上资本与劳动相互替代的比率 = **边际技术替代率（MRTS）**。

将等产量线放入埃奇沃斯盒状图 → 得到切点集合的曲线（生产契约曲线）→ **生产的帕累托最优**。


### 3.5 局部均衡分析

PPT 用总收益（TSB）/总成本（TSC）曲线和边际收益（MSB）/边际成本（MSC）曲线统一表达了帕累托最优条件：

**在 Q* 点**，TSB 与 TSC 的差最大，同时 MSB = MSC。所以帕累托最优的条件也可以写为：**每一种产品的社会边际成本等于其社会边际收益。**


### 3.6 两个福利定理——小结

PPT 第 15 页明确总结：

- **福利经济学第一定理**：完全竞争的市场条件下，经济会"自动地"实现有效的资源配置，无须任何中央集权式指导。或者说：完全竞争市场结构下，帕累托最优状态可以实现。
- **福利经济学第二定理**：政府唯一要做的事情是改变个人之间禀赋的初始分配状态，其余的一切都可以由市场来解决。每一种具有帕累托效率的资源配置都可以通过市场机制实现。或者说：每一种帕累托最优的资源配置状态，都可在要素于消费者之间进行了适当的再分配后，通过完全竞争市场下的一般均衡来实现。

**核心含义**：效率交给市场，公平交给分配。政府不需要替代市场（第一定理），但可以通过再分配达到想要的结果（第二定理）。

## 四、下半场：公平对效率的修正

### 4.1 效率是尺子，公平是选择

PPT 指出：帕累托最优是资源配置的理想状态，但**不是所有的人都喜欢经济效率**——因为资源配置的有效，有可能是和极大的收入分配差距并存的。

- **效率**：相对客观，可以较好地衡量
- **公平**：价值判断，更加难以衡量
- **起点公平原则**（福利经济学原则）vs **分配公平原则**（平均主义原则）

推荐书目：托马斯·皮凯蒂《21 世纪资本论》。

### 4.2 效用可能性曲线：效率不等于公平

将从契约曲线平移出来的帕累托效率点重新表达 → **效用可能性曲线**——已知一个人的效用，另一个人可达到的最大效用量。

关键洞见：P3 和 P5 都是帕累托最优点，q 点不是（q 在曲线内部），但**社会有可能更加偏好 q 点**——因为在 q 点社会分配更加"公平"。效率不是社会的唯一目标。


### 4.3 社会无差异曲线：叠出"既有效又公平"

**社会无差异曲线**表明社会愿意以一个人的效用替换另一个人的效用。向右上方移动 = 社会福利增加。

三个关键点：
- **i 点**：资源配置有效，但不一定"公正"
- **ii 点**：比 i 更"公正"，但不一定有效
- **iii 点**：**既有效，又"公正"** ← 社会无差异曲线与效用可能性曲线的切点


### 4.4 三种正义观：功利、罗尔斯、平均主义

PPT 画了三张图，对应三种社会福利函数：

| 函数 | 核心原则 | 政策逻辑 |
|------|------|------|
| **功利主义** | 社会福利 = 所有成员福利之和。每个人的福利等同。如果 A 变好而 B 变坏，只要 A 的好 > B 的坏 → 社会福利改进 | 从富人拿一块钱给穷人，边际效用递减意味着总福利增加 |
| **罗尔斯主义** | 社会福利只取决于最糟者的效用水平。只有改善了最糟者的福利，社会福利才改进 | 不管不平等多大，只盯住社会最底层 |
| **平均主义** | 所有社会成员效用水平必须相同。偏离了平等线 → 福利下降 | "不患寡而患不均" |


PPT 追问：**向高收入者征税，并补贴给低收入群体，是否可以改善一国社会福利总水平？** ——三种函数给出不同答案。

### 4.5 钟摆效应：社会政策的左右摇摆

PPT 用一个极简模型总结：

```
左回归线 ← → 右回归线
    ├──────┼──────┤
 社会崩溃  成熟社会  社会崩溃
           政策区间
```

成熟社会中的社会政策在两条回归线之间反复摇摆，但不会超越界限。更极端的立场（无论是极左还是极右）在成熟社会中无法成为政策——左右政策的斗争以这两条线为边界。

## 五、市场失灵与财政职能：政府登场的理由

### 5.1 六种市场失灵

福利经济学第一第二定理说市场能自动达到最优——但现实比理论复杂得多。PPT 列出六种失灵：

| 失灵类型 | 含义 |
|------|------|
| **垄断** | 市场竞争的结果本身产生了垄断，垄断者控制价格和产量 |
| **外部性** | 生产的外部性和消费的外部性——你的行为影响了别人但没付（或没得到）钱 |
| **信息不对称** | 逆向选择 + 道德风险 |
| **有限理性、机会主义与资产专用性** | 人不是完全理性的计算器；签完合同就变脸；投资一旦做了就收不回来 |
| **收入分配不公** | 公平与效率的复杂关系——市场有效率但可能极不平等 |
| **宏观经济波动** | 不可能三角——固定汇率、货币政策独立性、资本自由流动三者只能取其二 |

### 5.2 财政三大职能

PPT 明确给出：

**（一）资源配置职能**

核心问题：市场与政府之间的关系。十八届三中全会：**市场在资源配置中起决定性作用**。

政府资源配置职能的内容：
- 矫正外部性（正和负）
- 提供公共产品（这是政府解决正外部性的一种模式）
- 管制自然垄断

政府资源配置的机制和手段：
- 调整资源在政府和私人部门之间的配置
- 调整财政支出结构
- 运用财政收支政策
- 改善财政配置效率

**（二）收入分配职能**

核心问题：效率和公平之间的关系。

实现手段：
- **税收机制**：以支付能力原则确定——所得税调节个人收入和企业利润；财产税调节个人财富水平；商品（流转）税调节产品相对价格
- **政府资产性收入**：政府介入实体经济，通过财产权利取得股息、红利、利息、利润
- **工资补贴支出**：政府为公营事业提供工资、明补和暗补
- **社会保障制度**：养老、医疗、生育、工伤、失业、住房等基础保障——**社会保障制度的出现，极大地改变了市场经济的分配格局**

**（三）经济稳定职能**

四大目标：经济增长 + 充分就业 + 物价稳定 + 国际收支平衡。

实现方式：
- 反周期的财政货币政策
- 财政的自动稳定器功能（如累进所得税、失业保险）
- 财政与产业政策的配合
- 不可能三角的约束

### 5.3 财政职能的局限性

PPT 用两页篇幅指出：政府不是万能的。

**理论层面的局限**：
- **有机论 vs 机械论**：社会是一个有机生命体（政府是大脑）→ 政府应该做更多；社会是机器的集合（政府是修理工）→ 政府应该做更少。两种世界观，决定了你对政府规模的立场。
- 政府与市场的边界在哪里？
- 政府财政活动对市场的**挤出效应**
- 政府财政活动对收入分配的复杂影响（有时再分配非但没有缩小差距，反而加大了）

**实践层面的局限**：
- 政府信息的有限性
- 政府决策的滞后性
- 政治过程的复杂性
- 政府官员的代理问题

### 5.4 结语

> "经济学给不出答案，答案只在你的一念之间！"

福利经济学给了你分析工具——剩余、帕累托最优、市场失灵——但**政府到底应该多大、边界划在哪里**，经济学本身回答不了。那是政治过程决定的。

## 六、概念对照表

| 概念对 | 区别 |
|------|------|
| **帕累托最优 vs 帕累托改进** | 前者是不能再改了（改就伤人），后者是还能改（改了人人获益）|
| **消费者剩余 vs 生产者剩余** | CS = 愿意付但没付的钱；PS = 愿意接受但多赚的钱。总剩余 = 社会福利 |
| **第一定理 vs 第二定理** | 第一定理说"完全竞争市场自动有效率"，第二定理说"任何帕累托最优都可以通过市场+再分配实现" |
| **契约曲线 vs 效用可能性曲线** | 契约曲线在商品空间（埃奇沃思盒内的切点轨迹），效用可能性曲线在效用空间（是契约曲线的对偶表达）|
| **MRS vs MRT vs MRTS** | MRS = 消费替代率（我愿意怎么换），MRT = 生产转换率（技术上怎么换），MRTS = 要素替代率（资本换劳动）|
| **功利主义 vs 罗尔斯主义 vs 平均主义** | 功利：总和最大（允许贫富分化）；罗尔斯：最差者优先；平均：人人完全一样 |
| **起点公平 vs 分配公平** | 起点公平（福利经济学原则）：机会均等，结果不管；分配公平（平均主义原则）：结果也要均等 |
| **资源配置 vs 收入分配 vs 经济稳定** | 配置 = 做大蛋糕（效率逻辑）；分配 = 切蛋糕（公平逻辑）；稳定 = 蛋糕别忽大忽小（宏观逻辑）|
| **有机论 vs 机械论** | 有机论 → 大政府；机械论 → 小政府。这是财政哲学的根本分歧 |
| **自动稳定器 vs 反周期政策** | 前者是制度内置的（累进税/失业保险自动缓冲），后者是主动出击的（衰退时减税增支）|

## 七、整章逻辑链：从剩余到财政职能

```
市场交易 → 消费者剩余 + 生产者剩余 = 社会福利
    ↓
完全竞争条件下 → 自动达到帕累托最优（边际替代率相等/契约曲线/MSB=MSC）
    ↓
福利经济学第一定理：市场有效率；第二定理：任何分配结果都可通过市场+禀赋再分配实现
    ↓
但现实中有六种市场失灵：垄断、外部性、信息不对称、有限理性/机会主义/资产专用性、收入分配不公、宏观经济波动
    ↓
且帕累托最优不关心公平——效率高但可能极不平等
    ↓
需要引入公平修正：功利主义、罗尔斯主义、平均主义各有答案
    ↓
社会在公平与效率间钟摆式摇摆（有左右边界，不超线）
    ↓
政府出场 → 三大职能：
  1. 资源配置（矫正外部性/提供公共产品/管制垄断）
  2. 收入分配（税收/资产收入/补贴/社会保障）
  3. 经济稳定（反周期政策/自动稳定器/产业政策）
    ↓
但政府也有局限性：信息有限/决策滞后/政治复杂/代理问题/挤出效应
    ↓
→ 第四讲开始讨论具体怎么应对每一种市场失灵
```

## 八、考试最可能考的题型

1. **名词解释**：帕累托最优、帕累托改进、契约曲线、效用可能性曲线、福利经济学第一定理/第二定理
2. **简答**：比较功利主义、罗尔斯主义和平均主义三种社会福利函数的区别；财政三大职能的内容和实现手段
3. **论述**：用福利经济学的框架论证政府干预经济的必要性和边界
4. **画图分析**：埃奇沃思盒状图中如何从初始配置找到帕累托最优配置；效用可能性曲线与社会无差异曲线如何决定"既有效又公平"的点
