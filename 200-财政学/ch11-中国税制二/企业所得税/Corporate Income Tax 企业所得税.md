# 企业所得税 (Corporate Income Tax)

## 📌 定义
企业所得税是对企业和其他取得收入的组织的所得征收的税种。个人独资企业和合伙企业不适用。基本税率 25%，非居民企业 20%（实际减按 10%）。

## 🧠 机制

### 纳税人

| 类型 | 定义 | 纳税义务 |
|------|------|----------|
| **居民企业** | 依法在中国境内成立，或实际管理机构在中国境内 | 全球所得纳税 |
| **非居民企业** | 依外国法律成立，实际管理机构不在中国境内 | 仅中国境内所得纳税 |

> 有限责任公司存在**重复征税**问题：先缴企业所得税，股东分红再缴个人所得税。

### 应纳税所得额

$$\text{应纳税所得额} = \text{收入总额} - \text{不征税收入} - \text{免税收入} - \text{各项扣除} - \text{以前年度亏损}$$

**收入总额**（9类）：销售货物/提供劳务/转让财产/股息红利/利息/租金/特许权使用费/捐赠/其他

**不征税收入**：财政拨款、行政事业性收费、政府性基金

**免税收入**：国债利息、居民企业间股息红利、非营利组织收入

### 主要扣除标准

| 项目 | 扣除标准 |
|------|----------|
| 工资薪金 | 据实扣除 |
| 职工福利费 | ≤ 工资总额 14% |
| 工会经费 | ≤ 工资总额 2% |
| 职工教育经费 | ≤ 工资总额 8%（可结转） |
| 业务招待费 | 发生额 60%，≤ 营收 5‰ |
| 广告宣传费 | ≤ 营收 15%（可结转） |
| 公益性捐赠 | ≤ 利润总额 12%（可结转3年） |
| 研发费用 | **100% 加计扣除**；形成无形资产的 200% 摊销 |

**不得扣除**：股息红利分配、企业所得税税款、税收滞纳金、罚金罚款、超限捐赠、赞助支出、商业保险费等。

> 以**权责发生制**为原则；区分**收益性支出**（当期扣除）与**资本性支出**（分期扣除）。

### 税收优惠

| 优惠类型 | 内容 |
|----------|------|
| **高新技术企业** | 减按 15% |
| **小型微利企业** | 减按 25% 计算所得额，按 20% 税率（应纳税额 ≤ 300万/人数 ≤ 300/资产 ≤ 5000万） |
| **研发费用加计** | 100% 加计扣除，集成电路/母机 120% |
| **西部大开发** | 减按 15% |
| **海南自贸港** | 减按 15%，新增境外投资所得免税 |
| **软件/集成电路** | 两免三减半 |
| **环保专用设备** | 投资额 10% 抵免税额 |
| **创投企业** | 投资额 70% 抵扣 |

### 亏损弥补
- 一般企业：最长 **5年**
- 高新技术企业：最长 **10年**
- 疫情影响企业：最长 **8年**
- 境外亏损**不得**抵减境内盈利

### 境外税收抵免
境外已缴所得税可在限额内抵免，超过部分可在以后 **5个年度** 内抵补。

### 计算公式
$$\text{应纳税额} = \text{应纳税所得额} \times 25\% - \text{减免税额} - \text{抵免税额}$$

## 📄 来源原文
> 企业所得税的税率为25%。企业每一纳税年度的收入总额，减除不征税收入、免税收入、各项扣除以及允许弥补的以前年度亏损后的余额，为应纳税所得额。应纳税额=应纳税所得额×适用税率-减免税额-抵免税额。
