# 第十一讲 知识串讲：消费税、企税、个税——剩下的税收怎么征

## 一、一句话回答

**增值税之外还有三大税——消费税专挑烟酒汽车奢侈品调节消费，企业所得税从公司利润中切走 25%，个人所得税从你工资条上扣走 3-45%。剩下 14 个小税种各有分工。中国税制结构以间接税为主、直接税为辅。**

---

## 二、消费税：只征"不该多消费的"

### 2.1 基本定义与五大特征

消费税是对消费品和特定消费行为按流转额征收的一种商品税。

| 特征 | 含义 |
|------|------|
| **征收范围选择性** | 不像增值税啥都征——只挑 15 类 |
| **征收环节单一性** | 一般在生产、委托加工或进口**一个环节**征，不是道道征 |
| **平均税负差异大** | 烟 56% + 从量，高档手表 20%，税率差距悬殊 |
| **计税方法灵活** | 从价、从量、混合三种 |
| **属于中央税** | 收入全归中央 |

纳税人：在中国境内生产、委托加工和进口《消费税暂行条例》规定的消费品的单位和个人。

### 2.2 15 个税目

| 类别 | 具体税目 |
|------|------|
| **有害健康** | 烟、酒 |
| **奢侈炫富** | 高档化妆品、贵重首饰及珠宝玉石、高尔夫球及球具、高档手表、游艇 |
| **污染环境** | 鞭炮焰火、成品油、摩托车、小汽车、木制一次性筷子、实木地板、电池、涂料 |

**甲类/乙类区分**：
- 甲类卷烟：每标准条（200 支）调拨价 ≥ 70 元（不含增值税）
- 甲类啤酒：每吨出厂价 ≥ 3000 元（含包装物及押金，不含增值税）


### 2.3 三种计算方法

| 方法 | 公式 | 适用 |
|------|------|------|
| **从价定率** | 应纳税额 = 销售额 × 税率 | 化妆品 15%、高档手表 20% |
| **从量定额** | 应纳税额 = 销售数量 × 单位税额 | 汽油 1.52 元/升、啤酒 220-250 元/吨 |
| **复合计税** | 从量 + 从价双管齐下 | **卷烟**（56%/36% + 0.003 元/支）、**白酒**（20% + 0.5 元/500 克）|

消费税是**价内税**——含税价 = 不含税价 ÷ (1 - 消费税率)。从价定率下，销售额不含增值税但含消费税：
> **应税销售额 = 含增值税的销售额 ÷ (1 + 增值税税率或征收率)**

**从量定额下销售量的确定**：
1. 销售应税消费品 → 销售数量
2. 自产自用 → 移送使用数量
3. 委托加工 → 纳税人收回的数量
4. 进口 → 海关核定的进口征税数量

### 2.4 计税依据的特殊规定

| 情形 | 处理 |
|------|------|
| 自设非独立核算门市部销售 | 按**门市部对外销售额**征收 |
| 用于换取生产资料/消费资料/投资入股/抵偿债务 | 按同类消费品**最高销售价格**作为计税依据 |
| 卷烟 | 最低计税价格 = 批发环节销售价格 ×（1 - 适用批发毛利率），由税务总局核定 |
| 白酒 | 售价 < 正常价格 70%，或连续三个月累计涨跌幅 > 20% → 税务机关核定 |
| 金银首饰 | 既生产又销售且划分不清 → **从高**适用税率 |

### 2.5 自产自用与委托加工

**自产自用**：
- 用于连续生产应税消费品 → **不纳税**
- 用于其他方面（送人/发福利等）→ 移送使用时纳税
- 组成计税价格（从价）：**（成本 + 利润）÷（1 - 比例税率）**
- 组成计税价格（复合）：**（成本 + 利润 + 自产自用数量 × 定额税率）÷（1 - 比例税率）**

**委托加工**：由受托方在向委托方交货时**代收代缴**消费税。
- 按受托方同类消费品销售价格（当月 → 上月或最近月份 → 组成计税价格）
- 组成计税价格 = **（材料成本 + 加工费）÷（1 - 消费税率）**
- 收回后直接销售 → 不再征；收回后继续加工 → 已代扣消费税可抵扣

### 2.6 兼营与进口

- **兼营不同税率**：分别核算，未分别核算或组成成套 → **从高适用税率**
- **进口消费税**：组成计税价格 =（关税完税价格 + 关税）÷（1 - 消费税率）

### 2.7 计算举例

某白酒生产企业为一般纳税人，4 月销售白酒 50 吨，不含增值税销售额 200 万元。白酒比例税率 20%，定额税率 1 元/千克。
> 应纳消费税 = 50 × 1000 × 1 + 200万 × 20% = 5万 + 40万 = **45 万元**

## 三、企业所得税：公司挣的钱政府拿多少

### 3.1 纳税人与税率

在中华人民共和国境内，企业和其他取得收入的组织为企业所得税的纳税人。**个人独资企业、合伙企业不适用企业所得税法。**

| 纳税人类型 | 定义 | 纳税范围 |
|------|------|------|
| **居民企业** | 中国境内成立，或实际管理机构在中国境内 | **全球所得** |
| **非居民企业** | 外国成立且实际管理机构不在中国境内 | 仅中国境内所得，或与境内机构有实际联系的境外所得 |

税率：
- 居民企业：**25%**
- 非居民企业：**20%**（实际减按 10%）

**核心困境**：有限责任公司的利润先交企业所得税（25%），税后利润分给股东再交个人所得税 → **同一笔利润被双重征税。**

### 3.2 应纳税所得额

> **应纳税所得额 = 收入总额 − 不征税收入 − 免税收入 − 各项扣除 − 允许弥补的以前年度亏损**

**收入总额（9 类）**：销售货物、提供劳务、转让财产、股息红利等权益性投资、利息、租金、特许权使用费、接受捐赠、其他

**不征税收入（3 类）**：财政拨款、依法纳入财政管理的行政事业性收费/政府性基金、国务院规定的其他

**免税收入（4 类）**：国债利息、符合条件的居民企业间股息红利、非居民企业取得与境内机构有实际联系的股息红利、符合条件的非营利组织收入

### 3.3 扣除项目及标准（核心考点）

| 项目 | 扣除标准 |
|------|------|
| **工资薪金** | 合理支出，据实扣除 |
| **职工福利费** | ≤ 工资薪金总额 **14%** |
| **工会经费** | ≤ 工资薪金总额 **2%** |
| **职工教育经费** | ≤ 工资薪金总额 **8%**，超支可**结转**以后年度 |
| **社会保险费** | 基本五险一金，据实扣除 |
| **利息费用** | 非金融企业向金融企业借款：据实；向非金融企业借款：不超过金融企业同期同类贷款利率 |
| **业务招待费** | 实际发生额 **60%** 与当年销售（营业）收入 **5‰**——**取较小值** |
| **广告费和业务宣传费** | ≤ 销售收入 **15%**，超支可结转 |
| **公益性捐赠** | ≤ 年度利润总额 **12%**，超支可结转 **3 年** |
| **租赁费** | 经营租赁：按租赁期均匀扣除；融资租赁：分期折旧扣除 |
| **研发费用** | **100% 加计扣除**（未形成无形资产的）；形成无形资产的按 **200% 摊销**。集成电路和工业母机：120%/220% |
| **安置残疾人员** | 工资 **100% 加计扣除** |

### 3.4 不得扣除的支出

向投资者支付的股息红利、企业所得税税款、税收滞纳金、罚金罚款及被没收财物损失、超标准捐赠、赞助支出、未经核定的准备金、商业保险费（为投资者或职工）、企业间支付的管理费/租金/特许权使用费、境外营业机构亏损不得抵减境内盈利。

**权责发生制**：属于当期的收入和费用，不论款项是否收付，均作为当期。**收益性支出** → 当期直接扣除；**资本性支出** → 分期扣除或计入资产成本。

### 3.5 15 类税收优惠

| 优惠类型 | 内容 |
|------|------|
| **免税/减征** | 农林牧渔、公共基础设施、环保节能节水、技术转让、农村饮水安全等 |
| **高新技术企业** | 减按 **15%** |
| **技术先进型服务企业** | 减按 **15%** |
| **污染防治第三方** | 2024-2027 减按 **15%** |
| **小型微利企业** | 应税所得 ≤300 万/从业 ≤300/资产 ≤5000 万 → 减按 25% 计算所得，按 20% 税率 |
| **加计扣除** | 研发 100%/200%、残疾人工资 100%、基础研究 100% |
| **创投企业** | 投资初创科技企业满 2 年 → 投资额 **70%** 抵扣 |
| **加速折旧** | 2019 年起所有制造业 |
| **减计收入** | 资源综合利用减按 90%；农村小额贷款/农业保险按 90% |
| **税额抵免** | 环保节能专用设备投资额 **10%** 抵免 |
| **海南自贸港** | 全部企业减按 15%；新增境外直接投资所得免征 |
| **西部大开发** | 鼓励类产业减按 **15%** |
| **软件/集成电路** | **两免三减半** |
| **节能服务/电网** | **三免三减半** |

### 3.6 亏损弥补与境外抵免

- 亏损可用下一年所得弥补，最长 **5 年**（高新技术企业 **10 年**，受疫情影响严重企业 **8 年**）
- 境外已缴所得税可抵免，超过抵免限额部分可结转 **5 年**抵补

### 3.7 核定征收

无法完整提供会计资料的企业 → 税务机关核定：
> 应纳税所得额 = 应税收入总额 × 应税所得率
> 或 = 成本（费用）支出额 ÷（1 - 应税所得率）× 应税所得率

## 四、个人所得税：从你工资条上扣的钱

### 4.1 沿革与征收思路

个人所得税最早由**英国人 1799 年创立**。中国 1980 年正式开设，2019 年 1 月 1 日起最新《个人所得税法》实施。

三种征收模式：

| 模式 | 做法 | 特点 |
|------|------|------|
| **分类税** | 所得按来源分类，不同类别不同税率 | 控制税源、降低征收成本、可差别对待 |
| **综合税** | 所有收入汇总，按整体收入水平定税率 | 更公平、符合支付能力原则、征税成本高 |
| **混合税** | 分类 + 综合结合 | 中国现行模式 |

### 4.2 纳税人

| 类型 | 判定标准 | 纳税义务 |
|------|------|------|
| **居民纳税人** | ①在中国境内有住所；②无住所但一个纳税年度内居住**满 183 天** | **无限纳税义务**：全球所得都要交 |
| **非居民纳税人** | 无住所又不居住，或居住累计不满 183 天 | **有限纳税义务**：仅中国境内所得 |

**跨国纳税与国际重复征税**：税收管辖权有属人原则（居民/公民）和属地原则（地域）。两国对同一所得同时征税 → 国际重复征税。通过**国际税收协定**、**税收抵免**（扣除境外已纳税额）、**税收饶让**（外国税收优惠也给予扣除）来免除。

### 4.3 九类征税范围与税率

**综合所得**（七级超额累进 **3-45%**）：工资薪金、劳务报酬、稿酬、特许权使用费

**经营所得**（五级超额累进 **5-35%**）：个体工商户、个独、合伙企业

**其他所得**（比例税率 **20%**）：利息股息红利、财产租赁、财产转让、偶然所得

### 4.4 综合所得的计算

> **应纳税所得额 = 年收入额 − 6 万元基本减除 − 专项扣除 − 专项附加扣除 − 其他扣除**

- 劳务报酬、特许权使用费：按收入的 **80%** 计入
- 稿酬：按收入的 **80% × 70%** 计入

**七级超额累进税率**（年应纳税所得额）：
| 级数 | 累计额 | 税率 | 速算扣除数 |
|------|------|------|------|
| 1 | ≤ 3.6 万 | 3% | 0 |
| 2 | ≤ 14.4 万 | 10% | 2,520 |
| 3 | ≤ 30 万 | 20% | 16,920 |
| 4 | ≤ 42 万 | 25% | 31,920 |
| 5 | ≤ 66 万 | 30% | 52,920 |
| 6 | ≤ 96 万 | 35% | 85,920 |
| 7 | > 96 万 | 45% | 181,920 |


### 4.5 六项专项附加扣除

| 扣除项目 | 标准 | 备注 |
|------|------|------|
| **子女教育** | 1,000/月/孩 | 年满 3 岁至博士；父母各 50% 或一方 100% |
| **继续教育** | 400/月（≤48 月）或取证当年 3,600 | 境内接受 |
| **赡养老人** | 2,000/月 | 满 60 岁；独生子女全额，非独生子女分摊 |
| **大病医疗** | 自付超 1.5 万部分，≤ 8 万 | 年度据实 |
| **住房贷款利息** | 1,000/月（≤240 月）| 首套房 |
| **住房租金** | 1,500（直辖市/省会）/ 1,100（>100 万人）/ 800（≤100 万人）| 与贷款利息互斥 |

### 4.6 其他所得的费用扣除

| 类型 | 扣除标准 |
|------|------|
| 财产租赁 | 每次 ≤4000 元 → 扣 800 元；> 4000 元 → 扣 **20%** |
| 财产转让 | 收入 − 财产原值 − 合理费用 |
| 利息股息红利、偶然所得 | **不扣费用**，全额计征 |
| 公益捐赠 | ≤ 收入 30% 部分可扣除 |

### 4.7 征收方式与优惠政策

**全员全额扣缴申报**：每月发工资时由扣缴义务人预扣或代扣代缴，次年**汇算清缴**（个人所得税 APP）。

**主要免税项目**：省级以上奖金、国债利息、国家统一补贴、保险赔款、军人转业费、安家费/退休费/离休费、五险一金、见义勇为奖金、生育补贴、工伤补助、满五唯一住房转让、拆迁补偿款、持股超 1 年股息红利、单张有奖发票 ≤800 元/彩票 ≤10,000 元

**主要减征**：残疾/孤寡老人/烈属所得、自然灾害重大损失、远洋船员满 183 天→工资 **50%** 计入

### 4.8 计算举例

某居民为独生子女，年度社保后收入 20 万，劳务报酬 1 万，稿酬 1 万。两个小孩在学，父母均满 60 岁。
> 应纳税所得额 = 200,000 + 10,000×80% + 10,000×70%×80% − 60,000 − 12,000×2 − 24,000
> = 200,000 + 8,000 + 5,600 − 60,000 − 24,000 − 24,000 = **105,600 元**
> 应纳税额 = 105,600 × 10% − 2,520 = **8,040 元**

## 五、其他 14 个税种详解

### 5.1 城市维护建设税与教育费附加

| 税种 | 计税依据 | 税率 |
|------|------|------|
| **城建税** | 实际缴纳的增值税 + 消费税 | 市区 7%、县城/镇 5%、其他 1%（进口不征）|
| **教育费附加** | 实际缴纳的增值税 + 消费税 | 3% |
| **地方教育费附加** | 同上 | 2% |

### 5.2 烟叶税

收购烟叶的单位，按实际支付价款（收购价款 × 1.1）的 **20%** 缴纳。

### 5.3 关税

| 要素 | 内容 |
|------|------|
| **纳税人** | 进出口货物收发人、进出境物品所有人 |
| **税率类型** | 最惠国（WTO 成员）、协定（区域贸易协定）、特惠（特殊关税优惠）、普通、暂定（更优惠，每年公布）、配额 |
| **完税价格** | 以成交价格为基础，含货价 + 运抵输入地点起卸前的运输及相关费用 + 保险费 |

### 5.4 船舶吨税

境外船舶进入中国港口，按吨位征收，分优惠税率和普通税率。

### 5.5 资源税与环境保护税

**资源税**：开发应税资源的单位和个人。5 大类：能源矿产、金属矿产、非金属矿产、水汽矿产、盐。税率 1-20%。进口不征，出口不退税。

**环境保护税**：排放应税污染物。4 个税目：大气污染物、水污染物、固体废物、噪声。定额税率。

### 5.6 城镇土地使用税与耕地占用税

| 税种 | 纳税人 | 税率 |
|------|------|------|
| **城镇土地使用税** | 城市/县城/建制镇/工矿区范围内使用土地的单位和个人 | 定额税率；大城市 1.5-30 元/㎡、中等 1.2-24、小城市 0.9-18、县城等 0.6-12 |
| **耕地占用税** | 占用耕地建房或非农建设的单位和个人 | 按人均耕地面积差别定额 |

### 5.7 房产税

| 方式 | 计税依据 | 税率 |
|------|------|------|
| **从价计征** | 房产原值减除 10-30% 后的余值 | **1.2%** |
| **从租计征** | 租金收入 | **12%**（个人出租住房 4%）|

个人所有非营业用房**免征**（居民住房目前全免）。

### 5.8 契税

在中国境内转移土地、房屋权属，**承受方**缴纳。税率 **3-5%**。90 平米以下唯一住房 **1%**。

### 5.9 土地增值税

转让房地产并取得收入的单位和个人。**四级超额累进**：

| 增值额占比 | 税率 | 速算扣除数 |
|------|------|------|
| ≤ 50% | 30% | 0 |
| 50-100% | 40% | 5% |
| 100-200% | 50% | 15% |
| > 200% | 60% | 35% |

扣除项目：取得土地使用权支付的金额、房地产开发成本、开发费用、相关税金等。

### 5.10 车辆购置税、车船税、印花税

| 税种 | 要点 |
|------|------|
| **车辆购置税** | 购置应税车辆时一次征收，**统一 10%**。新能源车免税 |
| **车船税** | 车辆船舶所有人和管理人，按年缴纳，定额税率（按排量）|
| **印花税** | 书立/领受应税凭证（合同、账簿、证照），税率 0.005%-0.1% |

## 六、中国税制全景图

```
18 个税种 = 三大流转税 + 两大所得税 + 两类附加 + 11 个小税种

流转税（间接税 — 消费者最终负担）
  ├── 增值税（第一大，占 38%）— 普遍征收、道道征收
  ├── 消费税（调节性）— 15 个税目、单一环节、中央税
  └── 关税 — 进出口

所得税（直接税 — 挣钱者自己掏）
  ├── 企业所得税（25%）— 居民全球纳税
  └── 个人所得税（3-45%）— 综合+分类混合制

附加税
  ├── 城市维护建设税（7/5/1%）
  └── 教育费附加（3%+2%）

财产与行为税
  └── 房产税 / 契税 / 土地增值税 / 印花税 / 车船税 / 车辆购置税 / 船舶吨税 / 城镇土地使用税 / 耕地占用税 / 资源税 / 环境保护税 / 烟叶税
```

中国税制最显著的特征：**间接税为主、直接税为辅**。增值税 + 消费税占税收大头，个人所得税占比远低于发达国家。

## 七、概念对照表

| 概念对 | 区别 |
|------|------|
| **增值税 vs 消费税** | 增值税普遍征（道道征/价外税），消费税选择性征（单一环节/价内税）|
| **从价定率 vs 从量定额 vs 复合计税** | 从价 = 价格 × %；从量 = 数量 × 定额；复合 = 两者并用（卷烟/白酒）|
| **甲类卷烟 vs 乙类卷烟** | 每标准条调拨价 ≥ 70 元（不含增值税）→ 甲类；< 70 元 → 乙类 |
| **居民企业 vs 非居民企业** | 居民：全球所得纳税（25%）；非居民：仅中国境内（20%，实际 10%）|
| **居民纳税人 vs 非居民纳税人** | 居民：有住所或住满 183 天 → 全球所得 3-45%；非居民：仅中国境内 |
| **综合所得 vs 经营所得 vs 分类所得** | 综合：工资+劳务+稿酬+特许权（3-45%）；经营：个独/合伙（5-35%）；分类：利息/租赁/转让/偶然（20%）|
| **不征税收入 vs 免税收入** | 前者根本不在征税范围（财政拨款），后者在范围但免了（国债利息）|
| **权责发生制 vs 收付实现制** | 前者按权利责任归属期确认（企税用），后者按实际收付款确认 |
| **国际税收抵免 vs 税收饶让** | 抵免 = 扣除境外实缴税额；饶让 = 外国税收优惠部分也给予扣除 |
| **房产税从价 vs 从租** | 从价 = 余值 × 1.2%（自用）；从租 = 租金 × 12%（出租）；个人出租 4% |
| **土地增值税扣除 vs 企税扣除** | 土地增值税扣除 = 取得土地成本+开发成本+开发费用+税金；企税扣除 = 成本/费用/税金/损失 |

## 八、最重要的计算链

### 8.1 消费税完整计算
```
销售额确定（不含增值税、含消费税）
  → 选择计征方法（从价/从量/复合）
  → 如有自产自用/委托加工/进口 → 用组成计税价格
  → 应纳税额
```

### 8.2 企业所得税完整计算
```
收入总额（9类）
  − 不征税收入（3类）
  − 免税收入（4类）
  = 征税收入
  − 各项扣除（工资/福利14%/工会2%/教育8%/招待费60%&5‰/广告15%/捐赠12%/研发加计100%等）
  = 应纳税所得额
  × 税率（25%/15%/20%/10%）
  − 减免税额 − 抵免税额
  = 应纳税额
```

### 8.3 个人所得税完整计算
```
年综合收入（工资+劳务×80%+稿酬×80%×70%+特许权×80%）
  − 6万基本减除
  − 专项扣除（五险一金）
  − 专项附加扣除（子女1k/继教400/赡养2k/大病/房贷1k/租金0.8-1.5k）
  − 其他扣除
  = 应纳税所得额
  × 对应级数税率 − 速算扣除数
  = 全年应纳税额
  → 汇算清缴：多退少补
```
