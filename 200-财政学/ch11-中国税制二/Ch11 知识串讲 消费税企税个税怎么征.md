# 第十一讲 知识串讲：消费税、企税、个税——剩下的税收怎么征

## 一、一句话回答

**增值税之外还有三大税——消费税调节奢侈行为，企业所得税从公司利润里切，个人所得税从你工资里扣。剩下 14 个小税种补位收尾。**

---

## 二、消费税：只征"不该多消费的"

增值税啥都征（普遍征收），消费税只挑几样征（选择征收）。征什么？烟、酒、汽车、化妆品、珠宝、游艇——要么有害健康，要么炫富，要么污染环境。

### 2.1 为什么在增值税之外再加一道消费税

增值税税率统一 → 不能体现"这个东西该多征"。消费税补上了这个功能：**通过差别税率引导消费、调节收入分配。**

### 2.2 怎么征

三种计算方法配合使用：

| 方法 | 怎么算 | 征什么 |
|------|------|------|
| **从价定率** | 价格 × 税率 | 化妆品 15%、高档手表 20% |
| **从量定额** | 数量 × 单位税额 | 汽油 1.52元/升、啤酒 220-250元/吨 |
| **混合** | 从价 + 从量双管齐下 | **卷烟**（56%+0.003元/支）、白酒（20%+0.5元/500克） |

### 2.3 关键规则

- 一般在**单一环节**征收（生产/委托加工/进口），不是道道征
- 是**价内税**——不含税价 = 含税价 ÷ (1+税率)
- 属于**中央税**——收入全归中央
- 自产自用（送人、发福利）→ 照样纳税
- 委托加工 → 受托方代收代缴

## 三、企业所得税：公司挣的钱政府拿多少

### 3.1 基本框架

税率 **25%**。高新技术企业 15%，小微企业最低 5%。

**居民企业**（中国注册或实际管理机构在中国）：全球所得都要交。**非居民企业**（外国注册、管理在外）：只交在中国赚的那部分。

### 3.2 应纳税所得额怎么算

```
收入总额 − 不征税收入 − 免税收入 − 各项扣除 − 以前年度亏损 = 应纳税所得额
```

扣除项是整个税制的核心——哪些能扣、扣多少：

| 项目 | 扣除标准 | 记忆要点 |
|------|------|------|
| 工资薪金 | 据实扣除 | 发了多少扣多少 |
| 职工福利费 | ≤ 工资 14% | 和工会在一个模块 |
| 工会经费 | ≤ 工资 2% | — |
| 职工教育经费 | ≤ 工资 8% | 可结转以后年度 |
| 业务招待费 | 实际发生额 60%，≤ 营收 5‰ | **两个条件取较小值** |
| 广告宣传费 | ≤ 营收 15% | 可结转 |
| 公益性捐赠 | ≤ 利润总额 12% | 可结转 3 年 |
| **研发费用** | **100% 加计扣除** | 资本化：200% 摊销 |

### 3.3 税收优惠

15 类优惠政策，重点：

- **高新技术企业**：25% → 15%
- **西部/海南自贸港**：15%
- **小微企业**：实际税率最低 5%
- **研发加计**：100%（集成电路/工业母机 120%）
- **软件/集成电路**：两免三减半
- **创投企业**：投资额 70% 抵扣

### 3.4 一个核心困境：重复征税

有限责任公司先交企业所得税（25%），税后利润分给股东，股东还要交个人所得税。同一笔利润被征了两道税——这是企税和个税之间最基础的制度矛盾。

## 四、个人所得税：从你工资条上扣掉的钱

### 4.1 谁要交

**居民纳税人**（有住所或一年住满 183 天）：全球所得无限纳税。**非居民**：仅中国境内所得。

### 4.2 九类收入，三种计法

| 计法 | 适用 | 怎么算 |
|------|------|------|
| **综合征收** | 工资、劳务、稿酬、特许权 | 累进税率 3-45% |
| **分类征收** | 利息股息、财产租赁/转让、偶然所得 | 20% 统一税率 |
| **经营所得** | 个独/合伙企业 | 累进税率 5-35% |

### 4.3 七级超额累进

```
月薪 5000 起征，年收入 6 万以内不交税
超过 6 万 → 3%（≤3.6万）→ 10% → 20% → 25% → 30% → 35% → 45%（>96万）
```

最高边际税率 45% 意味着——你年终奖多拿 100 块，其中 45 块归税务局。

### 4.4 六项专项附加扣除（节税关键）

| 扣除 | 标准 | 条件 |
|------|------|------|
| 子女教育 | 1,000/月/孩 | 学龄前到博士 |
| 继续教育 | 400/月 或 3,600/年 | 在职提升 |
| 赡养老人 | 2,000/月 | 父母 ≥60岁 |
| 大病医疗 | 自付超 1.5万部分，≤8万 | 按年据实扣 |
| 住房贷款 | 1,000/月 | 首套房贷款 |
| 住房租金 | 800-1,500/月 | 按城市级别 |

### 4.5 征收方式

每月发工资时**预扣预缴**，次年 3-6 月**综合汇算清缴**——补税或退税。个人所得税 APP 是汇算入口。

## 六、中国税制全景图

```
18 个税种 = 三大流转税 + 两大所得税 + 13 个小税种

流转税（间接税 — 消费者实际负担）
  ├── 增值税（第一大，38%）
  ├── 消费税（调节有害/奢侈消费）
  └── 关税（管进出口）

所得税（直接税 — 挣钱的自己掏）
  ├── 企业所得税（25%，公司利润）
  └── 个人所得税（3-45%，个人收入）

小税种（补位）
  └── 财产税/资源税/行为税等 13 个
```

中国税制结构的一个显著特征：**间接税为主、直接税为辅。** 增值税+消费税占到税收的大头，而个人所得税占比远低于发达国家——这和前面第九讲的 7.8% 税收/GDP 比率是一脉相承的。

## 七、概念对照表

| 概念对 | 区别 |
|------|------|
| **增值税 vs 消费税** | 增值税普遍征（道道征），消费税选择性征（单一环节）；增值税是价外税，消费税是价内税 |
| **从价 vs 从量** | 从价 = 按价格百分比（价格×税率），从量 = 按实物数量（数量×定额） |
| **居民企业 vs 非居民企业** | 前者全球所得纳税，后者仅中国境内所得 |
| **综合所得 vs 分类所得** | 综合 = 四种收入合并算（更公平），分类 = 各算各的（更简便） |
| **预扣预缴 vs 汇算清缴** | 前者每月发工资时扣，后者次年一次性多退少补 |
| **企业所得税 vs 个人所得税** | 前者从公司利润征，后者从个人收入征；利润分到个人时双重征税 |
| **一般纳税人 vs 小规模纳税人** | 500万为界，抵扣制 vs 简易计税 |

## 八、最重要的故事：你的工资单发生了什么

```
公司发你工资 10,000 元
  ├→ 公司：这笔工资作为成本扣除（少交企业所得税 2,500）
  └→ 你：
      ├→ 先扣五险一金（假设 ~1,500）
      ├→ 再扣 6 项专项附加扣除（假设子女教育+赡养老人 3,000）
      ├→ 再减基本减除 5,000
      └→ 应纳税所得额 = 10,000-1,500-3,000-5,000 = 500
          → 500 × 3% = 15 元
          → 到手 ≈ 10,000−1,500−15 = 8,485 元
```

年收入不超过 9.6 万（税后）的中等收入群体，在专项附加扣除拉满的情况下，实际个税负担非常低。中国个税的主要贡献者是年入超百万的最高两档。
