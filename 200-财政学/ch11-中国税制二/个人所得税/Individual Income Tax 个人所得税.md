# 个人所得税 (Individual Income Tax / IIT)

## 📌 定义
个人所得税是以自然人取得的各类应税所得为征税对象征收的所得税。1799年英国首创，中国1980年开设，2019年1月1日实施最新《个人所得税法》。实行分类与综合相结合的混合税制。

## 🧠 机制

### 纳税人

| 类型 | 标准 | 纳税义务 |
|------|------|----------|
| **居民纳税人** | 在中国境内有住所，或一个纳税年度居住满 **183天** | 无限纳税义务（全球所得） |
| **非居民纳税人** | 无住所且不居住，或居住不满 183天 | 有限纳税义务（仅中国所得） |

### 征税范围（9个税目）

| 税目 | 内容 |
|------|------|
| 工资、薪金所得 | 非独立个人劳动所得（奖金、津贴等） |
| 劳务报酬所得 | 独立提供技艺、劳务所得 |
| 稿酬所得 | 以图书报刊形式出版发表所得 |
| 特许权使用费所得 | 专利、商标、著作权等 |
| 经营所得 | 个独/合伙企业生产经营所得 |
| 利息、股息、红利所得 | 拥有债权、股权所得 |
| 财产租赁所得 | 出租财产所得 |
| 财产转让所得 | 转让财产所得 |
| 偶然所得 | 中奖、得奖等 |

### 三种征收模式
- **分类税**：按来源分类，分别规定税基和税率（优点：控制税源、成本低）
- **综合税**：按整体收入水平确定税率（优点：公平、符合支付能力原则）
- **混合税**：中国现行模式——分类与综合结合

### 税率

**综合所得（七级超额累进）**：

| 级数 | 全年应纳税所得额 | 税率 | 速算扣除数 |
|:---:|------|:---:|:---------:|
| 1 | ≤36,000 | 3% | 0 |
| 2 | 36,000-144,000 | 10% | 2,520 |
| 3 | 144,000-300,000 | 20% | 16,920 |
| 4 | 300,000-420,000 | 25% | 31,920 |
| 5 | 420,000-660,000 | 30% | 52,920 |
| 6 | 660,000-960,000 | 35% | 85,920 |
| 7 | >960,000 | **45%** | 181,920 |

**经营所得（五级超额累进）**：5%-35%

**其他所得**：统一 20% 比例税率

### 费用减除标准

| 项目 | 标准 |
|------|------|
| 基本减除 | **60,000元/年**（5,000元/月） |
| 劳务报酬/稿酬/特许权 | 减除 20%（稿酬再减按 70%） |

### 专项附加扣除

| 项目 | 标准 |
|------|------|
| 子女教育 | 1,000元/月/子女 |
| 继续教育 | 400元/月（≤48月）+ 证书当年 3,600元 |
| 赡养老人（≥60岁） | 独生 2,000元/月；非独生分摊 |
| 大病医疗 | 医保后自付 > 15,000元部分，≤ 80,000元据实扣除 |
| 住房贷款利息 | 1,000元/月（≤240月） |
| 住房租金 | 1,500/1,100/800元/月（按城市级别） |

### 计算公式
$$\text{全年应纳税所得额} = \text{全年收入额} - 60,000 - \text{专项扣除} - \text{专项附加扣除} - \text{其他扣除}$$

$$\text{应纳税额} = \text{应纳税所得额} \times \text{适用税率} - \text{速算扣除数}$$

### 国际重复征税免除
- **国际税收协定**：双边免除协议
- **税收抵免**：允许扣除境外已纳税额
- **税收饶让**：境外税收优惠部分也给予扣除

### 征收方式
- 扣缴义务人按月**预扣代缴**
- 纳税人年终**汇算清缴**（个人所得税APP）
- 全员全额扣缴申报，次月15日内报送

## 📄 来源原文
> 个人所得税最早由英国人创立(1799年)，在我国1980年正式开设。2019年1月1日起最新《个人所得税法》实施。居民综合所得适用七级超额累进税率3-45%，以每年收入减除费用60000元以及专项扣除、专项附加扣除和其他扣除后的余额为应纳税所得额。
