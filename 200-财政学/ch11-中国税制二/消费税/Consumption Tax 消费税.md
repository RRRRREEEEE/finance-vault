# 消费税 (Consumption Tax)

## 📌 定义
消费税是对消费品和特定消费行为按流转额征收的一种商品税，属于中央税。征收范围具有选择性（15个税目），一般在生产/委托加工/进口单一环节征收。

## 🧠 机制

### 五大特征
1. 征收范围具有**选择性**（非普遍征收）
2. 征收环节具有**单一性**（通常只在一个环节征收）
3. 平均税负**差异较大**（奢侈品高、普通消费品低）
4. 计税方法具有**灵活性**（从价/从量/混合）
5. 属于**中央税**范畴

### 15个税目与税率

| 税目 | 税率 |
|------|------|
| **烟** | 甲类卷烟 56%+0.003元/支（生产），11%+0.005元/支（批发）；雪茄36%；烟丝30% |
| **酒** | 白酒20%+0.5元/500克；啤酒甲250元/吨、乙220元/吨；黄酒240元/吨 |
| **高档化妆品** | 15% |
| **贵重首饰珠宝** | 金银5%，其他10% |
| **鞭炮焰火** | 15% |
| **成品油** | 汽油1.52元/升，柴油1.2元/升等 |
| **小汽车** | 1-40%（按排量七档），超豪华+10% |
| **摩托车** | 3%-10%（按排量） |
| 高尔夫球具/游艇 | 10% |
| 高档手表 | 20% |
| 木制一次性筷子/实木地板 | 5% |
| 电池/涂料 | 4% |

### 三种计算方法

| 方法 | 公式 | 适用 |
|------|------|------|
| **从价定率** | 应纳税额 = 销售额 × 税率 | 多数税目 |
| **从量定额** | 应纳税额 = 销售数量 × 单位税额 | 成品油、啤酒、黄酒 |
| **混合计算** | 应纳税额 = 销售数量 × 定额税率 + 销售额 × 比例税率 | 卷烟、白酒 |

> 消费税是**价内税**，不含税销售额 = 含税销售额 ÷ (1 + 消费税税率)

### 自产自用
- 用于**连续生产**应税消费品 → 不纳税
- 用于**其他方面**（福利、赠送等）→ 移送使用时纳税
- 组成计税价格（从价）=（成本 + 利润）÷（1 - 比例税率）

### 委托加工
- 受托方代收代缴消费税
- 组成计税价格 =（材料成本 + 加工费）÷（1 - 消费税率）
- 收回后直接销售 → 不征；收回后继续加工 → 已代扣消费税可抵扣

### 进口
- 从价：组成计税价格 =（关税完税价格 + 关税）÷（1 - 消费税率）
- 复合：组成计税价格 =（关税完税价格 + 关税 + 进口数量 × 定额税率）÷（1 - 消费税率）
