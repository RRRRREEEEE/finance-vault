# 排放标准管制 (Emission Standard)

## 📌 定义
解决外部性的第二个思路：由政府公布一个确定的污染指标，所有人按照统一的标准排放或污染，一旦违反就面临制裁。

> 💬 通俗理解：政府划一条线——所有工厂的排污都不能超过这个数，超标就罚。一刀切，简单粗暴。

## 🧠 机制

### 运作方式

- 政府设定统一的排放上限
- 所有受管制企业必须将排放控制在标准以下
- 违规面临法律制裁

### 核心问题：企业间的分配不公

当受管制的企业超过一个时，统一的排放标准会分配给边际收益曲线不同的企业。如原文所述：

- 厂商 X 和厂商 Z 面临相同的边际成本 MPC（Marginal Private Cost，边际私人成本），即 MPC₁ = MPC₂，以及相同的排放标准
- 但 X 和 Z 的边际收益曲线 MR（Marginal Revenue，边际收益）不同（MR₁ ≠ MR₂）
- 统一标准下，X 和 Z 减少的产出并不相等
- **结果**：打破了原有市场均衡，出现企业之间的不公

![[财政学Ch04 排放标准企业差异图.png]]

> 图形说明：两企业面临相同排放标准（MPC₁+d = MPC₂+d），但因 MR 曲线不同，减少的产出不相等（X₂ ≠ Z₂），造成企业间的分配不公。

### 实例

规定所有的汽车都以相同的标准排放尾气——无论车型、用途、地区。

## 🏛️ 应用

| 方面 | 评价 |
|------|------|
| 优点 | 简单直接、易于执行、法律威慑力强 |
| 缺点 | 不考虑企业差异、成本效益比不一定最优、缺乏动态激励（达标后不再改进） |
| 与庇古税对比 | 标准管制是"量"的控制，庇古税是"价"的控制——经济学家普遍认为庇古税更有效率 |
