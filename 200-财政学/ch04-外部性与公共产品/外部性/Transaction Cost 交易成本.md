# 交易成本 (Transaction Cost)

## 📌 定义
交易成本是交易双方为达成交易而付出的谈判、搜寻和履约的成本，涵盖事前、事中和事后三个阶段。由罗纳德·科斯引入经济学。

> 💬 通俗理解：买卖不能白做——你要花时间找买家、讨价还价、签合同、盯着对方履约。这些"摩擦"就是交易成本。

## 🧠 机制

### 交易成本的三个维度

| 阶段 | 内容 |
|------|------|
| **事前** | 搜寻交易对象、信息收集、评估 |
| **事中** | 谈判、讨价还价、起草合同 |
| **事后** | 监督履约、违约处理、法律诉讼 |

### 为什么交易成本重要

- **交易成本为零** → [[Coase Theorem 科斯定理]]成立：无论初始产权归谁，自由谈判可达最优配置
- **交易成本不为零** → 产权的初始分配就很重要，它直接影响资源配置结果
- **产权清晰界定** → 减少交易成本 → 促成交易达成

> 科斯本人的结论：产权的界定是市场交易的必要前提。

## 🏛️ 应用

- **企业存在的理由**：科斯在《企业的性质》中提出——企业之所以存在，是因为市场交易有成本，企业内部用管理指令替代市场交易可以节约交易成本
- **政策设计**：法律和制度的核心功能之一就是降低交易成本（清晰的产权、标准化合同、有效的司法系统）
- **环境政策**：参与方极多时交易成本高昂（如气候变化），科斯方案失效 → 需要政府干预

## 📖 相关术语

- [[Coase Theorem 科斯定理]]：交易成本为零时的理想结果
- [[Property Rights 产权]]：清晰产权是降低交易成本的关键
- [[Market Failure 市场失灵]]：交易成本过高可能导致市场失灵
