# 第四讲 知识串讲：市场越界了怎么办——外部性与公共产品

## 一、一句话回答

**市场交易的代价有时不反映在价格里——你排污别人买单（负外部性），你做好事没报酬（正外部性），你需要的东西人人等着搭便车（公共产品）。这五种情况市场都搞不定。五种解决方案——庇古税、排放标准、科斯谈判、社会惯例、排污权交易——从不同角度修正价格信号，让私人成本回归社会成本。**

---

## 二、外部性是什么？五个特征一把抓

PPT 开宗明义：**外部性是一个经济主体的活动以没有反映在市场价格中的方式直接影响他人福利。**

五个特征，每一个都有深意：

| 特征 | 含义 |
|------|------|
| **没有或不能确立产权的结果** | 如果空气有产权，污染就得付钱——外部性就内部化了 |
| **企业和消费者都可能产生** | 工厂排烟是外部性，你在公共场合吸烟也是 |
| **本质上具有相互性** | 施害者和受害者是相互的——限制工厂也损害了工厂 |
| **可能是正的也可能是负的** | 蜂农养蜂→附近果园增产（正）；工厂排污→下游渔业受损（负）|
| **无法被消灭** | 外部性只能被管理和内部化，不可能从世界上消失 |

PPT 把外部性分为五类：
1. 生产的正外部性：养蜂场放飞蜜蜂提高了附近果园水果产量
2. 消费的正外部性：个人注射疫苗减少他人感染可能性
3. 生产的负外部性：工厂向河流排放污水影响下游
4. 消费的负外部性：公共场合吸烟导致他人吸入二手烟
5. **共有资源与外部性**：个体的消费给稀缺的共有资源带来压力 ← 这是公共地悲剧的伏笔

## 三、三种外部性的微观机制

### 3.1 负外部性：私人成本 < 社会成本 → 产量过剩

PPT 用造纸厂 + 渔民案例：

- **MPC**（边际私人成本）：工厂实际掏的钱
- **MD**（边际损害）：渔民的损失
- **MSC = MPC + MD**（边际社会成本）：社会真正承担的成本
- **MR**（边际收益）：工厂的收益

工厂按 MPC = MR 做决策 → 产量 Q1。但社会最优要求 MSC = MR → 产量 Q2。

**Q1 > Q2**：负外部性下，市场产量高于社会最优产量。工厂不承担 MD → 生产太多。


### 3.2 正外部性：社会收益 > 私人收益 → 产量不足

PPT 用基础教育举例：

- **MR**（私人边际收益）：个人教育回报
- **SMR**（社会边际收益）：全社会因为一个人受教育获得的回报
- **MC**（边际成本）

市场按 MR = MC 定产量 Q1，但社会最优要求 SMR = MC 定产量 Q2。

**Q1 < Q2**：正外部性下，市场产量低于社会最优。你做好事没报酬 → 做得太少。


### 3.3 公共地悲剧：个体理性 → 集体毁灭

公共资源（如公共草地）的场合，所有生产者都对人均产量产生影响。

- **AR**（个人平均收益）：你加一头牛你获得全部牛肉
- **SR**（社会边际收益）：你加一头牛对草地总量的影响由所有人承担
- **MC**（边际成本）

市场均衡 Q1（AR = MC），社会最优 Q2（SR = MC）。**Q1 > Q2**——每个牧民都往里加牛，草地被吃秃。

> "毁灭是所有人的共同宿命"——PPT 原话。

## 四、五种解决方案：从不完美中找最优

PPT 先讲负外部性的解决方案，再讲正外部性。

### 方案一：庇古税与庇古补贴

**庇古税**：对污染者征税，税额 = 边际损害（cd）。征税后 MPC 被推到 MPC + cd = MSC → 私人自动选择社会最优产量 Q2。价格上涨、产量下降、社会回到最优。


**PPT 追问**：如何找到庇古税的实际税收水平？——这恰恰是实施中最难的问题。你需要准确知道边际损害曲线，但现实中几乎不可能精确度量。

**庇古补贴**：反过来，向污染企业支付补贴（fhcd 面积），要求其回到最优产量。企业得到补贴后收益达到市场水平 → 放弃污染。


**PPT 追问**：庇古补贴带来逆向激励——企业可能为了拿补贴而先污染，或者更多企业进入污染行业等补贴。

| | 庇古税 | 庇古补贴 |
|------|------|------|
| 机制 | 征税提高成本 | 补贴换取减产 |
| 谁出钱 | 污染者出钱给政府 | 政府出钱给污染者 |
| 风险 | 税负转嫁给消费者 | 逆向激励（为补贴而污染）|

### 方案二：排放总量标准管制

政府公布统一的污染指标，所有人按相同标准排放，违反就制裁。例如：规定所有汽车以相同标准排放尾气。

**问题**：当受管制企业超过一个时，不同企业的边际收益曲线不同（MR1 ≠ MR2），统一标准要求它们减少相同产量（X1 = Z1），但它们的减排成本不同（X2 ≠ Z2）。


这会导致：
- 打破了原有市场均衡
- **企业之间的不公平**——减排成本低的企业和减排成本高的企业被同等对待

**PPT 追问**：固定标准执行中有哪些挑战？

### 方案三：科斯定理——让当事人自己谈

PPT 用了两页的完整数字案例来讲这个逻辑。这是考试高频考点：

**假设**：一条河，渔民初始收益 $200。上游建了造纸厂后，渔民收益降到 $100。

**场景一：法律规定渔民拥有清洁权**

| 造纸厂收益 | 造纸厂买清洁权出价 | 渔民最终收益 | 造纸厂最终收益 | 社会总收益 |
|------|------|------|------|------|
| $300 | $100 | $200 | $200 | $400 |
| $1000 | $300 | $300（退出生产）| $700 | $1000 |

**场景二：法律规定造纸厂拥有污染权**

| 渔民清洁收益 | 渔民买污染权出价 | 造纸厂最终收益 | 渔民最终收益 | 社会总收益 |
|------|------|------|------|------|
| — | — | $300 | $100 | $400 |
| $1000 | $300 | $300（退出生产）| $700 | $1000 |

**结论**：无论初始产权归谁（渔民有清洁权还是纸厂有污染权），协商的结果都使权利最终落到收益更高的人手中。**社会收益达到最大化。**

### 科斯定理的正式表述与边界

PPT 给出了两种等价表述：
1. 在**交易成本为零**的条件下，资源配置的效率与初始的产权归属无关
2. 在完全竞争的条件下，**私人成本等于社会成本**

但现实中交易成本不为零。科斯引入**交易成本**概念：交易双方为达成交易而付出的谈判、搜寻和履约成本——涵盖**事前、事中、事后**三个阶段。

**交易成本 > 0 时，产权的初始分配至关重要**——它直接影响资源配置结果。科斯本人说："产权的界定是市场交易的必要前提。"产权的充分界定本身就可以极大地减少交易成本，促成交易的达成。

**围绕科斯定理的争议**：
- 科斯定理表述的同义反复（tautology）
- 科斯定理对分配问题的忽视——这也是整个效率经济学的共同争议

推荐阅读：科斯《社会成本问题》、《企业的性质》。

### 方案四：社会惯例、道德压力、企业责任

不靠法律，不靠税收——靠社会规范。

- 欧盟诸多国家：环境友好已成为社会道德和企业社会责任的基本组成部分，所有人都生活在这样的环境中 → 共同体成员不得不遵守一种短期成本更高的生活方式
- 社会惯例通过各种方式迫使人们遵守商业道德和社会公德
- 遵守的人在**长期**中得到更高回报，但短期不一定
- 社会通过自发组织形成行业协会、政党、NGO 等，协调交易方行为，建立行为规范

**PPT 追问**：怎样的情形下才能形成这种强大的社会力量？

### 方案五：排污权交易——政府指导下的市场

当市场参与方复杂或权利没有清晰界定时，外部性当事人无法协商对话。解决思路：**先界定权利，再推动市场竞争**。

三步走：
1. 政府确定全年排放污染物**总量**
2. 将污染总量分成等份 → **形成产权基础**
3. 公开市场拍卖 → 出价高者获得污染权

至此，**污染权的界定成为促成交易的关键**。减排成本低的企业多减排 → 卖配额赚钱；减排成本高的少减排 → 买配额省成本。减排总量达标，且总成本最低。

### 五个方案的现实综合应用：气候与可持续发展

PPT 专门用一页讲这个时代最重大的外部性问题。气候变化是典型的全球负外部性——每个国家排温室气体，全球一起承担后果。

人类的应对工具箱，五种方案全部上阵：

| 方案 | 气候领域的对应 |
|------|------|
| 政府定总量（排放标准）| 全球气候谈判、欧盟内部协商、中国双碳目标 |
| 庇古税/庇古补贴 | 碳税、环境税 |
| 排污权交易（科斯+管制混合）| 碳交易——政府总量控制下的市场交易 |
| 金融杠杆 | 碳金融、气候金融、绿色金融——促进绿色项目融资 |

## 五、正外部性与公共产品

### 5.1 正外部性为什么导致产品不足

正外部性 = 私人收益 < 社会收益 → 产品供应不足。

两种解决方法：
1. 政府直接生产和供应
2. 政府给私人部门提供补贴，请私人部门供应

但正外部性产品面临两种困境：
- **免费搭车**（Free-rider Problem）：总有不愿意付出的参与者，对愿意付出者的积极性造成损害
- **拥挤成本**（Congestion Cost）：即使有人提供了，由于免费搭车倾向，人们很难享有高质量的公共产品

**公共产品的定义**：对于特定的产品，市场无法提供帕累托效率的最优数量——这样的产品就是公共产品。

### 5.2 萨缪尔森定义与两个特性

萨缪尔森：公共产品是所有人共同享用的物品，一个人对它进行消费不会导致其他人对该物品消费量的减少。私人产品可以为不同消费者分割消费，公共产品必须整体消费。

| 特性 | 含义 |
|------|------|
| **非排他性** | 一旦提供出来，不能排除任何人对它的消费；技术上排他不可能或成本很高；任何人不得不消费它；任何人消费同样的数量 |
| **非竞争性** | 消费的增加不导致生产成本的增加；生产不需要追加投资；单个人的消费不能减少其他人的消费 |

### 5.3 公共产品的几个重要性质

PPT 特别指出的三点，经常被忽略：

1. 即便每个人对公共产品的消费数量相同，**人们的效用评价可以不同**（有人觉得国防值很多钱，有人觉得不值）
2. 对于什么是公共产品，划分时要参考**当时的市场和技术条件**（古代祭祀是公共产品，今天不是；收费技术改变排他性）
3. 许多公共产品是由**私人部门提供**的，也有些私人产品是由**公共部门提供**的

### 5.4 产品光谱：从纯私人到纯公共

| 类型 | 特征 | 例子 |
|------|------|------|
| **纯私人产品** | 竞争性 + 排他性（消费独占、相互排斥）| 苹果、衣服 |
| **准公共产品** | 具有一定的竞争性或排他性。在约束内可共同消费，超过约束就拥挤；收费可行 | 收费公路、公园 |
| **俱乐部产品** | 准公共产品的一种——由人们自愿组成具有门槛的消费群体 | 会员制健身房 |
| **纯公共产品** | 非竞争 + 非排他 | 国防、灯塔 |

### 5.5 私人 vs 公共产品的需求加总

这是理解公共产品供给困境的钥匙：

**私人产品**：价格是公共的，购买量是私人的。个人需求曲线**水平加总**——在同一价格下把每个人的需求量加起来。


**公共产品**：供应数量是公共的，付费意愿是私人的。个人需求曲线**纵向加总**——对同一数量，把每个人的支付意愿加起来。


**税收 = 公共产品的影子价格**。你没法直接给国防定价——外交和国防对你来说值多少钱？人们对公共产品供给的决策通过**集体选择（公共选择）**来实现——选民选择主张某一种"税收-公共产品"组合的代议制政府。**你选的不是公共产品数量，而是"税收+公共产品"的组合。**这直接通往第五讲。

### 5.6 公共产品的私人提供

PPT 强调：公共产品不一定完全由政府提供。

- 远古时代：祭祀、墓葬是极其重要的公共产品 ← 技术条件变了，分类就变了
- 政府外包：城市绿化、扫雪外包给私人公司
- 准公共产品直接由私营企业提供：高速公路、高等院校、医院
- **私有化能否改善公共产品提供的效率？** 竞争压力与私有化之间的异同——这是一个开放问题

## 六、常被忽略：外部性政策也是收入分配

PPT 最后专门提醒：**所有政策都将导致社会收入再分配，使某些人受益，某些人受损。**

| 维度 | 谁受益 | 谁受损 |
|------|------|------|
| 环境保护 | 环境敏感人群、后代 | 污染企业、相关就业 |
| 公共产品提供 | 低收入群体（更依赖公共服务）| 高收入群体（承担更多税负）|
| 降低污染 | 受污染影响大的居民 | 减排成本高的行业 |

**任何环境政策同时都是一项收入分配政策。** 不是纯技术问题——选择哪种方案、标准多严、配额怎么分，都是利益博弈。

## 七、概念对照表

| 概念对 | 区别 |
|------|------|
| **负外部性 vs 正外部性** | 负：MSC > MPC → Q_market > Q_opt（做太多）；正：SMR > MR → Q_market < Q_opt（做太少）|
| **MPC vs MSC** | MPC = 私人承担的边际成本；MSC = MPC + MD = 社会承担的边际成本。外部性的本质是两者分离 |
| **庇古税 vs 庇古补贴** | 前者让污染者付钱（成本内部化），后者给污染者付钱（买减少产量）。补贴存在逆向激励风险 |
| **庇古税 vs 科斯定理** | 前者靠政府征税修正价格，后者靠私人谈判（前提：交易成本为零）|
| **科斯定理两个版本** | 版本一：零交易成本→产权初始分配不影响效率；版本二：完全竞争→私人成本=社会成本 |
| **排放标准 vs 排污权交易** | 前者政府定每家上限（可能不公平，不同企业减排成本不同），后者政府定总量+市场分配配额（总成本最低）|
| **公共产品 vs 准公共产品** | 纯公共：非竞争+非排他；准公共：只满足一个。俱乐部产品是准公共的一种（有门槛）|
| **水平加总 vs 纵向加总** | 私人产品：水平（同价异量）；公共产品：纵向（同量异价）|
| **免费搭车 vs 公共地悲剧 vs 拥挤成本** | 搭车：不出钱但受益；公地：个体理性→集体资源毁灭；拥挤：太多人用→质量下降 |
| **非排他性 vs 非竞争性** | 排他：能不能不让没付钱的人用；竞争：我用会不会影响你用。两个不同维度 |

## 八、最重要的三条逻辑链

### 8.1 负外部性的完整分析框架
```
MPC < MSC（外部成本无人承担）
    → Q_market > Q_opt（生产过剩）
    → 五种修正方案
    → 同时导致收入再分配
```

### 8.2 排污为什么不能靠自觉
```
一家企业排污 → 成本外部化 → 产品更便宜
    → 竞争对手要么跟着排污，要么被市场淘汰
    → 不排污成为竞争劣势
    → 市场本身在奖励排污行为
    → 必须外部干预（税收/管制/产权界定）
```

### 8.3 从外部性到公共选择
```
正外部性 → 产品供应不足 → 需要公共产品
    → 无法通过市场定价（免费搭车）
    → 纵向加总 → 税收 = 影子价格
    → 人们选择"税收+公共产品"组合 → 公共选择（第五讲）
```

## 九、考试最可能考的题型

1. **名词解释**：外部性、科斯定理、交易成本、公共产品、免费搭车、公共地悲剧、庇古税、排污权交易
2. **计算/案例分析**：给出造纸厂和渔民的数据，计算不同产权归属下的谈判结果和社会总收益
3. **简答**：比较庇古税和科斯定理在解决外部性上的思路差异；私人产品与公共产品需求加总的区别
4. **论述**：结合气候变化的案例，分析五种外部性解决方案如何综合运用；为什么说"任何环境政策同时都是收入分配政策"
