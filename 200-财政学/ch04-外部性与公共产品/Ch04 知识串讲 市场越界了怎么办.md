# 第四讲 知识串讲：市场越界了怎么办——外部性与公共产品

## 一、一句话回答

**市场交易会产生"溅射伤害"（外部性），也会有"谁都不想买单"的东西（公共产品）。这两种情况市场都搞不定，政府必须出手。**

---

## 二、这张地图

```
一个问题（市场越界）
    ├→ 外部性：我的行为影响了你，但价格里没体现
    │     ├→ 负外部性：干了坏事不赔钱 → 做太多
    │     └→ 正外部性：做的好事没报酬 → 做太少
    │
    └→ 公共产品：大家都要用但谁都不想付钱
          └→ 免费搭车 → 市场供应为零或不足

五种解决方案（从理想派到实操派）
    庇古税 → 排放标准 → 科斯定理 → 社会惯例 → 排污权交易
```

---

## 三、外部性：看不见的账单

### 3.1 负外部性——你赚了钱，让别人买单

一家化工厂排废水。工厂的成本是原料+人工，但下游渔民的损失**不在工厂的成本里**。

```
私人成本（MPC）：工厂实际掏的钱
社会成本（MSC）：MPC + 边际损害（MD）
```

工厂按 MPC 做决策（Q_market），但社会最优要求 MSC = MSB（Q_optimum）。因为工厂不承担 MD，**产量必定过剩**。

反过来，**正外部性**是社会收益 > 私人收益（SMR > MR），所以产出**不足**。比如你打疫苗，保护了自己（私人收益），也降低了别人的感染风险（外部收益），但你没为后者收钱 → 打疫苗的人比社会最优少。

### 3.2 公共地悲剧——你的理性就是我们的灾难

一片公共草地，每个牧民都往上面加牛。加一头牛：你有全部牛肉，但草被消耗的成本所有人平摊。

```
个人决策：AR（平均收益）> 个人成本 → 继续加
集体结果：草地被吃秃 → 所有人一起倒霉
```

**公共地悲剧 = 个体理性加总为集体非理性。** 公共资源没有排他性产权 → 每个人过度使用 → 资源毁灭。

## 四、五种解决方案：从不完美中找最优

### 方案一：庇古税（理想派）

对负外部性征税，税额 = 边际损害（MD）。征税后 MPC 被推到 MSC 位置 → 私人自动选择社会最优产量。

反过来，对正外部性给**庇古补贴** → 激励更多生产。

**核心逻辑**：把外部成本内部化。谁制造的外部性谁买单。

### 方案二：排放标准管制（直给派）

政府直接规定：每个工厂最多排多少。简单粗暴，但有一个硬伤——不同工厂减排成本不同，一视同仁的标准会导致**企业间不公平**。

### 方案三：科斯定理（自由派）

> **交易成本为零时，无论初始产权归谁，双方谈判都能达到有效率的资源配置。**

造纸厂污染了渔民的河。如果渔民有权要求赔偿 → 造纸厂赔。如果造纸厂有权排污 → 渔民凑钱让造纸厂减排。两种情况下减排量一样，只是谁付钱不同。

但科斯定理有一个致命前提：**交易成本为零。** 现实中谈判要花时间、搜集证据要花钱、执行合同要找律师——交易成本大于零时，**初始产权的分配至关重要**。谁掌握产权谁就掌握了谈判筹码。

### 方案四：社会惯例（道德派）

企业的社会责任、社区的道德压力、行业的自发组织——不靠法律也不靠税收，靠面子和社会规范。

### 方案五：排污权交易（混合派）

综合了管制（政府定排放总量）和科斯（市场分配配额）：
1. 政府发排放配额（总量控制）
2. 企业之间可以买卖配额
3. 减排成本低的企业多减排 → 卖配额赚钱
4. 减排成本高的企业少减排 → 买配额省成本
5. 最终减排总量达标，且总成本最低

碳交易市场就是这个逻辑的实际应用。

## 五、公共产品：为什么没人愿意买单

### 5.1 公共产品的两个基因

| 特性 | 含义 | 例子 |
|------|------|------|
| **非竞争性** | 我用了不影响你用 | 国防、知识、灯塔 |
| **非排他性** | 你没法阻止没付钱的人用 | 干净空气、广播信号 |

两者都满足 = **纯公共产品**。只有一种 = **准公共产品**（俱乐部产品：会员制健身房，可排他但非竞争）。

### 5.2 核心困境：免费搭车

别人花钱提供路灯，你不出钱照样享受照明 → 每个人都等着搭别人的车 → **没人提供路灯**。

这就是**免费搭车问题**——市场不可能自发提供足够数量的公共产品。为什么？因为私人产品的需求是**水平加总**（每个人按同一价格决定买多少），公共产品的需求是**纵向加总**（每个人对同一数量愿意付多少钱）。你愿付 10 块他愿付 5 块——但你会如实告诉政府你愿意付多少吗？你不会，你想让别人多出。

### 5.3 税收 = 公共产品的影子价格

既然没人自愿付费，就强制收费——这就是**税收**。税收是公共产品的强制性价格。

### 5.4 公共选择

从个人偏好到集体决策的过程就是**公共选择**。你的意愿 + 他的意愿 + 投票程序 → 公共产品的数量。但这个过程本身不完美——第五讲会详细展开投票悖论和阿罗不可能定理。

## 六、一个常被忽略的点：外部性政策也是再分配

所有外部性政策都有人在受益、有人在受损：
- 庇古税 → 被征税的污染企业受损，下游居民受益
- 排放标准管制 → 减排成本高的企业受损
- 排污权交易 → 拿到免费配额的企业受益

**任何环境政策同时都是一项收入分配政策。**

## 七、概念对照表

| 概念对 | 区别 |
|------|------|
| **负外部性 vs 正外部性** | 负：干了坏事不赔钱，做太多（MSC > MPC）；正：做了好事没报酬，做太少（SMR > MR） |
| **庇古税 vs 科斯定理** | 前者靠政府征税修正价格，后者靠私人谈判（假设零交易成本） |
| **排放标准 vs 排污权交易** | 前者政府定每家的上限（可能不公平），后者政府定总量+市场分配（成本最低） |
| **公共产品 vs 准公共产品** | 纯公共：非竞争+非排他（国防）；准公共：只满足一种（收费公路=排他但非竞争，公共渔场=竞争但非排他） |
| **水平加总 vs 纵向加总** | 私人产品：水平（同一价格，数量累加）；公共产品：纵向（同一数量，支付意愿累加） |
| **免费搭车 vs 公共地悲剧** | 前者：不付费但受益（光享受不出钱）；后者：个体理性导致集体毁灭（大家都用，资源用光） |

## 八、最重要的故事：排污为什么不能靠自觉

```
一家企业排污 → 成本外部化 → 产品更便宜（比不排污的竞争对手划算）
    ↓
竞争对手要么跟着排污，要么被淘汰
    ↓
不排污成为竞争劣势
    ↓
没有外部干预（税收/管制/交易机制），市场本身**奖励**排污行为
```

这就是为什么"靠企业自觉"不靠谱——不是企业家道德败坏，是竞争本身在惩罚守规矩的人。外部性的本质是**价格信号失真**，修价格（庇古税）比修人心（道德呼吁）有效。
