# 免费搭车问题 (Free Rider Problem)

## 📌 定义
免费搭车是指总有人不愿意为公共产品付费，却仍然享受公共产品的好处。这种行为对愿意付费参与者的积极性造成损害，导致公共产品供给不足。

> 💬 通俗理解：班级群里说凑钱买空调，总有人说"我不热，不出钱"。但空调装好以后，他照样吹。如果每个人都这么想——完了，空调永远买不成。

## 🧠 机制

### 为什么会出现免费搭车

1. 公共产品的**非排他性**：一旦提供，无法排除任何人消费
2. 个人的**理性自利**：不付费也能享受 → 最优策略是假装不需要
3. 结果：没有人愿意表露真实偏好 → 市场无法提供帕累托最优数量

### 后果

- 正外部性产品的**两种困境**之一
- 导致公共产品私人供给不足 → 政府干预的经典理由
- 与 [[Positive Externality 正外部性]] 直接相关：正外部性意味着私人收益 < 社会收益 → 私人不愿足量提供

## 🏛️ 应用

| 场景 | 免费搭车表现 | 解决方案 |
|------|-------------|----------|
| 国防 | 每个人都受保护，无法排除任何人 | 政府强制征税提供 |
| 公共广播 | 不捐款也能收听 | 政府拨款（BBC）或广告模式 |
| 路灯 | 路过的人都受益 | 政府税收提供 |
| 国际气候协议 | 某国不减排也享受全球减排好处 | 国际条约 + 碳边境调节 |
| 工会 | 不交会费也享受工会谈判的涨薪 | 强制会员制（部分国家）|
