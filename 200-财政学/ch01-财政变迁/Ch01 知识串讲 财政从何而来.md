# 第一讲 知识串讲：财政从何而来——人类历史中的财政变迁

## 一、一句话回答

**财政不是凭空设计的制度，而是在三千年人地博弈、中西分岔、战争与革命中被逼出来的生存方案。从人头税到土地税、从实物税到货币税、从领地国家到预算国家——财政变迁的本质是国家权力与个人财产之间边界的不断重划。**

---

## 二、这门课在讲什么？

PPT 开篇交代了课程的底层逻辑：**从理论和实践两个角度，系统阐述现代财政。**

- **理论线**：财政和社会福利的关系 → 代议制政府的优势和困境 → 经济发展和财政的关系 → 财政结构、绩效、预算、税收的原理
- **实践线**：中国财政政策逻辑 → 中国预算体系和预算法 → 中国财政支出结构 → 中国财政收入结构（预算管理、税收制度、政府债务）

指定教材：《公共财政概论》（马工程重点教材），参考：哈维·罗森《财政学》

**一句破题**："帝国，兴于雄心，而败于财政！"

## 三、中国财政史：三千年只做了一件事——把人头税打死

PPT 将中国财政史分为四个阶段，对应四个帝国时代。


### 3.1 城邦时代（夏商周）：力役财政

最早的"财政"不是收钱，是**出人**。

**贡助彻**（夏商周三代）：《孟子·滕文公上》——"夏后氏五十而贡，殷人七十而助，周人百亩而彻，其实皆什一也。"学术界对贡助彻的性质有争议：封建社会者认为助是劳役地租、彻是实物地租；奴隶社会者认为助是奴隶制剥削；土地国有论者认为助和彻是地租与赋税的合一。

**井田制**：出现于商朝，成熟于西周。土地被分隔成方块（形如"井"字），属周王所有分配给庶民使用。领主不得买卖转让，周边为私田，中间为公田。

**初税亩**（前 594 年鲁国）：中国税收从雏形走向完全形态的标志。不分公田私田，凡占有土地者按亩交纳土地税，税率亩产 1/10。首次承认土地私有权和地主经济的合法地位。《左传》说："初税亩，非礼也，谷出不过藉，以丰财也。"

**城邦时代总结**——PPT 给出三点特征：

| 特征 | 含义 |
|------|------|
| 以人为原则、以力役为形式组织财政收入 | 税就是出人干活 |
| 财政支出项目简单，祭祀支出突出 | "国之大事，在祀与戎"——《左传》 |
| 公私不分、"式法制财" | 以法定程式规范收支：九赋（收入九类）+ 九式（支出九类），专赋专用、量入为出 |

### 3.2 第一帝国（秦汉）：人丁税的黄金时代

**商鞅变法**是这一阶段的起点。PPT 用一个对比表揭示法家与其他各家的根本分歧：

| 法家 vs | 分歧 |
|------|------|
| 墨家 | 利己 vs "兼爱" |
| 儒家 | "进化史观"vs"退化史观"；"法治"vs"德治" |
| 道家 | "竞争"vs"无为" |

具体做法七条：户籍制度（乡里什伍制）→ 军功爵位制 → 废井田开阡陌 → 重农抑商 → 统一度量衡 → 郡县制 → 弱民愚民。这是一套把国家权力扎到每户人家的制度工程。

**人丁税**（算赋、口赋）：按户籍对自然人征收的简单直接税。汉高祖四年（前 203 年）定制——十五以上到五十六岁，每人一百二十钱为一算。特点是完全不考虑纳税人财产状况，只认人头。

**徭役与兵役**：秦分更役（非军事）和戍役（兵役）。本质是"国家对劳动人民实行普遍的人身奴役制"。

**平准、均输、盐铁专卖**（汉武帝/桑弘羊）：
- **均输**：各地贡物按市价折算为土特产品，运往价格高地区销售赚差价（理论上节省运费，实际上农民被迫购买指定产品再上交）
- **平准**：京师设平准机构，价低时买入、价高时抛售，平抑物价
- **盐铁专卖**：国家垄断盐铁利润

**第一帝国总结的三大问题**（PPT 直接给了）：
1. **课税对象背离支付能力原则**：财富集中后，有能力的人少纳税，没能力的人多纳税
2. **家族与功勋双体系运行**：分封与郡县并存，皇权与相权分立，既形成专业公务人员集团，又鼓励外戚宦官发展，限制了国家机器的专业化
3. **官僚集团的腐败和崩溃**：谶纬之学流行、诬告之术甚嚣尘上

### 3.3 第二帝国（唐宋）：从人丁走向土地

**均田制与租庸调制**（北魏→隋→唐初）：
- 均田制：每丁授田百亩（永业田 20 亩 + 口分田 80 亩）
- 租庸调：每丁每年输粟 2 石（租）+ 服役 20 日或不役纳绢（庸）+ 输绢 2 丈绵 3 两（调）
- 府兵制：全民皆兵，配合以人口为中心的赋税制度

**两税法**（唐建中元年 780 年，宰相杨炎）：均田制瓦解后的根本性改革——
- 租庸调并入户税、地税
- 不分主户客户，一律按现居地登籍纳税
- **放弃按丁征税，改为按资产和田亩征税** ← 这是关键转折
- 每年分夏秋两季征收

意义：国家对农民的人身控制开始松弛，开了中国封建社会后期改革人丁税的先河。

**宋朝的新发展**（PPT 特别强调）：
- **工商税大发展**：盐酒茶榷税 + 自由贸易商税，史无前例地增加
- **公共财政萌芽**：养兵、养官、养民——体制内工作的雏形
- **王安石变法**：探索市场与政府的边界
- 大规模物资调配能力：基础设施和社会管理的提升

### 3.4 第三帝国（明清）：人丁税彻底死亡

元代打断了第二帝国传统，不足百年退出中原。明代恢复两税法 + 人丁征税，但差役制度很快难以执行。

**一条鞭法**（明万历/张居正）：赋役合并为一，一律折银缴纳，差役由政府用银雇人充当。简化了税目和征收手续，但**并未取消丁赋**——丁银与田赋仍为两个税目，只是合并缴纳。因阻力重重，并未完全实行。

**摊丁入亩**（清雍正）：康熙五十一年（1712 年）固定丁数、"滋生人丁永不加赋" → 雍正推广摊丁入亩，丁税平均摊入田赋，征收统一的"地丁银"。地丁银与实际人丁数已毫无联系——**中国历史上延续二千余年的人丁税正式步入尾声。**

**第三帝国小结**（PPT 六点精华）：
1. 疆域辽阔、人口增加、社会平稳，中华帝国走向了稳定
2. **火耗归公**——雍正朝将预算外收入纳入预算，高薪养廉
3. **官商关系**：政府一方面重农抑商，另一方面又纵容官家营商
4. **量入为出**：第三帝国的财政原则
5. **财富南北再平衡**：南方经济发达，向北方输送财富
6. **官僚标准化**：沉重的财政负担

### 3.5 历代财政收入数据——硬对比

| 朝代 | 盛世时期 | 折合白银岁入 |
|------|------|------|
| 西汉 | 武帝时代 | 1000 万两 |
| 唐代 | 天宝年间 | 5230 万两 |
| 北宋 | 英宗时期 | 1.16 亿两 |
| 明代 | 万历初年 | 3450 万两 |
| 清代 | 乾隆中期 | 4000 万两 |

**北宋财政收入远超汉唐明清**——工商税的力量。宋代是中华帝国财政能力的巅峰，明清反而退步了。


### 3.6 中国财政史的五大趋势

PPT 第一讲总结给出：
1. 从人头税 → 土地税和商业税
2. 从实物税 → 货币税
3. 大一统国家的权力逐步增强
4. 盐铁专卖带来的**国有垄断传统**（到今天还在影响）
5. 历史在现实中的光影投射

## 四、西方财政史：税收为什么需要议会点头？

### 4.1 中世纪欧洲：碎片化的起点

PPT 给出了中世纪的六个关键特征：
- 典型封建采邑制度——庄园是实际的基层主体，拥有巨大自治权
- 农奴依附于庄园主，通过法律和习俗建立稳定依附关系
- 土地私有财产权残缺，农奴没有自由
- 教会提供统一信念结构，组织意识形态

**东西方中世纪的制度分岔**：

| 维度 | 中国 | 西方 |
|------|------|------|
| 政治结构 | 集权、一统 | 分封、协商 |
| 宗教与王权 | 皇权高于宗教 | 宗教权利高于王权 |
| 土地私有 | 地主为核心 | 领主为核心 |
| 人才选拔 | 科举制（流动性强） | 世袭制（流动性弱） |
| 税收体制 | 规模巨大的统一税制 | 碎片化的地方税权 |
| 封建时长 | 2000 年 | 1000 年 |


### 4.2 采邑制：以土地为纽带的等级体系

采邑制是欧洲中世纪早期（8 世纪起）的土地占有制度。查理·马特在法兰克王国（715-741 年）实施，英国从威廉一世开始实行。国王 → 大封建主 → 下属 → 下属的下属……形成一个以土地为纽带的领主与下属关系链。没有统一财政——每个领主自己收自己的。

### 4.3 罗马帝国财政：包税制的教训

- **帝国初期**：以"行省税"为核心，按土地征收，统一 10% 税率，实行**腐败的包税制**——政府把征税外包给私人包税商
- **帝国中期**：商业税收成体系——遗产税、拍卖税、销售税
- **东罗马帝国**：废除行省制，推行土地人头合并征收，佃农固定在土地上 → 社会僵化、经济衰退
- **优士丁尼法典**：以私有产权为基础，奠定了大陆法系的基本原则

### 4.4 西方世界兴起的关键机制

PPT 用一个动态因果链来解释：

**14 世纪**：
- 军事技术进步 → 抵御侵略成本上升 → 庄园规模被迫扩大或多庄园联合
- 战争和瘟疫 → 人口减少 → 土地-人口比变化 → 庄园间劳动力竞争
- 人口减少、税收下降 → **两条路分岔**：有的国家加强独裁，有的国家鼓励商业、保护产权以换取税收 ← **这就是资本主义的起点**

**16 世纪**：
- 人口大规模恢复 → 土地-人口比再次变化 → 边疆运动（大量开垦新土地）
- 农奴脱离控制，取得土地

**西班牙 vs 荷兰的对比**（PPT 关键案例）：
- 西班牙：用加强专制来对付财政危机 → 低地省份独立
- 荷兰：通过建立产权体系 → 成为欧洲经济中心

**低地国家**（荷兰、比利时、卢森堡）：处于帝国边缘，与南部核心地区路径相反——南部走向集中，低地走向多中心发展。


### 4.5 路径依赖：为什么变革是渐进的？

PPT 给出了一个重要的认知框架：

```
可感知的现实 → 信念 → 制度 → 政策 → 改变了的可感知的现实（循环）
```

真正的社会发展不是一次革命能实现的，本质上是渐进的——**路径依赖**。社会通过规则把选择导向更小的行动集，改进控制环境的能力。人们把解决不了的问题通过魔法、巫术、宗教、意识形态等方式来解释，降低共同体内的认知成本。

### 4.6 光荣革命：财政权的转移

1689 年英国议会通过《权利法案》和《王位继承法》，奠定国王"统而不治"的宪政基础：
- 国王未经议会同意无权征税 → **现代财税出现**
- 成立英格兰银行 → **现代金融出现**
- 实质是资产阶级新贵族与部分大土地所有者的政治妥协

这不仅是政治革命，更是**财政革命**——税收权从国王手里移到了议会手里。

### 4.7 现代西方财政的思想地基与制度变迁

四本奠基著作：
- 洛克《政府论》（1690）——天赋人权、限权政府
- 孟德斯鸠《论法的精神》（1748）——分权、法治
- 卢梭《社会契约论》（1762）——主权在民、自由民主
- 亚当·斯密《国富论》（1776）——市场机制、廉价政府

**两次制度跃迁**：

| 跃迁 | 直接原因 | 深层原因 |
|------|------|------|
| 领地国家 → 税收国家 | 战争 + 债务清偿 | 经济发展/经济社会结构转型，要求通过税收界定和保护私人产权 |
| 税收国家 → 预算国家 | 完整统一规范透明地约束政府支出 | 1816 年英国第一个完整政府预算；1921 年美国《预算和会计法》 |

## 五、新中国财政：从统收统支到分税制

PPT 按七个阶段展开，每一个都有具体的历史困境和政策应对。

### 5.1 三年恢复时期（1949-1952）

**困境**：经济凋敝 + 国际紧张（美军仁川登陆）；军费一度达总支出 50% 以上；人民币发行：1949 年 11 月比 1948 年增加近 100 倍，到 1950 年 2 月增加近 270 倍——物价大涨，货币贬值。

**应对**：
- "边抗、边稳、边建"三线并行
- 发行**人民胜利折实公债**——以煤炭、小米、大米、布匹、面粉为计价单位（物价不稳，只能以实物计价）
- 先建立"高度集中、统收统支"的财政管理体制——财政、物资、现金全部统一管理
- 后调整为"统一领导、分级管理"——全国财政划分为中央-大区-省三级

### 5.2 一五时期（1953-1957）

局面好转：国民经济全面恢复，物价稳定，财政收支平衡略有结余。核心任务：为工业化筹集资金 + 促进社会主义改造。

1953 年预算扩张过度，出现"底子过大"问题。中央任命邓小平兼任财政部长，提出**"六条方针"**：

| 方针 | 内容 |
|------|------|
| 预算归口 | 所有预算必须有部门负责 |
| 包干使用 | 收入不准降低，支出不准增加 |
| 自留预备费 | 结余留用，不再上缴 |
| 控制人员 | 人员不得随意增加 |
| 动用预备费需中央批准 | — |
| 加强财政监察 | — |

**邓小平原话**："有些同志不懂得预算底子打大了的问题的严重性。底子大了是上了马，而且是一匹烈马，上马必然还要下马，下马必须削减预算，问题就很多，所以预算底子打大了，是一个政治问题。"

### 5.3 大跃进和五年调整（1958-1965）

财税体制围绕计划经济体制的适应性探索期。三阶段：配合大跃进放权 → 应对困难收缩财权 → 财政收支规模从急剧膨胀回归常态。

### 5.4 文化大革命（1966-1977）

财税体制**频繁变动**期。主要任务：保证国家最低限度的经常性开支、保持财政运行的相对稳定。这一时期为后来包括财税体制改革在内的经济体制改革**埋下了深刻的动因**——知道不能再这样下去了。

### 5.5 放权让利（1978-1993）

改革开放起点。"权"= 财政管理权，"利"= 财政在国民收入分配格局中的份额。
- 承包制、租赁制改变原国营企业纳税制度
- 打破"财权集中过度、分配统收统支、税种过于单一"的传统体制

### 5.6 制度创新（1994-1997）

意识到"放权让利"不可持续，从利益格局调整转向新型体制建立。1994 年起踏上制度创新之路：
- 建立**流转税 + 所得税双主体税种**的完整税制
- 通过了《预算法》
- 搭建了适应社会主义市场经济的财税体制基本框架
- **分税制**：中央税 + 地方税 + 共享税（增值税央地 75:25）

### 5.7 公共财政建设（1998-2012）

强调财政的"公共性"：
- **费改税**：计划外资金进入计划内管理
- 国有企业"关停并转"：推动自负盈亏，减少政府支出义务
- 探索央地权责利边界

### 5.8 新变革时期（2012-今）

当前处于焦灼状态，四大问题彼此缠绕：
1. 中央财权集中 → 地方积极性受限
2. 地方财政去地产化开启但没完成
3. 财务约束增强但财政效率低下
4. 债务迅速上升 → 财政风险累积

## 六、结语：财政史的底层逻辑

PPT 结尾给了三个哲理性的总结：

1. **同样的条件，不同结果**：有的社会经济走向成功，有的走向失败；有的分散，有的一统。社会变化是渐进的过程——政权更替不一定能改变制度或意识形态；生产的制度结构变化才从根本上改变经济和财政状态。

2. **财政是中间观察指标**：在意识形态不变条件下，财政制度的变化会导致社会产出的巨大变化。一套制度组合旨在建立激励体系——给社会参与者明确的信号，告诉他们怎样最大化自己的利益。

3. **历史的周期律**：我们每个人都受制于所处时代。在更长历史长河中，经济规律发生作用的模式超出我们的想象。财政经济的规律与历史的其他规律结合在一起，形成中国历史的周期律和世界历史的周期律。

## 七、概念对照表

| 概念对 | 区别 |
|------|------|
| **贡助彻 vs 初税亩** | 前者以力役为税（为公田劳动），后者按亩征实物（土地成为课税对象，承认私有产权）|
| **人丁税 vs 土地税** | 前者按人头收（算赋口赋，不管贫富），后者按田亩收。中国三千年史就是前者消亡的过程 |
| **租庸调制 vs 两税法** | 前者以人丁为本（丁→租庸调），后者以资产为本（按资产田亩）。两税法是中国税制从人丁走向土地的分水岭 |
| **一条鞭法 vs 摊丁入亩** | 前者赋役合并折银（简化税目但丁赋仍存），后者丁税彻底摊入田赋（人丁税终结）|
| **均输 vs 平准** | 均输调运输（各地贡物→价高地区销售），平准稳物价（京师贱买贵卖）。两者配合构成国家经济干预体系 |
| **包税制 vs 现代税收** | 前者外包给私人（罗马教训：腐败温床），后者政府直接征收（制度化）|
| **领地国家 → 税收国家 → 预算国家** | 领主管自己领地 → 国家统一征税（战争驱动/产权保护）→ 征税和支出都有预算框着（1816 英国/1921 美国）|
| **西班牙道路 vs 荷兰道路** | 专制应对财政危机 → 失败；保护产权鼓励商业 → 经济中心 |
| **统收统支 vs 分税制** | 前者中央管一切（计划经济），后者央地分税各自过（市场经济）。1994 年是分水岭 |
| **中西分岔** | 中国：大一统 + 强国家 + 弱产权 + 科举流动 + 统一税制；西方：分裂竞争 + 议会制衡 + 强产权 + 世袭 + 碎片税权 |
| **火耗归公** | 雍正将预算外收入纳入预算内管理，配合高薪养廉——把灰色收入阳光化 |

## 八、最重要的故事线

### 8.1 中国：人→地→货

```
夏商周（力役/初税亩）
    → 秦汉（人丁税/徭役/盐铁专卖）→ 国家权力高涨
    → 唐宋（两税法：人→地；宋朝工商税爆发）→ 税收巅峰
    → 明清（一条鞭→摊丁入亩）→ 人丁税正式消亡
    → 结论：从人头到土地到商业，从实物到货币，国家权力越来越强
```

### 8.2 西方：战争逼出议会

```
军事技术进步 → 战争规模扩大 → 成本飙升 → 国王没钱
    → 找贵族商人借 → 对方：要钱可以，交出一部分权力
    → 1689 光荣革命：税收权归议会
    → 英格兰银行成立 → 政府可以规范借债
    → 领地国家 → 税收国家 → 预算国家
    → 结论：没有议会同意不纳税——西方现代财政的基因
```

### 8.3 新中国：放权与收权的反复

```
1949-1957（统收统支/集中力量）→ 1958-1977（放→乱→收→文革）
    → 1978-1993（放权让利：企业活了/中央穷了）
    → 1994 分税制（制度创新/央地分家）
    → 1998-2012（公共财政/费改税/国企关停并转）
    → 2012-今（央地博弈/去地产化/债务累积）
    → 结论：调整税制就是调整利益分配。财政改革从来不是纯技术问题
```
