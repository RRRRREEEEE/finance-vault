# 第一讲 知识串讲：财政从何而来

## 一、一句话回答

**财政不是凭空设计的制度，而是在三千年人地博弈、中西分岔、战争与革命中被逼出来的生存方案。**

---

## 二、三条线，一个答案

第一讲横跨三千年、覆盖中西方、结束于当代中国。信息量极大，但只问一个问题：**今天的财税体制是怎么变成这样的？**

答案拆成三条时间线来回答：

| 线 | 时间跨度 | 核心问题 |
|------|------|------|
| 中国财政史 | 先秦 → 清末 | 国家怎么从人身上收钱，变成从地和货上收钱？ |
| 西方财政史 | 中世纪 → 近代 | 西方为什么走出了一条"税收需要议会同意"的路？ |
| 新中国财政 | 1949 → 今 | 从统收统支到分税制，我们经历了什么？ |

---

## 三、中国线：三千年只做了一件事——把人头税打死

### 3.1 从头开始：城邦时代的力役财政

最早的"财政"不是收钱，是**出人**。

**贡助彻**（夏商周三代）：井田制下，农户以力役形式向领主交"税"——实际上是为公田干活。没有货币、没有税单、没有财政部。国家就是领主，财政就是徭役。

**初税亩**（鲁国，公元前594年）是一个分水岭：按亩征税确立了，不再只看你出多少力，还看你占多少地。

然后是**商鞅变法**（秦）：户籍制度把每个人钉在国家的台账上，军功爵制把战场和个人利益挂钩，废井田开阡陌让土地私有化——国家能力飞跃。

### 3.2 人丁税的千年统治

从秦汉到唐前期，**人丁税**是财政收入的主骨架。你这个人头在，你就得交钱（算赋、口赋）。同时还有**徭役**（更役、戍役）——国家需要修长城、挖运河，大家排队去干活。

汉武帝时期桑弘羊搞了**盐铁专卖**和**平准均输**：国家垄断盐铁利润，调剂物资运输平抑物价。这是中国历史上国家干预经济的高峰。

### 3.3 转折：两税法 → 一条鞭 → 摊丁入亩

这三个改革有一条共同线索：**把人头税一步步打死**。

```
唐代两税法（杨炎）
  → 从"按丁征收"改为"按资产田亩"
  → 人丁税退居次要

明代一条鞭法（张居正）
  → 赋役合并、折银征收
  → 实物税变货币税，税目简化

清代摊丁入亩（雍正）
  → 丁税彻底摊入田赋
  → 人丁税正式终结，中国征税对象从"人"变成"地"
```

三千年财政史一句话：**从头税到土地税/商业税，从实物税到货币税，国家权力越来越强。**

## 四、西方线：税收需要议会点头

### 4.1 罗马到中世纪：包税制的教训

**罗马帝国**的财政有一个反复出现的病灶——**包税制**：政府把征税任务外包给私人包税商。包税商向政府交一笔保证金，然后自己去压榨纳税人。差价就是利润。结果：行省被榨干，腐败遍地。

中世纪欧洲进入**采邑制**：土地是纽带，领主—封臣是等级关系，农奴依附于土地。没有统一的国家财政——每个领主自己收自己的。

### 4.2 光荣革命：税收权归议会

近代欧洲的财政变革背后有一个共同推力：**战争**。

军事技术进步 → 战争规模扩大 → 成本飙升 → 国王没钱了 → 必须找贵族和商人借钱 → 议会：要钱可以，你得交出一部分权力。

**光荣革命**（1689年）是关键节点：税收权归议会，英格兰银行成立。一个现代财税国家诞生了——政府想花钱，议会说了算；国家想借钱，有英格兰银行兜底。

从**领地国家**（国王靠自己的地租活）→ **税收国家**（靠税收养政府）→ **预算国家**（税收怎么花有预算管着）。洛克、孟德斯鸠、卢梭、亚当·斯密为这条路提供了思想地基。

### 4.3 中西分岔的关键变量

| 变量 | 中国 | 西方 |
|------|------|------|
| 税收合法性 | 皇权天授，无需议会 | 国王征税需代表机构同意 |
| 产权保护 | 弱，国家可随时征用 | 强，新兴产权体系 |
| 战争驱动 | 集权大于分权 | 竞争中形成民族国家 |
| 资本主义萌芽 | 官商并行，无法独立 | 商人阶级成为独立力量 |

## 五、新中国线：从统收统支到分税制

### 5.1 三个时期的摇摆

新中国财政史本质上是**放权与收权的反复**：

| 时期 | 关键词 | 结果 |
|------|------|------|
| 1949-1957 恢复+一五 | 统收统支、折实公债 | 集中力量办工业 |
| 1958-1977 大跃进+文革 | 放权→困难→收权 | 财税体制频繁震荡 |
| 1978-1993 放权让利 | 承包制、打破统收统支 | 企业活了，中央穷了 |

### 5.2 1994：分税制——当代财政的基石

1994年以前，中央财政穷到要向地方借钱。分税制改革一刀劈开了央地财政关系：
- **中央税**：关税、消费税等
- **地方税**：营业税、个人所得税等
- **共享税**：增值税（央地 75:25 → 后来调整为 50:50）

同时确立流转税+所得税双主体税制，通过了《预算法》。这是当代中国的财政宪法。

### 5.3 1998到现在的四条主线

| 阶段 | 主线 |
|------|------|
| 1998-2012 公共财政建设 | 费改税、国企关停并转、缩小政府支出范围 |
| 2012至今 新变革 | 央地博弈、去地产化、债务风险累积 |
| 贯穿全程 | 地方融资平台 + 土地财政兴起 |

### 5.4 最新挑战

三个问题彼此缠绕：**地方债高企** → 土地财政收缩 → 央地财权事权不匹配 → 改革压力。财政改革从来不是纯技术问题——调整税制就是调整利益分配。

## 六、概念对照表

| 概念对 | 区别 |
|------|------|
| **贡助彻 vs 初税亩** | 前者以力役为税（为公田劳动），后者按亩征实物（土地成为课税对象） |
| **人丁税 vs 土地税** | 前者按人头收（算赋口赋），后者按田亩收（田赋）。中国三千年史就是前者消亡、后者主导的过程 |
| **一条鞭法 vs 摊丁入亩** | 前者赋役合并折银（简化税目），后者丁税摊入田赋（消灭人丁税） |
| **包税制 vs 现代税收** | 前者外包给私人（腐败温床），后者政府直接征收（制度化管理） |
| **领地国家 vs 税收国家 vs 预算国家** | 领主管自己领地 → 国家统一征税 → 征税和支出都有预算框着 |
| **统收统支 vs 分税制** | 前者中央管一切（计划经济逻辑），后者央地分税各自过（市场经济逻辑） |
| **中西分岔** | 中国：大一统+强国家+弱产权；西方：分裂竞争+议会制衡+强产权 |

## 七、最重要的故事：光荣革命为什么是财政革命

```
国王打仗缺钱
  → 找贵族商人借钱 → 对方要求拿权力换
  → 1689年税收权归议会
  → 英格兰银行成立 → 政府可以规范借债
  → 现代财税国家诞生
```

和中国对比：中国皇帝不需要议会——天下都是我的。但代价是，当国家过度攫取（如明末加税征饷），没有制度机制刹车，只能等朝代崩溃。
