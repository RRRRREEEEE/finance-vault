# 第一帝国财政

## 📌 定义
第一帝国（秦汉）的财政制度始于商鞅变法，完善于盐铁会议，形成了强大的国家机制。以人丁税、徭役、国家专卖为核心，以户籍制度为征管基础。

## 🧠 机制

### 核心制度
- 户籍制度（乡里什伍制）→ 人丁税 + 徭役
- 平准均输 + 盐铁专卖 → 国家控制经济流通
- 军功爵位制 + 郡县制 → 打破世袭、中央集权

### 三大问题
1. 财富集中导致课税背离支付能力原则——有能力的人少纳税，没能力的人多纳税
2. 家族/功勋双体系 + 分封/郡县双体制 → 外戚宦官崛起，限制国家机器专业化
3. 官僚集团腐败崩溃——谶纬之学流行、诬告之术甚嚣尘上
