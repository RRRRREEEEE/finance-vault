# 西方世界的兴起

## 📌 定义
14-16世纪欧洲从封建庄园经济走向资本主义的关键转型期。核心机制：军事技术进步→庄园联合→人口变化→劳动力竞争→产权建立以换取税收→资本主义起点。

## 🧠 机制

### 14世纪的转折
- 军事技术进步 → 抵御侵略成本上升 → 庄园规模被迫扩大或联合
- 战争、瘟疫导致人口减少 → 改变土地-人口比 → 庄园间劳动力竞争
- 人口减少、税收下降 → 两种路径分岔：
  - 西班牙：加强独裁 → 治理失败 → 低地省份独立
  - 荷兰/英国：鼓励商业、保护产权 → 提高政府收入

### 路径依赖理论
社会变迁遵循"可感知的现实 → 信念 → 制度 → 政策 → 改变了的可感知的现实"的循环。真正的社会发展是渐进的——即**路径依赖**。

### 不确定性的处理
每个社会都需要处理不确定性。人们通过规则把选择导向更小的行动集。无法解释的问题通过魔法、巫术、宗教、意识形态来解释——降低共同体内部的认知成本。

## 📄 来源原文
> 一个社会的变迁中，经常是"可感知的现实-信念-制度-政策-改变了的可感知的现实"，而我们发现，真正的社会发展从来都不是可以通过一次革命实现的，本质上都是渐进的，我们称之为"路径依赖"。14世纪欧洲的变化：军事技术的进步，使得抵御侵略的成本上升；战争、瘟疫导致人口的减少，改变了土地—人口比，导致了庄园之间的劳动力竞争；人口的减少，税收的下降，对欧洲各国产生了不同的影响，有的地区走上了加强独裁的道路，有的地区走向了鼓励商业、保护产权以便提高政府收入的道路——建立产权以便获得税收，这是资本主义的起点。
