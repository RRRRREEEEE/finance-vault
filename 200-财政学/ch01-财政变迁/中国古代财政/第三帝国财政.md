# 第三帝国财政

## 📌 定义
第三帝国（明清）的财政特征：疆域辽阔、人口增加、社会平稳，以田亩和人丁双轨征税（两税法 + 差役），经历了一条鞭法和摊丁入亩两次重大改革后，最终走向以土地为核心的税制。

## 🧠 机制

### 明代
- 恢复以田亩（两税法）和人丁（差役）征税的传统
- 打击工商业——工商税收地位一落千丈
- 张居正一条鞭法：部分丁役摊入田亩、各类税合并折银

### 清代
- 雍正摊丁入亩：丁税彻底摊入田赋，征收统一地丁银
- 人丁税正式终结

### 第三帝国财政特点
- 官僚体系庞大 → 对弹性收入/非正式收入的依赖
- 火耗归公：雍正将预算外收入纳入预算，高薪养廉
- 官商关系：一方面重农抑商，一方面纵容官家营商
- 财政原则：量入为出
- 财富全国再平衡：南方经济发达，向北方输送财富
- 国家与官僚制度标准化 → 沉重的财政负担
