# 中世纪欧洲财政

## 📌 定义
中世纪欧洲（17世纪以前）的财政建立在封建采邑制度之上——庄园是实际基层主体，拥有巨大自治权。农奴依附于庄园主，土地私有产权残缺，教会提供统一的信念结构。

## 🧠 机制

### 中西欧制度分岔
| 欧洲 | 中国 |
|------|------|
| 分封、协商 | 集权、一统 |
| 宗教权利高于王权 | 皇权高于宗教权利 |
| 地主为核心的土地私有 | 领主为核心的土地私有 |
| 人才世袭制 | 人才科举制 |
| 社会流动性弱 | 社会流动性强 |
| 碎片化的地方税权 | 规模巨大的统一税制 |
| 约1000年封建时代 | 约2000年封建时代 |

## 📄 来源原文
> 希腊-罗马传统下的欧洲中世纪社会，有几个重要特点：典型的封建采邑制度——所有的生产集中在一个个庄园中，庄园是实际的基层主体，拥有巨大的自治权。农奴依附于庄园主，他们之间通过法律、习俗建立起稳定的依附关系。土地的私有财产权残缺，农奴没有自由。教会提供一个统一的信念结构，组织意识形态。分封、协商 VS 集权、一统；宗教权利高于王权 VS 皇权高于宗教权利。
