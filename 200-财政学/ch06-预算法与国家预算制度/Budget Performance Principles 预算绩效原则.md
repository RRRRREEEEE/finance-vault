# 预算绩效原则 (Budget Performance Principles)

## 📌 定义
预算绩效评价的原则体系包括绩效原则（3E 维度）和良治原则（法定授权、受托责任、透明度、预见性、全面性、公民参与）两大类。

## 🧠 机制

### 一、绩效原则

**3E 维度评价**（合规基础上的绩效评价）：
1. **总量绩效**：宏观层面的支出总量控制效果
2. **配置绩效**：资源配置效率与公平
3. **运营绩效**：项目层面的投入产出效益

**其他绩效原则**：
- 实质与形式相结合：合规优先，效率兼顾
- 投入产出配比：资源转化为成果

### 二、良治原则

| 原则 | 含义 |
|------|------|
| **法定授权** | 财政权力需要系列法律授权 |
| **受托责任** | 确保权力要向公众负责 |
| **透明度** | 确保财政信息的自由获取和流动 |
| **预见性** | 财政规划的可预知和统一有效 |
| **全面性** | 尽可能把政府经济行为全部纳入预算流程 |
| **公民参与** | 公众参与预算决策和监督 |
