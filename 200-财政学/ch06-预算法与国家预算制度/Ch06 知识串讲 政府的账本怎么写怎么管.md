# 第六讲 知识串讲：政府的账本怎么写怎么管——预算法与国家预算制度

## 一、一句话回答

**政府预算是一本有法律效力的收支计划书——"四本账"按复式编制，走"二上二下"审批流程，用四大制度管住钱，最后靠三级绩效框架和人大监督来确保钱花得值。**

---

## 二、预算是什么

**国家预算**（National Budget）就是政府的基本财政收支计划。三种身份：形式上是一张收支对照表，政治上是具有法律效力的文件，手段上是调节国民经济的工具。

这个账本有三件事必须做（三大职能）：
1. **法律与行政管理职能**——预算本身就是依法行政的工具
2. **财政政策职能**——用收支调节宏观经济
3. **财政监督职能**——通过预算约束政府行为

而要当好一本合格的账本，它必须遵守五大原则：

| 原则 | 一句话 |
|------|--------|
| **公开性** | 必须经人大讨论批准并向社会公布 |
| **可靠性（权威性）** | 内容必须符合《预算法》 |
| **完整性** | 不得在预算外另列预算 |
| **统一性** | 收支按统一程序、口径编制 |
| **年度性** | 采用历年制（1月1日到12月31日） |

---

## 三、记账的方法：怎么分类、怎么算

预算编制有两套交叉维度：按结构和按算法。

### 3.1 按结构分：单式 vs 复式

| 方法 | 做法 | 特点 |
|------|------|------|
| **单式预算**（Single Budget） | 全部收支编入一本总账 | 简单直观，看不出钱的具体性质 |
| **复式预算**（Double Budget） | 按经济性质分成两本或以上 | 分类清晰，便于分析——中国采用复式 |

### 3.2 按算法分：零基 vs 增量

| 方法 | 做法 | 现实中的问题 |
|------|------|-------------|
| **零基预算**（Zero-Based Budget） | 以零为起点，每个项目从头论证 | 理想很丰满——工作量极大，实践中很少完全执行 |
| **增量预算**（Incremental Budget） | 在去年基础上增减调整 | 现实很骨感——容易"路径依赖"，历史基数不合理的永远不合理 |

### 3.3 按层级分：总预算 → 部门预算 → 单位预算

- **总预算**（Overall Budget）——政府整体收支计划，最顶层
- **部门预算**（Department Budget）——一个部门（如教育部）的全部收支
- **单位预算**（Unit Budget）——具体单位（如北外），预算体系的细胞

三层关系：单位 → 部门 → 总预算，层层汇总。

## 四、四本账：《预算法》定下的复式预算体系

**《预算法》**（Budget Law）是中国预算的法律基石，1994年通过、2014年和2018年两次修正，确立了复式预算体系——政府有四本账：

| 预算类型 | 资金来源 | 用途 | 一句话理解 |
|----------|----------|------|-----------|
| **一般公共预算**（General Public Budget） | 以税收为主体 | 保民生、促发展、维国安、养机构 | **核心账本**——政府日常运转全靠它 |
| **政府性基金**（Government Funds） | 向特定对象征收，有期限 | 专项用于特定公共事业 | **专款专用账**——最大的项目是**土地出让金收入** |
| **国有资本经营预算**（State Capital Operations Budget） | 国有资本收益 | 国企改制、资本金注入、政策补贴，也可调入一般公共预算和补充社保 | **国企分红账**——国企赚的钱怎么花 |
| **社会保险基金预算**（Social Insurance Fund Budget） | 社保缴款 + 财政安排 | 专项用于社会保险 | **保命钱账**——按统筹层次和险种分编，必须收支平衡 |

《预算法》规定后三本账必须与一般公共预算衔接。关键认知：**土地出让金**归政府性基金，不是一般公共预算——地方"土地财政"依赖的是第二本账。

## 五、管账的四大制度

| 制度 | 核心 | 关键细节 |
|------|------|----------|
| **部门预算** | 按部门编，全面反映收支 | 取代按功能分类的老办法；批复后 **20 日内**公开 |
| **政府采购**（Government Procurement） | **公开、公正、竞争** | 公开招标投标是基石；《政府采购法》2003/2014 |
| **国库集中收付**（Centralized Treasury） | 国库单一账户，全流程监控 | 学习西方；先集中支付，再集中收付 |
| **收支两条线**（Separation of Revenue & Expenditure） | **收缴分离、收支脱钩** | 严禁"坐收坐支"；最终目标是取消预算外资金 |

## 六、账本的旅程：从编制到报告

### 6.1 编制与审批：二上二下

中央部门预算走**"二上二下"**：

| 阶段 | 方向 | 做什么 |
|------|------|--------|
| 一上 | 部门 → 财政部 | 部门报预算建议数 |
| 一下 | 财政部 → 部门 | 财政部审核后反馈 |
| 二上 | 部门 → 财政部 | 部门报正式预算数 |
| 二下 | 财政部 → 部门 | 人大批准后财政部批复 |

全国人大审查批准（第43条），财政部一个月内批复到各部门。

> 结构性矛盾：预算年度1月1日开始，人大3月才开会——1-3月支出缺合法性基础。《预算法》第54条打了补丁（允许参照上年同期先行支出），但没根本解决。

### 6.2 执行与调整

执行主体是各部门各单位——对自己的执行结果负责。但现实中下级常受上级干扰：年中出台政策要求支出、转移支付要求配套——年初编制和年中执行脱了节。

**预算调整**分两类：**全面调整**（宏观大变动，很少）和**局部调整**（动用预备费、追加追减、经费流用、预算划转，经常发生）。

《预算法》第67条规定了必须调整的四种情形：增减总支出、调入预算稳定调节基金、调减重点支出、增加举债。第69条要求预算调整方案必须在人大常委会开会 **30 日前**送交初审。

### 6.3 决算与财务报告

**决算**（Final Accounts）是预算执行的最终结果——预算是事前计划，决算是事后成绩单。**政府财务报告**（Government Financial Report）更全面——除收支数字外，还记录运营绩效和现金流量，供公众评价政府受托责任。

## 七、花得值不值：绩效评价与四大难题

长期以来只问"合规"——钱有没有按规矩花。近年来转向"绩效"——钱花出了什么效果。

### 7.1 三级评价框架

| 层级 | 名称 | 关注什么 |
|------|------|----------|
| 第一层 | **总额财政纪律**（Aggregate Fiscal Discipline） | 宏观——总支出有没有失控 |
| 第二层 | **优先性配置**（Priority-Based Allocation） | 中观——钱有没有用到战略重点上 |
| 第三层 | **运营绩效**（Operational Performance） | 微观——每个项目值不值 |

三级递进，但实施有四大障碍。

### 7.2 四大难题

**财政公共地问题**（Fiscal Commons Problem）——最核心难题。公共财政的钱没有主人，谁花都不心疼。后果：支出膨胀、成本社会化收益私人化、财政幻觉。根在公共产权的**非排他性**——人人都想多切一块"公家"的蛋糕。

**租值耗散**（Rent Dissipation）——同样的资产，私人产权下有租值，公共产权下价值流失。平壤的高楼和纽约的高楼，物理上一样，经济价值天差地别。

**财政幻觉**（Fiscal Illusion）——你不知道公共服务的真实成本，税收是隐性的，支出看似"免费"，政府借此扩张而不被惩罚。

**财政代理问题**（Fiscal Agency Problem）——管理者（代理人）和出资者（公众）分离。信息优势、自由裁量、契约不完备、监督困难、委托人偏好不一致——五重叠加，腐败风险巨大。

### 7.3 评价原则与我国实践

两套指导框架：**3E**（总量绩效、配置绩效、运营绩效）+ **良治六原则**（法定授权、受托责任、透明度、预见性、全面性、公民参与）。

我国两类评价：上级对下级（进展慢）和财政支出绩效评价（进展快——"谁拨款谁问效，谁用款谁负责"）。2014年财政部出台《地方财政管理绩效综合评价方案》，涵盖透明预算、规范编制、优化收支、盘活存量、债务管理、省以下体制、约法三章、严肃财经纪律八个方面。

## 八、谁来盯账：预算监督

监督主体是人大（第83条）：全国人大监督中央和地方，地方人大监督本级和下级。政府须在**6-9月**报告执行情况（第86条）。现实中的三重困境：

| 维度 | 问题 |
|------|------|
| **监督能力** | 主体力量薄弱、审查时间太短 |
| **监督权力** | 缺少预算修正案提案权、审查结果缺法律效力、监督范围窄、预算不够细化 |
| **监督方式** | 法规系统性失调、初审走形式、审批形式主义、缺实时监控、缺绩效监督 |

改进：强化主体能力、延长审查时间、细化审批制度、增强修改权力、用好审计和绩效工具。

## 九、概念对照表

| 概念对 | 区别 |
|------|------|
| **单式预算 vs 复式预算** | 前者一本账（结构简单），后者多本账按经济性质分开（分类清晰）——中国用复式 |
| **零基预算 vs 增量预算** | 前者从零论证每个项目（理想高效但成本高），后者在去年基数上调整（现实省力但固化历史不合理） |
| **总预算 vs 部门预算 vs 单位预算** | 层层汇总：单位是细胞，部门是器官，总预算是全身 |
| **一般公共预算 vs 政府性基金** | 前者靠税收养运转（核心账），后者特定征收专项专用（专款账，卖地是大头） |
| **国有资本经营预算 vs 社会保险基金预算** | 前者管国企红利怎么用，后者管社保缴款怎么保 |
| **决算 vs 政府财务报告** | 决算是数字汇总，财务报告更全面（含绩效、现金流） |
| **财政公共地问题 vs 财政代理问题** | 前者：公家钱没人珍惜→过度消耗；后者：管钱的和出钱的两拨人→腐败风险 |
| **租值耗散 vs 财政幻觉** | 前者：公共产权让资产贬值；后者：不了解公共成本瞎表态 |
| **3E vs 良治** | 3E 关注"效率"（值不值），良治关注"制度"（对不对） |
| **预算调整 vs 预算执行** | 预算=计划，执行=按计划走，调整=走到一半要改——第67条定什么时候必须改 |

## 十、最重要的故事：从"二上二下"到"钱花得值"——一条完整的预算生命线

```
国家预算 = 政府一年的收支承诺（有法律效力）
    │
    ├─ 按复式分四本账：一般公共预算 / 政府性基金 / 国有资本经营 / 社保基金
    │
    ├─ 用四大制度管：部门预算 + 政府采购 + 国库集中收付 + 收支两条线
    │
    ├─ 走"二上二下"程序 → 人大审批 → 执行
    │          ↓
    │     预算调整（第67/69条）→ 如果情况变了，按法律程序改
    │          ↓
    │     决算（年底算总账）+ 政府财务报告（向公众交底）
    │
    ├─ 三级绩效评价：总额纪律 → 优先性配置 → 运营绩效
    │      ↑ 面临四大难题 ↑
    │   财政公共地 → 租值耗散 → 财政幻觉 → 财政代理问题
    │
    └─ 人大监督兜底：能力、权力、方式都还要补课
        ↓
结论：预算不只是会计问题——它是权力分配、利益博弈、制度约束的综合载体。一本预算写得好不好、管得严不严、监督实不实，决定了政府用的是"人民的钱"还是"他们自己的钱"。
```
