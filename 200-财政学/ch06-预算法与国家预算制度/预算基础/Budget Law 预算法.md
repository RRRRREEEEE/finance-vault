# 预算法 (Budget Law)

## 📌 定义
《中华人民共和国预算法》是规范政府财政收支计划编制、审批、执行、调整、决算和监督的基本法律，是财政领域的"宪法"，确立了**法治财政**的制度框架。

## 🧠 机制

### 立法历程（三次关键节点）

| 时间 | 事项 | 要点 |
|------|------|------|
| 1994.3.22 | 第八届全国人大第二次会议通过 | 首部预算法，确立预算制度基本框架 |
| 2014.8.31 | 第一次修正（**大修**） | 全口径预算管理、允许地方发债、强化公开透明、跨年度预算平衡 |
| 2018.12.29 | 第二次修正 | 第十三届全国人大常委会第七次会议 |
| 2020.8.20 | 《预算法实施条例》最新修订 | 细化操作性规定 |

### 2014年修正四大突破
1. **全口径预算管理**：将预算外资金全部纳入预算，堵住"小金库"
2. **允许地方政府发债**：过去完全禁止 → 地方通过融资平台隐性举债，风险更大 → 开正门堵偏门
3. **增强预算公开透明**：预算信息向社会公开
4. **建立跨年度预算平衡机制**：设立预算稳定调节基金

### 一级政府一级预算
《预算法》总则规定：国家实行**一级政府一级预算**。五级政府（中央→省→市→县→乡），对应五级预算。

```
全国预算 = 中央预算 + 地方预算
地方预算 = 各省、自治区、直辖市总预算
地方各级总预算 = 本级预算 + 汇总的下一级总预算
```

### 法定复式预算——四本账
国家预算实行**复式预算**，包括四本预算：
1. **一般公共预算**：以税收为主体，保民生、运转
2. **政府性基金预算**：向特定对象征收、专项用于特定事业
3. **国有资本经营预算**：国有资本收益安排的收支
4. **社会保险基金预算**：社保缴款专项用于社保

> 四种预算应当保持**完整、独立**，后三种预算应当与一般公共预算相衔接——意味着有缺口时可以从一般公共预算调入资金。

## 📄 来源原文
> 《中华人民共和国预算法》于1994年3月22日第八届全国人民代表大会第二次会议通过，根据2014年8月31日第一次修正，2018年12月29日第十三届全国人民代表大会常务委员会第七次会议第二次修正；《中国人民共和国预算法实施条例》于2020年8月20日最新修订；《预算法》总则规定，国家实行一级政府一级预算，全国预算由中央预算和地方预算组成，地方预算由各省、自治区、直辖市总预算组成，地方各级总预算由本级预算和汇总的下一级总预算组成。国家预算实行复式预算，包括一般公共预算、政府性基金预算、国有资本经营预算、社会保险基金预算。四种预算应当保持完整、独立，后三种预算应当与一般公共预算相衔接。
