# 财政绩效评价三大难题

> 三级框架（总额纪律→优先配置→运营绩效）理论上是好的，但现实中有三道坎拦在前面。

## 📌 总览

| 难题 | 层面 | 核心 | 一句话 |
|------|------|------|--------|
| **财政公共地问题** | 产权制度 | 公共产权非排他 → 人人可捞 | 公家的钱，不花白不花 |
| **预算程序问题** | 制度设计 | 程序有漏洞 → 放大公共地 | 流程管不住，想管也白管 |
| **财政代理问题** | 人性行为 | 管理者≠出资人 → 腐败风险 | 花别人的钱，替别人办事 |

## 🧠 逻辑链

```
公共产权非排他（根源）
  → 公共地问题：人人想从公共池子多占
    → 程序漏洞放大：权力分配失当、流程不透明，管不住
      → 代理人有空可钻：信息不对称 + 自由裁量 → 腐败低效
```

三道题层层递进：**公共产权是根子**，**程序缺陷是放大器**，**代理问题是人的因素**。

---

## 一、财政公共地问题

公共资金池无排他边界，每个部门/官员/利益集团都想多占——反正成本大家摊。

**四大表现**：支出赤字债务膨胀 / 成本社会化收益私人化 / 财政幻觉 / 权益性支出刚性（棘轮效应）

**附带概念**：租值耗散（公共产权让财产贬值，同是高楼，纽约有租值、平壤只有物理意义）

**解法**：建立法规 → 建立等级制 → 建立私人产权

> 详见：[[Fiscal Commons Problem 财政公共地问题]]

## 二、预算程序与现金管理问题

程序缺陷**放大**公共地问题。三个维度：

| 维度 | 问题 |
|------|------|
| **预算程序** | 准备→审查→批准→执行流程如何设计才能限制自利？太松管不住，太紧没弹性 |
| **权力分配** | 行政 vs 立法、上级 vs 下级、政府 vs 公众——谁说了算？ |
| **现金管理** | 分散管理 → 决策困境；集中化 → 加强透明度，见 [[Centralized Treasury Receipt and Payment 国库集中收付制度]] |

---

## 三、财政代理问题

管理者（官员）≠ 出资者（纳税人），分离必然产生信息不对称。

| 风险 | 财政领域的体现 |
|------|--------------|
| **代理人优势** | 部门知道真实成本，人大不知道 |
| **不完全契约** | 预算不可能穷尽所有支出细节 |
| **自由裁量权** | 预算执行中的"灵活处理"空间 |
| **监管困境** | 审计走过场、人大审查形式化 |
| **委托人偏好不一致** | 有人要减税、有人要加福利，难以统一监督 |

> 详见：[[Fiscal Agency Problem 财政代理问题]]

## 📊 对比

| | 公共地 | 程序 | 代理 |
|------|------|------|------|
| 层面 | 产权制度 | 制度设计 | 人性行为 |
| 根源 | 公共产权非排他 | 权力分配+流程缺陷 | 所有权与经营权分离 |
| 解法 | 法治+产权界定 | 程序改革+集中化 | 监督+激励相容 |
| 理论 | 哈丁公地悲剧 | 制度经济学 | 委托-代理理论 |
