# 第十二章 知识串讲：中央和地方的钱与权怎么分

## 一、一句话回答

**政府之间通过划分事权和财权、建立转移支付制度来协调多级治理，中国的核心矛盾是财权向上集中、事权向下分散，导致了当前地方财政的系统性困难。**

---

## 二、政府不是一块铁板——为什么要讨论多级政府间的财政

前面十一章我们一直把"政府"当成一个整体来讨论：政府征税、政府支出、政府提供公共产品。但到了第十二章，我们把这个整体拆开——中央、省、市、县，每一级政府都是相对独立的利益主体。

> 你想想：北京财政部和你们县财政局，想的事情能一样吗？中央操心的是全国经济稳定、地区之间贫富差距；县长操心的是本县教师工资能不能按时发、垃圾有没有人收。

这就产生了一个根本问题：**各级政府之间的责、权、利怎么划分，才能让整个国家治理体系的社会效率最高？** 这就是政府间财政关系的核心。

---

## 三、三大职能怎么分——经济稳定归中央，资源配置看受益

政府有三大经济职能：经济稳定、收入分配、资源配置。在各级政府之间分配时，遵循一个朴素的原则——**谁能干得更好就归谁**。

**经济稳定职能**必须是中央的。为什么？地方政府如果自己搞经济稳定政策——比如刺激本地经济——好处可能溢出到隔壁省，成本却自己担。这叫"政策外溢"，跟外部性是一个逻辑。

**收入分配职能**也归中央。市场经济下资本和劳动力到处流动——上海收了富人的税补贴穷人，结果富人搬到苏州去了，上海的再分配政策就失灵了。只有中央政府才不怕这个。

**资源配置职能**就不是一刀切了。原则是：**公共产品的受益范围在哪一级，就归哪一级提供。** 这就是辖区受益原则——能下面解决的别往上报。一个公共产品如果没有跨辖区的外溢性，交给上级纯属多此一举。

> 用一个图来说：如果中央统一给北京人和贵州人提供同样的公共服务，两边都不满意——北京人要的多、贵州人消化不了。那两个三角形的面积就是社会福利损失。

---

## 四、集权好还是分权好——蒂伯特的答案

集权有集权的好处：能协调全国、能搞收入再分配、能动员大规模资源、能用规模经济。但它最大的毛病是**信息不对称**——中央不知道你村里最需要什么。

分权的优势恰好补了这个短：地方政府更了解本地偏好，能有效提供地方公共产品。更重要的是，分权鼓励**政府间竞争**和**地方创新**——浙江的"最多跑一次"就是从地方试点推向全国的。

但你可能会问：居民怎么表达自己需要什么公共产品？有两种方式：
- **"用手投票"**：通过参政议政，用选票表达偏好
- **"用脚投票"**：不喜欢这个地方？搬家。

**蒂伯特定理**就是给"用脚投票"做理论背书的。蒂伯特说：如果居民可以在辖区之间自由流动，每个辖区提供不同的"税收-公共服务"组合，那么居民通过选择住在哪来"购买"他们最喜欢的组合——就像在超市挑商品一样。这个流动信号倒逼地方政府竞争性地提高效率。一个分权制的财政体制，能让公共产品的提供更有效率。

> 这就是**财政联邦制**的思想基础：在中央和地方之间划分财权与事权，不是为了政治分权，而是为了让公共产品提供更有效率。

---

## 五、钱怎么分——哪些税归中央，哪些归地方

事权分完了，钱从哪来？这就涉及**收入在政府间的划分**。课件给了一套非常清晰的原则：

**应该归中央的税有四个特征**：流动性、再分配性、分布不平衡、收入波动大。

你想：个人所得税和企业所得税的税基是人和公司——会跑的。如果归地方，上海把税率压低，企业和人才全跑上海去了，中西部更穷。所以个税和企业所得税必须是中央税。增值税也一样——产地型和消费地分离时，产地征税不公平。关税就更不用说了，涉及国家主权。

**应该归地方的税有一个共同特征**：税基**动不了**。土地和房子搬不走，所以土地税和房产税天然是地方税。销售税如果是消费地型（终点型）的，税负落在本地消费者身上，也适合地方。

> 全世界的普遍现实是：税收权力**向上集中**（中央收大头），支出责任**向下分散**（地方干苦活）。这就产生了纵向财政缺口——中央有钱，地方有活，必须靠转移支付来填。

---

## 六、转移支付——怎么把钱从中央搬到地方

转移支付就是上级给下级钱。按有没有附带条件，分四种：

| 类型 | 条件 | 地方能随便花吗 |
|------|:---:|:---:|
| 一般性补助 | 无 | 随便 |
| 条件补助 | 有 | 满足条件就行 |
| 专项补助 | 指定用途 | 必须花在指定项目 |
| 配套补助 | 指定用途 + 地方出配套 | 地方也要掏钱 |

不同补助形式的经济效应不一样。无条件补助温和地扩大地方总支出；有条件补助（尤其是配套补助）会定向撬动特定领域的支出。

有一个有意思的现象叫**粘蝇纸效应**：中央拨下来的100块钱，比本地居民收入增加的100块钱，更容易被政府花掉。为什么？因为拨款直接到了具体部门和项目手里，绕过了居民的预算约束——钱像粘在了收到它的手上一樣。这解释了为什么上级补助的支出扩张效应通常大于居民收入增长带来的支出增长。

**转移支付的标准化测算**用的是公式法：

> 某地区均衡性转移支付 = (标准财政支出 − 标准财政收入) × 转移支付系数 + 增幅控制调整 + 均等化努力奖励

标准支出考虑了各地提供公共服务的成本差异（在高原修路跟在平原修路能一样吗），标准收入考虑了各地征税难度和政府征税努力程度（有些地方不是收不到税，是不努力收）。

转移支付有两个方向：
- **纵向**：中央 → 地方（中国的绝对主力）
- **横向**：发达 → 落后（德国西部给东部，中国各省对口援建西藏新疆）

## 七、地方财政为什么这么难——中国当前的问题

课件用非常实在的数据和案例讲了当前中国地方财政的系统性困境。

**收入端三重打击**：土地财政断崖式下滑（房子卖不动了）、税收增长乏力（经济增速下行）、减税降费（政策主动让利）。

**支出端三座大山**：三保支出（保工资、保运转、保基本民生）占比畸高，债务付息负担越来越重，再加上财权事权不匹配这个老毛病。

结果是：体制内降薪拖欠工资、工程烂尾（贵阳12公里环城高速修到一半停了）、包工头破产潮。

四个典型案例：
- **贵州**：债务率超60%，某县隐性债务80亿但年税收仅3亿
- **云南**：2024年收入约2195亿，支出超6900亿，缺口靠中央填
- **广西**：工程烂尾集中爆发——柳州铁路预算126亿、恭城文旅16.5亿全停了
- **天津**：东部的样本——经济增长压力大，2025年社零增速仅0.3%，东部也在紧平衡

> 还有一个结构性因素：人口流动。超过25%的人口不在户籍地居住——税缴给了北京上海，但孩子上学、老人看病在老家。公共服务供给地和税收归属地错配，进一步加剧了"支出刚性"和"税基弹性"的矛盾。

---

## 八、怎么改——开源、节流、放权

课件给的改革方向很清晰：

**开源**：开征新税——房产税（替代土地财政）、遗产税和赠与税（调节财富传承）、碳税（绿色转型）、数字资产税（覆盖新经济）。

**节流**：回归保障型政府——政府少做直接投资；回归小政府——缩减政府社会功能、减少政府层级。

**放权**：向地方下放权力，调动地方政府和民营企业的积极性。

**转型**：加速经济结构转型，提升高附加值产业比重——从量的扩张转向质的提升。

最终目标是：**回归长期、稳定、中速、均衡、高质量的增长**。

```
政府间职能怎么分（三大职能） 
  → 集权还是分权（各有利弊，蒂伯特定理说分权更有效率）
    → 分权后钱怎么分（流动性税归中央，不动产税归地方） 
      → 缺口怎么补（转移支付：纵向+横向，公式法定量） 
        → 当前中国的麻烦（收入端土地财政崩了 + 支出端三保刚性 + 人口流动错配）
          → 出路在哪（开源新税种 + 节流缩政府 + 放权给地方）
```

## 十、概念对照表

| 概念对 | 区别 |
|--------|------|
| **集权 vs 分权** | 集权节约协调成本但信息不对称严重；分权能有效提供地方公共产品但可能恶性竞争 |
| **用手投票 vs 用脚投票** | 前者通过政治程序表达偏好（参政议政），后者通过迁徙行为表达偏好（搬家） |
| **纵向缺口 vs 横向缺口** | 纵向是中央与地方之间的财力与事权不匹配；横向是不同地区之间的财力差距 |
| **一般性补助 vs 条件补助** | 前者无附加条件随便花；后者附带条件（专项指定用途/配套要地方也掏钱） |
| **不征税收入 vs 免税收入** | 前者压根不算收入（财政拨款）；后者算收入但国家不征（国债利息）<br>回顾[[税收计算详解 增值税消费税企业所得税个人所得税]]里的区分 |
| **专项补助 vs 配套补助** | 都指定用途，但配套补助要求地方也掏配套资金，有杠杆效应 |
| **蒂伯特定理 vs 财政联邦制** | 蒂伯特定理是理论（用脚投票带来效率），财政联邦制是制度安排（划分财权事权） |

## 十一、最重要的故事 / 考试最易考点

**1. 蒂伯特定理 + 用脚投票的逻辑链条**（大概率考）
蒂伯特说：居民用脚投票 → 辖区管理者看到人口流动信号 → 竞争性地优化税收-公共服务组合 → 分权体制让公共产品更有效率 → 财政联邦制。

**2. 收入划分原则**（大概率考）
记住四个特征决定归中央：流动性、再分配性、分布不均、波动大。具体对应：个税、企税、增值税、关税、资源税归中央；房产税土地税和终点型销售税归地方。

**3. 转移支付公式**（可能考计算）
均衡性转移支付 = (标准支出 − 标准收入) × 转移支付系数 + 增幅调整 + 奖励资金。关键是理解"标准"二字——不是实际数，是按人口、GDP、资源等测算出来的"应该"的收支水平。

**4. 粘蝇纸效应**（名词解释高频）
补助所增加的支出 > 等额居民收入增加所形成的支出。因为补助直接给到了具体项目和部门，绕过了居民偏好约束。
