# 政府间财政补助 (Intergovernmental Grants)

## 📌 定义
政府间财政补助是上级和下级政府之间以及不同地区的政府之间，出于财政平衡、社会公平的要求而实行的财政资金转移。按是否附带条件分为一般性补助（无条件补助）和条件补助（专项补助、配套补助）。

## 🧠 机制

### 四种补助类型

| 类型 | 特征 | 用途限制 | 是否需要地方配套 |
|------|------|:---:|:---:|
| **一般性补助**（无条件补助） | 对补助资金的使用方向不做规定 | 无 | 否 |
| **条件补助** | 发放时附加一定条件 | 有 | 视具体类型 |
| **专项补助** | 规定了资金用途 | 指定用途 | 否 |
| **配套补助** | 地方政府需拿出配套资金 | 指定用途 | 是 |

### 不同补助形式的经济效应

不同补助形式通过影响地方政府的**预算线**位置和形状来改变支出行为：

- 专项补助和配套补助影响预算线的位置和形状
- 居民无差异曲线处于偏下方位置 → 经济效益与普通补助一致
- 居民无差异曲线处于偏上方位置 → 受到较大影响

### 粘蝇纸效应 (Flypaper Effect)

> 社区收到补助所增加的支出，**大于**社区等额收入增加所形成的支出。

即：上级拨下来的钱比本地自己增收的钱更容易被花掉——钱"粘"在了收到它的部门手中。可能原因是补助直接给到了具体的项目和部门，绕过了居民的偏好约束。

| | 无条件补助 | 条件补助 |
|--|:---:|:---:|
| 扩张效应 | 温和扩大支出 | 定向扩大特定支出 |
| 替代效应 | 可替代本级支出 | 不可替代 |
| 居民福利 | 增加居民选择自由 | 可能扭曲居民偏好 |
| 粘蝇纸效应 | 较弱 | **更强** |
