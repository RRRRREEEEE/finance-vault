# 收入在政府间划分的原则 (Principles of Revenue Assignment)

## 📌 定义
收入在政府间划分的原则是指在中央和地方之间分配税收权力时遵循的一般规则。核心思想：流动性税基、再分配性质、税基分布不平衡、收入易波动的税种应划分为中央税；产地型产品税应为中央税，终点型产品税及房产土地税应为地方税。

## 🧠 机制

### 应划为中央税的税种

| 原则 | 税种 | 理由 |
|------|------|------|
| 流动性税基 | 个人所得税、企业所得税、增值税 | 流动性强，地方征收导致税基流失和恶性竞争 |
| 再分配性质（以支付能力为基础） | 个人所得税 | 收入再分配应由中央统一实施 |
| 税基在全国分布不平衡 | 资源税 | 避免资源富集地区获得不公平优势 |
| 收入易发生波动 | 企业所得税 | 中央有更强的抗波动能力 |
| 产地型产品税 | 增值税（产地型） | 产地型和消费地分离时，产地征税不公平 |

### 应划为地方税的税种

| 税种 | 理由 |
|------|------|
| 货物税或销售税（终点型/消费地型） | 税负落在本地消费者，适合地方征收 |
| 土地、房产税收 | 税基不可移动，天然属于地方 |

### 关税
关税应为中央税——涉及国家主权和统一对外贸易政策。

### 补充机制

> 中央与地方的**税收分享制**，可以有效缓解央地矛盾。

实践中，完全按原则划分难以实现，各国普遍采用**税收分享制**作为补充：同一税种由中央和地方按比例分享（如中国增值税中央与地方 50:50 分享）。

| 应划为中央税 | 应划为地方税 |
|------------|------------|
| 个人所得税 | 货物税/销售税（终点型） |
| 企业所得税 | 土地税 |
| 增值税 | 房产税 |
| 关税 | |
| 资源税 | |
