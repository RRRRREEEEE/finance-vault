# 蒂伯特定理 (Tiebout Theorem)

## 📌 定义
蒂伯特（Tiebout）认为，个人在管辖区之间移动的能力产生了一个类似市场的解决地方公共物品的办法。个人可以"用脚投票"表达自己的公共产品消费偏好，为社区管理者提供了一种类似于私人产品市场价格的信号。社区基于这些信号，为居民提供最优的公共服务与税收组合。

> 💬 通俗理解：如果北京市公共服务好但税高、河北省税低但公共服务差，每个人根据自己的偏好选择住在哪——这个选择过程像一只"看不见的手"，逼着各地方政府竞争性地提高效率。政府之间的竞争，就像企业之间的市场竞争。

## 🧠 机制

### 核心逻辑

1. 居民在不同辖区之间**自由流动**
2. 每个辖区提供不同的**税收-公共服务组合**
3. 居民选择最符合自己偏好的辖区 → 搬过去
4. 辖区管理者观察到人口流动信号 → 调整公共服务供给
5. 最终达到类似市场均衡的效率状态

### 理论意义

- 蒂伯特定理说明了：**一个分权制的财政体制，将使公共产品的提供更加有效率**
- 为[[Fiscal Federalism 财政联邦制]]提供了理论基础
- 解释了"用脚投票"的经济学逻辑

### 理论局限

- 假设居民可以无成本流动（现实中搬家有成本）
- 假设居民充分了解各辖区的税收和公共服务信息
- 假设有足够多的辖区可供选择
