# 地方财政困难 (Local Fiscal Difficulties)

## 📌 定义
当前中国地方政府面临的财政困境：收入端土地财政断崖式下滑与税收增长乏力叠加，支出端三保支出和债务付息刚性攀升，加上财权与事权不匹配的结构性矛盾，导致部分地方财政自给率偏低、工资拖欠、工程烂尾等问题。

## 🧠 机制

### 困难表现

| 表现 | 具体内容 |
|------|----------|
| **工资奖金拖欠与减薪** | 体制内工资奖金不能按时足额发放 |
| **工程停工与项目烂尾** | 基础设施和投资项目因资金断裂停摆 |
| **社保资金与三保危机** | 保工资、保运转、保基本民生压力巨大 |
| **财政自给率普遍偏低** | 多数县市本级收入远不足以覆盖支出 |
| **人口净流出地区税基流失** | 人走了但公共服务支出不减 |
| **包工头破产潮** | 政府工程款拖欠传导至下游 |

### 困难原因

**收入端：多重冲击叠加**
- 土地财政收入断崖式下滑
- 税收增长乏力，区域差异显著
- 减税降费政策的阶段性影响

**支出端：刚性持续攀升**
- 三保支出占比畸高
- 地方政府债务付息负担沉重
- 财权事权不匹配的结构性问题
- 低效无效支出增加负担

### 典型案例

| 地区 | 困难特征 |
|------|----------|
| **贵州** | 债务率超60%，某县隐性债务80亿、年税收仅3亿；贵阳环城高速12公里中途停工 |
| **云南** | 2024年一般公共预算收入约2195亿，支出超6900亿，高度依赖转移支付 |
| **广西** | 工程烂尾集中爆发区：柳州铁路（预算126亿）、恭城文旅项目（16.5亿）烂尾 |
| **天津** | 东部发达地区样本：经济增长压力加大，2025年社零增速仅0.3%，债务率持续攀升 |
