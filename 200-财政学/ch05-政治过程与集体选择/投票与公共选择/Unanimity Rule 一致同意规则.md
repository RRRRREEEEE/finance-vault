# 一致同意规则 (Unanimity Rule)

## 📌 定义
一致同意规则是指一项决策或议案需经全体投票人一致赞同才能通过的投票规则，即一票否决制度。它是政治市场上的帕累托状态——任何通过的决策都至少不使任何人变差。

> 💬 通俗理解：所有人点头才算通过，任何一个人都可以一票否决。公平是公平，但想让大家同时点头？难。

## 🧠 机制

### 优势：帕累托最优
一致同意规则是政治市场上的帕累托状态——任何通过的议案都意味着没有人受损（否则会否决），因此一定是社会福利的改善。

### 林达尔的理论支撑
林达尔认为，只要能够找到一种合理的税制，使每个人都能够明确其在税收中承担的份额，一致意见的达成就是有可能的。

### 局限

| 局限 | 说明 |
|------|------|
| 决策成本极高 | 每个参与者都有否决权，达成一致需要漫长的谈判和妥协 |
| 参与人数限制 | 参与人数越少越有效；人数越多，达成一致的难度指数级上升 |
| 一票否决的"威胁" | 参与者可利用否决权进行非对称表达——以否决为要挟谋取更多利益 |

## 🏛️ 应用

一致同意规则在现实中应用于少数重要场合：联合国安理会常任理事国否决权、某些国际条约的批准程序、合伙企业重大决策。但日常公共决策中因成本太高而极少使用。

实际中我们降低一致同意的条件，过渡到[[多数票规则]]。
