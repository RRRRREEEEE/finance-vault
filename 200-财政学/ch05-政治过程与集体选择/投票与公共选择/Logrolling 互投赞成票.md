# 互投赞成票 (Logrolling)

## 📌 定义
互投赞成票（投票交易）是指意见不一致的投票方通过相互妥协，利用不同领域的项目做交易——我支持你关心的议题，你支持我关心的议题——从而促进社会福利提升的投票行为。它是解除[[投票悖论]]的途径之一。

> 💬 通俗理解：你在农业补贴上支持我，我就在国防预算上支持你。虽然咱俩观点不同，但各取所需，互相成全。

## 🔗 前置知识
[[投票悖论]]、[[多数票规则]]

## 🧠 机制

### 解除投票悖论的途径之一

[[投票悖论]]产生于偏好的不可传递性。互投赞成票通过**跨议题交换**打破僵局：
- 各方在最看重的问题上得到满意结果
- 在不太关心的问题上做出让步
- 总体社会福利可能因此提升

### 解除投票悖论的其他途径（同页）

| 途径 | 说明 |
|------|------|
| 打分投票（基数论） | 改为打分而非排序，投票悖论自动解除 |
| 相对多数投票 | 放弃绝对多数要求，只取相对多数 |
| 代议制缩小选择集 | 通过代议制将复杂方案简化为两个方案的对决 |
| 互投赞成票 | 跨议题妥协交换 |

## 📖 相关术语

- [[多数票规则]]：互投赞成票运行于多数票规则框架内
- [[利益集团]]：利益集团常通过互投赞成票影响立法
