# 官僚制度 (Bureaucracy)

## 📌 定义
官僚制度是指政府机构中由非选举产生的职业官员（官僚）组成的行政体系。官僚作为理性人，同样追求自己最大化效用，利用手中权力实现利益最大化——但政府机构不核算利润，没有效率竞争，缺少节省成本的动力。

> 💬 通俗理解：公务员不是圣人——他们也追求自己的利益：更大的预算、更多的下属、更高的薪水。但政府不像企业有盈亏压力，所以官僚没有"省钱"的动力。

## 🧠 机制

### 官僚的行为特征

- **预算最大化**：官僚追求权力和预算的最大化 → 意味着更多的资源控制权
- **浪费最大化**：追求高薪、福利，有时倾向于加大无所谓的开支
- **信息操纵**：争取预算时夸大社会面临的问题与挑战；总结成绩时淡化问题和成本
- **不核算利润**：政府机构没有利润约束，无效率竞争，缺乏节省成本的动力

### 官僚与政治家的区别

| | 政治家 | 官僚 |
|------|------|------|
| 产生方式 | 选举 | 任命/考试 |
| 目标 | 选票最大化 | 预算最大化 |
| 约束 | 选举周期 | 职权范围 |
| 效率激励 | 选民问责 | 弱（无利润核算） |
