# 第五讲 知识串讲：投票能把私心变成公益吗——政治过程与集体选择

## 一、一句话回答

**政府不是一个人，是一群人按规则投票的结果。投票规则本身决定了结果——选票怎么数、议程怎么排、利益集团怎么施压、官僚怎么执行——不是理性加总出公共选择，而是制度选择公共结果。政府对市场失灵的矫正，本身就可能失灵。**

---

## 二、从规范到实证：政府到底长什么样？

第三讲用福利经济学论证了政府应该做什么——矫正市场失灵，资源配置、收入分配、经济稳定。那是**规范分析**（ought to be）。第五讲切换到**实证分析**（what is）——真实的政府是怎么做决策的？

| 第三讲（规范） | 第五讲（实证） |
|------|------|
| 政府应该矫正市场失灵 | 政府连自己的决策都可能是失灵的 |
| 资源配置 + 收入分配 + 经济稳定 | 投票悖论 + 寻租 + 官僚膨胀 |
| 经济学家笔下的无私智者 | 现实中追求选票和预算的理性人 |

经济学家在描述政府职能时，容易制造一种幻觉：政府无私且充满智慧。但那是幻觉。

---

## 三、政府为什么会失灵？

PPT 列出了五大困境，这比"政府失灵"四个字要具体得多：

| 困境          | 含义                                   |
| ----------- | ------------------------------------ |
| **信息不对称**   | 市场有信息不对称，政府的有时更严重——它离交易现场更远          |
| **认知成本**    | 经济目标不是政府追求的唯一目标（还有政治目标、维稳目标…）        |
| **决策时滞**    | 识别→讨论→决策→实施，每一个环节都有时间差，等政策出来问题可能已经变了 |
| **官僚主义与寻租** | 官员的个人利益 ≠ 政府的公共利益，权力幻觉 + 寻租动机        |
| **挤出效应**    | 政府的经济行为可能挤占私人部门的经济资源（政府借钱多了，企业借钱就贵了） |

**记法**：信息差 → 目标杂 → 速度慢 → 动机歪 → 挤出效应。五个环节，每一个都可能让政府矫正市场失灵的努力变成新的失灵。

---

## 四、代议制政府：委托-代理的先天缺陷

现代政府几乎都是代议制——选民选代理人来决策，而不是自己直接管。

**为什么需要代议制？** 降低决策成本、提升决策效率。全国十四亿人每天投票，什么也干不了。

**代价是什么？** 代理人会偏离委托人的目标函数——政客的利益 ≠ 选民的利益。这是代议制与生俱来的缺陷，不是某个政客的品德问题。

**三层经济属性**：
1. 政府产生过程：理性人显示偏好 → 投票决策（每个人按自身利益投票）
2. 执政代理人：理性人、个人利益最大化（不是公众利益最大化）
3. 官僚在职权范围内：预算最大化（不是效率最大化）

记住这个三层框架——后面讲投票、政党、官僚，都是在这个委托-代理链条上展开的。

---

## 五、政治市场：它不是商品市场

**政治市场**是一个隐喻：选票是货币，政策是商品，政客是卖家，选民是买家。

但政治市场和商品市场有一个本质区别：**商品市场创造价值，政治市场只是再分配财富。** 它本身不生产任何东西——只是决定蛋糕怎么切。但它的规则会极大地影响市场预期——因为切蛋糕的刀法，决定了明天谁还愿意做蛋糕。

在这个市场里，政治家的目标函数分两类：
- **选举产生的** → 选票最大化
- **上级任命的** → 上级信任最大化

人们通过斗争或妥协，追求个人、组织、利益集团的利益最大化。

## 六、投票制度：从理想到现实

### 6.1 集体选择的两大困难

PPT 一针见血：通过投票把个人偏好变成集体决策，有两个根本难题：

**问题一：偏好显示问题（你愿意说真话吗？）**
- 免费搭车动机：希望别人多承担成本，自己少承担 → 不愿表露真实支付意愿
- 投票的理性无知：在一场选举中，你一票投或不投，对结果几乎没影响 → 很多人放弃投票
- 社会是否提供了让每个人都能表达偏好的机制？

**问题二：偏好加总问题（说了真话怎么合成？）**
- 每个人的偏好可能相互矛盾
- 有人要更多公共产品，有人要更低税收——怎么把这些加总成一个集体决策？

这两个问题贯穿整个投票制度设计。

### 6.2 林达尔均衡——理想中的公平

两个人在林达尔均衡下各自承担一部分公共产品成本（"林达尔价格" = 各自的税收份额 = 税收是公共产品的影子价格），总量恰好等于公共产品最优数量。

**关键洞察**：选民需要做选择的，不是公共产品数量本身，而是"公共产品 + 税收"的组合。你投的不是"我要多少国防"，而是"我要这么多国防，并愿意承担这么多税"。

**但林达尔均衡只存在于两人世界。** 现实中每个人都会撒谎——低报自己的支付意愿，让别人多出。


### 6.3 一致同意 → 多数票

| 规则 | 优点 | 缺点 |
| -------- | -------------- | ---------------- |
| **一致同意** | 政治市场上的帕累托最优：任何人变差就否决 | 决策成本太高，一个人能卡死所有人；少数人可用"否决威胁"不对称表达意愿 |
| **多数票** | 降低门槛，能干事 | 投票悖论、忽略偏好强度（只能排序，不能表达"我有多在乎"）|

林达尔认为只要找到合理税制，一致同意是可能的。但现实中人数越多越无效——一致同意是数学上漂亮、现实中不可用的理想。

### 6.4 税价与收入：谁更喜欢大政府？

这是 PPT 上容易被忽略但非常重要的一个机制：

- **一次总付税（平均分摊）** → 高收入者喜欢更多公共产品（他们交的税占比低，享受同样的公共产品）
- **累进税** → 低收入者喜欢更多公共产品（富人承担更大比例，穷人获益更多）

**核心概念——税价（Tax Price）**：增加一单位公共产品需要承担的额外支出。穷人的需求曲线总是在富人以下——因为穷人每多交一块钱税的痛苦（边际效用损失）比富人大。

这解释了为什么不同收入阶层对"大政府 vs 小政府"有根本分歧。


### 6.5 投票悖论：数学上无解的问题

PPT 用了两个版本的例子来讲这个故事：

**版本一（有效投票）**：亚当、夏娃、上帝三个人，三个方案 ABC。无论怎么排序，B 总是胜出 → 投票是有效的。


**版本二（投票悖论）**：改一下三个人的偏好次序——
- A vs B：A 胜
- B vs C：B 胜
- C vs A：C 反胜 A

**A 赢 B，B 赢 C，C 又赢 A。结果取决于议程顺序。** 谁排议程，谁就部分掌握了结果——这就是**议程操纵**的权力。


### 6.6 单峰偏好救场

如果每个人的偏好是**单峰**的（只有一个最爽的点，越远越不爽），投票悖论消失，结果稳定。但如果有人是**多峰偏好**（有两个"爽点"），悖论就会出现。

单峰偏好是多数票规则能稳定运行的数学条件。


### 6.7 阿罗不可能定理

阿罗（诺贝尔经济学奖得主）用数学严格证明：**不存在任何一种投票机制，能同时满足所有民主要求（传递性、非独裁、不相关方案独立性等）而又不出现循环投票。**

翻译成人话：没有完美的投票制度。你必须在某些维度上让步。这是一个数学上的**悲观结论**——民主投票的天花板，不是制度设计问题，是数学约束。

### 6.8 投票悖论的四种解除方式

PPT 给了四条出路，这一点当前串讲完全漏了：

| 方法 | 原理 |
|------|------|
| **基数打分投票** | 不排序，直接打分——投票悖论一下子消失（基数代替序数）|
| **相对多数制** | 放弃绝对多数，只取相对多数 → 结果稳定 |
| **代议制缩减选项** | 把复杂方案口号化、意识形态化，N 个方案 → 两个方案对决 |
| **互投赞成票（Logrolling）** | 意见不一致的双方相互妥协，用不同领域的项目做交易，促进社会福利提升 |

互投赞成票不是贬义词——它是政治市场上克服偏好分散的可行工具。你在 A 议题上支持我，我在 B 议题上支持你，两边的福利都可能提升。

### 6.9 中间投票人定理：为什么政策总往中间跑

如果偏好单峰 → 中间投票人的偏好获胜。中间投票人的定义：相对于他而言，偏好更大支出的人数，恰好等于偏好更小支出的人数。

PPT 给出了三个层次的解释：

1. **极端主张难以当选**：公开选举很少选出极端立场的人，政客立场偏中间
2. **政党趋同**：在一个秩序平稳的国家，政客和政党的主张会趋同——他们会刻意制造"并不实际存在的区别"来竞争。选民实际上没有太多不同选择。
3. **承诺与实施的鸿沟**：选前承诺 vs 选后实施之间存在巨大差距，政客需要在声誉和夸大承诺之间权衡

## 七、利益集团与寻租：选票之外的权力

### 7.1 利益集团如何操纵投票

**利益集团**是把在某议题上有共同利益的人联合起来、试图影响公共政策的组织实体。

利益集团的力量来源有三：
- **更高的投票比例**（组织化 → 投票率高）
- **额外资金支持**（能做广告、能捐款）
- **地区或人口优势**（集中在关键选区）

利益集团通过大量非均衡手段影响投票——游说（Lobbying）、捐款、情感表达——这些统称为**院外活动**。
利益集团有助于社会多元化的实现，使得多种政治经济观点都可以自由地表达

利益集团由于其内在的原因，倾向于追求社会收入再分配，而不是社会收入的创造。因此，当利益集团的利益产生严重分化时，社会将因此而变得动荡
奥尔森认为，利益集团增加自己在分配中的份额而不是增加整个国民收入是因为第一种做法没有正外部性。



### 7.2 奥尔森的洞见

奥尔森指出：利益集团倾向于追求**再分配而非价值创造**。不是把蛋糕做大，而是多切一块给自己。

**为什么？** 因为第一种做法（创造）有正外部性——别人也受益了。第二种做法（再分配）没有正外部性——好处全归自己。理性选择的必然结果。

当利益集团的利益严重分化时，社会会变得动荡——因为大家都在抢蛋糕，没人在做蛋糕。

### 7.3 寻租：用政府权力变现

**寻租**（Rent-Seeking）：利用政府权力获取高于正常收益的超额收益。

PPT 以花生市场为例：政府限制产量 → 价格抬升 → 生产者获得超额利润。abcd 四块是生产者拿走的"租"，cde 三块是社会福利的无谓损失——租没拿到手，蛋糕先打翻了。
![[财政学Ch05 寻租花生市场.png]]

寻租不做蛋糕，只抢蛋糕，抢的过程中还打翻了蛋糕。


### 7.4 寻租的具体形式与选举贿赂

PPT 列出的寻租类型，当前串讲一字未提：

- **特许经营权**（政府给牌照 → 持牌者获租）
- **进出口配额、非关税壁垒**（限制竞争 → 国内生产者获益）
- **政府采购指定**（定向采购 → 供应商获租）
- **其他地下交易**（权力直接变现）

选举政治中的贿赂链条：
```
政党/政治家 ← 选举补助、资金补助
     ↑
企业家/特殊利益集团 ← 政治献金
     ↓
取得特许经营权、配额、采购合同……
```

## 八、政党制度：两党还是多党？

### 8.1 政党的目标函数

**政党是为了赢得选举而制定政策，不是为了政策理想而参加选举。**（PPT 原话）

政党的实际得票取决于三个变量：
- 选民的效用函数（你推行的政策他们满不满意）
- 在野党的战略（对手怎么做影响你的得票）
- 在野党的行动又取决于执政党的行动和选民效用

这是一个三方博弈——政党、选民、对手。

### 8.2 两党制的两种形态

PPT 区分了两种截然不同的两党制：

| 形态 | 机制 | 结果 | 案例 |
|------|------|------|------|
| **趋同型** | 两党都往中间靠拢，提出相似主张 | 制度稳定 | 传统欧美（象驴之战）|
| **极化型** | 各有铁票阵营，谁都不愿调整立场 | 政治撕裂、稳定性下降 | 以色列/巴勒斯坦选举政治 |

趋同型符合中间投票人定理。极化型说明中间投票人定理有前提条件——当社会本身是撕裂的，政党不一定往中间走。


### 8.3 多党制

- **多数票选举（赢者通吃）** → 容易产生两党制
- **比例代表制** → 容易产生多党制
- **总统制国家** → 偏向两党制；**议会制国家** → 偏向多党制
- 多党制 → 容易出现联合执政 → 政府组成更不稳定


### 8.4 政党制度的复杂性

PPT 特别强调：政治市场理论不是普适的。当不同议题的选民意见差别很大时，连"中间选民"都不好定义。社会文化对选举制度的影响很大。

三个案例：
- **中国台湾**：从多党制到两党制的演化
- **中国香港**：特别行政区政党制度的演化
- **日本、新加坡**：一党独大的政治模式

这些都不完全符合"赢者通吃 → 两党制"的简单公式。

## 九、官僚制度：政策落到谁手里？

### 9.1 官僚的行为特征

官僚是理性人，同样追求效用最大化。但官僚机构有一个根本特征：**不核算利润，没有效率竞争，缺少节省成本的动力。**

这导致三个"最大化"：
- **权力最大化**（预算越大 → 权力越大）
- **预算最大化**（花更多钱 → 控制更多资源）
- **浪费最大化**（追求高薪、福利，甚至加大无所谓开支——因为省下来的钱明年预算被砍）

信息策略也很机智：**要预算时夸大问题与挑战，总结成绩时反过来淡化。** 这是理性选择，不是道德败坏。

### 9.2 唐斯官僚理论

唐斯分析的是**政治家**的行为逻辑：

1. 政治家追求**个人利益**，而非公众利益或意识形态
2. 凭借**选票获取最大利益** → 提出社会方案争取选票
3. 只有**符合中间投票人**的利益 → 才能实现选票最大化
4. **有竞争的公共部门才会更有效率**

唐斯的改革主张：
- 公共部门权力**分散化**（打破垄断）
- 公共产品生产**私人化**（引入竞争）
- 提倡公共部门与非公共部门**之间的竞争**

### 9.3 尼斯卡宁官僚模型

尼斯卡宁分析的是**官僚**（非民选官员）的行为：

- V 线 = 社会总效益（斜率 = 边际社会效益）
- C 线 = 产出成本（斜率 = 边际成本）
- **Q1** = MC = MR → 公共产品最优供应点
- **Q2** = 官僚实际产出 → 大于最优产出
![[财政学Ch05 尼斯卡宁官僚模型.png]]

官僚利用信息不对称——向出资者（立法机关/公众）提供不完全信息，把产出推到 Q2。例如：国防部强调安全受威胁，卫生署夸大儿童营养不良的程度……

**结论**：官僚会在宣传性活动上花大量力气以提高公众对其产品的评价，**实际产出总是大于最优产出。** 因为没有利润约束 → 没有自动刹车机制。


### 9.4 政府规模增长

近代以来，全球政府支出占 GDP 比重持续上升。

两种解释：
- **收入效应**：公共产品是奢侈品——人们收入越高，愿意为公共产品支付更高比例的税收
- **制度效应**：代议制本身在推高政府成本

**2023 年中国政府支出占 GDP 约 30%。**

## 十、概念对照表

| 概念对 | 区别 |
|------|------|
| **偏好显示 vs 偏好加总** | 前者：你愿不愿说真话（个人层面）；后者：说了真话怎么合成决策（集体层面）|
| **一致同意 vs 多数票** | 一致同意 = 帕累托效率但成本极高；多数票 = 能干事但不能表达偏好强度 |
| **单峰偏好 vs 多峰偏好** | 单峰 = 一个最爱点 → 投票稳定；多峰 = 两个最爱点 → 投票悖论 |
| **阿罗定理 vs 中间投票人定理** | 阿罗说不存在完美制度（悲观结论）；中间投票人说单峰条件下多数票可行（乐观结论）|
| **一次总付税 vs 累进税下的公共产品偏好** | 平均分摊 → 高收入者要更多公共产品；累进税 → 低收入者要更多公共产品 |
| **利益集团 vs 寻租** | 利益集团是组织形态（一群人）；寻租是行为（用政府权力拿超额收益）|
| **唐斯 vs 尼斯卡宁** | 唐斯分析政治家（选票最大化）；尼斯卡宁分析官僚（预算/产出最大化）|
| **赢者通吃 vs 比例代表制** | 前者 → 两党制（总统制国家）；后者 → 多党制（议会制国家）|
| **趋同型两党制 vs 极化型两党制** | 前者 → 都往中间靠、制度稳定；后者 → 各有铁票不妥协、政治撕裂 |
| **投票悖论 vs 议程操纵** | 投票悖论是数学现象（偏好非单峰时循环）；议程操纵是利用悖论的人为干预（排顺序决定结果）|

## 十一、整章逻辑链：从投票到政府失灵的一条龙

```
政治市场：选票是货币，政策是商品
    ↓
代议制：委托-代理的先天缺陷（政客利益 ≠ 选民利益）
    ↓
偏好显示困难 → 免费搭车 + 理性无知 + 低投票率
    ↓
偏好加总困难 → 投票悖论 → 议程操纵
    ↓
阿罗不可能定理：数学上不存在完美投票制度
    ↓
中间投票人定理：单峰条件下多数票可行（但前提苛刻）
    ↓
利益集团 → 游说 → 少数人的利益压倒多数人
    ↓
寻租 → 政府权力变成私人利润（花生市场：abcd 租 + cde 无谓损失）
    ↓
政党行为：选票最大化 → 趋同或极化
    ↓
官僚制度：不核算利润 → 预算/权力/浪费最大化（Q2 > Q1）
    ↓
政府规模持续膨胀：收入效应 + 代议制效应
    ↓
结论：政府不是市场失灵的万能解药——政府自己也会失灵
```
