# 多数票规则 (Majority Rule)

## 📌 定义
多数票规则是指将一致同意条件降低至多数人同意即可通过决策的投票规则。决策成本较一致同意规则大幅减小，但部分社会成员合意做出的决策需全体社会成员承担成本。

> 💬 通俗理解：不用全票通过了，一半以上同意就行。决定做得快了，但少数反对的人也得跟着买单。

## 🧠 机制

### 从一致同意到多数票
一致同意规则成本太高 → 降低条件至多数人同意 → 多数票规则。

### 多数票规则的特征

**优势**：
- 决策成本远低于一致同意规则
- 在人数众多的大型社会中具有可操作性

**问题**：

| 问题 | 说明 |
|------|------|
| 少数承担成本 | 部分社会成员合意做出的决策，全体社会成员承担成本 → 可能出现"多数人的暴政" |
| 选民放弃表达 | 单个参与者的行为不具备决定性，因此更多人放弃表达偏好，不参加投票 |
| 投票悖论 | 存在投票程序无法选出稳定结果的可能（[[投票悖论]]） |
| 只能排序，无法表达强度 | 多数票规则只能显示偏好的顺序（选A还是B），无法表达偏好的强度（我有多喜欢A、多讨厌B） |

## 🏛️ 应用

多数票规则是现代民主制度中最常用的投票规则，适用于议会投票、全民公投、委员会决策等场景。但其内在缺陷——尤其是[[投票悖论]]和议程操纵问题——是政治经济学分析的核心议题。
