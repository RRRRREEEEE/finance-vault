# 唐斯官僚理论 (Downs' Bureaucracy Theory)

## 📌 定义
唐斯（Anthony Downs）的官僚理论认为，政治家追求个人利益而非公众利益或意识形态，凭借选票获取最大利益，通过提出社会方案争取选票——只有符合[[中间投票人定理|中间投票人]]利益才能实现选票最大化。

> 💬 通俗理解：政客不是圣人——他提出的政策不是为了"让世界更美好"，而是为了"让更多人投票给我"。中间选民喜欢什么，他就说什么。

## 🧠 机制

### 核心命题
1. 政治家追求**个人利益**而非公众利益或意识形态
2. 政治家凭借选票获取最大利益
3. 他们通过提出社会方案来争取选票
4. 只有符合[[中间投票人定理|中间投票人]]的利益才能实现选票最大化

### 唐斯的公共部门改革建议

| 建议 | 说明 |
|------|------|
| 有竞争的公共部门 | 引入竞争使公共部门更加有效 |
| 公共部门权力分散化 | 打破垄断，分散权力 |
| 公共产品生产私人化 | 将公共产品生产外包给私人部门 |
| 提倡公共—非公共部门竞争 | 允许私人企业进入公共服务领域 |
