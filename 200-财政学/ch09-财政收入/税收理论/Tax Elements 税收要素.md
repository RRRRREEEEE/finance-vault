# 税收要素 (Tax Elements)

## 📌 定义
税收要素是构成税收制度的基本元素，包括纳税人、课税对象、计税依据、税率、减免税、纳税环节和违章处理等。

## 🧠 机制

### 核心要素

| 要素 | 定义 |
|------|------|
| **纳税人（课税主体）** | 税法规定直接负有纳税义务的单位或个人 |
| **扣缴义务人** | 负有代扣代缴税收义务的单位或个人 |
| **课税对象** | 税收征收的对象（如收入、所得） |
| **计税依据** | 对课税对象的具体计量方法：从价税、从量税 |
| **税目** | 对课税对象的具体范围、项目和类别 |
| **税率** | 应纳税额与计税依据的比例 |

### 调节机制
- **附加**：在正税以外附加征收的额外部分
- **加成**：对部分纳税人的加税措施
- **减税与免税**
- **免征额**：超过后仅对超过部分征税
- **起征点**：超过后对全部（含未超过部分）征税

### 纳税环节
生产环节、销售环节等

### 违章处理
- 行为：偷税、欠税、抗税
- 措施：限期缴税、加收滞纳金、罚款、扣押或查封财产、纳税人银行暂停付款
- 救济：行政复议、提起诉讼
