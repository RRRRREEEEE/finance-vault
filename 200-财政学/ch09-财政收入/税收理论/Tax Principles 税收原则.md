# 税收原则 (Tax Principles)

## 📌 定义
税收原则是指导税制设计的规范性准则。从亚当·斯密古典四原则到瓦格纳四大原则，再到现代公平/效率/稳定三维框架。

## 🧠 机制

### 亚当·斯密古典四原则

| 原则 | 含义 |
|------|------|
| **平等** | 公民应根据能力来承担政府经费支出 |
| **确实** | 税收制度稳定可预期 |
| **便利** | 各种手续从简，采用最便利的方式 |
| **节省** | 征税费用最少 |

### 瓦格纳四大原则

| 维度 | 原则 |
|------|------|
| 财政收入 | 收入充足 + 有弹性适应支出变化 |
| 国民经济 | 慎选税源保护税源 + 慎选税种避免干扰市场 |
| 社会正义 | 普遍 + 平等 |
| 税务行政 | 确定 + 便利 + 节省 |

### 现代税制原则

| 维度 | 原则 | 含义 |
|------|------|------|
| **公平** | 横向公平 | 同等境况同等税负 |
| | 纵向公平 | 不同境况不同税负 |
| | 受益原则 | 税负应与获得的服务对等 |
| | 支付能力原则 | 根据纳税能力纳税 |
| **效率** | 税收中性 | 尽可能不影响市场相对价格 |
| **稳定与增长** | 充分和弹性 | 税收随经济自动调节 |
| **行政** | 节约便利 | 降低征管成本 |
