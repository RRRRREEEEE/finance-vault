# 税收的转嫁与归宿 (Tax Shifting and Incidence)

## 📌 定义
税收转嫁是纳税人通过种种途径将税收负担转移给他人的过程；税收归宿是税收负担最后的落脚点。税收的**经济归宿与法定课征对象无关**——无论向谁征收，最终由供需双方根据弹性分摊。

> 💬 通俗理解：政府说"向生产者征税"，但不代表生产者真的全付——价格涨了消费者买单，价格不涨生产者自己扛。最终谁付看弹性。

## 🧠 机制

### 转嫁形式
| 形式 | 方式 |
|------|------|
| **前转** | 提高售价，转嫁给消费者 |
| **后转** | 压低原料价格或税收资本化，转嫁给供应商 |
| **混转** | 前转 + 后转结合 |
| **消转** | 通过提高效率自我消化 |

### 局部均衡分析
税收的经济归宿与课征对象是**生产者或消费者无关**：
- 向消费者征税 → 需求曲线下移 → 税款由双方分摊
- 向生产者征税 → 供给曲线上移 → 分摊比例相同

### 需求弹性决定归宿

| 需求弹性 | 归宿 |
|----------|------|
| **完全无弹性** | 税款**全部转嫁**给消费者 |
| **完全弹性** | 生产者**全部负担** |
| **富有弹性** | 生产者负担大部分 |
| **缺乏弹性** | 消费者负担大部分 |

> 核心结论：需求弹性越大，税收越不容易转嫁；弹性越小，越容易转嫁。
