# 税收竞争 (Tax Competition)

## 📌 定义
税收竞争是辖区政府通过税式支出（豁免、扣除、减免等）吸引其他辖区资源流入、扩张税基以增加政府收入的行为。税负输出则是在不增加本辖区居民税负的条件下，通过对某些资源的控制从其他辖区获取税收收入。

## 🧠 机制

### 税收竞争的双重效应

| 正面 | 负面 |
|------|------|
| 促进自治地方税收负担水平平均化 | 一国政府总收入减少 |
| 防止辖区内官员滥用权力 | 各自为政现象严重 |

### 税负输出的典型
- 对本地区垄断资源征收重税
- 对外销资源征收重税
- 旅游点对旅客征收重税

### 税收划分的一般原则

| 适合做中央税 | 原因 |
|-------------|------|
| 税基流动性强的税种（增值税、消费税） | 否则引起严重税收竞争 |
| 强再分配性质的税种（个人所得税） | 以支付能力为基础，地方征收引致竞争 |
| 税基不平衡的税种 | 防止地区间极不平衡 |
| 产地垄断产品的税收 | 防止税负输出 |

### 我国税收收入的央地划分

| 归属 | 主要税种 |
|------|----------|
| **中央固定收入** | 消费税、车辆购置税、关税、船舶吨税、海关代征进口增值税 |
| **央地共享** | 增值税（50:50）、企业所得税（60:40）、个人所得税（同企业所得税分成） |
| **地方固定收入** | 城镇土地使用税、耕地占用税、土地增值税、房产税、车船税、契税、环境保护税、烟叶税 |
