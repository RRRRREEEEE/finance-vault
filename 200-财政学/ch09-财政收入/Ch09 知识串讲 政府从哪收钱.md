# 第九讲 知识串讲：政府从哪收钱——税收的逻辑与制度

## 一、一句话回答

**政府收钱主要靠税收——固定、强制、无偿。但税收从来不只是"收钱"：谁出、出多少、怎么转嫁、弹性决定了谁真正买单，每一步都在重新分配社会财富。中国的 18 个税种在中央和地方之间按一套严密规则划分，背后是 1994 年分税制奠定的财政联邦主义框架。**

---

## 二、财政收入从哪来：四个口袋

PPT 给出了一个完整的财政收入形成逻辑图：

| 类型 | 逻辑 | 具体形式 |
|------|------|------|
| **强制性收入** | 你不交也得交 | 税收、罚款、强制性公债 |
| **非强制收入** | 你自愿参与 | 普通公债、公有财产收入（出租/出售国有资产）|
| **服务性收入** | 你用了才交 | 公共收费（办护照、高速公路费）|
| **间接性收入** | 偷偷收了你没感觉 | 货币发行——通货膨胀稀释你的购买力 |

这个分类的关键在于：政府收钱的手段不只有税收。**公债**（借）、**印钱**（通胀税）、**收费**都是收入，但税收永远是主体。

PPT 同时指出："**税收是政府取得财政收入的最佳形式。**" 税收征收形式的历史变化：实物 → 货币，农业税 → 工商业税，简单税制 → 复杂税制。

## 三、税收的基本定义与内在矛盾

### 3.1 定义

税收是用以满足政府开支需要，通过事先确定的程序，由政府**强制、无偿**征收的一种收入。三个基本形式特征：**固定、强制、无偿**。

### 3.2 国家税收理论的局限

PPT 特别批判了传统的国家税收理论：

> "国家税收理论是建立在政府对公民强权的假设基础上的，仅仅倡导公民的义务，而不说明公民所应该具有的权利。"

税收除了强制属性以外，还应该强调：
- 税收属于**纳税人的贡献**，其筹集与使用应充分与纳税人沟通
- 尊重纳税人的**监督权**
- **任何政府的税收都是对私人财产的重新分配**——不是简单的"取之于民，用之于民"，而是在不同利益主体之间进行利益分配

### 4.1 税率类型

| 税率类型     | 规则       | 注意                                          |
| -------- | -------- | ------------------------------------------- |
| **比例税率** | 统一百分比    | 具有**事实上的累退性质**——穷人富人交同一比例，但穷人的负担更重          |
| **累进税率** | 收入越高税率越高 | 全额累进（极不公平，到档全按高档）vs **超额累进**（现代标配，超过部分才按高档） |
| **累退税率** | —        | **工薪税往往实行累退税率**——高收入者工薪占比低                  |
| **定额税率** | 每单位固定金额  | 一包烟固定多少税                                    |

| 财产税 | — | 中央与地方共享税 |
| 资源税 | — | — |
| 行为税 | — | — |

## 五、税收的超额负担：税比你想象中更贵

政府对汽油征税。你以为只是每升贵了几毛钱——但损失的远不止这些。

征税造成**两个价格**：消费者支付的价格（PD）≠ 生产者实际得到的价格（PS）。两者之差一部分进了政府腰包（税收收入 = 方形区域），但还有一个**三角形区域**是白白损失的——这不属于任何人，纯粹是因为征税导致产量从 Q1 萎缩到 Q2。这就是**超额负担（Excess Burden，即无谓损失）**。


### 5.1 收入效应与替代效应

| 效应 | 机制 | 影响范围 |
|------|------|------|
| **收入效应** | 征税相当于使收入减少 → **所有**商品的消费都减少 | 全面收缩 |
| **替代效应** | 商品比价关系改变 → 人们相对减少课税商品的消费 | 结构调整 |


**对生产者也一样**：替代效应 → 减少课税商品生产、增加无税产品产量（生产可能性边界上从 A 移到 B）；收入效应 → 课税后生产能力降低、收入减少。

## 六、税收的转嫁与归宿：法定谁交 ≠ 实际谁付

### 6.1 基本概念

- **税收转嫁**：纳税人将税收负担转移给他人的过程
- **税收归宿**：税收负担最后的落脚点

PPT 举例：某种酒精饮料税前售价 $10，现每瓶征 $1 税：
- 售价仍为 $10 → 生产者是负税人
- 售价为 $11 → 消费者是负税人
- 售价介于 $10-$11 → 双方共同负担

四种转嫁形式：

| 形式     | 做法                                          |
| ------ | ------------------------------------------- |
| **前转** | 提高售价，税推给消费者                                 |
| **后转** | 压低原料价格，税推给供应商；或**税收资本化**（购买时从价格中预先扣除未来应纳税额） |
| **混转** | 前转 + 后转同时用                                  |
| **消转** | 自己提高效率消化掉                                   |

### 6.2 关键结论：弹性决定归宿

PPT 给出了完整的需求弹性分析（对生产者征税的四种情况）：

| 需求弹性 | 结果 |
|------|------|
| **完全无弹性** | 税款**全部**转嫁给消费者（你离不开它，涨价也得买）|
| **完全弹性** | 税款**全部**由生产者承担（一涨价你就不买了）|
| **富有弹性** | 生产者负担大部分 |
| **缺乏弹性** | 消费者负担大部分 |


**一般结论**：
- 需求弹性越大 → 越不容易转嫁；供给弹性越大 → 越容易转嫁
- 需求弹性 > 供给弹性 → 不容易转嫁
- 需求弹性 = 0 → 可全部转嫁；供给弹性 = 0 → 不能转嫁
- 需求完全弹性 → 不能转嫁；供给完全弹性 → 可以全部转嫁

**最重要洞见**：税收的经济归宿与课征对象是生产者还是消费者**无关**——无论向谁征收，最终由弹性决定分摊比例。

## 七、要素课税与税收资本化

### 7.1 工资与地租

- **工资收入的课税**：大部分由工人承担（详见社会保障一章的社会保险税归宿）
- **地租收入**：土地的供给量保持不变。如果对未来地租收入征税，土地购买者会通过**压价**的方式将税负转嫁给土地供给方


### 7.2 税收资本化

**税收资本化**：生产要素购买者**将所购要素未来应纳税额**通过从购入价格中**预先扣除**的方法转嫁税收的过程。

**土地价格 = 土地未来收入的现值**。对地租收入征税后，土地的交易价格会相应下降——未来税负被**折现**到了当前价格中。

## 八、哈伯格模型：一般均衡分析

PPT 介绍了一个简化的两部门一般均衡模型：

**假设**：经济体中只有食品（F）和机械制造（M）两个部门；生产要素只有资本（K）和劳动力（L）；家庭部门没有储蓄；资源自由流动；完全竞争。

税收体系：
- tKF：以食品业的资本要素收入为课税对象
- tLF：以食品业的劳动要素收入为课税对象
- tF：以食品业的产出为课税对象
- tM：以制造业的产出为课税对象
- tK、tL：对资本和劳动要素的普遍征税

**重要等效关系**：
1. 如果政府既征 tF 又征 tM 且税率相同 → 相当于对所有产品征 tT
2. 如果政府既征 tK 又征 tL 且税率相同 → 也相当于对所有产品征 tT
3. **因此，不考虑行政成本，在经济学意义上，对产品的普遍征税和对要素的普遍征税是等效的。**

## 十、地下经济与逃税：贝克尔模型

地下经济逃避政府管制秩序，有自己依靠信誉运行的秩序。**逃税手段**：帐目舞弊、现金交易、以物易物不做记录。

**最佳逃税规模模型**：逃税的边际成本 = 被查处概率（P）× 查处后的边际罚款金额（M）。在 R* 点，逃税的边际成本等于边际收益。


政府打击逃税至少有两种方法：**提高查处概率**和**增加罚款金额**。

## 十一、发达国家 vs 发展中国家税收体系对比

| 维度 | 发达国家 | 发展中国家 |
|------|------|------|
| **主体税种** | 所得税为主 | 流转税为主 |
| **财产税** | 重要地位 | 资源税的重要地位 |
| **产业侧重** | 制造业和服务业税收比例高 | 制造业和农业税收比例高 |
| **税制复杂度** | 体系复杂，但成本相对较低，私人分摊成本较多 | 体系简单，但成本相对较高，私人分摊成本较少 |

中国处于从发展中国家税制向更复杂税制过渡的阶段。

### 12.1 税收法律关系与立法权

**税收法律关系**：权利主体（一方是税务机关，另一方是纳税义务人）+ 权利客体（征税对象）。

**按效力和内容分类**：
- **税收基本法**：框定税收基本架构——**我国尚没有基本法**
- **税收普通法**：对税收具体事项分类立法

**按职能分类**：
- **税收实体法**：确定具体税种的立法
- **税收程序法**：税务管理方面的法律

**立法机关**：
- 全国人大及其常委会行使国家立法权：《企业所得税法》《个人所得税法》《税收征收管理法》
- 全国人大或常委会**授权国务院**：增值税、消费税、资源税、土地增值税等暂行条例
- 国务院税收行政法规：各种实施细则
- 地方人大：海南、自治区有受限制的立法权，**其他地方人大不得制定地方性税收法规**

## 十三、财政联邦主义与央地分权

### 13.1 财政联邦主义

一个单一制国家在中央和地方政府之间实行一种类似于联邦制国家的财政制度——明确划分各级**政府的财权与事权，实行财政分权**，以便更有效地提供各种公共产品。

1994 年中国改革各级政府之间的财政体制，实行在**单一制基础上的地方分权**，**各级政府和预算在法律上仅对本级人民代表大会负责**。地方政府根据中央确定的法律收取地方税收。

### 13.2 夏普劝告与夏普三原则

日本战后财政改革的经典参考：

**夏普劝告**（奠定日本战后所得税体系）：
1. 建立以**所得税**为主体的税收体系
2. 个人所得税采取超额**累进综合所得税制**，最高税率 55%
3. 个人资产净值超过 500 万日元部分，以四级超额累进税率**课征富裕税**
4. 企业所得税（法人税）比例税制，税率 35%
5. 防止对**股利分红双重课税**，允许从应缴税额中***扣除股利收入的 25%**

**夏普三原则**（确定日本央地财政划分基本原则）：
1. **明确行政职责**
	1. **大权统一，小权分散**
2. **优先市町村**（基层优先）
3. **效率原则**

### 13.3 税收竞争与税负输出

**税收竞争**：辖区政府通过税式支出（豁免、扣除、减免等）**吸引其他辖区资源流入**而扩张税基进而增加政府收入。

**税负输出**：在**不增加本辖区居民税负的条件下**，通过**对某些资源的控制**从其他辖区获得税收收入——例如对本地垄断资源征重税、旅游点对旅客征重税。

税收竞争的作用：促进自治地方税负水平平均化、防止辖区内官员滥用权力，但也导致**一国政府总收入减少**和**各自为政现象严重**。

### 13.4 央地税种划分原则

| 适合做中央税的                      | 原因            |
| ---------------------------- | ------------- |
| 税基流动性较强的税种（如增值税、消费税）         | 否则会引起严重的税收竞争  |
| 具有强再分配性质、以支付能力为基础的税种（如个人所得税） | 地方征收会引发**税收竞争 |
| **税基不平衡的税种**                 | 防止地区间极不平衡     |
| **产地垄断产品的税收**                | 防止税负输出        |

## 十五、公债：今天花明天的钱

### 15.1 基本概念

政府以发行债务的方式向社会募集收入。按主体分国债和地方债；按期限分短期和长期：
- **短期公债**：经常采用**折价发行**模式
- **长期公债**：采用**面值 + 票面利率还本付息**模式

政府公债被市场认为风险相对较低 → 收益率相对企业债要低。有第三方机构发布**主权信用评级**：标准普尔、惠誉、穆迪。

### 15.2 公债的经济作用

- 转化部分储蓄**进入投资** → 促进经济增长
- 把**私人支出变为公共支出** → 促进资源在不同主体间转换
- 带来政府的**还款压力** → 成为政府风险的来源之一

### 15.3 规模数据

- 美国国债：34.6 万亿美元（2022 年 10 月突破 31 万亿，超过 GDP）
- 中国中央政府显性债务：25.87 万亿人民币（2023 年 1 月）
- 中国地方政府显性债务：35.7 万亿人民币
- 中国 GDP：121.02 万亿

### 15.4 隐性债务与地方债排名

**隐性债务**：地方政府除了正式发行的债务以外，还有非正式的、具有潜在还款风险的债务——地方政府担保债务、担保的外债、地方金融机构的呆坏账、社会保障资金缺口等。

**地方融资平台**：地方政府发起设立，通过划拨土地、股权、规费、国债等资产，包装出资产和现金流均可达融资标准的公司，辅之以财政补贴作为还款承诺，将资金运用于市政建设、公用事业等。

## 十六、概念对照表

| 概念对                      | 区别                                               |
| ------------------------ | ------------------------------------------------ |
| **强制性收入 vs 非强制收入**       | 前者不交不行（税收/罚款/强制性公债），后者自愿参与（普通公债/公有财产出租）          |
| **从价税 vs 从量税**           | 前者按价格计征（如 13% 增值税），后者按数量计征（如一包烟固定税额）             |
| **免征额 vs 起征点**           | 免征额 = 超过后仅就超过部分征税；起征点 = 超过后对全部征税——天壤之别           |
| **比例税率 vs 累进税率 vs 累退税率** | 比例 = 统一%（事实累退）；累进 = 收入越高税率越高；累退 = 工薪税（高收入者工薪占比低） |
| **收入效应 vs 替代效应**         | 收入效应 = 变穷了省着花（量变）；替代效应 = 比价变了换着花（结构变）            |
| **超额负担 vs 税收收入**         | 超额负担 = 净福利损失（三角形）；税收收入 = 政府拿走的部分（方形）             |
| **前转 vs 后转 vs 税收资本化**    | 前转 = 提价推给消费者；后转 = 压价推给供应商；资本化 = 从购入价中预先扣除未来税负    |
| **法定归宿 vs 经济归宿**         | 法定说谁交（法律条文），经济看谁真正付（弹性决定）——与向谁征收无关               |
| **直接税 vs 间接税**           | 直接税 = 税负不能转嫁（所得税）；间接税 = 税负可以转嫁（流转税）              |
| **哈伯格等效关系**              | 对产品的普遍征税 = 对要素的普遍征税（不考虑行政成本）                     |
| **税收基本法 vs 普通法**         | 基本法 = 框定税收基本架构（我国尚无）；普通法 = 具体税种立法                |
| **显性债务 vs 隐性债务**         | 显性 = 正式发行计入统计；隐性 = 担保/平台/缺口等非正式但真实存在的风险          |
| **国债 vs 地方债**            | 国债由财政部发行（国家信用），地方债由省级政府发行（有中央兜底预期）               |
| **财政联邦主义 vs 税收竞争**       | 前者 = 央地财权事权划分的制度框架；后者 = 地方之间用税收优惠争夺资源的行为         |

## 十七、最重要的逻辑链

### 17.1 税收转嫁的完整逻辑
```
政府征税（法定归宿）
    → 被征税方尝试转嫁（前转/后转/混转/消转）
    → 弹性决定转嫁程度（需求弹性越大越难转嫁）
    → 最终归宿（谁真正付了这笔税）
    → 结论：向谁征税不重要，弹性决定谁买单
```

### 17.2 中国央地财税关系的完整框架
```
1994 分税制
    → 财政联邦主义（单一制下的分权）
    → 税种三分：中央税 / 地方税 / 共享税
    → 划分原则：流动性强的归中央（防税收竞争）
    → 结果：中央占收入大头，地方靠转移支付 + 土地财政
    → 副作用：地方债（显性 + 隐性）膨胀
```
