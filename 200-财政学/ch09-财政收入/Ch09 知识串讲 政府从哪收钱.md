# 第九讲 知识串讲：政府从哪收钱——税收的逻辑

## 一、一句话回答

**政府收钱主要靠税收，但税收从来不只是"收钱"——谁出、出多少、怎么转嫁，每一步都在重新分配社会财富。**

---

## 二、财政收入从哪来

政府收入有四个口袋：

| 类型 | 逻辑 | 例子 |
|------|------|------|
| **强制性收入** | 你不交也得交 | 税收、罚款 |
| **非强制收入** | 你自愿参与 | 买国债、公有财产出租 |
| **服务性收入** | 你用了才交 | 办护照收费、高速公路收费 |
| **间接性收入** | 偷偷收了你没感觉 | 通货膨胀（央行印钱稀释你的购买力） |

注意这个国际比较：中国税收占 GDP 仅 **7.8%**。法国 43.6%、英国 39%、德国 23.5%——中国不是高税负国家，这是理解中国财政结构的第一个关键数据。

---

## 三、税收的基本要素

任何一个税种都由七个要素定义：

**纳税人**（谁来交）、**课税对象**（对什么征税）、**计税依据**（按什么算）、**税目**（具体项目）、**税率**（交多少）、**减免**（优待谁）、**违章处理**（不交怎样）。

税率有三种基本形式：

| 税率类型 | 规则 | 例子 |
|------|------|------|
| **比例税率** | 统一百分比 | 企业所得税 25% |
| **累进税率** | 收入越高税率越高 | 个人所得税 3%→45% |
| **定额税率** | 每单位固定金额 | 一包烟固定多少税 |

累进税率中还有**全额累进**和**超额累进**之分——前者是到了某档全部按高档算（极不公平），后者是超过部分才按高档（现代税制的标配）。

---

## 四、税收的超额负担：税收比你想象中更贵

假设政府对汽油征税。你以为只是每升油贵了几毛钱——但损失的远不止这些。

征税造成**两个价格**：消费者付的价 ≠ 生产者拿到的价。中间的差额一部分进了政府腰包（税收收入），但还有一个**三角形区域**是白白损失的——这不属于任何人，纯粹是因为征税导致产量从最优水平萎缩。这就是**超额负担（无谓损失）**。

税收还有两个效应：
- **收入效应**：你变穷了，所有消费都省着来
- **替代效应**：被征税的东西相对变贵了，你换买别的——需求被扭曲

## 五、税收的转嫁与归宿：谁在真正付钱

政府说"向企业征企业所得税"。谁付？企业老板付吗？

不一定。**税收的法定归宿和实际归宿是两回事。**

| 转嫁形式 | 怎么做的 |
|------|------|
| **前转** | 提价，税加到消费者头上 |
| **后转** | 压供应商的价，把税往后推 |
| **混转** | 前转+后转同时用 |
| **消转** | 自己提高效率消化掉 |

### 决定性因素：需求弹性

```
需求完全无弹性 → 税全部转嫁给消费者（你离不开它，涨价也得买）
需求完全弹性 → 税全由生产者扛（一涨价你就不买了）
需求富有弹性 → 生产者扛大部分
需求缺乏弹性 → 消费者扛大部分
```

一句话：**需求弹性越大，税收越难转嫁。** 为什么烟酒重税？因为需求缺乏弹性——你戒不了，税都转嫁给你了。

## 六、税收原则：好税长什么样

从亚当·斯密到今天，好税的标准：

| 斯密四原则（1776） | 瓦格纳四原则（19世纪） | 现代框架 |
|------|------|------|
| 平等 | 财政政策 | 公平 |
| 确定 | 国民经济 | 效率 |
| 便利 | 社会正义 | 稳定 |
| 最少征收费用 | 税务行政 | — |

## 七、地下经济与逃税

每个人都算一笔账：逃税被抓住的概率 × 罚款 = 预期成本。预期成本 < 应纳税额 → 逃。

这就是**贝克尔模型**：P × M 决定最佳逃税规模。监察越严、惩罚越重 → 逃税越少。反过来，税负越重、逃税越赚 → 地下经济越大。

---

## 八、公债：今天花明天的钱

### 8.1 公债的逻辑

政府不够花 → 发债借钱 → 现在有钱了 → 将来要还。**公债就是跨期税收**——今天的赤字是明天的税单。

### 8.2 规模有多大

- 美国国债：34.6 万亿美元，超过 GDP
- 中国中央政府显性债务：25.87 万亿人民币
- 中国地方显性债务：35.7 万亿人民币
- 中国 GDP：121 万亿 → 中央+地方显性债务约 61.6 万亿，约占 GDP 51%

### 8.3 更可怕的是隐性债务

地方政府平台公司——政府注入土地、股权包装出来的融资实体——欠的钱不算法定政府债务，但最后还不起还是政府兜。青海、贵州、吉林等省份负债率已超 50%。

## 九、概念对照表

| 概念对 | 区别 |
|------|------|
| **强制性收入 vs 非强制收入** | 前者你不交不行（税收），后者你自愿（买国债） |
| **法定归宿 vs 经济归宿** | 法定说谁交，经济看谁真正付——两者经常不同 |
| **前转 vs 后转** | 前转推给消费者（提价），后转推给供应商（压价） |
| **收入效应 vs 替代效应** | 前者是人变穷了省着花（量变），后者是比价变了换着花（结构变） |
| **超额负担 vs 税收收入** | 前者是净福利损失（三角形/无谓损失），后者是政府拿走的收入（方形） |
| **显性债务 vs 隐性债务** | 前者计入官方统计，后者不体现但风险真实存在 |
| **国债 vs 地方债** | 国债由财政部发行（国家信用），地方债由省级政府发行（有中央兜底预期） |

## 十、最重要的故事：税收转嫁链条

```
政府说：向生产者征消费税
    ↓
生产者：行，我把价格涨上去
    ↓
消费者：这个东西一定要用吗？
    ├→ 需求有弹性：不买了 → 税由生产者扛
    └→ 需求无弹性：硬着头皮买 → 税全转嫁给你
    ↓
最终结论：政府向谁征税不重要，弹性决定谁真正买单
```
