# 第十讲 知识串讲：增值税为什么是中国第一大税

## 一、一句话回答

**增值税占全国税收的 38%（2024 年 6.57 万亿），因为它用一种巧妙的设计解决了传统流转税"越流转越沉重"的死结——只对新增价值征税，进项抵销项，道道征收但不重复。**

---

## 二、增值税凭什么成为第一大税

增值税起源于法国，已被约 170 个国家或地区广泛采用。不同国家叫法不同：增值税（VAT）、商品服务税（GST）、日本独有的消费税（JCT）。

**2024 年全国国内增值税征收总额 6.57 万亿元，占全部税收收入 38%；2025 年 6.89 万亿元，增长 3.4%。**

**2024 年 12 月 25 日**，十四届全国人大常委会审议通过了《中华人民共和国增值税法》，国务院颁布了《实施条例》——增值税正式从条例升级为法律。

应税交易范围：在中华人民共和国境内**销售货物、服务、无形资产、不动产以及进口货物**。

---

## 三、增值税的发展历程

| 时间 | 事件 |
|------|------|
| 1993.12.13 | 国务院颁布《增值税暂行条例》|
| 1994.1.1 | 正式实施（生产型增值税）|
| 2009 | 从**生产型**向**消费型**转型 |
| 2012 | 上海市开展**营改增**试点 |
| 2016.5.1 | 全国范围营改增试点，营业税逐步退出 |
| 2017 | 修订《增值税暂行条例》，废止《营业税暂行条例》|
| 2024.12.25 | **《增值税法》审议通过** |

1993 年定义 vs 2017 年定义的核心变化：征税范围从"货物 + 加工修理修配劳务"扩展到**"货物 + 劳务 + 服务 + 无形资产 + 不动产"**——营改增把服务业全部纳入了增值税体系。

---

## 四、为什么需要增值税：传统产品税的困局

增值税由托马斯（美）与威尔海姆（德）最早提出，1936 年法国最早采用，专门针对传统流转税的**重复征税**问题。

传统流转税下：商品从生产到消费经历的环节越多，承担的税款越多——每销售一次都按总销售收入全额征税。这等于**惩罚专业分工**。

**增值税的对比**（PPT 用两张表展示）：

| | 传统产品税 | 增值税 |
|------|------|------|
| 征收方式 | 每次按全额征 | 每次只对增值额征 |
| 税负 | 流转环节越多税越重 | 无论多少环节，总税负恒定 |
| 公平性 | 鼓励纵向一体化（少流转少交税）| 税收中性，不扭曲生产方式 |


### 六大特点

| 特点 | 含义 |
|------|------|
| **税收中性** | 大部分商品税率相同，尽可能不影响商品间相对价格，减少影响市场运行 |
| **普遍征收** | 税基宽阔，涉及绝大部分商品和服务，基本没有免税条目 |
| **道道征收** | 商品每流转一个环节就征一次税 |
| **税款抵扣** | 进项抵销项——如果没有进货发票，进项部分得不到抵扣 → **商家之间互相监督** |
| **出口退税** | 出口环节零税率，退还出口商品中已含的增值税 → 提升竞争力但不违反自由贸易规则 |
| **价外征收** | 税款以不含税价乘以税率计算，发票分列"商品价格 + 商品税款" |

## 五、增值税的三种类型

根据固定资产投资的税务处理不同：

| 类型 | 购进资产能抵吗 | 税基 | 增值额 = |
|------|:---:|------|------|
| **生产型** | ❌ 不能 | 国民生产总值（最宽）| 消费额 + 净投资 + 折旧 = 工资 + 利息 + 租金 + 利润 + 折旧 |
| **收入型** | 🔄 按折旧逐年抵 | 国民收入 | 消费额 + 净投资 = 工资 + 利息 + 租金 + 利润 |
| **消费型**（中国现行）| ✅ 一次全额抵 | 消费资料（最窄）| 消费额 |

**2009 年 1 月 1 日起，中国实行消费型增值税**——准许一次性全部抵扣当期购进的固定资产价款。这对鼓励投资有重要意义：买设备不再被惩罚。

## 六、纳税人与征税范围

### 6.1 一般规定

纳税人：在中华人民共和国境内销售货物、服务、无形资产、不动产以及进口货物的单位和个人。

- 货物：有形动产、电力、热力、气体
- 服务：交通运输、邮政、电信、建筑、金融、信息技术、文化体育、鉴证咨询、教育医疗、广播电视等
- 无形资产：技术、商标、著作权、商誉、自然资源使用权等
- 不动产：建筑物和构筑物

### 6.2 视同应税交易

虽无直接销售但视同销售征税：
1. 单位和个体工商户将自产或委托加工的货物用于**集体福利或个人消费**
2. 单位和个体工商户**无偿转让**货物
3. 单位和个人无偿转让无形资产、不动产或金融商品

### 6.3 非应税交易

以下不属于增值税征收范围：
- 员工为受雇单位提供取得工资薪金的服务
- 收取行政事业性经费、政府性基金
- 依法被征收征用而取得补偿
- 取得存款利息收入

### 6.4 扣缴义务人

境外单位或个人在境内发生应税行为、在境内未设有经营机构的，以**购买方**为增值税扣缴义务人。应扣缴税额 = 接收方支付的价款 ÷（1 + 税率）× 税率。

## 七、一般纳税人 vs 小规模纳税人

按会计核算水平和经营规模分：

| 维度 | 一般纳税人 | 小规模纳税人 |
|------|------|------|
| **认定标准** | 年销售额 > 500 万 | 年销售额 ≤ 500 万 |
| **计税方式** | 销项 − 进项（抵扣制）| 销售额 × 征收率（简易计税）|
| **发票** | 可开增值税**专用发票**（供下游抵扣）| 不可开专票（只能用普通发票）|

## 八、税率与征收率

### 8.1 四档税率

| 税率 | 适用范围 |
|------|------|
| **13%**（基本税率）| 销售货物、加工修理修配劳务、有形动产租赁、进口货物（除特殊规定外）|
| **9%**（低税率）| ①粮食等农产品/食用植物油/食用盐；②自来水/暖气/天然气/煤气；③图书/报纸/杂志/电子出版物；④饲料/化肥/农药/农机/农膜；⑤交通运输/邮政/基础电信/建筑/不动产租赁/销售不动产/转让土地使用权 |
| **6%**（低税率）| 增值电信服务、金融服务、现代服务（不动产租赁除外）、生活服务、销售无形资产（转让土地使用权除外）|
| **零税率** | 出口货物 |

### 8.2 征收率（小规模纳税人/简易计税）

法定征收率 **3%**，少量行为为 5%、2%、1.5%、1%、0.5%。

## 九、计税方法

### 9.1 三种计算方法

| 方法 | 公式 | 适用 |
|------|------|------|
| **一般计税** | 当期应纳税额 = 当期销项税额 − 当期进项税额 | 一般纳税人 |
| **简易计税** | 应纳税额 = 销售额 × 征收率 | 小规模纳税人 |
| **扣缴计税** | 应扣缴税额 = 接收方支付价款 ÷（1+税率）× 税率 | 境外单位扣缴 |

### 9.2 销项税额

**销项税额 = 销售额 × 税率**

增值税实行**价外税**——销售额是不包含税款的不含税额。价款和税款合并定价的：
> **销售额 = 含税销售额 ÷（1 + 税率）**

销售额包括全部价款和价外费用：手续费、补贴、基金、集资费、返还利润、奖励费、违约金、包装费、包装物租金、储备费、运输装卸费、代收代垫款项等——**无论会计上是否列入销售额，均应并入计算。**

**特殊销售方式**：

| 方式 | 处理 |
|------|------|
| **折扣销售** | 折扣额与销售额在同一发票注明 → 可按折扣后余额；另开发票 → 不得减除 |
| **现金折扣** | 属于融资性费用，**不得**从销售额中减除 |
| **销售折让** | 因质量引起，以折让后货款为销售额 |
| **以旧换新** | 按新货物同期销售价格，**不得**扣减旧货收购价 |
| **还本销售** | 不得从销售额中减除还本支出 |
| **以物易物** | 双方都做购销处理 |
| **包装物押金** | 1 年以内且未过期 → 不计税；逾期或超 1 年 → 计销项税额 |

### 9.3 进项税额

进项税额是纳税人购进货物、劳务、服务、无形资产、不动产所支付或负担的增值税额。

**准予抵扣的进项税额**（限于以下凭证注明的税额）：
1. 从销售方取得的**增值税专用发票**
2. 从海关取得的**海关进口增值税专用缴款书**
3. 自境外购入，从税务机关或扣缴义务人取得的**完税凭证**
4. 购进农产品，按规定扣除率**计算进项税额**

**不得抵扣的进项税额**：
1. 没有取得相关发票和完税凭证的
2. 简易计税方法征收的
3. 非正常损失的
4. 用于个人消费的

销项税额不足抵扣进项税额的，可以**结转下期继续抵扣**。

### 9.4 计算举例

某生产企业为一般纳税人，适用 13% 税率，2021 年 8 月业务：
- 销售甲产品，不含税销售额 80 万元 + 送货运输费 5.65 万元（含税）
- 销售乙产品，含税销售额 22.6 万元
- 自产产品用于集体福利，成本 20 万元，成本利润率 10%
- 购进货物专用发票注明税额 7.8 万元；支付运输费取得专票注明税额 0.54 万元

## 十、进口与出口

### 10.1 进口环节增值税

申报进入中国海关的商品均需申报增值税，税率 13% 或 9%，不使用征收率。特殊情况：进口抗癌及特殊用途药品按 3% 缴纳。

> **组成计税价格 = 关税完税价格 + 关税 + 消费税**
> **应纳税额 = 组成计税价格 × 税率**

### 10.2 出口退（免）税

我国的出口退税是对出口货物退还或免征国内各环节已缴纳的增值税和消费税——出口零税率，消费税出口免税。

三种方式：

| 方式 | 含义 |
|------|------|
| **出口退税并免税** | 出口环节不征 + 退还以前已征的税款 |
| **出口免税不退税** | 出口环节不征，但前道环节是免税的（价格不含税）→ 无税可退 |
| **出口不免税也不退税** | 国家限制或禁止出口的货物 → 照常征税，不退以前税负 |

退税率：现行有 13%、9%、6%、0 等，该部分税率经常调整。

两种计算方式：
- **"免、抵、退"办法**：适用于自营和委托出口自产货物的企业
- **"先征后退"办法**：主要用于收购货物出口的外（工）贸企业

## 十一、主要税收优惠

### 11.1 起征点

小规模纳税人：以一个月为计税期间的起征点 **10 万元**，一个季度 **30 万元**。按次为每次 1000 元。起征点以下免税，超过起征点**全额**纳税。

### 11.2 免税项目（PPT 列了约 30 项）

| 类别 | 具体项目 |
|------|------|
| **农业** | 农业生产者自产农产品、农业机耕/排灌/病虫害防治、农牧保险、农业生产用地收入 |
| **医疗/教育/福利** | 医疗机构医疗服务、学校学历教育/学生勤工俭学、托儿所/幼儿园/养老/残疾人机构育养服务、残疾人组织进口专用物品 |
| **文化/文物** | 纪念馆/博物馆/文化馆等文化活动门票收入、古旧图书 |
| **科研/援助** | 直接用于科研的仪器设备、外国政府/国际组织无偿援助进口物资 |
| **金融** | 个人金融商品转让收入、港深通/沪港通投资收入、人民银行对金融机构的贷款服务 |
| **土地/住房** | 土地所有者出让使用权或归还、个人销售自建自用住房、家庭财产分割无偿转让不动产/土地使用权 |
| **其他** | 自然人销售自己使用过的物品、被撤销金融机构清偿债务、执法部门罚没财产收入 |

## 十二、其他征管事项

- **纳税义务发生时间**：收讫销售货款或取得销售款凭据当天；先开票的为开票当天；进口为报关当天
- **计税期间**：10 日、15 日、一个月或一个季度，由税务机关核定
- **纳税地点**：固定业户 → 机构所在地；非固定业户 → 业务发生地；进口货物 → 报关地海关
- **增值税专用发票**：分纸质和电子票，具有同等法律效力。**2024 年 1 月 1 日起全面推行"数电发票"**。1995 年出台《关于惩治虚开、伪造和非法出售增值税专用发票犯罪的决定》

## 十三、概念对照表

| 概念对 | 区别 |
|------|------|
| **增值税 vs 传统产品税** | 传统产品税道道全征无抵扣→重复征税；增值税进项抵销项→只对增值额征 |
| **生产型 vs 收入型 vs 消费型** | 买设备：不能抵（生产型）→ 按折旧抵（收入型）→ 一次全额抵（消费型，中国现行）|
| **一般纳税人 vs 小规模纳税人** | 500 万为界；前者用抵扣制（销−进），后者简易计税（销售额×征收率）|
| **销项税额 vs 进项税额** | 销项 = 卖东西收的税（要交税务局）；进项 = 买东西付的税（可抵扣）|
| **价内税 vs 价外税** | 价内：税款含在价格里（消费者看不到）；价外：发票分列价格+税款 |
| **出口退税 vs 出口免税** | 退税 = 退还已征的税（有税可退）；免税 = 不征也没得退（前道环节免税）|
| **免抵退 vs 先征后退** | 前者适用自营出口生产企业（免→抵→退三步），后者适用外贸企业（先征后退）|
| **视同应税交易 vs 非应税交易** | 视同应税=不销售也要交（福利/无偿转让）；非应税=不属于增值税范围（工资/存款利息）|
| **起征点 vs 免征额** | 起征点：超过后全部计税；免征额：超过后仅就超过部分计税 |

## 十四、最重要的故事线

```
传统产品税：每流转一次征一次全额 → 环节越多税越重 → 惩罚分工
    ↓
增值税（1936法国/1994中国）：每个环节只对增值额征税 → 税负恒定
    ↓
六大特点齐发力：中性/普遍/道道/抵扣/退税/价外
    ↓
关键升级：
  2009 生产型→消费型（固定资产可抵 → 鼓励投资）
  2012-2016 营改增（服务纳入 → 全产业链打通）
  2024 立法（条例升级为法律）
    ↓
2024年6.57万亿，占全国税收38% → 中国第一大税
```
