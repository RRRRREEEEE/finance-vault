# 第十讲 知识串讲：增值税为什么是中国第一大税

## 一、一句话回答

**增值税占了全国税收的38%，因为它用一种巧妙的设计解决了传统流转税"越流转越沉重"的死结——只对新增价值征税，不重复。**

---

## 二、为什么需要增值税：传统产品税的困局

假设一根木头经过三个环节到消费者手里：原材料（30元）→ 产成品（70元）→ 零售（100元）。每个环节征 10% 的税。

传统产品税下：原材料环节交 3 元，产成品环节交 7 元，零售环节交 10 元 → **总共 20 元税**，实际税负率 20%。

问题在哪？**重复征税。** 产成品环节的 70 元里面已经包含了原材料环节那 30 元的成本——这笔钱已经被征过一次税了，再征就是"税上加税"。

增值税的做法：每个环节只对**新增的价值**征税——原材料增值 30 元 → 交 3 元；产成品增值 40 元 → 交 4 元；零售增值 30 元 → 交 3 元。**总税负 10 元，整体负担率恒定为 10%**，不受流转环节数量影响。

---

## 三、增值税怎么做到"不重复"

核心机制：**销项税额 − 进项税额 = 应纳税额**。

你卖了 100 块钱的货（销项 100×13%=13），生产这批货时买原料花了 60 块钱已经交过税（进项 60×13%=7.8）。你只交 13-7.8=5.2，相当于对增值的 40 块钱征 13%。

这就是**税款抵扣制**——每一环节只对增值部分征税，上游交过的税下游抵扣回来。

### 六大特点

| 特点 | 含义 |
|------|------|
| **税收中性** | 大部分商品同税率，不打乱比价关系 |
| **普遍征收** | 覆盖绝大部分商品和服务 |
| **道道征收** | 每个流转环节都征 |
| **税款抵扣** | 进项抵销项，只对增值额征税 |
| **出口退税** | 出口时退还全部已征增值税（出口零税率） |
| **价外征收** | 发票上货款和税款分两行，你一眼能看到交了多少税 |

## 四、中国增值税的演化

| 时间 | 事件 |
|------|------|
| 1993 | 《增值税暂行条例》颁布 |
| 1994 | 正式实施，当时是**生产型**（买设备不能抵扣） |
| 2009 | 转型为**消费型**（买设备一次全额抵扣） |
| 2012 | 上海"营改增"试点 |
| 2016 | 全国营改增——营业税彻底退出 |
| 2024.12.25 | **《增值税法》审议通过**，从条例升级为法律 |

增值税从 1994 年至今，经历三次关键升级：**生产型→消费型**（2009）、**营改增**（2012-2016）、**立法**（2024）。

三种类型的区别在于固定资产怎么处理：

| 类型 | 买设备能不能抵 | 实际税基 |
|------|:---:|------|
| 生产型 | ❌ 不能 | 国民生产总值（最宽） |
| 收入型 | 🔄 按折旧逐年抵 | 国民收入 |
| **消费型**（中国现行） | ✅ 一次全额抵 | 仅消费资料（最窄） |

## 五、纳税人与税率

**一般纳税人 vs 小规模纳税人**：年销售额 500 万是分界线。

| 纳税人 | 计税方式 | 发票 |
|------|------|------|
| 一般纳税人 | 销项−进项，可开增值税专用发票 | 别人能从你这里拿到进项抵扣 |
| 小规模纳税人 | 简易计税（征收率），不可开专票 | 别人从你这买就没办法抵扣 |

四档税率：**13%**（基本：货物/加工修理）、**9%**（民生：粮食/图书/交通）、**6%**（现代服务业）、**0%**（出口）。

---

## 六、出口退税的逻辑

出口商品到国外，如果还含着中国的增值税，到了国外加上目的国的税，中国商品的竞争力就打了折扣。

出口退税 = 出口环节零税率 + 退还以前环节已征的增值税。三种情况：

| 方式 | 谁适用 |
|------|------|
| **退税** | 有进项可退的出口货物 |
| **免税** | 没进项可退的（比如小规模纳税人出口） |
| **不免不退税** | 国家限制出口的（稀土等资源品） |

## 七、概念对照表

| 概念对 | 区别 |
|------|------|
| **增值税 vs 传统产品税** | 传统产品税道道全征无抵扣→重复；增值税进项抵销项→只对增值额征 |
| **生产型 vs 消费型增值税** | 前者买设备不能抵（税负重/抑制投资），后者一次全额抵（鼓励投资） |
| **一般纳税人 vs 小规模纳税人** | 500万为界，前者开专票用抵扣制，后者简易计税 |
| **销项税额 vs 进项税额** | 前者是你卖东西收的税（你要交给税务局的），后者是你买东西付的税（你可以抵扣的） |
| **出口退税 vs 免税** | 退税=退还已征的税，免税=不征税也没得退 |

## 八、最重要的故事：为什么营改增划时代

```
1994年前：营业税+产品税并行
    ↓
营业税最大的问题：服务业不能抵扣
    ↓
你开了家物流公司，买卡车要交增值税（进项不能抵营业税）
    ↓
服务业成本被高估 → 行业被抑制
    ↓
2012-2016营改增：货物和服务统一用增值税
    ↓
所有行业都能抵扣 → 产业链条打通
    ↓
2024年增值税6.57万亿，占税收38% → 第一大税
```
