# 增值税 (Value-Added Tax / VAT)

## 📌 定义
增值税是对商品和服务在流转过程中产生的**增值额**为计税依据而课征的一种流转税。起源于法国（1936年），已被约170个国家或地区采用。2024年中国国内增值税征收 6.57 万亿元，占全部税收收入 **38%**。

> 💬 通俗理解：普通流转税按"卖多少钱"征税，卖得越多税越重；增值税只对"赚了多少"（增值部分）征税，避免重复课税。

## 🧠 机制

### 六大特点

| 特点 | 含义 |
|------|------|
| **税收中性** | 大部分商品税率相同，尽可能不影响商品间相对价格 |
| **普遍征收** | 税基宽阔，涉及绝大部分商品和服务 |
| **道道征收** | 每个流转环节都征收 |
| **税款抵扣** | 销项税额 - 进项税额 = 应纳税额 |
| **出口退税** | 出口环节零税率，退还已含增值税款 |
| **价外征收** | 税款以不含税价 × 税率计算，发票上价款与税款分列 |

### 与传统产品税对比

| | 传统产品税 | 增值税 |
|--|:-----:|:-----:|
| 原材料环节（30元→10%） | 3 | 3 |
| 产成品环节（70元→10%） | 7 | 4（仅对增值40元） |
| 零售环节（100元→10%） | 10 | 3（仅对增值30元） |
| **总税负** | **20** | **10** |
| **整体负担率** | 20% | **10%** |

> 传统产品税下经历的环节越多税越重；增值税整体负担率恒定为 10%，不受环节影响。

### 三种类型

| 类型 | 固定资产处理 | 税基 |
|------|-------------|------|
| **生产型** | 购进资产不允许抵扣 | 国民生产总值（消费+净投资+折旧） |
| **收入型** | 按当期折旧额抵扣 | 国民收入（消费+净投资） |
| **消费型** | 购进时一次性全额抵扣 | 消费资料（仅消费额） |

> 2009年1月1日起，中国实行**消费型**增值税，准许一次性全部抵扣当期购进固定资产价款。

### 中国发展历程
| 年份 | 事件 |
|------|------|
| 1993 | 国务院颁布《增值税暂行条例》 |
| 1994.1.1 | 增值税正式实行 |
| 2009 | 生产型→消费型转型 |
| 2012 | 上海营改增试点 |
| 2016.5.1 | 全国范围营改增 |
| 2017 | 修订暂行条例，废止营业税 |
| 2024.12.25 | 《中华人民共和国增值税法》审议通过 |
