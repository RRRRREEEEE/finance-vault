# 增值税计算方法 (VAT Calculation)

## 📌 定义
增值税的计算核心是"销项减进项"：当期应纳税额 = 当期销项税额 - 当期进项税额。小规模纳税人和特定情形使用简易计税方法。

## 🧠 机制

### 三种计税方法

| 方法 | 公式 | 适用 |
|------|------|------|
| **一般计税** | 应纳税额 = 销项税额 - 进项税额 | 一般纳税人 |
| **简易计税** | 应纳税额 = 销售额（不含税）× 征收率 | 小规模纳税人 |
| **扣缴计税** | 应扣缴税额 = 接收方支付价款 ÷ (1 + 税率) × 税率 | 境外单位扣缴 |

### 销项税额
$$\text{销项税额} = \text{销售额} \times \text{税率}$$

- 增值税是**价外税**，销售额是不含税款的不含税额
- 含税销售额换算：$\text{销售额} = \frac{\text{含税销售额}}{1 + \text{税率}}$

**价外费用**（均应并入销售额）：
手续费、补贴、基金、集资费、返还利润、奖励费、违约金、包装费、包装物租金、储备费、优质费、运输装卸费、代收代垫款项等。

**特殊销售方式**：
| 方式 | 处理 |
|------|------|
| 折扣销售 | 同票注明可扣除，另开发票不可扣除 |
| 现金折扣 | 融资性费用，不得扣除 |
| 销售折让 | 以折让后货款为销售额 |
| 以旧换新 | 按新货同期售价，不扣旧货价 |
| 以物易物 | 双方均做购销处理 |
| 包装物押金 | 1年内且未过期不征税；逾期或超1年计税 |

### 进项税额
纳税人购进货物、劳务、服务、无形资产、不动产所支付的增值税税额。

**准予抵扣**：
- 增值税专用发票上注明的税额
- 海关进口增值税专用缴款书上注明的税额
- 境外购入的完税凭证上注明的税额
- 农产品按扣除率计算

**不得抵扣**：
- 未取得相关发票和完税凭证的
- 简易计税方法征收的
- 非正常损失的
- 用于个人消费的

> 销项税额不足抵扣进项税额的，可结转下期继续抵扣。

### 进口环节增值税
$$\text{组成计税价格} = \text{关税完税价格} + \text{关税} + \text{消费税}$$
$$\text{应纳税额} = \text{组成计税价格} \times \text{税率}$$

- 进口环节税率：13% 和 9%，不使用征收率
- 进口抗癌及特殊用途药品：按 3%

## 📄 来源原文
> 一般计税方法的计算公式是：当期应缴纳的增值税税额=当期销项税额—当期进项税额。销项税额=销售额×税率。增值税实行价外税，销售额是不包含税款的不含税额。进项税额是纳税人购进货物、劳务、服务、无形资产、不动产所支付或者负担的增值税税额。
