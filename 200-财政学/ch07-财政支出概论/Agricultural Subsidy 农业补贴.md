# 农业补贴 (Agricultural Subsidy)

## 📌 定义
农业补贴有两种基本模式：高价补贴模式（市场机制 + 政府托底）和低价补贴模式（低于成本收购 + 巨额财政补贴）。两种模式的效率和公平结果截然相反。

## 🧠 机制

### 高价补贴模式（欧盟/美国）

| 维度 | 特征 |
|------|------|
| **起源** | 法国 1933 年起规定小麦保证价（高于市价），逐渐演变为欧盟和美国模式 |
| **价格水平** | 农业价格一直维持在相对较高水平 |
| **政府角色** | 市场机制发挥基础作用，政府仅在价格低于保证价时入市干预 |
| **效果** | 农业现代化问题基本解决 |
| **分配效应** | 消费者利益转移到农产品生产商处 |

### 低价补贴模式（苏联）

| 维度 | 特征 |
|------|------|
| **操作** | 早期以相当于生产成本 13% 的收购价收购粮食；勃列日涅夫九次提高收购价，使收购价高于销售价格 |
| **农业后果** | 农业部门无法自给自足，不能完成自身积累，长期落后，农民极度贫困 |
| **财政后果** | 政府承担巨额财政补贴，不堪重负 |
| **社会后果** | 粮食极度短缺与浪费、特权现象并存 |
| **分配效应** | 生产者利益转移给消费者和工业部门 |

### 模式对比

| 维度 | 高价模式 | 低价模式 |
|------|----------|----------|
| 市场化程度 | 高 | 低 |
| 农业现代化 | ✅ 基本解决 | ❌ 长期落后 |
| 财政负担 | 可控 | 不堪重负 |
| 利益流向 | 消费者 → 生产者 | 生产者 → 消费者/工业 |

## 📖 相关术语
- [[Implicit Subsidy 暗补]]：低价模式本质是暗补
- [[Explicit Subsidy 明补]]：高价模式更接近明补逻辑
- [[Fiscal Expenditure 财政支出]]：农业补贴是转移性支出的重要内容

## 📄 来源原文
> 高价补贴模式：法国1933年起规定小麦的保证价（高于市价），逐渐演变成欧盟和美国的补贴模式。农业价格一直维持在相对较高的水平，市场机制发挥基础作用，政府在价格低于保证价时才入市干预，农业现代化问题基本解决。低价补贴模式：苏联早期以相当于生产成本13%的收购价收购粮食，勃列日涅夫九次提高收购价，使收购价高于销售价格。农业部门无法自给自足，不能完成自身积累，长期落后，农民极度贫困，政府承担了巨额财政补贴，不堪重负。
