# 瓦格纳法则 (Wagner's Law)

## 📌 定义
瓦格纳（Wagner）的公共支出增长法则：随着经济发展和人均收入的提高，公共部门的相对规模会有所提高。原因在于公共产品的收入需求弹性较高，人们对公共产品增长的需求会比一般产品更高。

> 💬 通俗理解：国家越富，人民对教育、医疗、社保等公共服务的需求增长越快，政府支出占 GDP 的比重自然上升。

## 🧠 机制

### 核心逻辑
- **高收入弹性**：公共产品的需求收入弹性 > 1
- 随着收入增长，人们对公共产品的需求增长快于对一般产品的需求
- 公共部门相对规模随经济发展持续扩大

### 实证数据：中央政府支出总额占 GDP 比例

| 国家组别   | 中央政府支出占 GDP（%） |
| ------ | :------------: |
| 发展中国家  |      26.4      |
| 低收入国家  |      20.8      |
| 中等收入国家 |      27.5      |
| 高收入国家  |      28.6      |

> 数据表明：收入水平越高，政府支出占 GDP 比重越大。
