# 国防支出 (Defense Expenditure)

## 📌 定义
国防经费主要由人员生活费、训练维持费和装备费三部分组成，各部分大约各占三分之一。支出科目包括军费、国防科研、专项工程、国防动员等。

## 🧠 机制

### 三大部分（各占约三分之一）
1. **人员生活费**
2. **训练维持费**
3. **装备费**

### 规模与趋势
- 国防经费占 GDP 和财政支出的百分比近年整体稳定：分别约 **2%** 和 **5%**
- 2025年国防支出 17,846.65 亿元 → 2026年计划 19,095.61 亿元

### 国际比较（2025年）
美国 : 中国 : 俄国 ≈ **8 : 3.1 : 2.1**（差距巨大）

### 公共安全支出
2025年中央公共安全支出 2,438.48 亿元 → 2026年 2,582.69 亿元

> ⚠️ AI备注：中央预算列示的分类预算总额不代表政府在该项事业中的实际支出。例如教育部给工科院校的部分预算可能用于支持国防科研。
