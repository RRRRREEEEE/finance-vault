# 第八讲 知识串讲：中国政府的钱具体花在哪

## 一、一句话回答

**政府的钱花在四块：自身运行、教育科技医疗、社会保障、投资建设——其中后两块最具中国制度特色。**

---

## 二、先看整体：中国财政支出的四块版图

如果你翻开每年两会期间财政部的预算报告，你会发现所有支出条目都可以归入四大类。把这几块串起来，你就能看懂政府花钱的完整逻辑：

| 四大板块 | 核心内容 | 占财政比重参考 |
|---------|---------|:---:|
| **一般公共服务** | 行政、国防、外交——政府自身运转 | 行政~8%、国防~5%、外交<1% |
| **教育科技医疗** | 最大单项公共服务 | 教育15%、科技4%、医疗7%+ |
| **社会保障** | 保险、救济、福利、优抚 + 脱贫攻坚 | 随覆盖面扩大持续增长 |
| **政府投资与土地财政** | 基建、平台公司、土地出让 | 一度占地方支出近半 |

你看，前两块是所有国家都有的常规支出；后两块，尤其是土地财政，才是真正"中国特色"的地方。

---

## 三、第一块：一般公共服务——政府自身运行的代价

### 3.1 行政支出 (Administrative Expenditure)：政府有多"大"

**行政费用代表政府的基本规模**，包括人大、政协、党派、社团、政府各项支出。拆开来看：

- **人员经费**：工资、津贴、奖金、福利、离退休金
- **公用经费**：公务费、修缮费、设备费、业务费——这里面藏着著名的**三公经费**（公费接待、公费出国、公费购车）

历史走势很有意思：1978年只占财政支出4.37%，此后一路膨胀，2005年冲到14.3%。转折点出现在**2015年中央八项规定**——三公经费大幅下降，行政费此后稳定在**8%**左右。

> 政府规模是不是最优？答案是：税收是公共产品的影子价格——你看人们愿不愿意为目前的公共产品付这个税。

### 3.2 国防支出 (Defense Expenditure)：钱花在三块

国防经费结构极其稳定——**人员生活费、训练维持费、装备费各占约三分之一**。2026年计划19,095.61亿元，约占GDP的**2%**、财政支出的**5%**。

国际比较让你有体感：美国、中国、俄国的军费比大约是 **8 : 3.1 : 2.1**——差距巨大。

> 注意一个容易忽略的点：预算表上的分类数字不等于实际支出。比如教育部拨给工科院校的钱，可能有一部分实际支持了国防科研——跨科目交叉是常态。

### 3.3 外交支出 (Diplomatic Expenditure)：小而关键

外交管理、驻外机构、对外援助、国际组织会费等，2026年计划**709.75亿元**。量不大，但体现的是国际存在感。

---

## 四、第二块：教育、科技、医疗——最大的公共服务支出

这是整个一般公共服务中体量最大的部分，而且增速分化明显：

| 领域 | 占中央预算 | 关键数据 |
|------|:---:|------|
| **教育** | ~15% | 基层可达20%+，占GDP的4% |
| **科技** | 3.5%~4% | 全国约1万亿，中央本级3,215亿 |
| **医疗** | 7%+ | 2010年以来增速最快 |

趋势很清楚：**医疗支出增速跑赢教育和科技**，背后是老龄化和医改双重推动。

教育的全球性摇摆也值得你关注：美国奥巴马大规模推公立教育引发争论，中国2018年以来推"双减"、基础教育公益化、高等教育去产业化——政府这只手在教育领域到底该伸多长，全世界都没有共识。

## 五、第三块：社会保障——从理论到中国实践

### 5.1 四种形式与两大失灵

**社会保障 (Social Security)** 是国家在公民失去正常收入能力时提供的物质帮助。它有四种形式：

| 形式 | 要缴费吗 | 典型例子 |
|------|:---:|------|
| **社会保险** | 是 | 养老、失业、工伤 |
| **社会救济** | 否 | 灾民救济 |
| **社会福利** | 否 | 老人、儿童、残疾人保障 |
| **优抚安置** | 否 | 军人及其亲属 |

但为什么社保一定要政府来搞，不能全靠私人保险？三个关键词：

- **逆向选择 (Adverse Selection)** == 死亡螺旋：高风险的人抢着投保 → 赔款增加 → 费率提高 → 健康人退出 → 费率再提高 → 循环崩溃
- **道德风险 (Moral Hazard)**：第三方支付下，医院和患者都有过度消费的动机——反正不是我掏钱
- **家长主义 (Paternalism)**：保险太复杂，普通人根本算不清成本收益，政府替你做主强制参保

> 医疗市场和保险市场都存在严重的信息不对称。经济学到现在也没找到简便方法解决——目前全球没有一种模式敢说"我成功了"。

### 5.2 四种国际模式

1881年德国俾斯麦政府开了头。各国走出四条路：

| 模式 | 代表国家 | 保障对象 | 一句话 |
|------|----------|----------|------|
| **保守模式** | 法国、德国 | 雇员 | 你工作了才保你 |
| **自由模式** | 美国、澳大利亚 | 弱势群体 | 只兜底，其余靠市场 |
| **社会民主模式** | 北欧 | 全体公民 | 从摇篮到坟墓 |
| **苏东模式** | 前苏联、东欧、早期中国 | 城镇居民/国有单位 | 单位包办一切 |

每一种都是**保障广度 vs 财政负担**的权衡——保障越全，花钱越多；保障越少，社会风险越大。PPT 警告：**"如果使用得当，社会保障可以降低贫困水平、减少收入差距，防止社会动荡，但社会保障也可能会极大地增加政府行政成本，导致政府崩盘。"**

**美国案例：奥巴马 vs 川普的社保拉锯**

| 奥巴马政府改革 | 川普政府再改革 |
|------|------|
| 强制全民参保（企业、个人）| 废除个人强制参保 |
| 禁止保险歧视 | 砍掉给保险公司的补贴 |
| 扩大医疗补助 | 承诺退休金保障 |
| 补贴中低收入者购买保险 | 收紧残障与低保 |
| 对高收入者加征 3.8% 医保税 | 减税、福利紧缩、优待老年人 |

美国整体社会保障程度一般（尤其相对欧洲），核心是"保障有工作的人"——雇员和雇主各 50% 分担工薪税，退休后取得退休金。医疗保险不涵盖全体国民。

### 5.3 社会保险税：老板交的那份，本质上还是你的钱

**社会保险税 (Social Insurance Tax)**，也叫工薪税，雇主和雇员共同缴纳。但经济学的**税收归宿**分析告诉你一个反直觉的结论：**不管法律上怎么分，大部分负担最终落在雇员身上**。

逻辑很简单：老板替员工交的那部分，通过压低工资转嫁了——它本质上就是你的劳动成本。中国目前养老保险资方交16%、劳方交8%，医疗保险资方6-8%、劳方2%，且已经从社保部门征收改为**税务部门征收**，征收力度更强。

> 对收入特别低的国家，社会保险税没有意义——它甚至可能加大贫富差距。


### 5.4 中国社保：从"单位管你"到"五险一金"

中国社保的发展是一条从苏东模式转型的路径：

- **1995年以前**：苏东模式**单位保障**——你的生老病死全靠你所在的单位
- **现在**：**五险一金**（养老、医疗、失业、工伤、生育 + 住房公积金），与市场经济配套
- **关键变化**：五险缴纳标准从最低基数调整为**税前收入标准**；保险费从社保部门征收调整为**社保监管、税务部门征收**——征收力度显著增强

但中国人的社保水平因身份而异，PPT 明确指出三种境遇：

| 身份 | 保障水平 |
|------|------|
| 体制内 | 较高，含**公费医疗** |
| 体制外 | 参差不齐 |
| 农村户口 | 最低 |

农村养老：每年交100-8000元，60岁以上每月领163-350元。农村医保：个人交380元/年，政府补724元/人/年。改革方向很明确：从**高保障、小范围**走向**低保障、大范围**，推进全国统筹。

### 5.5 脱贫攻坚 (Poverty Alleviation)：中国动员能力的极致体现

脱贫标准你记住六个字：**一收入、两不愁、三保障**。

- **一收入**：按2010年不变价，农民人均收入2,300元（折现价4,000元+）
- **两不愁**：不愁吃、不愁穿
- **三保障**：义务教育、基本医疗、住房安全

时间线：2015年通过决定 → **2020年宣布消除极端贫困** → 2021年转向乡村振兴。这套工程的中国特色在于：**政府表现出了不可比拟的动员能力**。

## 六、第四块：政府投资与土地财政——中国特色的增长引擎

### 6.1 政府投资 (Government Investment)：乘数效应 vs 挤出效应

中国政府有着规模巨大的投资行为。按**投资乘数**理论，政府投一块钱能撬动更多GDP增长；但风险是**挤出效应 (Crowding-Out)**——政府投资多了，私人投资空间被挤压。

政府投资要考虑三个问题：**规模多大？结构投哪？效益如何？** 钱来自两个渠道：**一般公共预算中的资本性投资，以及政府基金和国有资本预算中的基建支出。

五种运行模式，从完全公办到公私深度合作：

| 模式              | 机制                              | 实例        |
| --------------- | ------------------------------- | --------- |
| **政府完全筹资建设**    | 财政全包，免费或收费                      | 三峡大坝、航空航天 |
| **私人出资 + 收费补偿** | 政府授权私方投资收费                      | 地方公路桥梁    |
| **政府法人团体运作**    | 城投公司开发新城                        | 城投公司      |
| **BOT**         | Build-Operate-Transfer，建设—经营—移交 | 高速公路      |
| **PPP**         | 政府与社会资本合资平台                     | 比BOT合作更紧密 |

### 6.2 地方政府平台公司 (Local Government Platform Company)：融资"壳"

地方政府平台公司（俗称**城投公司**）是地方政府融资的核心工具。运作逻辑：政府注入财政拨款、**土地**或股权 → 拥有独立法人资格 → 以市场化方式融资 → 投向基础设施。

典型的如合肥建投、深圳城投、深创投、苏州工业园区——它们本质上是政府的"融资手臂"。

### 6.3 土地财政 (Land Finance)：曾经撑起地方支出的半壁江山

**土地财政**是90年代以来形成的以土地出让为核心的地方发展模式，也是本章最重要的枢纽概念。它的完整链条：

```
分税制(1994) → 中央拿走大头，地方靠土地
      ↓
土地征收制度 → 政府垄断土地一级市场（低价征收农地）
      ↓
住房分配货币化 → 释放商品房需求
      ↓
土地出让（高价卖出）→ 出让金成为地方核心财源
      ↓
资金投向基建 → 地价上涨 → 更多土地出让 → 循环
```

地方政府将**分税制、土地征收、住房货币化、公共工程建设、产业引进** 统筹协调，形成了一套独特的GDP增长模式。从财政支出角度看，土地财政**一度构成地方政府支出近一半的资金来源**——深刻影响了中国地方政府的全部行为逻辑。

> 理解土地财政，你就理解了过去三十年地方政府为什么拼命征地、建新城、招商引资——这是一整套环环相扣的制度安排，不是某个环节单兵突进。

## 七、概念对照表

| 概念对 | 区别 |
|------|------|
| **行政支出 vs 国防支出** | 前者是政府自身运行的"行政成本"（三公、人员经费），后者是国家安全的外在防御成本 |
| **社会保险 vs 社会救济** | 前者要缴费（如养老保险），后者不缴费（如灾民救济）——有义务vs无义务 |
| **逆向选择 vs 道德风险** | 逆向选择发生在投保阶段（高风险者扎堆），道德风险发生在理赔阶段（过度消费） |
| **保守模式 vs 社会民主模式** | 前者保雇员，后者保全体公民——保障范围的本质差异 |
| **投资乘数 vs 挤出效应** | 政府投资拉动GDP增长 vs 挤占私人投资空间——同一个行为的正反两面 |
| **BOT vs PPP** | BOT是建设-经营-移交的阶段性合作，PPP是更深度的合资平台 |
| **平台公司 vs 土地财政** | 平台公司是运作载体（融资工具），土地财政是整套制度逻辑（以地生财的模式） |
| **单位保障 vs 五险一金** | 单位保障是计划经济下的"单位全包"，五险一金是市场经济的社保制度转型 |
| **法定归宿 vs 经济归宿** | 社会保险税法规定雇主雇员共缴，但经济学分析表明大部分负担落在雇员身上 |
| **脱贫攻坚 vs 社会保障** | 脱贫攻坚是消除极端贫困的攻坚战（2020已收官），社会保障是长期制度安排 |

## 八、枢纽逻辑：中国财政支出的三元结构

把整章串起来，你会发现中国政府的钱花出去，遵循一个**三元结构**：

```
一般公共服务（行政/国防/外交）
      ↓ 维持政府运转
教育科技医疗（最大公共服务）
      ↓ 培育人力资本
社会保障（保险/救济/福利/扶贫）
      ↓ 兜底社会稳定
      +
政府投资 + 土地财政（中国特色的增长引擎）
      ↓ 推动经济增长
```

前三元和所有现代国家大同小异，真正让中国支出结构与众不同的是**第四元——政府投资和土地财政**。它把土地作为核心杠杆，一端撬动基础设施和城市化，另一端撬动了地方政府的行为模式。理解了这一点，你就理解了中国过去三十年的增长密码，也理解了今天地方债务等问题的制度根源。
