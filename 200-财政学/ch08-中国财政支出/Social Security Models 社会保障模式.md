# 社会保障模式 (Social Security Models)

## 📌 定义
现代社会保障制度始于1881年德国俾斯麦政府。各国形成了四种基本模式：保守模式、自由模式、社会民主模式和苏东模式。

## 🧠 机制

### 四种模式

| 模式 | 代表国家 | 保障对象 |
|------|----------|----------|
| **保守模式** | 法国、德国 | 雇员 |
| **自由模式** | 美国、澳大利亚、新西兰 | 弱势群体 |
| **社会民主模式** | 斯堪的纳维亚半岛（北欧） | 全体公民 |
| **苏东模式** | 前苏联、东欧、中国（早期） | 城镇居民/国有单位工作人员 |

### 核心权衡
每一个社会都需要权衡如何设定适合自己的保障水平——保障越全面，财政负担越重；保障越少，社会风险越大。

## 📄 来源原文
> 1881年德国俾斯麦政府颁发了《社会保障法》。我们可以用比较粗的方式，把现代社会保障分成几种不同的模式：保守模式（法德，保障雇员）、自由模式（美澳新，保障弱势群体）、社会民主模式（斯堪的纳维亚，保障全体公民）、苏东模式（前苏联东欧中国，保障城镇居民或国有单位工作人员）。
