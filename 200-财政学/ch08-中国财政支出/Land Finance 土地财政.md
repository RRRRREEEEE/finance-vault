# 土地财政 (Land Finance)

## 📌 定义
土地财政是 90 年代以来形成的以土地出让为核心的地方经济发展和财政收支模式。地方政府将分税制、土地征收制度、住房分配货币化、大型公共工程建设、产业引进等统筹协调，形成了一整套独特的 GDP 增长模式。土地财政一度构成地方政府支出近**一半**的资金来源。

## 🧠 机制

### 构成要素
地方政府将以下制度和工具统筹协调：
1. **分税制**（1994）：中央集中财政收入，地方依赖土地出让
2. **土地征收制度**：政府垄断土地一级市场
3. **住房分配货币化**：释放住房需求
4. **大型公共工程建设**：以地融资
5. **产业引进和聚集**：招商引资，以地换产业

### 核心逻辑
```
土地征收（低价） → 土地整理 → 土地出让（高价）
                                    ↓
              基础设施建设 ← 土地出让金 ← 地方财政支出
                                    ↓
              地价上涨 → 更多土地出让 → 循环
```

### 深刻影响
- 从财政支出角度看，土地财政一度构成地方政府支出近一半的资金来源
- 深刻影响了中国地方政府行为模式
