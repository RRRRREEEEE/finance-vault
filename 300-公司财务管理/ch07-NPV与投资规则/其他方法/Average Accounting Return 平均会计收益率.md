# 平均会计收益率 (Average Accounting Return)

## 📌 定义
平均会计收益率（AAR）= 项目平均净利润 ÷ 项目平均账面价值。接受标准：AAR > 管理层设定的目标收益率。AAR 是唯一**不使用现金流**而使用会计利润的投资决策方法。

> 💬 通俗理解：不用现金流而用会计利润来评价一个投资项目——就像用"账面身高"而非真实身高来判断一个人够不够高。折旧怎么摊、存货怎么计价都会影响 AAR，但它跟项目到底产生多少现金无关。

## 🧠 机制

### 公式

$$AAR = \frac{Average\ Net\ Income}{Average\ Book\ Value\ of\ Investment}$$

与 ROA 类似——都衡量"会计利润 ÷ 资产账面价值"。

### 决策规则

| 规则 | 说明 |
|------|------|
| 接受标准 | AAR > 管理层设定的目标会计收益率 |
| 排序标准 | 选择 AAR 最高的项目 |

### 优缺点

| 优点 | 缺点 |
|------|------|
| 会计信息随手可得 | ❌ **忽略货币时间价值** |
| 计算简单 | ❌ **基于账面价值而非现金流和市场价值** |
| | ❌ 目标收益率完全主观 |
| | ❌ 受会计政策影响（折旧方法、存货计价等） |

> 🎓 **学术评价**：AAR 是现代公司金融中最不被推荐的方法。它用应计会计的数字（净利润、账面价值）替代了项目估值真正需要的东西——现金流。

- 学术和实务共识：**NPV 和 IRR 是更优选择**

## 📄 来源原文
> Average accounting return (AAR): average project earnings after taxes and depreciation, divided by the average book value of the investment during its life. Decision-making: Ranking Criteria: choose highest AAR. Advantages: Accounting information is usually available, Easy to calculate. Disadvantages: Ignores the time value of money, Based on book values, not cash flows and market values.
