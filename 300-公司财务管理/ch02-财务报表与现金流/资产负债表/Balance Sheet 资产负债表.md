# 资产负债表 (Balance Sheet)

## 📌 定义
资产负债表是会计师对企业会计价值在某一特定时点的快照（snapshot），核心恒等式为：**资产 ≡ 负债 + 股东权益**（Assets ≡ Liabilities + Stockholders' Equity）。

> 💬 通俗理解：资产负债表就像给企业拍一张"财务X光片"——左边是公司拥有什么（资产），右边是这些资产怎么来的（借的 + 股东投入的），两边永远相等。

无

## 🧠 机制

资产负债表的结构逻辑：

- **资产 (Assets)**：按**流动性**从高到低排列——转换成现金所需时间越短越靠前。流动资产（现金、应收账款、存货）在前，固定资产（厂房设备）在后
- **负债 (Liabilities)**：按**到期日**排列——流动负债（应付账款）在前，长期负债在后
- **股东权益 (Stockholders' Equity)**：资产减去负债后的剩余索取权，是企业的"净值"

三大分析关注点（财务经理读表时必须注意）：
1. **会计流动性**：企业能偿还短期债务吗？
2. **债务与权益**：企业通过债务承担了多少风险？
3. **价值与成本**：会计数字反映真实经济价值吗？

## 🏛️ 应用
- 资产负债表是评估企业财务健康状况的基础工具——银行放贷前、投资者决策前都先看
- 三大报表（资产负债表 + 利润表 + 现金流量表）必须一起读，单独看任何一张都会得出片面结论

## 📖 相关术语
- [[Accounting Liquidity 会计流动性]]：资产负债表分析的第一大关注点
- [[Debt versus Equity 债务与权益]]：资产负债表右侧结构——债务与权益的比例关系
- [[Market Value vs Book Value 市场价值与账面价值]]：资产负债表上的价值是账面价值，非市场价值
- [[Income Statement 利润表]]：与资产负债表互补——资产负债表是存量的快照，利润表是流量的记录
- [[Net Working Capital 净营运资本]]：流动资产 − 流动负债，直接从资产负债表上半部分计算

## ⚖️ 易混点对照
- 与 [[Bank Balance Sheet 银行资产负债表]] 的区别：企业资产负债表资产端以存货和固定资产为主，银行资产负债表资产端以贷款和证券为主，负债端以存款为主。会计恒等式相同（A=L+E），但科目含义不同

## 📄 来源原文
> An accountant's snapshot of the firm's accounting value at a specific point in time. The Balance Sheet Identity is: Assets ≡ Liabilities + Stockholders' Equity.
> Assets are listed in order of liquidity. Liabilities are listed in the order in which they come due.
> When analyzing a balance sheet, the Finance Manager should be aware of three concerns: Accounting liquidity, Debt versus equity, Value versus cost.
