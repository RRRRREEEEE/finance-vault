# 市场价值与账面价值 (Market Value vs Book Value)

## 📌 定义
财务报表中的资产价值通常基于**历史成本**（Historical Cost）——购买时支付的价格。而**市场价值**（Market Value）是资产、负债和权益在当前市场上实际可以买卖的价格，与历史成本是完全不同的概念。

> 💬 通俗理解：你十年前买的房子，账本上记的是买入价 $100 万（账面价值），但现在市场价是 $300 万（市场价值）。资产负债表给你看的是 $100 万——这对决策有用吗？

## 🔗 前置知识
[[Balance Sheet 资产负债表]]

## 🧠 机制

### 账面价值 vs 市场价值的差异来源
- **账面价值** = 历史成本 − 累计折旧 → 面向过去
- **市场价值** = 买卖双方在当前市场愿意成交的价格 → 面向未来
- 通货膨胀、技术进步、品牌价值等因素导致两者差距可能极大

### 对决策的启示
- **财务报表提供的数字不等于真实经济价值**
- 财务经理和投资者做决策时应基于市场价值而非账面价值
- 但在实践中，市场价值难以客观获取（尤其是非上市公司的资产），所以账面价值仍是分析的起点

## 🏛️ 应用
- 资产负债表分析的第三大关注点——会计数字是否反映真实？
- 并购估值：买方关注的是市场价值，而非卖方账面上的历史成本
- 资产减值测试：当市场价值持续低于账面价值时需计提减值

## 📖 相关术语
- [[Balance Sheet 资产负债表]]：市场价值与账面价值的差异是理解资产负债表的第三个关键
- [[Income Statement 利润表]]：利润表中的折旧基于历史成本而非市场价值，进一步加深了会计利润与实际现金流的偏差

## ⚖️ 易混点对照
- 与 [[Shareholder Wealth Maximization 股东财富最大化]] 的关系：股东财富取决于市场价值（股价），而非账面价值（每股净资产）
