# 公司财务管理 ch02 财务报表与现金流

> 62 页 PPT → 17 篇新知笔记 + 1 篇扩展。四组主题：资产负债表 → 利润表 → 财务现金流 → 会计现金流量表。

## 本章知识点一览

| 页码 | 知识点 | 一句话概括 |
|------|--------|-----------|
| 1-11 | 案例引入（瑞幸/康美/恒大） | 三大财务造假案例：收入、利润、现金都可以造假——必须三张报表一起读 |
| 13-17 | [[Balance Sheet 资产负债表]] | 企业价值的快照：Assets ≡ Liabilities + Equity，左边按流动性排，右边按到期日排 |
| 18 | 资产负债表分析三关注点 | 会计流动性、债务与权益、价值与成本——财务经理读表时必须回答的三个问题 |
| 19 | [[Accounting Liquidity 会计流动性]] | 资产变现金的速度且无显著损失——流动性越高越安全但回报越低 |
| 20 | [[Debt versus Equity 债务与权益]] | 债权人优先索取 vs 股东剩余索取，两者的成本不同，关系影响盈利能力 |
| 21 | [[Market Value vs Book Value 市场价值与账面价值]] | 账面价值基于历史成本，市场价值是现在能卖的价格——两者是完全不同的概念 |
| 22-28 | [[Income Statement 利润表]] | 一段时期的业绩记录：收入 − 费用 = 利润，分经营/非经营/净利润三部分 |
| 29-30 | [[Matching Principle and Accrual Basis 配比原则与权责发生制]] | 收入与费用在交易发生时确认而非现金收付时——CAS/GAAP/IFRS 共同遵循 |
| 31 | [[Noncash Items 非现金项目]] | 折旧是最大非现金费用——净利润 ≠ 现金流的核心原因 |
| 32 | [[Time and Costs 时间与成本]] | 短期有固定成本（设备/承诺），长期所有成本都可变 |
| 33-34 | [[Net Working Capital 净营运资本]]（扩展） | CA − CL，NWC 的增加 = 企业的投资，随企业规模增长 |
| 35-46 | [[Financial Cash Flow 财务现金流]] | CF(A) ≡ CF(B) + CF(S)：资产产生的现金 = 给债权人的 + 给股东的 |
| 36-37 | [[Operating Cash Flow 经营性现金流量]] | OCF = EBIT + Dep − Tax，企业经营创造现金的能力 |
| 38-39 | [[Capital Spending 资本性支出]] | 购买固定资产 − 出售固定资产，企业为未来的投资 |
| 42-43 | [[Cash Flow to Creditors 债权人现金流量]] | 利息 + 还本金 − 新借款 = 流向债权人的净现金 |
| 44-45 | [[Cash Flow to Shareholders 股东现金流量]] | 股利 + 回购 − 发行新股 = 流向股东的净现金 |
| 47-56 | [[Statement of Cash Flows 现金流量表]] | 官方会计现金流量表：经营 + 投资 + 筹资 = 现金变动 |
| 48-49 | [[Cash Flow from Operating Activities 经营活动现金流]] | 从净利润出发调整非现金项目和营运资本变动 |
| 51-52 | [[Cash Flow from Investing Activities 投资活动现金流]] | 资本性资产的变化 = 取得固定资产 − 出售固定资产 |
| 53-56 | [[Cash Flow from Financing Activities 筹资活动现金流]] | 权益和负债的变化：发债/还债/发股/回购/分股利 |
