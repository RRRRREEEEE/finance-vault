# 经营活动现金流 (Cash Flow from Operating Activities)

## 📌 定义
会计现金流量表中，经营活动产生的现金流从净利润出发，加回非现金项目（如折旧），并调整流动资产和流动负债（现金除外）的变化。

> 💬 通俗理解：利润表说赚了 $86M，调整后发现实际从经营中拿到了更多——因为折旧 $90M 没花现金，加上应付账款多了 $31M（还没付出去）。

## 🔗 前置知识
[[Statement of Cash Flows 现金流量表]]、[[Noncash Items 非现金项目]]

## 🧠 机制

### 计算过程（U.S.C.C. 2017）
| 调整步骤 | 金额 |
|----------|------|
| **净利润 (Net Income)** | $86M |
| + 折旧 (Depreciation) | +$90M |
| + 递延税款 (Deferred Taxes) | +$13M |
| **流动资产和流动负债变动：** | |
| − 应收账款增加 (A/R Increase) | −$24M |
| + 存货减少 (Inventory Decrease) | +$11M |
| + 应付账款增加 (A/P Increase) | +$31M |
| = **经营活动现金流** | **$207M** |

### 逻辑
- **加回折旧**：非现金费用，减少利润但未付现金
- **加回递延税款**：会计已确认但尚未支付
- **A/R 增加 → 减去**：收入已确认但现金未收
- **存货减少 → 加上**：存货变现为现金
- **A/P 增加 → 加上**：采购已完成但现金未付

## 📖 相关术语

- [[Cash Flow from Investing Activities 投资活动现金流]]：下一部分
- [[Cash Flow from Financing Activities 筹资活动现金流]]：最后一部分
- [[Noncash Items 非现金项目]]：经营活动调整的核心原因

## ⚖️ 易混点对照
- **会计经营现金流 vs OCF（财务现金流）**：会计经营现金流 = Net Income + Dep + Δ营运资本调整；OCF = EBIT + Dep − Tax。两者出发点不同（NI vs EBIT），结果也不同（$207 vs $238）
