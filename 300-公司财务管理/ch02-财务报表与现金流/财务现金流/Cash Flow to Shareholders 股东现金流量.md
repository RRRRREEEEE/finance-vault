# 股东现金流量 (Cash Flow to Shareholders)

## 📌 定义
股东现金流量（CF(S)）是企业支付给股东的净现金：**CF(S) = 股利 + 股票回购 − 新发行股票**。它是财务现金流恒等式中流向权益投资者的部分。

> 💬 通俗理解：企业今年给股东分了多少？发了 $43M 股利 + $6M 回购股票 = 共给股东 $49M，但又发行新股收回了 $43M，所以净付 $6M。

## 🔗 前置知识
[[Financial Cash Flow 财务现金流]]、[[Debt versus Equity 债务与权益]]

## 🧠 机制

### 计算公式
$$ CF(S) = Dividends + Repurchase\ of\ Stock - Proceeds\ from\ New\ Stock\ Issue $$

U.S.C.C. 2017 示例：
| 项目 | 金额 |
|------|------|
| 股利 (Dividends) | $43M |
| + 股票回购 (Repurchase of Stock) | $6M |
| = 流向股东的现金 (Cash to Stockholders) | $49M |
| − 发行新股所得 (New Stock Proceeds) | −$43M |
| = **流向股东的净现金** | **$6M** |

## 🏛️ 应用
- CF(S) > 0：企业在向股东返还现金（成熟期企业特征）
- CF(S) < 0：企业在从股东融资（成长期企业特征）
- 验证：CF(A) $42M = CF(B) $36M + CF(S) $6M ✅——现金流恒等式成立

## 📖 相关术语
- [[Financial Cash Flow 财务现金流]]：CF(S) 与 CF(B) 之和 = CF(A)
- [[Cash Flow to Creditors 债权人现金流量]]：流向投资者的另一端
- [[Shareholder Wealth Maximization 股东财富最大化]]：CF(S) 是股东实际收到的回报
