# 共同比资产负债表 (Common-Size Balance Sheet)

## 📌 定义
共同比资产负债表将资产负债表中每一项**除以总资产**（Total Assets = 100%），标准化后可跨公司和跨时期比较资产结构和融资结构。

## 🔗 前置知识
[[Balance Sheet 资产负债表]]、[[Common-Size Statement Analysis 共同比报表分析]]

## 🧠 机制

$$ 每项百分比 = \frac{\text{某项金额}}{\text{总资产}} \times 100\% $$

Walmart 示例（FY2023）：
| 资产 | % of Assets |
|------|-------------|
| 现金及等价物 | 3.5% |
| 应收账款 | 3.3% |
| **存货** | **23.3%** |
| 流动资产合计 | ~33% |
| 固定资产+其他 | ~67% |

- 存货占比高达 23.3% → 零售业的典型特征
- 现金仅 3.5% → 零售业的低流动性特征（钱在货上）

## 🏛️ 应用
- 回答"为什么存货占总资产比例这么高？"——零售业的核心就是存货周转
- 对比不同规模零售商（Walmart vs Costco）的资产结构
- 趋势分析：存货占比逐年上升 → 可能库存在积压
