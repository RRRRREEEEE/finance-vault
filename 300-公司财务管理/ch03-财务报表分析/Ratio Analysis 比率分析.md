# 比率分析 (Ratio Analysis)

## 📌 定义
比率分析通过计算和解读财务报表中不同项目之间的关系，深化对企业的理解。它与共同比分析互补——共同比看结构，比率看关系。

> 💬 通俗理解：每个比率问一个具体问题：能不能还债？借了多少钱？存货卖得快不快？每块钱销售剩多少利润？——单独的比率没意义，比较才有意义。

## 🧠 机制

### 五大比率类别

| 类别 | 回答的问题 | 核心比率 |
|------|-----------|----------|
| **流动性比率** | 能还短期债吗？ | Current Ratio、Quick Ratio、Cash Ratio |
| **杠杆比率** | 借了多少？还得起利息吗？ | TD/TA、D/E、EM、TIE、Cash Coverage |
| **营运效率比率** | 资产用得好吗？ | Inventory T/O、Receivables T/O、TAT |
| **盈利比率** | 赚钱能力如何？ | PM、ROA、ROE |
| **市值比率** | 市场怎么看？ | P/E、Market Cap、EV |

### 比率本身没有意义——必须比较
1. **时间趋势分析**（Time-Trend）：同一企业各年度 → 看改善还是恶化
2. **同行对比分析**（Peer Group）：同行业不同企业 → 看竞争优势
3. 跨行业比较危险——茅台和 Walmart 的存货周转完全不可比

### 比率分析的局限性
- 没有理论框架决定哪些比率最重要（依赖分析师判断）
- 多元化企业难以找到合适的同行
- 全球化和会计准则差异使跨国比较困难
