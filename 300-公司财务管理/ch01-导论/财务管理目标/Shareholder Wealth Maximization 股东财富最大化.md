# 股东财富最大化 (Shareholder Wealth Maximization)

## 📌 定义
公司财务管理的正确目标是**最大化现有股东权益的市场价值**（Maximize the current market value of shareholders' equity），而非最大化利润（Maximize Profit）、收入、市场份额（Market Share）或现金持有量。股东财富（Shareholder Wealth）= 公司未来现金流的现值（Present Value of future cash flows）。

> 💬 通俗理解：公司做一切决策的终极标准——这个决策能让股价涨还是让股价跌？

## 🔗 前置知识
- [[Cash Flow 现金流]] — 股东财富来源于企业未来产生的现金流
- [[Corporation 公司]] — 公司制下所有权与控制权分离，目标设定尤为重要

## 🧠 机制
### 为什么是"股东财富最大化"而非其他？
| 替代目标 | 为什么不对 |
|----------|-----------|
| 利润最大化 | 忽略现金流的时点和风险；可能鼓励短期行为 |
| 现金流最大化 | 现金闲置不创造价值，应投资或返还股东 |
| 成本最小化 | 过度压缩成本可能损害长期竞争力 |
| 市场份额最大化 | 市场份额 ≠ 利润，可能以亏损换规模 |

### 股东财富最大化 vs 公司价值最大化
幻灯片原文设问：股东权益价值最大化 = 公司价值最大化？答案不完全等同——当企业有债务时，公司价值 = 股权价值 + 债权价值。但正常经营下，最大化股东财富是合适的代理目标。

### 潜在矛盾
管理层并不总是天然愿意追求股东财富最大化——由此产生代理问题（见 [[Agency Problem 代理问题]]）。

## 🏛️ 应用
- 所有财务决策的终极评判标准：是否增加股东财富
- 资本预算：选择 NPV > 0 的项目（增加股东财富）
- 资本结构：选择最小化资本成本的融资组合（最大化股东财富的份额）

## 📖 相关术语
- [[Agency Problem 代理问题]]：管理层可能偏离股东财富最大化目标
- [[Corporate Finance 公司金融]]：三大决策都是为这个目标服务的工具
- [[Cash Flow 现金流]]：股东财富的本质是未来现金流的折现值

## ⚖️ 易混点对照
- 与"利润最大化"的区别：利润是会计概念（含非现金项目、忽略风险和时间价值）；股东财富是市场价值概念（折现现金流）
- 与"公司价值最大化"的区别：公司价值 = 股权 + 债权；股东财富最大化只关注股权部分
