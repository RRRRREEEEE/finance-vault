# 第1章 知识串讲：公司金融如何为企业创造价值？

## 一、一句话回答

公司金融通过三个决策——**投什么、怎么融、日常怎么转**——来最大化股东财富，但代理问题让这件事变得不简单。

---

## 二、一个让你记住整章的故事：Apple 的"有钱偏要借钱"

2013 年，Apple 账上趴着 **1550 亿美元**现金——富可敌国。但大部分钱在海外，汇回美国要交重税。

激进投资者 Carl Icahn 公开施压：把钱还给股东！要么分红，要么回购股票。

Apple 的选择出人意料：它在美国发行了 **170 亿美元公司债券**——明明不缺钱，偏要借钱。为什么？因为当时利率极低，借钱成本几乎是史上最便宜的。用借来的钱回购股票、发股利，海外现金继续留在海外，**绕开了巨额汇回税**。

> 这个故事串起了公司金融的全部问题：资本结构（发债还是发股？）、税务策略、股东压力 vs 长期发展、闲置现金的机会成本。

读完这章你应该能回答：为什么 Apple 这个"反常"决策，其实是最理性的选择？

---

## 三、资产负债表：一张图看懂公司金融

你没看错——公司金融的三个问题，全部能在一张资产负债表上讲清楚。

![[公司财务管理Ch01 资产负债表 三大决策总览.png]]

> 图形说明：Slide 1-8。资产负债表完整视图——上列 Current Assets (流动资产) + Fixed Assets 固定资产 (Tangible 有形 / Intangible 无形)，下列 Current Liabilities (短期负债) + Long-Term Liabilities (长期负债) + Shareholders' Equity (所有者权益)。Total Value of Assets = Total Firm Value to Investors。

公司金融的三个问题对应这张表的三个区域：

- **左侧：资本预算 Capital Budgeting**——买设备、建厂房、投研发，资产必须产生比成本更多的现金
- **右侧：资本结构 Capital Structure**——借钱（Debt）还是卖股份（Equity）？Apple 发债因为利息低且可抵税
- **上方：净营运资本 Net Working Capital**——流动资产 − 流动负债，手头有现金付工资、进货、收货款

## 四、企业长什么样：为什么公司制赢了？

人类发明了三种企业形式，越往后越复杂、越能做大：

**个人独资企业**：一个人开店。赚钱都是你的，亏了拿全部家当还债。简单但做不大。

**合伙企业**：几个朋友一起干。普通合伙——每个人都要对全部债务兜底（无限连带责任）。有限合伙进了一步——有钱的出钱（有限责任），有力的出力（无限责任管经营）。

**公司**：法律上的"人造人"。三大法宝：
- **有限责任**：股东最多亏完投入的股本
- **永续存续**：创始人死了公司还活着
- **股份自由转让**：随时买卖，流动性高

代价是**双重征税**——公司层面交一次企业所得税，分给股东再交一次个人所得税。但这代价值不值？看看世界上所有大企业——几乎全是公司制，你就知道答案了。

> 中国互联网公司偏爱 AB 股：李彦宏持百度 16.3% 股份但有 57.6% 投票权——用小股权撬动大控制权。

## 五、现金流：比利润更重要的东西

会计利润会说谎。折旧是费用但不花现金，应收账款是收入但钱还没到账。**利润表好看 ≠ 兜里有钱**。

公司金融关心的不是账面利润，是**真实的现金运动**（Cash Flow, not accounting income）：

![[公司财务管理Ch01 企业现金流循环图.png]]

> 图形说明：Slide 1-24。企业现金流循环——Firm raises cash from Financial markets（Short-term debts / Long-term debts / Equity shares）→ Invests in assets（Current + Fixed）→ Assets generate cash flow → 返还 Government、Financial markets。The cash flows from the firm must exceed the cash flows from the financial markets.

> 核心铁律：**A successful firm generates more cash than it uses.** 如果一家公司永远靠融资续命，利润再好看也是僵尸。

## 六、目标：一切为了股东？

财务管理的正确目标不是利润最大化（太短视）、不是现金流最大化（钱闲着不创造价值）、不是成本最小化（砍到伤筋动骨）、不是市场份额最大化（赔钱换规模可以吗？）。

**正确答案：最大化股东财富。**

股东财富 = 未来所有现金流的折现值。这个目标自动考虑了三个维度：**现金流的大小、时点和风险**。今天赚 100 万比明年赚 100 万值钱，确定的 100 万比可能打水漂的 100 万值钱。

但这里有一个致命的问题——下一节揭晓。

---

## 七、代理问题：为什么管理层不一定替你着想

你（股东）是委托人，雇了管理层（代理人）替你管公司。所有权在你手里，控制权在他们手里——这就是**代理关系**。

问题是：他们和你想的不一样。

你想股价涨，他们可能想：
- 坐更好的办公室、配更好的车（**高额福利**）
- 保住自己的位置（**生存需要**）
- 不想被董事会管太多（**独立性**）
- 把公司做大——体量越大薪酬越高，但规模大 ≠ 股东赚钱多（**规模偏好**）

当管理层为规避短期风险放弃一个能创造长期价值的项目时，那个**放弃的未来现金流 = 代理成本**。

## 八、瑞幸咖啡：代理问题怎么演变成灾难

2020 年，瑞幸咖啡承认伪造了超过 **20 亿人民币**销售额。

故事线：管理层拿了大量与**营收增长和股价**挂钩的股权激励 → 为达目标不惜造假 → 内控薄弱无人发现 → 曝光后股价暴跌 80%以上 → 纳斯达克退市 → 股东血本无归。

这不是偶然事件，是代理问题的经典配方：**激励太猛 + 指标单一 + 没人监督 = 道德风险爆炸**。

怎么防？
1. 审计要硬（内部审计 + 独立董事会）
2. 激励要长（挂钩长期、经审计的业绩指标）
3. 造假要罚（追回条款——Clawback：发现造假就把高管奖金要回来）
4. 监管要严（尤其是海外上市企业）

## 九、胡萝卜加大棒：如何让管理层替你干活

两条路：

**胡萝卜——薪酬激励**：不发工资发股票，把管理层也变成股东。但瑞幸告诉你，胡萝卜太大也会噎死人——激励强度要适度、指标要多元、内控要跟上。

**大棒——并购威胁**：股价跌到低于公司真实价值 → 就会有"野蛮人"来收购 → 收购完第一件事就是换管理层 → 这种恐惧本身就在逼管理层好好干。

> 公司治理 = 内部胡萝卜 + 外部大棒 + 董事会监督。三条腿少一条就瘸。

## 十、概念对照表

| 概念对 | 区别 |
|--------|------|
| 资本预算 vs 资本结构 | 资本预算 = 钱怎么花（资产负债表左侧），资本结构 = 钱从哪来（右侧） |
| 现金流 vs 会计利润 | 现金流 = 实际收到/支付的现金；会计利润含非现金项目（折旧、应收） |
| 股东财富最大化 vs 利润最大化 | 财富最大化考虑风险和时间价值；利润最大化是单期会计概念 |
| 代理关系 vs 代理问题 | 代理关系 = 所有权与控制权分离（中性制度）；代理问题 = 分离导致的利益冲突 |
| 代理成本 vs 代理问题 | 代理问题是现象（利益不一致）；代理成本是后果（价值损失） |
| 薪酬激励 vs 公司控制权 | 薪酬激励 = 内部治理（胡萝卜）；控制权市场 = 外部治理（大棒） |
| 普通合伙 vs 有限合伙 | 普通合伙全无限责任；有限合伙有普通合伙人（无限）+ 有限合伙人（有限） |
| 公司 vs 合伙企业 | 公司：有限责任 + 永续存续 + 双重税；合伙企业：无限责任 + 有限存续 + 单层税 |
| 个人独资 vs 普通合伙 | 独资 = 1人；普通合伙 = 2+人，均无限责任 |

## 十一、最重要的故事 / 考试最易考点

**🍎 Apple 案例（为什么有钱偏要借钱？）**：资本结构决策 + 税务约束 + 股东压力。几乎每次考试都会出现。

**☕ 瑞幸咖啡（代理问题的教科书级案例）**：股权激励设计不当 → 造假 → 退市。记住四个改进方向（审计、长期激励、追回条款、监管）。

**💰 三大决策问题**：资本预算、资本结构、净营运资本——分别对应资产负债表左侧、右侧、上方。这是整门课的地图，后续每一章都是在回答这三个问题的某一个。
