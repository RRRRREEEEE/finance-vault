# 管理层薪酬激励 (Managerial Compensation)

## 📌 定义
管理层薪酬激励（Managerial Compensation）是通过将管理层报酬与股东利益绑定的薪酬设计，来缓解代理问题（mitigate the agency problem）的治理机制。激励机制需审慎设计（structured carefully），确保实现预期目标而非适得其反。

> 💬 通俗理解：给管理层发股票而不是发工资——让他们自己也变成"股东"，自然就会替股东着想。但设计不好反而会催生造假（如瑞幸）。

## 🔗 前置知识
- [[Agency Problem 代理问题]] — 薪酬激励是缓解代理问题的主要手段
- [[Agency Cost 代理成本]] — 激励的目标是降低代理成本
- [[Shareholder Wealth Maximization 股东财富最大化]] — 激励应使管理层目标与股东目标一致

## 🧠 机制
### 激励的基本原理
通过将管理层薪酬与**股东价值指标**挂钩，使代理人利益与委托人利益趋同。常见形式：股票期权、限制性股票、业绩奖金。

### 激励失效的风险（瑞幸咖啡教训）
瑞幸管理层持有大量与**营收增长和股价**挂钩的股权激励：
- 激励强度过大 → 管理层为达目标不惜造假
- 指标单一（营收增长）→ 忽略盈利质量和真实性
- 内控薄弱 → 造假行为未被及时发现

### 改进方向（课件给的四条方案）
1. 强化内部审计和独立董事会监督
2. 将薪酬与**长期、经审计的业绩指标**挂钩
3. 引入**追回条款 (Clawback Provisions)**——发现造假后追回高管奖金
4. 加强对海外上市企业的监管

## 🏛️ 应用
- 设计激励时需平衡：激励强度 vs 造假风险、短期指标 vs 长期价值、财务指标 vs 非财务指标
- 后章节详述：股东与管理层的代理问题是公司治理的核心议题

## 📖 相关术语
- [[Agency Problem 代理问题]]：激励试图缓解的问题
- [[Corporate Control 公司控制权]]：与薪酬激励互补的外部约束机制
- [[Corporation 公司]]：公司制下所有权与控制权分离的治理挑战

## ⚖️ 易混点对照
- 与 [[Corporate Control 公司控制权]] 的区别：薪酬激励 = 内部"胡萝卜"（给你钱让你好好干），公司控制权 = 外部"大棒"（不好好干就会被收购踢走）
