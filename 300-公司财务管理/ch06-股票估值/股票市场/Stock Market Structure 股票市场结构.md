# 股票市场结构 (Stock Market Structure)

## 📌 定义
股票市场是股票发行和交易的场所，分为**一级市场**（IPO/增发，公司融资）和**二级市场**（投资者之间交易），按组织形式分为集中交易所和场外交易市场（OTC）。

> 💬 通俗理解：一级市场 = 厂家直销，公司卖股票拿钱；二级市场 = 二手市场，投资者互相买卖，钱不经过公司。

## 🧠 机制

### 美国股票市场（三层架构）

| 层级 | 市场 | 特点 |
|------|------|------|
| 全国集中交易所 | **NYSE**（纽约证券交易所） | 全球最大、最古老、上市门槛最高；IBM、GE、阿里巴巴 |
| 全国OTC | **NASDAQ**（纳斯达克） | 无实体场所，计算机报价系统，做市商制，科技股集中地（Apple、Google、Facebook、百度、京东） |
| 地区交易所 | 太平洋、辛辛那提、费城、波士顿 | 规模较小 |

**NASDAQ 三级分层**：
1. **Global Select Market**（全球精选市场）：大公司
2. **Global Market**（全球市场）：中型公司
3. **Capital Market**（资本市场）：小公司

> 底层 OTC 市场还包括 **OTCBB**（场外柜台交易系统）和 **Pink Sheets**（粉单市场）。

### 英国股票市场

| 层级 | 市场 |
|------|------|
| 全国集中交易所 | **London Stock Exchange**（伦敦证券交易所） |
| ├─ 主板 (Main Board) | 大型企业 |
| └─ **AIM**（另类投资市场） | 中小企业、成长型企业 |
| 地区交易所 | 伯明翰、曼彻斯特、利物浦、格拉斯哥 |
| 全国OTC | OFEX (Off Exchange) |

### 香港股票市场

| 层级 | 市场 |
|------|------|
| 集中交易所 | **SEHK**（香港联交所） |
| ├─ 主板 (Main Board) | 大型成熟企业 |
| └─ **GEM**（创业板） | 成长型企业 |
| OTC | 场外市场 |

## 🏛️ 应用
- **NYSE vs NASDAQ**：NYSE 是竞价市场（auction market），NASDAQ 是做市商市场（dealer market）
- 科技公司偏好 NASDAQ 上市，传统制造业偏好 NYSE
- 香港是内地企业海外上市的首选地之一
