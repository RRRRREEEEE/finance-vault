# 公司财务管理 · ch06 股票估值

> 77 页 PDF | 2026-06-30 完成 | 13 篇笔记

## 章节概要

本章从巴菲特买入 Coca-Cola 的案例出发，系统讲授股票估值的方法体系：先介绍全球股票市场结构与中国多层次资本市场，再区分普通股与优先股的特征，然后从通用股利折现模型（DDM）出发，依次推导零增长、固定增长和变动增长三种估值模型，进一步讲解模型参数 $g$ 和 $R$ 的估计方法、总支付（含回购）估值、增长机会净现值（NPVGO）、零股利公司的估值逻辑，最后深入分析市盈率的决定因素。

## 关键公式速查

| 模型 | 公式 |
|------|------|
| 通用 DDM | $P_0 = \sum_{t=1}^{\infty} \frac{Div_t}{(1+R)^t}$ |
| 零增长 | $P_0 = Div / R$ |
| 固定增长 | $P_0 = Div_1 / (R-g)$ |
| 变动增长 | $P_0 = P_A(\text{年金}) + P_B(\text{永续年金折现})$ |
| 增长率 | $g = b \times ROE$ |
| 必要回报率 | $R = Div_1/P_0 + g$ |
| NPVGO 分解 | $P_0 = EPS/R + NPVGO$ |
| P/E 分解 | $P/E = (1-b)(1+b \times ROE)/(R-b \times ROE) = 1/R + NPVGO/EPS$ |
