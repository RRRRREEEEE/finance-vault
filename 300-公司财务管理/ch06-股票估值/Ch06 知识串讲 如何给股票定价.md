# Ch06 知识串讲：如何给股票定价

> 77 页 → 13 篇笔记 → 一条主线：把未来股利折现到今天

---

## 一、巴菲特为什么要买 Coca-Cola？

1988 年底，Coca-Cola 市值 163 亿美元。过去 7 年增长率 20%。巴菲特以均价 $11/股买入——他看到了什么？

答案是：**市场对 Coke 的定价低于其未来所有股利的现值之和**。这个判断的逻辑，就是本章的全部内容。

---

## 二、三层框架：从通用模型到具体计算

### 第一层：DDM 的逻辑基础

买股票就是买**未来股利的索取权**。你持有一年后卖出——卖出价 $P_1$ 本身也等于未来股利的现值。无限迭代后得出：

$$P_0 = \frac{Div_1}{1+R} + \frac{Div_2}{(1+R)^2} + \cdots$$

> **股票的价值只取决于它未来能分多少钱**。卖出价、资本利得——最终都是未来股利的反映。

### 第二层：三个特例，由简到繁

| 模型 | 假设 | 公式 | 比喻 |
|------|------|------|------|
| **零增长** | 股利永远不变 | $P = Div/R$ | 一张永不给利息的"固定工资"支票——用年金公式定价 |
| **固定增长** | 股利每年增长 $g\%$ | $P = Div_1/(R-g)$ | Coca-Cola 般的永续增长机器——Gordon 模型 |
| **变动增长** | 前 $T$ 年高增长，之后稳定 | 两段折现 | 初创公司 → 成熟公司——最贴近现实 |

三者的关系：零增长 ⊂ 固定增长（$g=0$） ⊂ 变动增长（$g_1=g_2$）。

> 变动增长模型的计算要点：前半段 = 增长年金，后半段 = 增长永续年金折现到 $T$ 时刻再折回今天。两步折现，不要忘了第二步。

### 第三层：$g$ 和 $R$ 从哪来？

模型用起来的前提是知道 $g$ 和 $R$。

**$g$ 的来源**：公司挣了钱，留一部分再投资（留存率 $b$），再投资能赚 $ROE$：
$$g = b \times ROE$$

- $b = 0$（全部分红） → $g = 0$ → 零增长模型
- $b = 70\%, ROE = 12\%$ → $g = 8.4\%$

**$R$ 的来源**：从股价反推：
$$R = \frac{Div_1}{P_0} + g$$

- $Div_1/P_0$ = 股利收益率（分红的"利息"）
- $g$ = 资本利得收益率（股价上涨的"差价"）

---

## 四、NPVGO：增长的真正引擎

DDM 只告诉你 $g$ 大了股价高，但 **$g$ 大不一定是好事**。

把股价拆开：
$$P_0 = \underbrace{\frac{EPS}{R}}_{\text{吃老本}} + \underbrace{NPVGO}_{\text{增长机会}}$$

- 如果公司留存利润（$b > 0$）但 ROE < R → NPVGO < 0 → **留存反而毁掉价值**
- 只有 ROE > R 时，留存越多，股价才越高

> 核心洞见：**不是所有的增长都创造价值**。只有投入资本回报率超过资本成本的增长才是好增长。

---

## 五、P/E 不是"高就贵"

P/E 可以从基本面推导：
$$\frac{P}{E} = \frac{(1-b)(1+b \times ROE)}{R - b \times ROE}$$

三个因素决定 P/E 的高低：
1. **ROE vs R**：ROE 越高于 R → P/E 越大（留存创造价值）
2. **$R$ 的高低**：市场利率低 → P/E 整体高（$1/R$ 变大）
3. **NPVGO 的大小**：$P/E = 1/R + NPVGO/EPS$，增长机会多 → P/E 高

| P/E = 30 | 可能因为 | 也可能因为 |
|----------|----------|------------|
| ROE = 25%, R = 10% | 公司确实很会赚钱 | 不是泡沫 |
| ROE = 5%, R = 10% | 才不正常 | 确实高估 |

> 投资启示：**不要只看 P/E 高低，要看 P/E 背后的 ROE 和 $R$**。

---

## 六、DDM 的三个盲区

1. **零股利公司**（Google、早期 Amazon）：DDM 无法直接估值 → 公司价值全部来自投资者对未来 NPVGO 的信心
2. **股票回购**：只看股利会低估一半——应当用"股利 + 回购"的总支付代入公式
3. **$g \ge R$**：公式爆炸 → $g > R$ 只能在短期维持，不能代入永续公式

## 七、本章公式网络

```
通用 DDM: P = Σ Div_t/(1+R)^t
   │
   ├── 零增长: P = Div/R  ──────── 永续年金
   ├── 固定增长: P = D1/(R-g) ──── 增长永续年金 = Gordon Model
   └── 变动增长: 两阶段 ───────── 年金 + 永续折现
        │
        ├── g 的估计: g = b × ROE
        ├── R 的估计: R = D1/P0 + g
        │
        └── 股价拆解: P = EPS/R + NPVGO
              │
              └── P/E 分解: P/E = 1/R + NPVGO/EPS
```

---

## 八、关键公式卡片

| 公式 | 一句话 |
|------|--------|
| $P_0 = \frac{Div_1}{R-g}$ | 固定增长的股票值多少钱 |
| $g = b \times ROE$ | 增长不来自于魔法，来自留存 × 回报率 |
| $R = \frac{Div_1}{P_0} + g$ | 你的总回报 = 分红 + 涨价 |
| $P_0 = \frac{EPS}{R} + NPVGO$ | 股价 = 吃老本 + 增长想象力 |
| $\frac{P}{E} = \frac{1-b}{R-b \times ROE}$ | P/E 的"分子分母"都是基本面决定的 |

> **一句总结**：股利折现模型是一棵大树——根是"未来股利决定价值"，干是 DDM，枝是三种特例，叶是 $g$ 和 $R$ 的估计，果实是 P/E 和 NPVGO 的投资判断。
