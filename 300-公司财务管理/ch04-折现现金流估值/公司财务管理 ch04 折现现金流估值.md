## 本章知识点一览

| 页码 | 知识点 | 一句话概括 |
|------|--------|-----------|
| 3-7 | [[Time Value of Money 货币时间价值]] | 今天的一块钱 > 未来的一块钱；四变量(FV/PV/T/r)互求 |
| 8-25 | [[Future Value 终值]] | FV = PV × (1+r)^T；复利的指数增长效应 |
| 11-29 | [[Present Value 现值]]（扩展） | PV = FV/(1+r)^T；多期现金流的折现求和 |
| 13-18 | [[Net Present Value 净现值]] | NPV = PV − Cost；NPV > 0 接受项目——最核心决策准则 |
| 33 | [[Discount Rate 折现率]] | 资金的机会成本 = 无风险利率 + 风险溢价 |
| 19-26 | [[Compounding 复利]] | 利滚利；复利频率越高 FV 越大 |
| 38-48 | [[Effective Annual Rate 有效年利率]] | EAR = (1+APR/m)^m−1；真实年回报率 |
| 45-48 | [[APR 年百分比率]] | APR 是名义报价利率，EAR 是实际利率 |
| 49 | [[Continuous Compounding 连续复利]] | FV = PV × e^(rT)；m→∞ 的极限 |
| 51-57 | [[Annuity 年金]] | 有限期等额现金流；PV = C × [1−(1+r)^(−T)]/r |
| 58-60 | [[Growing Annuity 增长年金]] | 现金流以固定增长率 g 递增的有限期序列 |
| 61-63 | [[Perpetuity 永续年金]] | PV = C/r；永远支付等额现金流的资产 |
| 64-66 | [[Growing Perpetuity 增长永续年金]] | PV = C₁/(r−g)；戈登增长模型的数学基础 |
| 68-69 | [[Pure Discount Loan 纯折价贷款]] | T-bill/零息债券——期初折价买入，期末面值收回 |
| 70 | [[Interest-Only Loan 纯利息贷款]] | 每期付利息+期末还本金；公司债券标准结构 |
| 71-76 | [[Amortized Loan 分期偿还贷款]] | 等额本息还款；每期 C 用年金公式计算 |
