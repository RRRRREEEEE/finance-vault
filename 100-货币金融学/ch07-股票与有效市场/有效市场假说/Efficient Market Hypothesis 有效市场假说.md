# 有效市场假说 (Efficient Market Hypothesis)

## 📌 定义
有效市场假说（EMH）是理性预期在金融市场的应用：金融市场中，证券价格充分反映所有可得信息。股价遵循随机游走。

> 💬 通俗理解：市场价格已经把所有人知道的信息都算进去了——所以你看到新闻再买股票已经晚了。小额投资者的最佳策略：买指数放着不动。

## 🧠 机制

### EMH 的核心

$$ R^{of} = R^* $$

证券收益的最优预测（$R^{of}$）= 均衡收益（$R^*$）

所有未被利用的利润机会都会被套利消除。

### 支持 EMH 的证据

| 证据 | 含义 |
|------|------|
| 投资顾问/共同基金无法持续跑赢大盘 | 过去表现好≠未来表现好 |
| 已公开信息不引起股价变动 | 市场已经定价了 |
| 股价遵循**随机游走** | 未来价格变动不可预测 |
| 技术分析无效 | K线图无法预测未来 |

### 反对 EMH 的证据（市场异象）

| 异象 | 描述 |
|------|------|
| **小公司效应** | 小盘股历史回报高于大盘股 |
| **一月效应** | 一月份股票回报异常高 |
| **市场过度反应** | 过度反应后均值回归 |
| **过度波动** | 股价波动 > 基本面波动 |

### 行为金融学（Behavioral Finance）

新分支，研究心理因素如何影响投资行为：
- 并非所有投资者完全理性
- 套利有限制（limits to arbitrage）
- 市场基本面外因素也影响价格

**黑色星期一（1987）**和**科技泡沫（2000）**很难用基本面解释——可能是行为因素。

### 投资策略含义

- 对大多数投资者：**买入并持有（Buy and Hold）**
- 热门消息通常已反映在价格中
- 只有**新的、意外的**信息才引起股价变动
