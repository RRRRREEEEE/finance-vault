# 必要回报率 (Required Rate of Return)

## 📌 定义
必要回报率($k_e$)是投资者对投资所要求的最低预期回报率，反映投资的风险水平和替代投资机会的回报。在股票估值中，它是未来现金流的贴现率。

## 🔗 前置知识
[[Stock Valuation 股票估值]]、[[Gordon Growth Model 戈登增长模型]]

## 🧠 机制
- $k_e$ 是戈登增长模型的分母关键变量
- 影响因素：无风险利率 + 风险溢价
- 美联储降息 → 债券收益率下降 → 投资者接受更低的 $k_e$ → 股价上升
- 风险认知上升 → $k_e$ 上升 → 股价下跌
