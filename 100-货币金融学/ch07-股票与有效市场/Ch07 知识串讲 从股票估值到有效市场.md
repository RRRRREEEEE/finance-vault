# 第七章 知识串讲：从股票估值到有效市场

## 一、股票是什么

你买入一股股票，意味着你成为公司的**股东**（Stockholder），是公司的所有者之一。

作为股东，你拥有两项核心权利：
1. **投票权**：参与重大决策，选管理层
2. **剩余索取权**（Residual Claimant）：公司赚了钱，先还债、先交税、先发工资——剩下的才是你的

"剩余"这两个字很关键。它解释了为什么股票比债券风险大：债券的利息和本金是合同约定的，公司在赚钱之前就得付；而股东的股利，是扣完所有固定支出之后，有剩余才有，没剩余就没有。

这也意味着：**股票的未来现金流天生不确定**。债券你能知道每年拿多少利息、到期拿多少本金；股票的股利取决于公司业绩，业绩取决于经济环境、行业竞争、管理层决策……这一系列不确定因素。

那么问题来了：一个未来现金流这么不确定的东西，你凭什么给它定价？

---

## 二、如何给一只股票定价

### 2.1 所有估值模型的共同起点：现值

回答这个问题的核心工具，早在第四章就准备好了：**现值**（Present Value）。

任何资产的价值 = 它未来所有现金流的现值之和。这是金融学里最核心的定价原理。债券这么做，股票也这么做。

### 2.2 一期估值：最简单的起步

假设你打算持有一年。这一年里你会收到一笔股利 $Div_1$，年末把股票以 $P_1$ 卖掉。你要求这笔投资的回报率是 $k_e$（**必要回报率**）。

那么你今天愿意出的最高价 $P_0$ 是多少？

$$P_0 = \frac{Div_1 + P_1}{1 + k_e}$$

这本质上就是现值公式：把一年后的全部收入（股利 + 卖出价）折现回来。

### 2.3 广义股利估值模型：推到极致

但问题是：$P_1$ 又是谁定的？一年后买你股票的人，他愿意出多少钱？他也会用同样的逻辑——看第二年的股利 $Div_2$ 和卖出价 $P_2$。

一路推下去，推到无穷远。最终你会发现：**股票的最终售价根本不重要**——因为那个售价本身也是由未来更远的股利决定的，一路递归下去，最终售价被推到了无穷远处。

这就是**广义股利估值模型**（Generalized Dividend Model）：

$$P_0 = \sum_{t=1}^{\infty} \frac{Div_t}{(1 + k_e)^t}$$

一句话：**股票的价值 = 未来全部股利的现值总和。**

这个模型逻辑完美，但它有个致命问题：我怎么可能预测未来无穷多期的股利？

### 2.4 戈登增长模型：一个实用的简化

与其预测无穷多期每一期的股利，不如做一个大胆但简洁的假设：**股利每年以恒定速率 $g$ 永续增长。**

在这个假设下，无穷级数可以**化简**为一个极简公式——**戈登增长模型**（Gordon Growth Model）：

$$P_0 = \frac{Div_1}{k_e - g}$$

$Div_1$ 是下一期的预期股利，$k_e$ 是必要回报率，$g$ 是股利增长率。要求 $k_e > g$，否则股价爆炸到无穷大——这在现实中不可能。

这个公式极其有力，因为它把股价的决定因素浓缩成了**两个变量**：

| 变量 | 怎么影响股价 | 为什么 |
|------|------------|--------|
| $k_e$ ↑ | 股价 ↓ | 分母变大，折现更狠 |
| $g$ ↑ | 股价 ↑ | 分母变小，未来股利增长更快 |

### 2.5 货币政策如何影响股价：戈登模型的威力

有了戈登模型，美联储的一举一动怎么传导到股市就一目了然了：

**降息 → 两条路径同时推高股价：**
1. $k_e$ ↓：债券收益率下降，投资者对股票的替代回报要求降低 → 分母缩小 → $P_0$ ↑
2. $g$ ↑：低利率刺激经济，公司盈利前景改善 → 股利增长预期上升 → 分母进一步缩小 → $P_0$ ↑

**加息 → 两条路径反向压低股价。**

2007-2008 次贷危机就是反面的典型案例：经济崩溃，预期 $g$ 暴跌；风险恐慌，$k_e$ 飙升。分母急剧扩大，分子急剧萎缩——道琼斯遭遇 50 年来最严重的熊市。

2020 年 3 月新冠冲击，道琼斯从 29,551 跌到 18,561（-37%），逻辑完全一致。

## 三、但是——预期从哪来？

戈登模型很漂亮，但它有一个潜台词：你必须知道 $Div_1$（也就是下一年的股利）和 $g$（未来永续增长率）。

但这两样东西，你**现在**无法确切知道。你只能**预期**。

于是问题从"股价等于多少"变成了"预期是怎么形成的"。

### 3.1 三种预期形成方式

历史上，人们对预期形成有三种理解：

| 方式 | 怎么形成预期 | 问题 |
|------|------------|------|
| **静态预期** | 明天 = 今天 | 最原始，完全忽视变化 |
| **适应性预期** | 只看过去，根据历史误差慢慢修正 | 数据变化时调整太慢，被新古典宏观经济学批判缺乏微观基础 |
| **理性预期** | 使用所有可得信息，做出最优预测 | 获取全部信息成本可能很高 |

### 3.2 理性预期理论的三个含义

**理性预期理论**（Theory of Rational Expectations），John Muth 在 1961 年提出。核心命题：

> **预期 = 使用所有可得信息做出的最优预测**

它有三个关键含义：

**第一，预测误差平均为零，而且无法提前预测。**

你在做预测的时候，已经把所有已知信息都用上了。剩下的误差，是那些你无法预测的、真正随机的新信息造成的。所以误差本身也是随机的——平均来看是零，而且你不可能"提前知道你会错多少"（如果提前知道，你早把它修正进预测了）。

**第二，变量的运行方式变了，预期的形成方式也必须跟着变。**

这是理性预期最锋利的地方。适应性预期只是机械地看历史——过去是什么趋势，我继续这个趋势。但如果政策变了、结构变了（比如美联储把通胀目标从2%改成了3%），继续用历史数据做预测就会系统性出错。理性预期要求你消化新的结构信息，调整你的预测方式。

**第三，金融市场里理性预期的激励最强。**

预测天气不准，顶多淋场雨。预测股价不准，你会亏钱。预测得更准的人赚得更多，所以金融市场的参与者有最强的动机去搜集信息、理性预测。

这就引出了一个自然的问题——

### 3.3 如果金融市场里人人都在做理性预测，那市场会变成什么样？

让我们从一个具体的场景开始想。

假设某只股票当前的交易价格是 50 元。你搜集了一切能找到的信息——公司财报、行业前景、货币政策走向——用戈登模型算出来，这只股票的内在价值应该是 60 元。

也就是说，当前市价 50 元 **低于** 你的最优预测值 60 元。这意味着什么？你花 50 块买了一件值 60 块的东西。金融学里管这种情况叫**未被利用的盈利机会**（Unexploited Profit Opportunity）。

你会怎么做？立刻买入。不光是你——任何做了同样分析的人都会买入。

众人一起买入 → 需求暴增 → 股价被推高 → 从 50 向 60 靠拢。当股价涨到 60 的时候，盈利机会消失了——你再也买不到"50 块买 60 块的东西"了。

**这个过程就是套利**（Arbitrage）——利用价格低于价值的偏差，买入获利，而这个行为本身就把偏差消灭了。

现在把逻辑反过来：如果股价是 70 元，高于 60 元的内在价值呢？理性投资者会卖出（或者做空），把价格压回 60。

结论：在一个所有人都在做理性预测的市场里，**任何价格偏差都是转瞬即逝的**。套利者会让价格始终卡在"反映了全部可得信息"的水平上。没有免费的午餐——你不可能系统性地找到被错误定价的股票。

## 四、这个状态有一个名字：有效市场假说

上面用逻辑推出来的这个结论——价格充分反映全部可得信息——就是**有效市场假说**（Efficient Market Hypothesis）。

这个名字拆开看：
- **有效**（Efficient）：不是指"运行快速"，而是指**信息利用得充分**——没有任何可用的信息被浪费、被忽视
- **市场**：特指金融市场，尤其是股票市场
- **假说**：它是一个理论上推出来的命题，不是绝对真理。有支持它的证据，也有挑战它的反例（后面会讲）

用最简洁的一句话概括：

> **有效市场假说 = 证券的价格已经把所有公开可得的信息都算进去了。**

用公式表达：

$$R^{of} = R^*$$

左边 $R^{of}$ 是你用全部可得信息算出的最优预测收益率。右边 $R^*$ 是市场的均衡收益率。有效市场要求二者相等——如果不等，套利行为就会立刻把价格推到相等为止。

### 4.2 随机游走：有效市场的直接推论

如果价格已经充分反映了所有可得信息，那么未来价格的变化只能来自**新信息**。

而新信息的本质是什么？**不可预测。** 如果你能预测明天的头条新闻，那它就不是"新"闻了。

既然新信息不可预测 → 由新信息驱动的价格变化也不可预测 → 股价遵循**随机游走**（Random Walk）。

很多人对"随机游走"感到不舒服——这难道不是说市场是疯狂的、非理性的吗？

恰好相反。随机游走恰恰是**市场理性**的标志。因为理性投资者已经把一切已知信息都定价了，剩下的价格波动只反映了真正不可预测的新信息。如果你发现股价有可预测的规律（比如"每年11月最后一周都涨3%"），那恰恰说明市场没有充分定价——存在未被利用的盈利机会。

### 4.3 有效市场假说的三层形式（考研高频考点）

有效市场假说按"可得信息"的范围分三个层次：

| 层次        | 价格反映了什么信息             | 什么分析失效    |
| --------- | --------------------- | --------- |
| **弱式有效**  | 所有历史价格信息              | 技术分析无效    |
| **半强式有效** | 所有公开信息（历史 + 财报 + 新闻等） | 基本面分析也无效  |
| **强式有效**  | 所有信息（包括内幕和私人信息）       | 内幕交易也无法获利 |

考研中反复考的就是这个区分。借壳上市的新闻公布后股价涨停、随后回落——这恰恰说明公布那一瞬间市场没有完全吸收新信息，所以**没有达到半强式有效**。

### 4.4 证据：支持和反对

**支持有效市场假说的证据：**
- 投资顾问和共同基金**无法持续跑赢大盘**。今年表现好的基金，不代表明年也好
- 已公开信息不会引起股价大幅变动——市场已经把预期定价了
- 技术分析（K线图、形态分析）无法系统性地预测股价
- 股价确实表现出随机游走的特征

**反对有效市场假说的证据（市场异象）：**
- **小公司效应**：历史上小盘股的回报率高于大盘股
- **一月效应**：一月份股票回报异常高（可能与税损卖出有关）
- **市场过度反应**：股价对新闻过度反应后均值回归
- **过度波动**：股价的实际波动大于基本面能解释的范围

这些异象对有效市场假说构成了有意义的挑战，但它们自身也不稳定——一旦被发现、被广泛知晓，套利者就会利用它们，从而削弱甚至消除它们。

### 4.5 对投资者的实际含义

如果你相信有效市场假说，你的投资策略就很简单：**买入并持有**（Buy and Hold）。

- 热门消息、新闻头条——这些在你看的时候已经反映在股价里了
- 技术分析？K 线？——历史信息早已被充分消化
- 只有**新的、意外的**信息才会引起股价变动——而你不能系统性地预测意外

对小额投资者而言，买低成本的指数基金，长期持有，就是最理性的选择。

## 五、行为金融学：来自人性深处的挑战

### 5.1 有效市场假说的两根支柱，分别在现实中出了问题

有效市场假说有两根支柱：
1. **投资者理性**：投资者用所有可得信息做出最优预测
2. **套利无摩擦**：如果价格偏离了价值，套利者会立刻纠正

**行为金融学**（Behavioral Finance）直接攻击这两根支柱。

### 5.2 投资者并不完全理性

行为金融学发现了大量系统性偏离理性的证据：

- **损失厌恶**（Loss Aversion）：人们对亏损的痛苦远大于同等金额收益的快乐。避害大于趋利
- **过度自信**（Overconfidence）：人们高估自己判断的准确性——可以解释为什么交易量那么高（如果大家都理性，谁会在对面做反向交易？）
- **社会传染**（Social Contagion）：周围人都在买，我也买——羊群效应，催生泡沫
- **禀赋效应**（Endowment Effect）：一旦拥有某样东西，就会更高地评估它的价值
- **心理账户**（Mental Accounting）：在心理上把钱分到不同"账户"里分别对待——虽然钱就是钱

### 5.3 套利并不总是有效的

有效市场假说认为只要价格偏离价值，理性套利者就会纠正。但现实中有**套利限制**（Limits to Arbitrage）：

最关键的例子就是**卖空**（Short Sales）。如果某只股票价格明显过高，理性做法是做空它——先借来卖掉，等价格跌了再买回来还。

但在实践中：
- 卖空可能产生**无限损失**（股价可以一直涨，亏损没有上限，而做多的最大损失只是本金）
- 卖空在文化中常被视为"不道德"（赌别人失败）
- 很多机构投资者被禁止卖空

结果？即使你清楚地知道一只股票被高估了，你也不敢轻易做空。高估可以持续很久——**价格可以长时间偏离价值。**

### 5.4 泡沫：两种理论的正面交锋

**泡沫**（Bubble）是行为金融学和有效市场假说交锋的正面战场。

资产泡沫：价格相对于**市场基本面**（Market Fundamentals）决定的合理价格出现非平稳性向上偏离。

历史上的重大事件都很难用纯粹的有效市场假说来解释：
- 1987 年**黑色星期一**：一天跌 20%+，基本面一天内没有变那么多
- 2000 年**科技泡沫**：纳斯达克涨到完全不合理的水平然后崩盘
- 2008 年**次贷危机**前的房地产泡沫

从戈登模型的视角看泡沫：当市场参与者集体认为 $g$（股利增长）会迅猛上升，或 $k_e$（要求回报率）会大幅下降时，模型给出的 $P_0$ 会急剧膨胀。央行可以通过紧缩货币政策提高利率 → $k_e$ 上升 + $g$ 预期下降 → 刺破泡沫。

但有效市场假说的辩护者也有说法：事后看起来是泡沫，但当时技术进步如此之快，没人能确定科技公司的真正价值。市场参与者**确实在用他们当时能获得的所有信息做最优预测**——只是事后证明那些预测错了。

## 六、把整章串起来：一个完整的知识链条

现在我们可以从起点走到终点，画出一条完整的逻辑线：

```
1. 股票是什么？
   → 普通股 = 投票权 + 剩余索取权

2. 怎么给股票估值？
   → 一期模型 → 广义股利估值模型（P₀ = 未来所有股利现值）
   → 戈登增长模型（P₀ = Div₁/(kₑ - g)）

3. 戈登模型需要什么输入？
   → Div₁、kₑ、g —— 这三样都需要预期

4. 预期怎么形成？
   → 静态/适应性/理性预期
   → 理性预期 = 使用所有可得信息的最优预测

5. 理性预期放到金融市场会发生什么？
   → 任何盈利机会都会被套利消灭
   → 价格反映全部可得信息 = 有效市场假说

6. 有效市场的直接推论是什么？
   → 股价随机游走（新信息不可预测 → 价格变化不可预测）
   → 技术分析无效
   → 最佳策略：买入并持有

7. 有效市场完美吗？
   → 不完美：小公司效应、一月效应、过度波动等异象
   → 更深层的挑战：投资者并不总是理性的

8. 行为金融学补充了什么？
   → 损失厌恶、过度自信、社会传染 → 催生泡沫
   → 卖空受限 → 套利无效 → 价格可长期偏离价值
   → 市场有时疯狂，且疯狂可以持续
```

这一章本质上是在回答三个层层递进的问题：

1. **股票怎么定价？**（戈登模型——数学工具）
2. **市场定价有效吗？**（有效市场假说——理论判断）
3. **如果市场不总是有效的，为什么？**（行为金融学——人性解释）

三个问题，三种视角，合在一起就是我们对股票市场的完整认知框架。

## 七、三个最重要的计算（考试必备）

### 7.1 戈登增长模型基本计算

A 公司刚支付 2 元股利，股价 100 元，增长率 6%，求资本成本。

$$k_e = \frac{Div_1}{P_0} + g = \frac{2 \times 1.06}{100} + 6\% = 2.12\% + 6\% = 8.12\%$$

### 7.2 不变增长模型求未来价格

预期股利 2.12 元，年增长 10%，必要回报率 14.2%。

$$P_0 = \frac{2.12}{0.142 - 0.10} = 50.48$$

下一年价格（股利增长10%后）：
$$P_1 = \frac{2.12 \times 1.10}{0.142 - 0.10} = 55.52$$

注意：$P_1 / P_0 = 55.52 / 50.48 = 1.10$——股价也以 $g=10\%$ 增长，和股利同步。

### 7.3 零增长模型（永续年金）

公司年净利润全作股利发放，每股 4 元，折现率 10%。

$$P_0 = \frac{4}{10\%} = 40 \text{元}$$

这是戈登模型在 $g=0$ 时的特例：$P_0 = Div / k_e$，也就是永续年金公式。

## 八、概念对照表（易混点）

| 概念对 | 区别 |
|--------|------|
| **理性预期 vs 适应性预期** | 理性预期使用**全部可得信息**（包括对政策变化的理解）；适应性预期**只看历史**，反应迟钝 |
| **最优预测 vs 理性预期** | 理性预期是**最优预测的一种**——充分利用全部信息后的最优预测 |
| **弱式有效 vs 半强式有效** | 弱式：历史信息已定价；半强式：**所有公开信息**已定价。考研最爱混：说"弱式反映所有公开信息"是错的 |
| **套利 vs 未被利用的盈利机会** | 盈利机会是**现象**（价格低于最优预测）；套利是**行为**（利用这个机会获利） |
| **风险溢价 vs 必要回报率（$k_e$）** | $k_e$ = 无风险利率 + 风险溢价。$k_e$ 是总要求回报率，风险溢价只是其中超过无风险利率的那部分 |
| **市场基本面 vs 内在价值** | 基本面是影响现金流的**因素**（经济、行业、公司）；内在价值是这些因素**决定的理论价格**。泡沫 = 市场价偏离基本面决定的合理价 |
