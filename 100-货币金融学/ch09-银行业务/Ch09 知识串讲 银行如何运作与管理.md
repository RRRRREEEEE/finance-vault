# 第九章 知识串讲：银行如何运作与管理

## 一、先看一张银行的资产负债表

你要管理一家银行，首先得知道银行长什么样。

银行资产负债表和普通公司的一样：左边是资产（钱用到了哪里），右边是负债和资本（钱从哪里来）。恒等式永远成立：

$$\text{资产} = \text{负债} + \text{银行资本}$$

### 负债端：钱从哪来

| 来源        | 是什么                                  | 特点                |
| --------- | ------------------------------------ | ----------------- |
| **支票存款**  | 活期存款、NOW账户、货币市场存款账户（MMDA）            | 传统核心资金来源，可开支票，成本低 |
| **非交易存款** | 储蓄账户、定期存单（CDs）                       | 不能开支票，利率高于支票存款    |
| **借款**    | 从美联储借钱（贴现贷款）、从其他银行借（联邦基金）、从企业借（回购协议） | 主动获取，成本较高         |
| **银行资本**  | 总资产减总负债 = 股东的净值                      | 吸收损失的第一道缓冲        |

### 资产端：钱用到了哪

| 用途 | 是什么 | 特点 |
|------|------|------|
| **准备金** | 在美联储的存款 + 库存现金 | 法定要求，几乎不赚利息 |
| **证券** | 国债、市政债等 | 重要的盈利资产，流动性好 |
| **贷款** | 工商业贷款、按揭贷款、消费贷款 | 利润最高，但流动性最差、风险最大 |
| **其他资产** | 大楼、设备、系统 | 实物资本 |

### 资产负债表背后的商业模式

看一眼这个结构，银行的核心矛盾就暴露了：

> **负债端是短期的、随时可被提取的（存款人说取就取）。资产端是长期的、难以快速变现的（贷款要几年后才收回）。**

这个矛盾的学名叫**期限错配**（Maturity Mismatch）。银行不回避它——银行靠它赚钱。

## 二、核心商业模式：资产转换

银行的赚钱逻辑只有四个字：**借短贷长**。

用专业术语说，这叫**资产转换**（Asset Transformation）：出售具有一组特征（短期、低成本、高流动性）的负债，用所得资金购买具有另一组特征（长期、高收益、低流动性）的资产。

举个例子。你用 T-account 追踪一笔交易：

```
初始：银行吸收 $100 存款
   资产              负债+资本
准备金  +$100      支票存款  +$100

银行留下 $10 准备金（假设准备金率 10%），其余 $90 放贷：
   资产              负债+资本
准备金  +$10        支票存款  +$100
贷款    +$90
```

银行用 $100 的短期存款，创造了 $90 的长期贷款。存款利率假设 1%，贷款利率假设 6%，吃掉 5% 的利差——这就是银行的利润来源。

但问题来了：如果存款人突然要取钱，而你的 $90 已经以贷款的形式锁死在了外面……这就是银行管理的核心挑战。

## 三、四大管理原则

### 3.1 流动性管理：随时准备被挤兑

存款人什么时候来取钱，你不知道。但你知道的是：**法定准备金必须满足，短缺必须消除。**

假设某存款人提取了 $10：
```
提款后：
   资产              负债+资本
准备金   $0          支票存款  $90
贷款    $90
```

准备金从 $10 变成 $0，达不到法定准备金要求。银行有四种办法补上这 $10 的缺口：

| 方法 | 操作 | 成本 |
|------|------|------|
| **借款** | 从其他银行借（联邦基金市场） | 支付借款利息 |
| **出售证券** | 卖出持有的国债 | 经纪费和交易成本 |
| **找美联储** | 向美联储申请贴现贷款 | 支付贴现率利息 |
| **减少贷款** | 催客户还款，或把贷款卖给别人 | **成本最高**——惹恼客户，别的银行只愿大幅折价接盘 |

顺序是按成本从低到高排的。催客户还贷是最后一招，因为伤害的是银行的长期关系——你下次就别想再贷给他了。

**持有超额准备金的代价**：放弃了这些钱本来可以放贷赚到的利息。这就是持有超额准备金的机会成本。超额准备金本质上是银行买的保险——保险不便宜，但比没有保险出了事强。

### 3.2 资产管理：把钱投好

有了资金，怎么投？资产管理的目标是三角平衡：

1. **追求最高回报**：找到愿意付高利率且违约风险低的借款人
2. **降低风险**：通过多样化分散风险——不要把所有贷款都贷给一个行业
3. **保持充足流动性**：总有手头要留的现金和易变现的证券

买什么证券？买高收益且风险低的。贷给谁？贷给信用好且能接受高利率的。但这两件事天然冲突——高利率往往对应高风险——第八章的逆向选择讲过为什么。所以筛选借款人就是银行最核心的能力。

### 3.3 负债管理：1960 年代的思想革命

在 1960 年代之前，银行是被动经营的：等着储户来存款，有多少钱放多少贷。

1960 年代之后，银行的思路变了：**看到好的贷款机会，可以主动去筹钱。** 隔夜贷款市场（联邦基金）的扩张，加上可转让存单（Negotiable CDs）等新工具的发明，让银行可以在需要资金时去"买钱"，而不是等着储户来"存钱"。

这就是**负债管理**（Liability Management）的革命——银行从"资产管理"拓展到"资产负债联合管理"（ALM）。资产端和负债端不再是割裂的，而是一体两面。

这条变化的后果之一：**支票存款作为资金来源的重要性持续下降。** 银行不再只靠活期存款过日子了。

### 3.4 资本管理：最微妙的一个——赚钱和安全在拔河

**银行资本**（Bank Capital）= 总资产 - 总负债。它是银行的净值，也是吸收损失的护城河。

资本多一点好还是少一点好？这是一个经典的权衡。

**资本高的好处**：安全。资产贬值时，先亏的是股东的资本，不是储户的钱。资本越厚，银行越难破产。

**资本高的代价**：股东回报率被压低。看一个公式：

$$ROE = ROA \times EM$$

- $ROE$（股本回报率）：股东最关心的指标——我投的钱赚了多少
- $ROA$（资产回报率）：每单位资产创造多少利润——资产的赚钱效率
- $EM$（股本乘数，Equity Multiplier）：$EM = \frac{\text{总资产}}{\text{银行资本}}$，衡量杠杆程度

这个公式揭示了资本的残酷真相：

两家银行 ROA 完全一样（都是 1%），但只要资本结构不同，ROE 可以差很多：
- 银行 A：资本占资产 10%，$EM = 10$ → $ROE = 1\% \times 10 = 10\%$
- 银行 B：资本占资产 5%，$EM = 20$ → $ROE = 1\% \times 20 = 20\%$

银行 B 的杠杆是 A 的两倍，股东回报也是两倍。但杠杆高意味着资产只要跌 5%，资本就亏光了——破产。

**资本管理的三个手段**：发新股/回购股票、调股利、控制资产增速。

**次贷危机的教训**：银行持有 MBS（抵押贷款支持证券）→ 房价跌 → MBS 巨亏 → 银行资本被侵蚀 → 资本不够满足监管要求 → 无法发新股筹资（股价已经崩了）→ 只能缩减贷款 → **信贷紧缩**（Credit Crunch）→ 实体经济被拖下水。从银行的资本缺口到全经济的衰退，传导链非常直接。

## 四、信用风险管理：把第八章的工具搬进银行

银行的贷款业务天然面对信息不对称——逆向选择（借钱前分不出好人坏人）和道德风险（借了钱可能去赌博）。信用风险管理就是银行应对这两大问题的武器库。

### 工具一：筛选与监测

贷款之前下功夫调查（筛选），贷款之后盯住不放（监测）。专业化放贷——银行的信贷员不是通才，他们深入了解特定行业、特定区域的借款人。这本身就是一种信息生产。

### 工具二：长期客户关系

> 你在同一家银行借了十年钱，银行比你自己还清楚你什么时候会遇到麻烦。

长期关系是信息积累的容器。老客户的信息不对称程度远低于新客户，银行的筛选和监测成本也随之降低。

### 工具三：贷款承诺

银行事先和企业说好：未来一段时间内，你随时可以以市场利率提取不超过约定金额的贷款。企业获得了 **"需要时一定有"的安全感；银行则锁定了长期关系，方便收集信息。** 现代大部分工商业贷款都是在贷款承诺安排下发放的。

### 工具四：抵押品与补偿余额

**抵押品**你已经熟悉了——借款人违约，银行可以扣押资产，减少损失。

**补偿余额**（Compensating Balances）是银行独特的发明：借款企业必须在银行支票账户里保持一笔最低金额。这笔钱一石二鸟：①银行可以动用，增加可用资金；②银行可以监测企业的现金流——如果账户余额波动异常，马上知道有问题。

### 工具五：信贷配给——最反常识的一个

> 有人愿意付 20% 的年利率借钱，银行说不借。为什么？

因为提高利率本身会恶化借款人的质量。愿意接受 20% 利率的人，要么是**准备赌一把（道德风险），要么是自知极有可能违约（逆向选择）。低风险的借款人看到这么高的利率，自己就退出了。

所以银行在某一利率水平之上**拒绝放贷**，即便对方愿意付更高利率也一样。银行用数量（借不借、借多少）而非价格（利率）来分配信贷。这就是**信贷配给**（Credit Rationing）。

## 五、利率风险管理：当利率变动伤害银行

银行是借短贷长的——这个基本事实让它对利率变动格外敏感。

### 5.1 缺口分析

把银行的资产和负债分成两类：利率敏感的（短期内利率会重置的）和利率不敏感的。

$$\text{缺口（GAP）} = \text{利率敏感资产（RSA）} - \text{利率敏感负债（RSL）}$$

$$\Delta \text{利润} = GAP \times \Delta i$$

场景推演：

- **GAP > 0**（利率敏感资产多于利率敏感负债）→ 利率上升 → 资产收益增加 > 负债成本增加 → **利润增加**
- **GAP < 0**（利率敏感负债多于利率敏感资产）→ 利率上升 → 负债成本增加 > 资产收益增加 → **利润减少**

比如 GAP = -2000 万，利率上升 5%：利润变化 = -2000万 × 5% = **-100 万美元**。

银行可以通过调整资产和负债的利率敏感结构来管理这个缺口。**想要规避利率上升的风险？缩小负缺口——增加利率敏感资产，或减少利率敏感负债。**

### 5.2 久期分析——升级版

缺口分析只看"利率变不变"，不区分短期和长期资产负债对利率的敏感程度差异。久期分析弥补了这个缺陷：考虑每笔资产负债的久期（对利率的真实敏感性），加权汇总，看利率变动对银行**整体经济价值**（不只是当期利润）的影响。

金融工具的到期日越长、到期前支付的金额越少 →  久期越高 → 对利率越敏感 *。

## 六、表外活动：不写在资产负债表上，但风险是真的

银行越来越多地开展不在资产负债表上的业务——**表外活动**（Off-Balance-Sheet Activities）。

### 类型

| 类型         | 是什么                         | 风险                                            |
| ---------- | --------------------------- | --------------------------------------------- |
| **贷款销售**   | 把发放的贷款卖给其他机构                | 可能放松贷款标准（卖掉的贷款不用自己担风险）                        |
| **产生费用收入** | 为 MBS 提供管理服务、创建结构化投资工具（SIV） | SIV 在次贷危机中暴雷——银行在法律上不拥有 SIV，但当 SIV 出问题时银行被迫救助 |
| **交易与衍生品** | 金融期货、期权、利率互换、外汇投机           | 委托代理问题——交易员为奖金过度冒险（巴林银行倒闭）                    |

### 金融衍生品

**金融衍生品**（Financial Derivatives）是其收益"派生"自其他资产（标的）的金融工具。常见三类：

- **金融期货**：约定在未来按特定价格买卖的标准化合约。交易所交易，清算所居中——标准化带来了高流动性。
- **期权**（Option）：买或卖的**权利**而非义务。看涨期权 (Call) 给你买入权，看跌期权 (Put) 给你卖出权。买方付一笔期权费，亏损封顶（最多亏掉期权费），但收益不对称——期货锁住了风险也锁住了收益，期权只锁风险不锁收益。用看跌期权对冲就像买保险：花小钱，下行有保护，上行不封顶。
- **利率互换**（Interest Rate Swap）：双方交换基于同一名义本金的利息支付流，不交换本金，只结算利息差额。最经典的是固定利率换浮动利率（Plain Vanilla Swap）。银行的经典用法：放固定利率贷款但吸收浮动利率存款 → 怕利率涨 → 进入互换支付固定利率、收取浮动利率 → 利率涨了，互换中浮动收入跟着涨，弥补了存款成本的上升，让资产和负债的利率敏感特征更匹配。

衍生品可以**对冲风险**——在另一个市场建立反向头寸，让两个盈亏相互抵消——也可以**投机**赌方向。后者引发了委托代理问题：交易员赌赢了拿奖金，赌输了亏银行的钱。银行用三道防线管住这个危险：**风险敞口上限**（给每个交易员设硬天花板，超了就砍）、**VaR 在险价值**（用历史数据算正常波动下"一天 99% 概率最多亏多少"）、**压力测试**（假想利率飙 300 基点或股市崩 30% 这种极端灾难，看银行扛不扛得住）。三道防线互补——上限管纪律，VaR 管日常，压力测试管灾难。

（期权、互换、对冲、上限、VaR、压力测试的详细机制见独立笔记 [[Option 期权]] / [[Interest Rate Swap 利率互换]] / [[Hedging with Derivatives 衍生品对冲]] / [[Risk Exposure Limits 风险敞口上限]] / [[Value at Risk 在险价值]] / [[Stress Testing 压力测试]]）

次贷危机之后，美联储对所有大型银行做年度压力测试（CCAR/DFAST）——模拟极端经济衰退场景，看每家银行的资本充足率会不会掉到监管底线以下。结果是公开的——市场可以看到哪些银行扛得住，哪些悬。这既是审慎监管工具，也是市场纪律（支柱 3）的实现。

#### 三道防线的逻辑关系

```
交易员的每一笔衍生品头寸
        ↓
第一道：风险敞口上限 → 先划定边界，你不能赌多大
        ↓
第二道：VaR → 每天对正常波动下的潜在损失做量化，知道"日常可能亏多少"
        ↓
第三道：压力测试 → 假想极端场景，确保即使"不可能的事"发生了，银行也死不了
```

三者互补：上限管纪律，VaR 管日常，压力测试管灾难。缺一道都不算完整的风险控制。

## 七、银行行业正在经历什么

### 7.1 脱媒：钱不通过银行了

**脱媒**（Disintermediation）指资金绕过银行直接在供需双方之间流动。

推动力来自两边：
- **监管限制**：对存款利率的上限管制（如曾经的 Q 条例）让存款利率低于市场利率 → ** 储户把钱转到货币市场共同基金**
- **技术进步**：信息技术降低了其他金融机构的交易成本，削弱了银行的成本优势

结果：商业银行在金融中介总资产中的占比持续下降。

### 7.2 影子银行：银行之外还有人做着银行的事

**影子银行**（Shadow Banking）是在传统银行体系之外从事信用中介的机构——货币市场基金、对冲基金、结构化投资工具（SIV）等。

它们做的事情和银行类似（把钱从有闲钱的人导向需要钱的人），但不吸收传统存款，也不受同样的监管约束。次贷危机中，影子银行体系暴露了巨大的风险——它们是"跑得比监管快的银行"。

### 7.3 金融创新：猫鼠游戏

银行不断地创新，部分是为了应对新风险，部分是为了规避旧监管。几个例子：

- **可调整利率抵押贷款**（ARM）：利率随市场重置，把利率风险从银行转嫁给借款人。利率波动时代下的产物。
- **扫款账户**（Sweep Accounts）：日终自动将企业支票账户余额转入不受准备金要求的账户 → 规避准备金要求（"准备金相当于对存款征税"）→ 教科书级别的监管套利
- **证券化**（Securitization）：把缺乏流动性的贷款打包变成可交易的证券。银行获得了流动性，但也可能放松贷款标准——反正贷款会被卖掉。

```
  流动性管理  资产管理   负债管理   资本管理   风险管理
  （随时有钱  （把钱投好）（主动找钱）（安全vs赚钱）（信用+利率）
   应对提款）                              ↓
                                ┌───────┴───────┐
                                ↓               ↓
                           信用风险管理      利率风险管理
                           （5大工具）      （缺口+久期分析）
```

银行就是在期限错配上走钢丝。左边是存款人的钱（随时要还），右边是贷款人的债（几年后才回）。管理一家银行，就是在赚钱的冲动和安全的约束之间不停地做选择。

### 最核心的五个公式

| 公式 | 含义 |
|------|------|
| 资产 = 负债 + 银行资本 | 资产负债表的永恒约束 |
| ROE = ROA × EM | 利润（ROE）源于资产效率（ROA）和杠杆（EM）的双重放大 |
| GAP = RSA - RSL | 缺口衡量利率敏感资产和负债的错配 |
| Δ利润 = GAP × Δi | 利率变动一分，利润就变动缺口的倍数 |
| 准备金必须满足法定要求 | 短缺只有四条路：借钱、卖证券、找美联储、砍贷款 |

### 概念对照表

| 概念对 | 区别 |
|--------|------|
| **资产管理 vs 负债管理** | 资产端管"钱投哪"；负债端管"钱从哪来"。1960 年前银行只管资产端（被动等存款），之后主动管理负债端 |
| **流动性管理 vs 资本管理** | 流动性管的是"能不能拿出足够的现金应对提款"；资本管的是"资产贬值时够不够亏"。流动性危机和资本危机是两回事 |
| **缺口分析 vs 久期分析** | 缺口分析看短期利润对利率的敏感度；久期分析看整体经济价值对利率的敏感度。久期更全面但更复杂 |
| **ROA vs ROE** | ROA 衡量资产赚钱效率（1% 就算不错）；ROE 衡量股东赚钱效率（取决于杠杆）。ROE = ROA × EM |
| **信贷配给 vs 提高利率** | 提高利率用价格分配信贷 → 但利率高了逆向选择恶化；信贷配给用量分配 → 在最优利率之上直接拒绝 |
| **中间业务 vs 表外业务** | 中间业务（结算、代理、咨询）= 基本无风险的收费服务；表外业务（贷款承诺、担保、衍生品）= 有条件触发时转入表内的或有项目 |
| **脱媒 vs 影子银行** | 脱媒是**现象**（资金绕开银行）；影子银行是**机构**（替代银行做信用中介的非银行机构）。脱媒推动了影子银行增长 |
| **证券化 vs 贷款销售** | 证券化是把贷款打包成标准化证券出售（批量）；贷款销售是出售单笔或少数贷款（零售） |
