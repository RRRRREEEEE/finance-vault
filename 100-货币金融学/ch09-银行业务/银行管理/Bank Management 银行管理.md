# 银行管理 (Bank Management)

## 📌 定义
银行通过资产转换（借短贷长）盈利——出售具有一组特征（流动性、风险、收益）的负债，使用收益购买具有另一组特征的资产。银行管理围绕四大原则：流动性、资产、负债、资本充足。

> 💬 通俗理解：银行就是"资金的中转站"——吸收短期便宜存款，发放长期高息贷款。管理一家银行就是平衡赚钱和安全。

## 🧠 机制

### 银行资产负债表

| 资产（资金用途） | 负债+资本（资金来源） |
|------|------|
| 准备金（法定+超额） | 支票存款（活期/NOW/MMDA） |
| 收款中现金项目 | 非交易存款（储蓄+定期CDs） |
| 在其他银行存款 | 借款（贴现贷款/同业拆借） |
| 证券（国债/市政债） | **银行资本**（净值） |
| 贷款 | |
| 其他资产（建筑/设备） | |

恒等式：**资产 = 负债 + 银行资本**

### 银行经营：资产转换
- 借短（吸收存款）→ 贷长（发放贷款）
- T-account：吸收$100存款→准备金+$10，超额+$90→贷款+$90

### 四大管理原则

#### 1. 流动性管理
- 持有足够准备金应对存款提取
- 准备金不足→出售证券/借款/减少贷款
- 超额准备金的机会成本 = 放弃的利息收益

#### 2. 资产管理
- 找高利率低违约风险的借款人
- 买高收益低风险证券
- 分散化降低风险
- 平衡流动性与收益

#### 3. 负债管理
- 1960s后兴起：不再仅依赖支票存款
- 有贷款机会时→借入或发行CDs获取资金
- 资产负债管理（ALM）

#### 4. 资本管理
- 高资本→安全但ROE低；低资本→ROE高但易破产
- 资本充足监管要求
- 资本管理工具：发行/注销股票、调整股利、改变资产增速

**次贷危机案例**：银行持有MBS→巨额损失→资本减少→信贷紧缩（Credit Crunch）
