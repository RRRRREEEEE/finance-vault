# 当期收益率 (Current Yield)

## 📌 定义
当期收益率是到期收益率的一种估计值，定义为每年支付的息票利息金额与债券价格之比：$i_c = C/P$。其中 $i_c$ 为当期收益率，$P$ 为息票债券价格，$C$ 为年利息金额。

> ⚡ **与 [[Yield to Maturity 到期收益率]] 的区别**：YTM 把未来所有现金流（利息 + 本金）折现算进收益，当期收益率只算当年利息占买入价的比例——YTM 是全景，当期收益率是快照。

## 🔗 前置知识
- [[Yield to Maturity 到期收益率]]
- [[Coupon Bond 息票债券]]

## 🧠 机制

债券的回报率 = 当期收益率 + 资本利得率：$R = i_c + g$。

当期收益率的特征：
- 债券价格越接近面值、期限越长 → 当期收益率越近似到期收益率
- 债券价格越远离面值、期限越短 → 近似性越差
- 不管是否良好近似，当期收益率的变动总是与到期收益率**同方向**

当债券价格等于面值时，当期收益率 = 息票利率 = 到期收益率。

## 📖 相关术语（如有）
- [[Yield to Maturity 到期收益率]]：当期收益率是YTM的近似
- [[Rate of Return 回报率]]：$R = i_c + g$（当期收益率 + 资本利得率）

## ⚖️ 易混点对照（人类填写区域）
- 当期收益率 vs 到期收益率：当期收益率只考虑息票利息/价格，YTM含本金偿还
