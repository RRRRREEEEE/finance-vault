# 货币金融学 ch04 学习笔记思考题与考研真题

## 考研真题 📎

### 到期收益率与票面利率 [中国科学技术大学2016研]
其他情况不变，若到期收益率**高于**票面利率，则债券将**折价出售**。

解析：YTM>票面利率→价格<面值（折价）；YTM=票面利率→价格=面值（平价）；YTM<票面利率→价格>面值（溢价）。

### 折价债券的收益率 [清华大学2015研]
30年期限、10%票面利率、面值100元的债券售价98元，该债券的收益率应**大于10%**。折价发行的债券到期收益率大于票面利率。

### 永续债券定价 [中央财经大学2011研]
面值100元、票面利率10%的永久性债券，市场利率8%时，理论价格 = 100×10%/8% = **125元**。

### 期限与利率敏感性 [中山大学2019研]
同一公司5年期 vs 3年期债券，**期限风险收益率较大**。期限越长，利率风险越大。

### 价格波动率最大的债券 [清华大学2015研]
应持有**长期、零息票债券**。价格利率敏感性与期限正相关、与票面利率负相关。

### 实际利率计算 [浙江工商大学2011研]
一年期存款利率2.50%，CPI为3.80%，实际年利率 = (1+2.5%)/(1+3.80%)-1 = **-1.25%**。

### 实际利率与名义利率的符号 [中央财经大学2012研]
"实际利率和名义利率都可能为负值"——**正确**。日本20世纪90年代和美国2008年次贷危机期间出现过名义利率为负。当通胀率高于名义利率时实际利率为负。

## 课后思考题 📎

### 1. 不同利率下明天1美元的现值
利率20%时：1/(1+20%) = **0.83美元**；利率10%时：1/(1+10%) = **0.91美元**。利率越低，现值越高。

### 2. 今天收5000还是1年后收5500
需考虑利率水平。PV = 5500/(1+i)。若PV>5000则选1年后，若PV<5000则选今天。

### 3. 政府贷款的到期收益率
25年每年偿还126美元的1000美元固定支付贷款，正常YTM=12%。但毕业两年后才开始还款，现值必然小于1000美元，因此YTM**低于12%**。

### 4. 债券持有人希望YTM上升还是下降
希望YTM**下降**。因为YTM上升代表债券价格下降，资本收益减小（资本损失增大）。

### 5. 回报率公式中哪个部分包含未来价格变动
R = (C+P₁-P₀)/P₀ = ic + g。**资本利得率g = (P₁-P₀)/P₀**包含了债券价格的未来变动。

### 6. 抵押贷款利率上升但房价预期上涨
买房意愿**更高**。实际利率从3%（5%-2%）下降到1%（10%-9%）。虽然名义利率上升，但购置住宅融资的实际成本降低。

### 7. 当期收益率何时近似到期收益率
当债券价格**非常接近票面价值**或债券**到期期限超过十年**时。长期息票债券的价值接近具有相同票面利率的永续债券。

### 8. 政府为何发行永续债券
永续债券维持给定规模贷款的短期成本远小于固定支付贷款、贴现债券或息票债券。例如1000美元10年期5%利率：永续债每年50美元（共500）；固定支付贷款每年129.50美元（共1295）；贴现贷款10年一次性支付1628.89美元。

### 9. 贴现债券名义利率为负的条件
当前价格P > 面值F时，贴现债券名义利率为负。息票债券当C+F < P时也可能出现负名义利率。永续债券**不可能**出现负名义利率。

### 10. 判断："贴现债券的回报率等于资本利得率"
**正确**。回报率 = ic + g，贴现债券没有利息支付（ic=0），所以 R = g。

### 11. 利率下降时更愿持有什么债券
更愿意持有**长期债券**，因为利率下降使长期债券价格上升幅度大于短期债券。长期债券的**利率风险更大**。

### 12. 80年代中期实际利率高于70年代晚期
**正确**。70年代晚期名义利率低于通胀率，实际利率为负；80年代中期预期通胀率比名义利率下降更快，实际利率为正。

### 13. 退休人士抱怨低利率有道理吗
**不一定**。如果预期通胀与名义利率同步下降，实际存款回报率不受影响。但老年人和退休人员的生活费用通胀通常高于标准通胀率，低名义利率可能产生不利影响。

## 应用题 📎

### 14. 多期现金流现值计算
利率10%，第一年1100、第二年1210、第三年1331美元。
现值 = 1100/1.1 + 1210/1.1² + 1331/1.1³ = **3000美元**。

### 15. 贴现债券现值
1000美元贴现债券，5年到期，YTM=6%。
PV = 1000/(1.06)⁵ = **747.26美元**。

### 16. 彩票现值
1000万美元分5年支付，每年200万，第一笔当时兑现，利率6%。
PV = 200 + 200/1.06 + 200/1.06² + 200/1.06³ + 200/1.06⁴ = **893.02万美元**。

### 17. 负收益率国库券
1年后支付5000美元，售价5012美元。
YTM = (5000-5012)/5012 = **-0.239%**。不常见但存在——国库券电子储蓄比现金更便捷，负利率不一定意味绝对亏损。

### 18. 普通贷款到期收益率
100万美元普通贷款，5年后偿付200万美元。
100 = 200/(1+i)⁵，解得 i ≈ **14.87%**。

### 19. 两种债券YTM比较
- 20年期：售价800，当期收益率15%，YTM≈**15%**
- 1年期：售价800，当期收益率5%，YTM>(1000-800)/800=**25%**
**1年期债券的到期收益率更高。**

### 20. 息票债券：期限、YTM与价格关系
息票利率4%，面值1000美元。
- **YTM>息票利率时**：当前价格<面值（折价）
- **YTM<息票利率时**：当前价格>面值（溢价）
- 给定到期日，价格随YTM上升而**下降**
- 给定YTM，价值随到期日延长而**上升**
- YTM=票面利率时，价格=面值（与期限无关）

### 21. 计算到期收益率
面值1000，息票利率10%，售价1044.89，2年到期。
1044.89 = 100/(1+i) + 100/(1+i)² + 1000/(1+i)² → i = **7.5%**。

### 22. 永续债券价格与YTM变化
每年50美元利息，YTM=2.5%：价格 = 50/0.025 = **2000美元**。
YTM翻番至5%：价格 = 50/0.05 = **1000美元**（降到一半）。

### 23. 房产税现值
25万美元房产，每年4%房产税 = 10000美元，利率6%贴现。
PV = 10000/0.06 = **166666.67美元**。

### 24. 当期收益率、资本利得率与回报率
面值1000，息票利率10%（C=100），当前价格960，预期下年价格980。
- 当期收益率 ic = 100/960 = **10.4%**
- 预期资本利得率 g = (980-960)/960 = **2.1%**
- 预期回报率 R = 10.4% + 2.1% = **12.5%**

### 25. 名义利率与实际通胀
实际年利率3%，预期通胀1% → 名义利率 = 3%+1% = **4%**。
若实际通胀0.5%（低于预期1%），意味着以更高名义利率支付贷款，**贷款成本偏高**。
