# 回报率 (Rate of Return)

## 📌 定义
回报率（RET）衡量在**一定持有期内投资人持有债券实际获得的总收益**，等于**当期收益率加资本利得率**。YTM 是事前的内部报酬率，回报率是事后的外部报酬率。

> 💬 通俗理解：YTM 是"买之前算的预期收益"，回报率是"卖之后算的实际收益"。如果中途卖掉或利率变了，两者就不一样。

## 🧠 机制

### 回报率公式
$$ RET = \frac{C + P_{t+1} - P_t}{P_t} = i_c + g $$

> 变量说明：
> - $RET$ = 回报率（持有期）
> - $C$ = 息票支付
> - $P_t$ = 买入价
> - $P_{t+1}$ = 卖出价（或持有期末价格）
> - $i_c$ = 当期收益率（$C/P_t$）
> - $g$ = 资本利得率（$(P_{t+1}-P_t)/P_t$）

### YTM = 回报率的条件（三个同时满足）
1. **持有至到期日，中途不出售
2. 债券**无拒付风险，无提前赎回**
3. 所有**利息以相同 YTM （单期重复）再投资**

### 利率与回报率的关键事实

以下讨论中，"利率"或"市场利率"是一个**笼统的统称**。现实中存在多种利率——国债利率、公司债利率、房贷利率等——它们虽然数值不同，但**变动方向大多一致**（比如央行加息，各类利率普遍上行）。因此课程用"市场利率"泛指利率的整体水平，不区分具体品种。当市场利率变动时，你手里已买入的债券价格随之变化，从而影响你的实际回报率。

1. **持有期 = 到期日 → 回报率 = 买入时的 YTM，无利率风险。** 因为持有到期时，你收到的价格就是面值——这个金额是固定的，不受中途利率波动影响。

2. **持有期 < 到期日（中途卖出）→ 市场利率上升导致资本损失。** 原因：市场利率↑ → 新发债券更有吸引力 → 你手里的老债券必须降价才能卖掉 → 卖出价低于买入价 → 资本利得率为负。

3. **到期日越远，市场利率变动造成的价格波动越大。** 因为长期债券的未来现金流折现期数多，折现率一变，现值变化被层层放大。所以长期债券的利率风险大于短期债券。

4. **即使买入时 YTM 很高，市场利率上升也可能让回报率变成负数。** 因为资本损失的幅度可能超过全部利息收入。比如你买入时 YTM = 10%，但市场利率从 10% 飙到 20%，价格暴跌带来的资本损失吃掉了利息，最终净收益为负。
