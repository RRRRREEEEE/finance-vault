# 第四章 知识串讲：利率如何度量

## 一、这一章在回答什么问题

第一章说了利率很重要。但你听到的"利率"到底是什么？银行门口的"理财年化 4%"、国债收益率曲线上写的 3.5%、以及你每个月付的房贷利率——它们是一回事吗？

第四章告诉你：**"利率"有很多种算法，它们说的是不同的东西。** 全章围绕一个核心问题展开：如何精确衡量"借钱成本"或"投资收益"？

---

## 二、往前倒一步——钱有时间价值

### 2.1 今天的一块钱和明年的一块钱不一样

今天把 100 块存银行，利率 5%，一年后变成 105 块。所以**一年后的 100 块，今天值不到 100 块。**

这就是现值的概念：

$$PV = \frac{CF}{(1 + i)^n}$$

$PV$ = 现值，$CF$ = 未来现金流，$i$ = 利率，$n$ = 期数。

**核心原则**：不同时点的钱不能直接比较——必须全部折现到同一时点。

### 2.2 单利和复利

| | 单利 | 复利 |
|------|------|------|
| 怎么算 | 每期利息只按本金算 | 利息也产生利息 |
| 公式 | $FV = PV(1 + n \cdot i)$ | $FV = PV(1 + i)^n$ |
| 什么特点 | 简单 | 利滚利，越久差越大 |

72 法则：用 72 除以年利率，等于资金翻倍需要的年数。年利率 7.2% → 约 10 年翻倍。

## 三、到期收益率——最准确的利率指标

### 3.1 什么是 YTM

**到期收益率（Yield to Maturity, YTM）是使债务工具未来全部现金流入的现值，恰好等于当前价格的折现率。**

关键词：全部现金流、折现、等于当前价格。

同一个"利率"——看息票利率可能 5%，看 YTM 可能是 4.8%（如果债券折价买的）或 5.2%（溢价买的）。YTM 才是真实收益率。

### 3.2 四种信贷工具，四种 YTM 算法

| 工具 | 现金流模式 | YTM 怎么算 |
|------|------|------|
| **普通贷款** | 今天借 100，到期一次还 110 | $i = (FV-PV)/PV$ ——最简单的 |
| **固定支付贷款** | 今天借 100，分 5 年每年还 26.4 | 每期付等额，解方程 |
| **息票债券** | 每年付利息，到期还本金 | 把利息 + 面值都折现 |
| **贴现债券** | 不付息，折价发行，到期按面值兑 | $i = (F-P)/P$ |

**永续债券（Consol）** 的特例：英国政府发的永远不还本的债券。它的 YTM 极其简单：$i = C/P_c$（年息除以价格）。

### 3.3 一个关键规律

| 价格与面值的关系 | YTM 与票面利率的关系 |
|------|------|
| 价格 = 面值 | YTM = 票面利率 |
| 价格 < 面值（折价） | YTM > 票面利率 |
| 价格 > 面值（溢价） | YTM < 票面利率 |

**YTM 和债券价格负相关。** 利率涨 → 债券跌。

## 四、其他利率指标——为什么还有别的

### 4.1 当期收益率：只看当年的利息

$$i_c = \frac{C}{P}$$

只算当年收到的利息除以买入价，**不管**债券到期还你本金时你赚了还是亏了。折价买债券 → 到期按面值兑付 → 其实还有一笔隐藏的"涨价收益"——当期收益率没算这笔。所以它只能做近似。债券期限越长，当期收益率越接近 YTM。

### 4.2 贴现收益率：国库券的"方言"

国库券报价用的是贴现收益率：

$$i_{db} = \frac{F - P}{F} \times \frac{360}{t}$$

两个问题使它低估 YTM：①分母用面值 F 而非买入价 P（低估）；②用 360 天而非 365 天（略微高估）。低估远大于高估，所以**贴现收益率总是低于真实 YTM。** 这个惯例沿用了几个世纪——不是它精确，而是市场习惯了这个"方言"。

### 4.3 基点

1 个基点 = 0.01%。利率从 3.50% 涨到 3.75%，涨了 25 个基点。金融市场报利率几乎都用基点——避免百分比混淆。

---

## 五、回报率——YTM 是预期，回报率才是真相

### 5.1 YTM 和回报率不是一回事

- **YTM**：买之前算的，假设持有到期、无违约、利息以相同 YTM 再投资
- **回报率**：卖之后算的，真正落袋的收益

$$RET = \frac{C + P_{t+1} - P_t}{P_t} = i_c + g$$

回报率 = 当期收益率 + 资本利得率。如果买完之后利率大跌，债券价格大涨 → 你中途卖出 → 回报率可以远远高于买入时看到的 YTM。反过来也一样——利率飙升，中途卖就血亏。

### 5.2 两个致命事实

**持有期 = 到期日 → 回报率 = 买入时的 YTM。** 因为持有到期时你收到的价格就是面值，固定不变，利率波动不影响你。

**持有期 < 到期日（中途卖出）→ 利率上升导致资本损失。** 市场利率↑ → 新债券更有吸引力 → 你手里老债券必须降价才有人接 → 卖价低于买价 → 回报率可能变成负数——即使买入时 YTM 看起来很美。

**到期日越远，利率变动造成的价格波动越大。** 因为长期债券的未来现金流折现期数多，折现率一变，现值变化被放大。所以长期债券的利率风险远大于短期债券。

## 六、实际利率——把通胀剥掉

### 6.1 费雪方程式

$$i_r = i - \pi^e$$

- $i_r$ = 实际利率（真实借款成本）
- $i$ = 名义利率（票面上写的）
- $\pi^e$ = 预期通胀率

名义利率 5%，预期通胀 3% → 实际利率只有 2%。名义利率 10%，预期通胀 12% → 实际利率为 -2%，借钱的人反而赚了（货币贬值速度快于利息积累）。

### 6.2 事前 vs 事后

**事前实际利率**：用**预期**通胀算（$i - \pi^e$）。你在做投资决策时只知道预期通胀，这是你决策的依据。

**事后实际利率**：用**实际**通胀算（$i - \pi$）。到期后回头看才知道。和回报率一样——事后才知道真实收益是多少。

### 6.3 再剥一层——税后实际利率

利息收入要缴税。如果税率是 $t$，税后名义利率只剩 $i(1-t)$。再减通胀：

$$\text{税后实际利率} = i(1 - t) - \pi^e$$

名义利率 5%，25% 税率，预期通胀 3% → 税后实际利率 = 3.75% - 3% = 0.75%。看起来赚了 5%，落到口袋几乎没赚。

## 七、麦考利久期——一张债券对利率有多敏感

### 7.1 什么是久期

久期（Duration）是现金流入时间的加权平均，衡量债券的"有效期限"和对利率的敏感度。

- 零息债券：久期 = 到期期限（全部现金流在最后一笔）
- 息票债券：久期 < 到期期限（中间有利息分散了时间）

### 7.2 久期的三大规律

| 变量 | 怎么变 | 久期怎么变 |
|------|------|------|
| 息票利率 ↑ | 中间利息多 | 久期 ↓ |
| YTM ↑ | 远期现金流被折得更狠 | 久期 ↓ |
| 到期期限 ↑ | 最后一笔更远 | 久期一般 ↑ |

高息票债券的久期更短——因为你拿钱早，对利率没那么敏感。

## 八、把本章所有利率概念串在一起

以一张债券为例：息票利率 5%，你 95 块买入（面值 100），5 年到期。假设通胀预期 3%，你税率 20%。

```
息票利率 = 5%（债券合同上写的）
    ↓
买入折价后，你的真实收益不止 5%
    ↓
YTM = 约 6.2%（比息票利率高，因为折价买）
当期收益率 = 5/95 = 5.26%（只看当年息票，低估了折价收益）
    ↓
你持有了两年，市场利率意外下降，债券价格涨到 98，你卖出
    ↓
回报率 = 当年收到的利息 ÷ 95 + (98-95)/95
      = 5.26% + 3.16% = 8.42%  ← 远超 YTM！利率跌让资本利得赚了
    ↓
但这都是名义收益。名义 YTM 6.2% - 通胀预期 3% = 实际 3.2%
再交 20% 税 → 税后实际利率 = 4.96% - 3% = 1.96%
```

这个例子说明：**你听到的每一个"利率"都删掉了一层现实。YTM 比息票利率更准，回报率比 YTM 更真，实际利率比名义利率更切肤，税后实际利率才是你的钱。**
