# 第二章 知识串讲：金融体系由哪些部分构成

## 一、这张章在干什么

第一章说"金融体系很重要"。第二章把"金融体系"这个词拆开给你看——它到底由什么组成、怎么运转。

全章三块：**金融市场**（交易的场所和规则）→ **金融中介**（交易的主要玩家）→ **金融监管**（交易的规则执行者）。

---

## 二、金融市场的五种分类法——同一笔交易，五个维度同时定位它

金融市场不是大杂烩。任何一笔金融交易，都可以从五个维度同时给它贴标签。记住五个字：**期—阶—所—层—域**。

### 2.1 按期限：货币市场 vs 资本市场

| | 货币市场 | 资本市场 |
|------|------|------|
| 交易什么 | 1 年以内的短期债务 | 1 年以上的长期债务 + 股票 |
| 谁在用 | 银行、大企业、政府 | 企业、政府、个人 |
| 干什么 | 短期头寸调剂 | 长期投资融资 |
| 波动 | 小，收益稳定 | 大，收益弹性高 |

国库券（T-bills）、商业票据（CP）、回购协议（Repo）、联邦基金——这些都是货币市场工具，买家是机构，个人基本不直接参与。

### 2.2 按发行阶段：一级市场 vs 二级市场

**一级市场**：新证券首次发行的市场。公司 IPO、政府发新国债——投资银行承销的角色在这里。

**二级市场**：已经发行的证券在投资者之间转手的市场。纽交所、纳斯达克——你买股票的地方。二级市场的存在让一级市场成为可能（投资者知道买完还能卖，才敢买）。

二级市场的衍生功能不止交易：价格发现、公司并购、甚至倒逼公司改善治理（股价太低会被收购→管理层有压力）。

### 2.3 按交易场所：交易所 vs 场外 OTC

**交易所**：集中场所，竞价交易。上市有门槛，交易有规则。NYSE、上交所。

**OTC**：分散的交易商网络，电子报价系统连接。没有集中的"交易大厅"，也没有上市标准。债券、外汇、衍生品主要走 OTC。

### 2.4 按上市层次：主板 vs 二板

**主板**：成熟大企业，门槛高（盈利要求、市值要求）。NYSE、上交所主板。

**二板**：高新创业企业，门槛低。NASDAQ、科创板、创业板。企业年轻时上二板，长大了可以转主板。

### 2.5 按地域：国内 vs 国际

国际金融市场有三个独特角色：
- **外国债券**：外国发行人在本国市场以本币发债（比如日本企业在纽约发美元债）
- **欧洲债券**：在货币发行国以外发行的债券（在伦敦发美元债——"欧洲美元债券"）
- **欧洲货币**：存放在货币发行国以外的银行存款（存在伦敦的美元 = 欧洲美元）

注意：**"欧洲"在这里不是地理位置，是"离岸"的概念。** 欧洲美元、欧洲日元、欧洲人民币——逻辑一样。

## 三、两类融资方式

**直接融资**：借款人通过金融市场直接向贷款人出售证券。企业发公司债券，你直接买→你是直接借给企业。

**间接融资**：金融中介站在中间。你把钱存银行→银行把钱贷给企业。你不管企业是谁，你只管银行。

第八章会告诉你为什么**间接融资远比直接融资重要**——信息不对称让银行在信息生产上有天然优势。

## 四、金融市场的工具——哪些东西在交易

### 4.1 货币市场工具（短期债务，< 1 年）

| 工具                  | 发行方       | 怎么运作                    | 风险  |
| ------------------- | --------- | ----------------------- | :-: |
| **国库券**（T-bills）    | 美国财政部     | 贴现发行（不付息，折价卖，到期按面值兑）    | 极低  |
| **可转让存单**（NCD）      | 商业银行      | 银行发行的大额定存，可在二级市场转卖      |  中  |
| **商业票据**（CP）        | 大公司       | 企业短期借据，无担保，贴现发行         | 中高  |
| **银行承兑汇票**（BA）      | 企业开票、银行担保 | 国际贸易结算用的远期汇票，银行兜底       |  低  |
| **回购协议**（Repo）      | 持券机构      | 卖证券给你 + 承诺过几天买回来 = 抵押借款 |  低  |
| **联邦基金**（Fed Funds） | 银行间       | 银行间互相拆借准备金的隔夜贷款         |  中  |

### 4.2 资本市场工具（长期，> 1 年）

| 工具              | 属性      | 特征                |
| --------------- | ------- | ----------------- |
| **股票**          | 权益      | 永续，无到期日，有表决权，剩钱才分 |
| **公司债券**        | 中长期债权   | 按期付息，到期还本         |
| **抵押贷款**        | 长期债权    | 居民买房融资，非标准化       |
| **国债/市政债**      | 主权/地方债权 | 国债风险极低，市政债有税收优惠   |
| **银行贷款**（商业/消费） | 非标债权    | 几乎没有公开二级市场        |

**抵押贷款支持证券（MBS）** 是一个关键创新：银行把大量的抵押贷款打包→发行基于贷款池现金流的新证券→贷款变成可交易的标准品。这在次贷危机中是核心角色。

## 五、金融中介——谁在做搬运工

金融中介不是只有银行。三大类：

| 类型 | 主要机构 | 资金来源 | 资金用途 |
|------|------|------|------|
| **存款机构** | 商业银行、储蓄机构、信用社 | 存款 | 贷款、证券 |
| **契约储蓄机构** | 保险公司、养老基金 | 保费、养老金缴款 | 长期证券（债券+股票） |
| **投资中介** | 共同基金、货币市场基金、对冲基金 | 出售份额 | 多样化资产组合 |

存款机构的特点：它们的负债是**货币**（存款随时可以花）。这让它们比别的中介受监管更严。契约储蓄机构的特点：它们的资金来源稳定可预测，所以可以投长期资产。投资中介的特点：帮普通人实现分散投资（买一股基金 = 间接持有一篮子股票或债券）。

## 六、金融监管——为什么政府盯得最紧

金融体系是经济中监管最严的部门。三个核心目标：

1. **增加信息**（信息披露）：让投资者能区分好坏公司 → 减少逆向选择
2. **确保稳健**（审慎监管）：防止银行冒险，保护储户 → 减少道德风险
3. **控制货币**：货币政策需要通过银行体系传导
