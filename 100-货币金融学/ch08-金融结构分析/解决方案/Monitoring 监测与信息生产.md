# 监测与信息生产 (Monitoring)

## 📌 定义
监测（Monitoring）是指贷款人或股东对借款人或管理层行为**进行监督**，**并生产相关信息以减少信息不对称的活动**。它是解决道德风险问题的基础工具。

## 🔗 前置知识
[[Asymmetric Information 信息不对称]]、[[Moral Hazard 道德风险]]、[[Free-Rider Problem 搭便车问题]]

## 🧠 机制
- **监测的必要性**：信息不对称导致道德风险 → 需要**监测来获取借款人**/管理者的真实行为信息
- **搭便车问题**：**分散持股下，单个股东没有足够激励承担监测成本**（收益归所有股东，成本自己承担）
- **金融中介的优势**：银行/风投通过**集中持股**，将监测收益内部化，从而**有激励进行充分监测**

## 📖 相关术语
- [[Free-Rider Problem 搭便车问题]]：分散持股下监测不足的根源
- [[Costly State Verification 昂贵状态核查]]：监测的一种形式——核查借款人真实财务状况
- [[Venture Capital 风险投资]]：通过集中持股解决监测中的搭便车问题
- [[Equity Contract 股权合约]]：股权合约中监测是解决道德风险的关键
- [[Financial Intermediaries 金融中介]]：专业化监测的优势提供者
