# 昂贵状态核查 (Costly State Verification)

## 📌 定义
昂贵状态核查是指对借款人（代理人）的行为或财务状况进行监测需要花费成本。这是解决委托代理问题的一个基本障碍。

## 🔗 前置知识
[[Principal-Agent Problem 委托代理问题]]、[[Asymmetric Information 信息不对称]]

## 🧠 机制
- 解决委托代理问题需要监测（信息生产）以核实经理行为
- 但监测本身有成本——收集、核实公司实际经营状况需要资源
- 同时存在搭便车问题：小股东没有动力承担监测成本，却可享受监测收益
- 因此分散持股下监测不足，委托代理问题难以靠私人监测解决

## 📖 相关术语
- [[Principal-Agent Problem 委托代理问题]]：CSV是解决此问题的障碍
- [[Free-Rider Problem 搭便车问题]]：加剧了CSV问题
- [[Venture Capital 风险投资]]：风险投资机构集中持股，有动力承担CSV成本
