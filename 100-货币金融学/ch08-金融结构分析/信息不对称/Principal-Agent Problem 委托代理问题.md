# 委托代理问题 (Principal-Agent Problem)

## 📌 定义
委托代理问题源于公司股东（委托人）所有权与经理（代理人）控制权的分离。管理者可能追求个人利益和权力而非公司盈利能力。

## 🔗 前置知识
[[Moral Hazard 道德风险]]、[[Asymmetric Information 信息不对称]]

## 🧠 机制
- 股东（委托人）拥有公司但不直接经营
- 经理（代理人）控制公司但未必按股东利益行事
- 经理可能追求：个人薪酬最大化、在职消费、帝国构建（扩大企业规模以获取权力）
- 结果：公司利润低于股东期望水平

## 🏛️ 应用
解决委托代理问题的工具：
- **监测/信息生产 (Costly State Verification)**：核实经理行为——但存在搭便车问题
- **政府监管**：强制信息披露
- **金融中介（风险投资）**：风险投资机构集中持股并参与管理，规避搭便车
- **债务合约**：固定还款减少管理层的自由裁量空间（解释事实1：为何企业更多使用债务而非股权）

## 📖 相关术语
- [[Moral Hazard 道德风险]]：委托代理问题是道德风险在股权合约中的具体形式
- [[Costly State Verification 昂贵状态核查]]
- [[Venture Capital 风险投资]]

## ⚖️ 易混点对照（人类填写区域）
- 与 [[Moral Hazard 道德风险]] 的区别：委托代理问题是道德风险的子类型，特指股权合约
