# 柠檬问题 (Lemons Problem)

## 📌 定义
柠檬问题（次品问题）描述了信息不对称下优质品被劣质品驱逐的市场失灵现象。在证券市场：若无法区分好证券和坏证券，投资者只愿支付平均价格，导致好证券被低估而不发行、坏证券被高估而过度发行。

## 🔗 前置知识
[[Asymmetric Information 信息不对称]]、[[Adverse Selection 逆向选择]]

## 🧠 机制
1. 投资者无法区分好证券和坏证券 → 只愿支付平均价值
2. 好证券被低估 → 公司不愿发行
3. 坏证券被高估 → 发行过多
4. 投资者最终不愿购买 → 市场运行不良

此机制部分解释了事实1（股票非最重要外部融资来源）和事实2（可流通证券非主要融资方式）。知名企业信息不对称较少，柠檬问题较小（解释事实6）。

## 📖 相关术语
- [[Adverse Selection 逆向选择]]：柠檬问题是逆向选择在证券市场的具体表现
- [[Free-Rider Problem 搭便车问题]]：阻碍私人信息生产解决柠檬问题

## ⚖️ 易混点对照（人类填写区域）
- 与 [[Adverse Selection 逆向选择]] 的区别：柠檬问题是逆向选择的一个经典模型
