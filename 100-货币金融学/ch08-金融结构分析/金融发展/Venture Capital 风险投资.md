# 风险投资 (Venture Capital)

## 📌 定义
风险投资是金融中介的一种形式，通过集中持股并参与被投资公司的管理，解决委托代理问题。它避免了搭便车问题，因为风投机构是主要甚至唯一的权益提供者。

## 🔗 前置知识
[[Financial Intermediaries 金融中介]]、[[Principal-Agent Problem 委托代理问题]]

## 🧠 机制
- 风险投资机构集中持有大量股权（非分散持股）
- 参与公司董事会和战略决策 → 直接监测管理层
- 避免了搭便车问题：风投是主要股东，监测收益归己
- 解释了金融结构事实3（金融中介的重要性）

## 📖 相关术语
- [[Risk 风险]]：风险投资中的"风险"指被投企业的高不确定性和信息不对称
- [[Monitoring 监测与信息生产]]：风投通过集中持股解决监测中的搭便车问题
- [[Equity Contract 股权合约]]：风投是股权合约道德风险的解决方案
- [[Principal-Agent Problem 委托代理问题]]：风投是其金融中介解决方案
- [[Moral Hazard 道德风险]]：风投通过集中持股和参与管理解决道德风险
- [[Costly State Verification 昂贵状态核查]]：风投有动力承担监测成本
- [[Free-Rider Problem 搭便车问题]]：风投通过集中持股规避此问题
