# 贴现贷款 (Discount Loans)

## 📌 定义
贴现贷款是美联储向银行发放的贷款，收取的利率称为贴现率(Discount Rate)。贴现贷款是美联储资产的重要组成部分，也是基础货币的来源之一。

## 🔗 前置知识
[[Monetary Base 基础货币]]、[[Federal Reserve System 联邦储备体系]]、[[Lender of Last Resort 最后贷款人]]

## 🧠 机制
- 美联储发放贴现贷款 → 银行准备金增加 → MB增加
- 美联储收回贴现贷款 → 银行准备金减少 → MB减少
- 贴现贷款受银行借款决策影响 → 美联储不能单方面完全控制MB

将MB分为两部分：
- **非借入基础货币(MBn)**：由公开市场操作直接控制
- **借入基础货币/贴现贷款(A)**：受银行决策影响，控制较弱

市场利率上升 → 贴现贷款增加（银行以较低贴现率借款再以较高市场利率放贷）
