# 债券 (Bond)

## 📌 定义
债券是一种**债务证券**，承诺在规定时期内**定期进行支付**。它是政府、金融机构、工商企业等机构为**筹措资金**，依法定程序向投资者发行的、**约定在一定期限内还本付息的证券**。债券在投资者（持有人）与发行人之间形成债权债务关系——发行人为债务人，持有人为债权人。

> 💬 通俗理解：债券就是"借条"——你借钱给政府或公司，他们承诺按期还你利息，到期还本金。

## 🧠 机制

### 核心要素
1. **发行人**：政府（国债）或企业（公司债）
2. **面值**：到期偿还的本金金额
3. **票面利率**：定期支付的利息比率
4. **期限**：债券存续的年限

### 三大特征 📎
> 📎 以下补充自学习笔记，PPT未涉及。
1. **偿还性**：债券有确定的偿还期限，发行人必须按期偿还本金，并按约定利率支付利息
2. **流通性**：债券一般可以自由转让，在证券交易所上市交易或在其他合法交易场所转让
3. **低风险**：债券到期必须还本付息，有确定的预期收益，市场价格波动相对较小；即使发债企业破产，债券持有人的债权也优先于股东对企业清算财产的索取权

### 常见分类 📎
| 分类标准   | 类型                 |
| ------ | ------------------ |
| 发行主体   | 政府债券、企业（公司）债券、金融债券 |
| 偿还期限   | 短期债券、中期债券、长期债券     |
| 利率是否变化 | 固定利率债券、浮动利率债券      |
| 发行地    | 国内债券、国际债券          |

### 价格与利率的关系
- 债券价格与市场利率**反向变动**
- 利率上升 → 债券价格下跌；利率下降 → 债券价格上涨
