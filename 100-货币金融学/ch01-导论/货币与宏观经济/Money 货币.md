# 货币 (Money)

## 📌 定义
货币是在**支付商品或服务、或偿还债务**时被**普遍接受**的任何东西。

> 💬 通俗理解：货币就是"大家都认的钱"——不管是纸钞、硬币还是电子支付，只要买卖双方都接受，它就是货币。

## 🔗 前置知识
- [[Financial Markets 金融市场]]

## 🧠 机制

与货币密切相关的三个核心经济变量：
1. **商业周期（Business Cycles）**：经济总产出的上升和下降运动
2. **通货膨胀（Inflation）**：物价水平的持续上涨
3. **利率（Interest Rates）**：货币作为一种资源的价格

货币为什么重要？
- 货币供给的变化与总经济活动和价格水平密切相关（货币理论）
- 货币是货币政策的核心工具
- 货币影响所有人的日常生活——就业、物价、借贷成本

### 货币 vs 通货 vs 财富 vs 收入

| 概念               | 定义            | 关键区别        |
| ---------------- | ------------- | ----------- |
| **货币（Money）**    | 被普遍接受的支付手段    | 功能定义        |
| **通货（Currency）** | 流通中的现金（纸币+硬币） | 货币的子集，不含存款  |
| **财富（Wealth）**   | 用于储存价值的财产总和   | 存量概念，含股票房产等 |
| **收入（Income）**   | 单位时间的收入流量     | 流量概念        |

### 货币层次（Monetary Aggregates）

"货币"不是非黑即白的——有些资产（现金）一眼就是货币，有些（股票）明显不是，中间有一大片灰色地带。各国央行按**流动性**（变现的方便程度）把货币分成不同层次：

| 层次 | 美国 | 中国 |
|------|------|------|
| **最流动** | M1 = 通货 + 旅行支票 + 活期存款 + 其他支票存款 | M0 = 流通中现金 |
| | | M1 = M0 + 企业活期存款 |
| **次流动** | M2 = M1 + 小额定期 + 储蓄存款 + 货币基金（零售） | M2 = M1 + 准货币（定期+储蓄+其他） |

> 划分标准：**流动性**——作为流通手段和支付手段的方便程度。M1 是"马上能花的钱"，M2 是"很快能变现的钱"。

M1 和 M2 短期可能背离（2020 年美国 M1 飙升但 M2 相对温和），但长期趋势一致。政策制定者关注哪个口径取决于当前经济环境。

> 详见 [[Money Supply Measurement 货币供给度量]]——含美国/中国/欧元区/日本/英国的完整货币层次、M1 vs M2 政策含义、Divisia 加权总量。
