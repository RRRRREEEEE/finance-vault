# 中央银行 (Central Bank)

## 📌 定义
中央银行是负责货币政策的政府机构，其行动影响利率、信贷数量和货币供给。核心职能：调节货币价值、为政府融资、充当最后贷款人。

> 💬 通俗理解：中央银行就是"银行的银行"——它不跟老百姓打交道，但管着所有商业银行的钱和利率。

## 🧠 机制

### 历史起源

| 年份 | 事件 |
|------|------|
| 1668 | 瑞典国家银行成立（最早） |
| **1694** | **英格兰银行成立——央行制度事实上的起源** |
| 1791-1836 | 美国第一/第二银行（失败） |
| **1913** | 美联储成立（Federal Reserve Act） |
| 1999 | 欧洲中央银行（ECB） |

**美国为何迟迟没有央行？**：对中央集权的恐惧+对金融利益集团的不信任。1907年恐慌后公众才信服需要央行。

### 核心职能演进
1. 调节本国货币价值
2. 为政府融资
3. **最后贷款人**（Lender of Last Resort）——应对银行流动性/信用危机
4. 监督和监管金融机构

> *"必须有一个强有力的中央银行。美国在很长一段时期内没有中央银行，其后果是灾难性的。"* ——《伟大的博弈》
