# 均衡利率的变动 (Changes in Equilibrium Interest Rates)

## 📌 定义
均衡利率的变动研究什么因素推动债券市场的均衡利率上升或下降。有两条等价的路径：**可贷资金理论**（债券市场供求）和**流动性偏好理论**（货币市场供求）。核心方法是**比较静态分析**——假定其他条件不变（ceteris paribus），观察单一因素变动如何改变均衡位置。

> 💬 通俗理解：利率为什么会变？要么是想借钱的人变多了（债券供给↑ → 利率↑），要么是愿意借钱出去的人变少了（债券需求↓ → 利率↑）。下面把每一个具体因素拆开来看。

## 🧠 机制：可贷资金框架下的均衡利率变动

> **基本规则**：债券价格与利率反向变动。债券需求↑ → 债券价格↑ → 利率↓。债券供给↑ → 债券价格↓ → 利率↑。

### 一、债券需求曲线位移（$B_d$ — 谁在买债券？）

债券需求 = 可贷资金供给。需求曲线右移 → 利率下降。

![[货币金融学Ch05 债券需求曲线位移因素.png]]

> 图形说明：债券需求曲线位移的四个因素——财富、预期收益、风险、流动性。每个因素的变动方向决定了 $B_d$ 向左还是向右移动。来源：第二补充笔记 第24页。

| 因素 | 变动方向 | $B_d$ 位移 | 利率 | 机制解释 |
|------|:---:|:---:|:---:|------|
| **财富** | ↑ | 右移 | ↓ | 经济扩张/储蓄增加 → 财富↑ → 每种资产都想多买 → 债券需求↑ |
| **预期收益** | ↑ | 右移 | ↓ | 未来利率预期↓ → 债券价格预期↑ → 预期资本利得↑ → 债券更有吸引力 |
| | | | | 替代资产（如股票）预期收益↓ → 债券相对更有吸引力 |
| **风险** | ↓ | 右移 | ↓ | 债券自身风险↓（如评级上调）→ 债券更安全 |
| | | | | 替代资产风险↑（如股市崩盘）→ 债券相对更安全 |
| **流动性** | ↑ | 右移 | ↓ | 债券市场深度↑/交易成本↓ → 债券更易变现 |
| | | | | 替代资产流动性↓（如房产限售）→ 债券相对更灵活 |

### 二、债券供给曲线位移（$B_s$ — 谁在发债券？）

债券供给 = 可贷资金需求。供给曲线右移 → 利率上升。

![[货币金融学Ch05 债券供给曲线位移因素.png]]

> 图形说明：债券供给曲线位移的三个因素——投资机会、预期通胀、政府赤字。来源：第二补充笔记 第25页。

| 因素 | 变动方向 | $B_s$ 位移 | 利率 | 机制解释 |
|------|:---:|:---:|:---:|------|
| **投资机会/预期盈利** | ↑ | 右移 | ↑ | 经济扩张 → 企业看到可盈利投资机会 → 发债融资需求↑ |
| **预期通货膨胀率** | ↑ | 右移 | ↑ | $\pi^e$↑ → 实际借款成本↓（用更不值钱的钱还款）→ 企业多发债 |
| **政府赤字** | ↑ | 右移 | ↑ | 政府需借款弥补赤字 → 国债供给↑ → 债券总供给↑ |

## 🧠 机制：流动性偏好框架下的均衡利率变动

> **基本规则**：货币需求↑或货币供给↓ → 利率↑。货币需求↓或货币供给↑ → 利率↓。

详细分析见 [[Liquidity Preference Framework 流动性偏好理论]]。此处仅列总表：

| 因素 | 变动 | 受影响曲线 | 位移 | 利率 |
|------|:---:|------|:---:|:---:|
| 收入 | ↑ | $M_d$ | 右移 | ↑ |
| 物价水平 | ↑ | $M_d$ | 右移 | ↑ |
| 货币供给 | ↑ | $M_s$ | 右移 | ↓ |

## 🔑 费雪效应：双曲线同时位移

预期通胀 $\pi^e$↑ 是唯一同时推动**两条**曲线反向移动的因素：

![[货币金融学Ch05 费雪效应 双曲线位移.png]]

> 图形说明：预期通胀↑ → $B_s$ 右移（借款人发更多债）+ $B_d$ 左移（贷款人要求更高收益补偿）。两条曲线反向移动，均推高利率。来源：第二补充笔记 第26页。

**费雪方程式**：名义利率 = 实际利率 + 预期通货膨胀率，即 $i = i_r + \pi^e$。

在长期中，实际利率由实质经济因素（生产率、时间偏好等）决定，不受货币因素影响。当货币供给持续增长 → 预期通胀上升 → 名义利率同比例上升。费雪效应还涉及国际维度（国际费雪效应）：一国预期通胀上升 → 本币贬值 → 该国利率上升，公式 $R_a - R_b = \pi_a - \pi_b$。

考研真题：[厦门大学2024研]

> 详细图形推导见 [[Loanable Funds Framework 可贷资金理论#Fisher Effect（费雪效应）]]。

## 📊 全因素速查表

> 假定其他条件不变（ceteris paribus）。

### 可贷资金框架

| 因素变动 | $B_d$ | $B_s$ | 利率 |
|----------|:-----:|:-----:|:---:|
| 财富↑ | → | — | ↓ |
| 预期收益↑ | → | — | ↓ |
| 风险↓ | → | — | ↓ |
| 流动性↑ | → | — | ↓ |
| 投资机会↑ | — | → | ↑ |
| 预期通胀↑ | ← | → | ↑↑ |
| 政府赤字↑ | — | → | ↑ |

### 流动性偏好框架

| 因素变动 | $M_d$ | $M_s$ | 利率 |
|----------|:-----:|:-----:|:---:|
| 收入↑ | → | — | ↑ |
| 物价↑ | → | — | ↑ |
| 货币供给↑ | — | → | ↓ |

> 符号：→ = 右移，← = 左移，— = 不变，↑↑ = 两股力量同向推高

## 🏛️ 应用一：经济周期波动与利率

经济周期（商业周期）的扩张和衰退直接影响债券市场的供求，进而推动利率变动。

![[货币金融学Ch05 应用 经济周期与利率.png]]

> 图形说明：经济扩张期间 $B_d$ 与 $B_s$ 同时右移的综合效果。来源：第二补充笔记 第27页。

**经济扩张期**：两个力量同时发力——
- 财富↑ → 储蓄↑ → $B_d$ 右移（推低利率）
- 投资机会↑ → 企业发债需求↑ → $B_s$ 右移（推高利率）

两个力量方向相反。**实证数据表明**：供给曲线位移幅度通常**大于**需求曲线位移幅度 → 整体利率**上升**。利率是**顺周期**的（pro-cyclical）——经济好时利率升，经济差时利率降。

**经济衰退期**：一切反向。财富缩水+投资萎缩 → $B_d$ 左移 + $B_s$ 左移 → 供给位移更大 → 利率下降。

## 🏛️ 应用二：低通货膨胀与长期停滞

> 📎 来源：第二补充笔记 第27页。

![[货币金融学Ch05 应用 低通胀与长期停滞.png]]

> 图形说明：低通胀环境 + 长期停滞如何压低均衡利率。来源：第二补充笔记 第27页。

2008 年金融危机后，发达经济体普遍经历了长期低利率，可用可贷资金框架解释：

**低通货膨胀**：
- 预期通胀 $\pi^e$ 持续低迷 → 实际借款成本相对较高 → $B_s$ 左移（企业不愿发债）
- 同时 $\pi^e$↓ → 贷款人要求的通胀补偿降低 → 但名义利率已接近零下限

**长期停滞（Secular Stagnation）**：
- 人口老龄化 + 生产率增长放缓 → 投资机会减少 → $B_s$ 左移（企业缺乏可盈利项目）
- 储蓄过剩（财富集中在老年人）→ $B_d$ 右移（储蓄意愿强）
- 两股力量均推低利率 → 均衡利率长期被压制在低位

这也解释了为什么美联储等央行在 2010 年代长期维持零利率政策仍难以推高通胀和利率。
