# 资产组合选择理论 (Portfolio Choice Theory)

## 📌 定义
资产组合选择理论研究经济主体如何在多种资产之间配置财富。资产需求由四个因素决定：财富、预期收益、风险、流动性。

> 💬 通俗理解：你手里的钱怎么分配——存银行、买股票、买房？取决于你有多少钱、能赚多少、有多大风险、能不能随时变现。

## 🧠 机制

### 资产需求的四大决定因素

| 因素                        | 定义                        | 与需求量的关系 |
| ------------------------- | ------------------------- | :-----: |
| **财富（Wealth）**            | **个人拥有的包括所有资产在内的总资源**     |  ✅ 正相关  |
| **预期收益（Expected Return）** | **相对于**替代资产下一期的预期回报       |  ✅ 正相关  |
| **风险（Risk）**              | **相对于**其他资产收益的**不确定性程度**  |  ❌ 负相关  |
| **流动性（Liquidity）**        | **相对于**其他资产变现的**容易程度和速度** |  ✅ 正相关  |

### 财富效应的两个层次

**第一层：方向是一致的。** 假定其他因素不变，财富增长会增加对资产的需求数量。这是上述四大因素中"财富正相关"的直接推论——**你更有钱了，每一种资产你都倾向于买更多。财富效应是推动债券需求曲线位移的四个力量之一。**

**第二层：幅度是不同的。** 资产各种各样，对不同资产的需求对财富变动的反应是不同的。**比如财富增加 10%，你对活期存款的需求可能只增加 3%，但对股票的需求可能增加 25%。** 正因如此，不能只用"正相关"来刻画财富效应——需要一个量化工具来度量这种反应强度的差异。**这个工具就是财富弹性。**

### 财富弹性

$$ E_w = \frac{\Delta Q_d / Q_d}{\Delta W / W} $$

> - $E_w$ = 财富需求弹性（一个纯数字，无单位）
> - $Q_d$ = 对该资产的**需求量**（Quantity Demanded）——不是价格，是你想买多少
> - $\Delta Q_d / Q_d$ = 需求量变化的**百分比**
> - $W$ = 财富（Wealth）
> - $\Delta W / W$ = 财富变化的**百分比**

**公式在算什么**：财富每涨 1%，你对这个资产的需求量会变动百分之几。

| $E_w$ 值       | 含义                          | 举例                        |
| ------------- | --------------------------- | ------------------------- |
| $E_w > 1$     | **奢侈品**：财富涨 1%，需求涨超过 1%     | 股票、豪宅——有钱了买更多，而且比例比财富涨得还猛 |
| $0 < E_w < 1$ | **必需品**：财富涨 1%，需求也会涨，但不到 1% | 活期存款——有钱了多存点，但不会成比例猛增     |
| $E_w < 0$     | **劣等品**：财富涨了反而买更少           | 高利贷——有钱了就不借了              |

**和债券市场的关系**：经济扩张 → 财富增长 → 债券需求增加 → 债券需求曲线右移 → 债券价格↑、利率↓。不同资产的 $E_w$ 决定了这个传导有多强。

### 风险态度
- **风险规避（Risk-Averse）**：预期收益相同时偏好低风险（大多数人是）
- **风险偏好（Risk-Seeker/Lover）**：预期收益相同时偏好高风险
- **风险中性（Risk-Neutral）**：只关注预期收益，不关心风险

风险规避者的效用函数：
$$ U''(W) < 0 \quad \text{（凹函数）} $$
