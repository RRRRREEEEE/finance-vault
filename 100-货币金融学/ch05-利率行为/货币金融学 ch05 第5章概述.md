# 货币金融学 第五章 利率行为

> 本章回答一个核心问题：**名义利率由什么决定、哪些因素导致利率变动？**

本章的推理分五步走。**先**交代微观基础——人买资产时看财富、预期收益、风险、流动性四个维度（第1步）。**然后**把四个维度放进债券市场，用 $B_d$（需求曲线，4个位移因素）和 $B_s$（供给曲线，3个位移因素）构建可贷资金框架（第2步），逐一做比较静态分析，一张表统揽全部因素（第3步）。**再换**货币市场视角，用 $M_d$ 和 $M_s$ 两条曲线得到等价结论——这两个框架分析的是同一个均衡，$M_s - M_d = B_d - B_s$（第4步）。两个框架的关键差异在于推导利率的结构：可贷资金通过"债券价格"这个中介绕一圈，供给侧有企业/借款人/政府三条独立通道，擅长分析实体经济冲击；流动性偏好没有中介，利率直接出清货币市场，供给侧只有央行一条通道，擅长分析货币政策传导。**最后**聚焦一个具体问题：货币供给增加到底使利率升还是降？流动性偏好框架中 $M_s$↑ 的直接效应是利率↓，但货币扩张还会间接推高收入和物价，反过头来又推高利率——四条效应同时发生，方向相反，短期↓、长期可能↑（第5步）。

---

## 第1步：资产需求的四大决定因素（p5-11）

> **这一节在回答**：一个人决定买不买某种资产，看哪几个维度？

这是所有后续分析的**微观基础**。后面无论是债券需求曲线位移还是货币需求曲线位移，原因都归结为这四个因素中的某一个。

| 因素 | 定义 | 与需求量关系 | 在后面哪一步用到 |
|------|------|:--:|------|
| **财富** | 个人拥有的全部资源 | ↑ | 第2步：财富↑ → $B_d$ 右移 |
| **预期收益** | 相对于替代资产的预期回报 | ↑ | 第2步：预期收益↑ → $B_d$ 右移 |
| **风险** | 相对于替代资产的收益不确定性 | ↓ | 第2步：风险↓ → $B_d$ 右移 |
| **流动性** | 相对于替代资产的变现容易程度 | ↑ | 第2步：流动性↑ → $B_d$ 右移 |

## 第2步：把这些因素放进债券市场——可贷资金框架（p13-17）

> **这一节在回答**：第 1 步的四大因素，在债券市场里具体通过哪条曲线、往哪个方向推动利率？

**可贷资金理论** = 用债券市场供求来分析利率。债券需求 = 想买债券的人（可贷资金供给方），债券供给 = 想发债融资的人（可贷资金需求方）。均衡利率由 $B_d = B_s$ 决定。

关键规则：**债券价格与利率反向变动**。$B_d$↑ → 债券价格↑ → 利率↓。$B_s$↑ → 债券价格↓ → 利率↑。

## 第3步：逐一分析每个因素——均衡利率的变动（p18-29）

> **这一节在回答**：把所有影响利率的因素摆到一张桌子上，逐个做比较静态分析。

汇集可贷资金框架（7 因素）和流动性偏好框架（3 因素），假定其他条件不变，看单一因素变动如何改变均衡利率。

### 可贷资金框架：$B_d$ 位移的四个因素

| 因素变动 | $B_d$ | 利率 | 机制——为什么会这样？ |
|----------|:-----:|:---:|------|
| 财富↑ | 右移 | ↓ | 第1步"财富"→ 更富了，每种资产都想多买 → 债券需求↑ |
| 预期收益↑ | 右移 | ↓ | 第1步"预期收益"→ 未来利率↓则债券价格↑ → 预期资本利得↑ |
| 风险↓ | 右移 | ↓ | 第1步"风险"→ 债券自身风险↓或替代资产风险↑ → 债券更安全 |
| 流动性↑ | 右移 | ↓ | 第1步"流动性"→ 债券更容易变现 → 更有吸引力 |

### 可贷资金框架：$B_s$ 位移的三个因素

| 因素变动 | $B_s$ | 利率 | 机制——为什么会这样？ |
|----------|:-----:|:---:|------|
| 投资机会↑ | 右移 | ↑ | 企业看到可盈利项目 → 发债融资需求↑ |
| 预期通胀↑ | 右移 | ↑ | 未来用更不值钱的钱还款 → 实际借款成本↓ → 多发债 |
| 政府赤字↑ | 右移 | ↑ | 政府需借款弥补赤字 → 国债供给↑ |

### 费雪效应：唯一同时推动两条曲线的因素

预期通胀 $\pi^e$↑ 同时推动 $B_s$ 右移（借款人）+ $B_d$ 左移（贷款人要求更高收益补偿通胀），两股力量同向推高利率。费雪方程式：$i = i_r + \pi^e$。

### 应用

- **经济周期**：扩张期 $B_d$ 和 $B_s$ 同时右移，但供给位移更大 → 利率上升（顺周期）
- **低通胀与长期停滞**：投资机会匮乏 + 储蓄过剩 → 利率长期低迷（2010s 发达经济体）

## 第4步：换一个视角——流动性偏好框架（p30-39）

> **这一节在回答**：同样的均衡，用**货币市场**而不是债券市场来看，结论一样吗？

凯恩斯的视角。假设只有货币和债券两种资产。**货币需求**（$M_d$）向下倾斜——利率越高，持币机会成本越大，越不愿持有现金。**货币供给**（$M_s$）由央行外生决定，垂直线。$M_d = M_s$ 等价于 $B_d = B_s$。

### 流动性偏好框架：供求位移的影响因素

| 因素变动 | 受影响曲线 | 位移 | 利率 | 机制 |
|----------|:------:|:----:|:---:|------|
| 收入↑ | $M_d$ | 右移 | ↑ | 收入↑ → 交易性货币需求↑ → $M_d$ 右移 |
| 物价↑ | $M_d$ | 右移 | ↑ | 物价↑ → 购买同样商品需要更多货币 → $M_d$ 右移 |
| 货币供给↑ | $M_s$ | 右移 | ↓ | 央行增发货币 → 流动性效应 → 利率↓ |

**与第 2-3 步的关系**：可贷资金框架下的因素（财富↑/预期收益↑/风险↓/流动性↓/投资机会↑等）用债券曲线位移来表达；流动性偏好框架用货币曲线位移来表达。两套框架等价——货币市场均衡 $M_s=M_d$ 等价于债券市场均衡 $B_d=B_s$，因为 $M_s - M_d = B_d - B_s$。

> 📎 流动性偏好理论采用**资产市场方法**（存量分析），与可贷资金理论的流量分析相对。

|------|--------|--------|
| p30-39 | [[Liquidity Preference Framework 流动性偏好理论]] | $M_d$ 向下倾斜 + $M_s$ 垂直线 → 均衡利率；三种外生冲击的统一分析；资产市场方法定位 |

## 两个框架的关键差异：推导利率的结构不同

可贷资金框架和流动性偏好框架分析的是**同一个均衡**（$M_s - M_d = B_d - B_s$），但它们推导利率的**结构**有两个根本差异。

### 差异一：有没有"债券价格"这个中介？

**可贷资金**：要通过债券价格绕一圈。$B_d$↑ → 债券价格↑ → 利率↓。利率是债券价格的衍生物。

**流动性偏好**：不需要中介。$M_d$ 和 $M_s$ 的交点直接给出 $i^*$。利率就是持币的机会成本——利率越高，越不愿持有现金，$M_d$ 越低。

```
可贷资金：  因素 → 债券价格 → 利率           （两步推导）
流动性偏好： 因素 → $M_d$ / $M_s$ 位移 → 利率  （一步推导）
```

### 差异二：供给侧的因素丰富度完全不同（最关键差异）

**可贷资金**的 $B_s$ 有三个独立的位移因素，分属不同经济主体：

| $B_s$ 位移因素 | 主体 | 机制 |
|:--|:--|------|
| 投资机会↑ | 企业 | 看到可盈利项目 → 发债 |
| 预期通胀↑ | 借款人 | 用更不值钱的钱还款 → 多发债 |
| 政府赤字↑ | 政府 | 借款弥补赤字 → 国债供给↑ |

可贷资金可以分门别类地讨论：**企业**投资热情变了 → 利率怎么变？**政府**借钱多了 → 利率怎么变？供给侧是一个丰富的分析空间。

**流动性偏好**的供给侧只有一条垂直线 $M_s$，控制者只有**央行**。收入↑和物价↑都只能挤到需求侧 $M_d$ 上。所有实体经济冲击在流动性偏好框架里被压缩成了"货币需求变动"，分不开是谁造成的。

```
可贷资金供给侧（3条通道）：              流动性偏好供给侧（1条通道）：
  企业 → 投资机会 → $B_s$ 位移              央行 → $M_s$ 位移
  借款人 → 预期通胀 → $B_s$ 位移            其他所有因素 → 全挤到 $M_d$
  政府 → 赤字 → $B_s$ 位移
```

### 结论：两个框架各擅胜场

| | 可贷资金 | 流动性偏好 |
|------|------|------|
| **擅长分析** | 实体经济冲击如何传导到利率 | 央行动作如何传导到利率 |
| **供给侧** | 丰富（企业/借款人/政府三条通道） | 简单（只有央行） |
| **需求侧** | 四大因素（继承自资产组合选择理论） | 两个因素（收入、物价） |
| **中介** | 债券价格 | 无（利率直接出清货币市场） |

这也是为什么第 5 步（货币供给的四种效应）必须依托流动性偏好框架——只有这个框架把央行和 $M_s$ 放在舞台中央。

## 第5步：货币供给增加，利率到底怎么走？（p40-44）

> **这一节在回答**：第 4 步说 $M_s$↑ → 利率↓。但 $M_s$↑ 还会推高收入和物价 → 又反过来推高利率。那净效果是升还是降？

**问题出在哪**：第 4 步的"$M_s$↑ → 利率↓"只看了直接效应（流动性效应）。但现实中，货币供给增加后还会**间接引发**第 4 步中的另外两个 $M_d$ 位移因素——收入↑和物价↑。结果就是四条效应同时发生，方向彼此相反：

| 效应          | 机制                                | 利率方向 | 发挥作用的时间 | 对应第4步的哪个因素     |
| ----------- | --------------------------------- | :--: | :-----: | -------------- |
| **①流动性效应**  | $M_s$↑ 直接压低利率                     |  ↓   |   立即    | $M_s$ 右移       |
| **②收入效应**   | $M_s$↑ → 收入↑ → $M_d$↑             |  ↑   |   中期    | 收入↑ → $M_d$ 右移 |
| **③物价效应**   | $M_s$↑ → 物价↑ → $M_d$↑             |  ↑   |   中长期   | 物价↑ → $M_d$ 右移 |
| **④预期通胀效应** | $M_s$↑ → $\pi^e$↑ → Fisher Effect |  ↑   |  取决于预期  | 费雪效应（第3步）      |
|             |                                   |      |         |                |

**结论**：净效果不确定，看时间范围。短期内流动性效应主导（利率↓），长期后三条效应可能压倒（利率↑甚至超过初始水平）。
