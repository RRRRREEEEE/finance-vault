# Vault 笔记规则

> 操作 `D:\data\金融学\` 下文件时自动加载。

## 文件命名

- 知识点笔记：`{English Term} {中文术语}.md`
- 章节索引：`{课程名} ch{XX} {章节标题}.md`
- 英文首字母大写，中英文间一个空格

## 存放路径

- 单课程概念 → `{课程编号}-{课程名}/ch{XX}-{章节简称}/`
- 跨课程枢纽概念 → 首次出现课程的章节文件夹
- 章节索引 → 放章文件夹内

## 笔记 Frontmatter

```yaml
tags: [{课程名}, {chXX-章简称}, {知识类型}]
courses: [{课程名}]
aliases: [{英文}, {缩写}, {中文}]
importance: "{* 到 *****}"   # 必须加引号
```

## 图片

- 放在 `999-Attachments/`
- 命名：`{课程}Ch{XX} {描述}.png`
- 引用：`![[图片名.png]]` 紧跟 `> 图形说明：...`
- 图片来自课件原文件，🚫 禁止 matplotlib 自绘

## 链接

- 正文首次提到已有概念 → `[[文件名]]`
- 双向链接由 AI 完成
