# 笔记检查清单

> **用途**：每批笔记完成后逐项自检。全部通过才能标记批次完成。
> **执行频率**：每批笔记创建后，以及整章完成后各跑一次。

---

## 一、文件层面

- [ ] **存放路径正确**：笔记在 `{课程编号}-{课程名}/ch{XX}-{章节简称}/` 下，不在 Vault 根目录
- [ ] **文件命名合规**：`{English Term} {中文术语}.md`，英文首字母大写
- [ ] **无重复文件**：Grep 全 Vault 确认同名概念没有两个文件
- [ ] **章节索引存在**：`{课程名} ch{XX} {章节标题}.md` 在章文件夹内

---

## 二、Frontmatter

- [ ] `tags` 有三个：课程名、章节标签（`chXX-章简称`）、知识类型标签
- [ ] `courses` 用列表格式 `[课程名]`
- [ ] `aliases` 覆盖：英文全称、英文缩写（如有）、中文标准名、中文别名（如有）
- [ ] `created` 日期正确（`YYYY-MM-DD` 格式）
- [ ] `source` 精确到文件名 + 页码
- [ ] `chapter` 格式：`第X讲 标题`
- [ ] `importance` **加了双引号**：`"****"` 不是 `****`
- [ ] importance 评级合理：`*`=壳 `**`=次要 `***`=一般 `****`=重要 `*****`=枢纽

---

## 三、内容质量

- [ ] **只写原文内容**：无 AI 自己补充的知识、维基、课本
- [ ] **无编造数据/公式**
- [ ] **无编造跨课程链接**：原文没提的关联已删除
- [ ] **公式复杂时标了 TODO**：不确定的 LaTeX 写 `TODO: 截图替换`
- [ ] **无考试重点标注**：留给人类
- [ ] **来源原文节有内容**：摘录了原文关键句

---

## 四、板块按需启用

- [ ] `💬 通俗理解`：学术表述干涩时写了，否则已删除
- [ ] `假设条件`：原文有假设时写了，否则已删除
- [ ] `模型/推导`：原文有因果链时写了，否则已删除
- [ ] `公式与计算`：原文有公式时写了，否则已删除
- [ ] `📊 图形`：原文有图表时写了且插入了实际图片，否则已删除
- [ ] `🏛️ 应用`：原文涉及政策/实践时写了，否则已删除
- [ ] `📖 相关术语`：原文有可扩展概念时写了，否则已删除
- [ ] `🔀 跨课程链接`：只保留了原文提到的课程行
- [ ] `📝 个人批注`：AI 没有写入任何内容
- [ ] **模板字段顺序不变**

---

## 五、图片与附件

- [ ] **所有图片都是课件原图**：没自己用 matplotlib 画
- [ ] **无 TODO 占位**：Grep `TODO: 截图替换` 返回 0 结果
- [ ] **图片命名合规**：`{课程名}Ch{XX} {描述}.{png/jpg}`，描述性中文
- [ ] **图片在 999-Attachments**
- [ ] **图片引用格式正确**：`![[图片名.png]]`
- [ ] **每张图片下方有图形说明**

---

## 六、链接

- [ ] 正文中首次提到已有笔记的概念时用了 `[[链接]]`
- [ ] 相关术语节中的链接指向正确的文件名
- [ ] 跨课程链接指向的笔记确实存在（或合理空壳）
- [ ] Aliases 覆盖了链接中可能用的所有叫法
- [ ] **无死链**：Grep `[[` 找到的链接目标都能在 Vault 中找到对应文件

---

## 七、进度文件

- [ ] `阅读进度.json` 的 `processed_pages` 已更新
- [ ] `notes_generated` 列表已追加新笔记名
- [ ] `last_updated` 时间戳已更新
- [ ] 如果整章完成：`status` 改为 `"completed"`，统计了 content_notes/shell_notes

---

## 八、章节索引

- [ ] 章节索引文件已创建（整章完成时）
- [ ] 按页码线性列出所有知识点
- [ ] 每个知识点有一句话概括
- [ ] 底部统计了笔记数量和枢纽概念列表

---

## 九、最终确认

- [ ] Grep `TODO: 截图替换` → 0 结果
- [ ] Grep `TODO: 原文模糊` → 只有确实模糊的才保留
- [ ] `importance` 字段无空值（所有笔记都有星级）
- [ ] 新笔记文件数量 = 预期数量
- [ ] 向人类报告：笔记数量、TODO 项、待确认项

---

*每次检查不通过 → 修正 → 重新检查。不要带问题进入下一批。*
