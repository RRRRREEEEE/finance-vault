# 第3章 知识串讲：当比较优势和要素禀赋解释不了贸易时

## 一、一句话回答
当代国际贸易理论回答了一个传统理论回答不了的问题：为什么相似的国家之间也在大规模做贸易？

---

## 二、传统理论的尴尬

李嘉图告诉我们，比较优势决定贸易——国家之间差异越大，贸易越活跃。H-O 接着说，这种差异来自要素禀赋——穷国出口劳动密集型产品，富国出口资本密集型产品。

这个逻辑链条很美。但二战后出现了两个让传统理论闭嘴的现象：

1. **国际贸易的最大份额不在穷国和富国之间，而在富国和富国之间。** 美国最大贸易伙伴是加拿大和欧盟，不是孟加拉国。
2. **同一产业的产品双向流动。** 德国出口宝马到美国，美国出口福特到德国——这不是要素禀赋能解释的。

这就是**当代国际贸易理论**（新贸易理论）的出发点。

---

## 三、产业内贸易：同一个篮子里的双向流动

先厘清两个概念：
- **产业间贸易**：你出口纺织品，进口飞机。两个产业，单向流动。传统理论解释得了。
- **产业内贸易**：你既出口汽车又进口汽车。同一个产业，双向流动。传统理论解释不了。

怎么衡量一个产业的贸易是"内"还是"间"？Grubel 和 Lloyd 给了公式：

$$A_i = 1 - \frac{|X_i - M_i|}{X_i + M_i}$$

A 在 0 到 1 之间。越接近 1，产业内贸易越强。越接近 0，产业间贸易为主。

看数据：德国制造业产业内贸易指数从 1950 年的 35% 涨到 2025 年的 85%。全球都在朝产业内贸易走。

看看美国的例子：2005 年美国进口高尔夫球棒 1668 万根，均价 $18/根；出口 404 万根，均价 $79/根。同一类东西，低价从中国进口，高价向日本英国出口——这不是 H-O 能解释的。

---

## 四、偏好相似：林德的"需求视角"

瑞典人林德（Linder, 1961）换了一个角度：不从供给看（要素禀赋），从**需求**看。

逻辑很简单：
1. 一国人均收入水平 → 决定了这国消费者会买什么档次的东西
2. 厂商首先为**本国消费者**生产 → 本国需求催生了供给
3. 厂商把产品出口到哪？→ **收入水平相近的国家**，因为那里的消费者需求和本国重叠

所以富国跟富国做贸易——因为它们想要的东西差不多。穷国跟穷国的需求也有重叠。但穷国和富国的需求区间差距太大，重叠少，贸易潜力自然小。

这就是**重叠需求**（Overlapping Demand）的洞见：收入越近，重叠越大，贸易越多。

---

## 五、技术差距与产品生命周期：动态的比较优势

前面两章的比较优势和要素禀赋都是**静态**理论——假设技术不变。但技术一直在变。

**技术差距模型**（Posner, 1961）：美国发明了某样东西 → 美国出口 → 过段时间日本学会了 → 日本也开始生产 → 再过段时间中国也学会了 → 美国反而不生产了，变成进口国。

这个过程分三个"滞后"：
- **需求时滞**：新产品上市到消费者认可要时间
- **反应时滞**：进口国厂商反应过来"这东西有市场"要时间
- **掌握时滞**：从开始仿制到掌握技术要时间

**产品生命周期理论**（Vernon, 1966）把这个想法系统化了：

```
创新期（美国）：研发密集型 → 高价、高技术
    ↓
成熟期（日本、德国）：资本+熟练劳动密集型 → 技术扩散
    ↓
标准化期（中国）：劳动密集型 → 技术内置于设备，谁都能做
```

所以比较优势是**动态转移**的——跟着产品"变老"，从最发达的国家一路转移到发展中国家。没有哪个国家能永远占据一个产品的比较优势。

---

## 六、规模经济：两个一样的国家也能做贸易

如果两个国家完全一样——同样的要素禀赋，同样的技术水平，同样的偏好——传统理论会说：你们没有任何理由做贸易。

新贸易理论说：不，你们可以。

因为**规模经济**。一个工厂生产越多，平均成本越低。如果每个国家都只生产几个型号的车（专业化），然后互相贸易，全世界消费者能买到的车型反而更多，价格还更低。

这就是**垄断竞争与贸易**的逻辑：
- 每个国家的厂商数量减少 → 每家生产规模扩大 → 平均成本下降 → **价格更低**
- 虽然本国品种少了，但可以买到对方国家的品种 → **品种更多**

两个完全相同的国家之间发生贸易，纯粹是因为**企业对规模经济的追求**。

**外部规模经济**还有一个重要含义：**先发优势和锁定效应**。瑞士手表强不是因为瑞士工人手巧——是因为几百年前就开始了，产业集聚形成了专业化供应商、共享劳动市场、知识外溢。后来者很难打破这种"历史的偶然"。

## 七、公司内贸易与钻石模型：微观与宏观的两个补充

**公司内贸易**（内部化理论）：跨国公司的贸易有一大块发生在自己公司内部——母公司出口零部件给子公司，子公司出口成品给母公司。为什么？因为技术、专利、管理诀窍在外部市场上交易风险太大——容易泄露，不如"自己跟自己交易"。

**波特钻石模型**：德国为什么汽车强？日本为什么电子强？波特说，看六个因素：
- 生产要素：尤其是**高等要素**（人才、技术）而非自然资源
- 需求：挑剔的本国消费者逼企业进步
- 相关产业：强大的供应链
- 企业战略与竞争：激烈的国内竞争逼出国际竞争力
- + 机遇 + 政府

四阶段：要素驱动 → 投资驱动 → 创新驱动 → （危险）财富驱动

## 末、概念对照表

| 概念对 | 区别 |
|--------|------|
| **产业间贸易 vs 产业内贸易** | 间：不同产业单向流动（传统理论可解释）；内：同产业双向流动（需新理论）|
| **内部 vs 外部规模经济** | 内部：单个厂商规模决定成本（垄断竞争）；外部：产业规模决定成本（完全竞争）|
| **技术差距 vs 产品生命周期** | 技术差距：一次性的创新→模仿过程；产品生命周期：产品从生到死的三阶段动态转移 |
| **H-O vs 偏好相似** | H-O：供给角度（要素禀赋差异）；偏好相似：需求角度（收入→偏好→重叠需求）|
| **比较优势 vs 国家竞争优势** | 比较优势：静态的要素/技术差异；竞争优势：动态的综合国家创新体系 |

## 末+1、最重要的故事 / 考试最易考点

**六种理论的定位矩阵**：

| 理论 | 回答的核心问题 |
|------|-------------|
| 产业内贸易 | 同一产业为什么双向流动？→ 规模经济+产品差异 |
| 偏好相似 | 为什么富国跟富国做贸易？→ 收入相近→重叠需求 |
| 技术差距 | 技术创新如何创造贸易？→ 模仿时滞→贸易时间差 |
| 产品生命周期 | 比较优势如何转移？→ 创新→成熟→标准化三阶段 |
| 规模经济 | 完全相同的国家为何也做贸易？→ 规模效应+品种多样化 |
| 钻石模型 | 一国的竞争力从哪来？→ 四核心+两辅助+四阶段 |
