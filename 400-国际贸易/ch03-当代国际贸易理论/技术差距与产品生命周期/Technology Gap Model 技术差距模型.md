# 技术差距模型 (Technology Gap Model)

## 📌 定义
技术差距模型（Posner, 1961）认为**技术差距是国际贸易的原因**：创新国凭借技术领先优势生产和出口新产品，模仿国需要一段"模仿时滞"才能掌握该技术。持续的创造发明过程导致贸易不断产生——即使要素禀赋和偏好相似的国家之间，技术进步也会导致贸易。

## 🧠 机制

### 模仿时滞（Imitation Lag）
从新技术发明到其他国家掌握该技术的时间段，分为三类：

| 时滞类型 | 英文 | 含义 |
|----------|------|------|
| **需求时滞** | Demand Lag | 新产品出口后，消费者尚不了解，不能取代老产品的时间差 |
| **反应时滞** | Reaction Lag | 进口国生产商感受到需求压力，开始调整生产新产品的时间差 |
| **掌握时滞** | Mastery Lag | 仿制国从开始生产到达到创新国技术水平的时间间隔 |

### 贸易的时间线
```
创新国：发明 → 生产/出口 → 消费（国内生产+出口）
模仿国：进口 → 需求增长 → 开始仿制 → 掌握技术 → 出口
```

### 核心观点
> 持续的创造发明过程会导致贸易的产生。即使是要素禀赋和偏好相似的国家，在一个静态的 H-O 世界里不会展开贸易的国家之间，技术进步也会导致贸易。
