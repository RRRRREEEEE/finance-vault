# 重叠需求 (Overlapping Demand)

## 📌 定义
重叠需求是林德假说的核心概念：每个国家的人均收入水平决定了其代表性消费者的**需求品质区间**。两国收入水平越接近，需求区间的**重叠部分**越大，贸易潜力越大。这解释了为什么发达国家之间的贸易占全球贸易的最大份额。

> 💬 通俗理解：月薪 5 万的人和月薪 5 千的人想买的东西完全不一样——前者要进口车，后者要国产车。但两个月薪 5 万的人，想要的东西差不多，所以富国之间更有贸易动力。

## 🧠 机制

### 收入-需求-贸易链条
```
人均收入 → 代表性需求区间（商品品质范围）
    ↓
两国收入接近 → 需求区间重叠大
    ↓
重叠部分 = 潜在的贸易空间
    ↓
两国企业都可以为这个重叠市场生产 → 产业内贸易
```

### 为什么富国-富国贸易多
- 美国人均收入 ~$80,000，德国 ~$50,000 → 需求区间大幅重叠
- 美国人均收入 ~$80,000，孟加拉 ~$2,000 → 需求区间几乎不重叠
- 所以美国最大贸易伙伴是加拿大和欧盟，不是穷国
