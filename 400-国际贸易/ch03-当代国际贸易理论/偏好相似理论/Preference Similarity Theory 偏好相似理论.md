# 偏好相似理论 (Preference Similarity Theory / Linder Hypothesis)

## 📌 定义
偏好相似理论（林德假说，Linder 1961）是需求导向的贸易理论：一国的人均收入水平决定了其代表性消费者的偏好模式，进而产生对特定产品的需求和生产。**收入水平相近的国家之间存在重叠需求（Overlapping Demand）**，这是它们之间发生贸易的基础。

> 💬 通俗理解：富国和富国做贸易——因为有钱人的需求差不多，都想买差不多的好东西。穷国和富国需求差异大，反而不容易做同类产品的双向贸易。

## 🧠 机制

### 逻辑链条
```
一国人均收入水平↑
  → 代表性消费者的消费偏好变化 → 对工业制成品的需求增加
  → 厂商增加供给 → 出口
  → 出口对象：收入水平相近、有重叠需求的国家
```

### 重叠需求（Overlapping Demand）
- 每个国家的人均收入水平决定了其需求的商品品质区间
- 两国收入水平越接近 → 重叠的需求区间越大 → 贸易潜力越大
- 解释了为什么**发达国家之间的贸易**占全球贸易的绝大部分

## ⚖️ 易混点对照
- 与 H-O 理论的区别：H-O 从**供给**角度（要素禀赋）解释贸易，偏好相似从**需求**角度（收入→偏好→需求）解释贸易

## 📄 来源原文
> 1961年，瑞典经济学家林德提出了偏好相似理论。The per capita income level of a country will yield a particular pattern of tastes. Trade between countries with similar per capita income levels.
