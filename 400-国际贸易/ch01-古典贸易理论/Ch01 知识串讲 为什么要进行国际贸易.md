# 第1章 知识串讲：为什么要进行国际贸易？

## 一、一句话回答
因为各国在不同产品上的生产效率存在差异——按各自优势专业化生产，再通过贸易交换，双方都能获得比自给自足更多的产品。

---

## 二、重商主义：贸易是零和游戏

国际贸易理论的起点不是"自由贸易好"，恰恰相反——最早的贸易思想是**重商主义**（Mercantilism，1500-1750）。

重商主义者的世界观很简单：**货币就是财富**。一个国家有多少金银，就有多富。那金银怎么来？出口赚外汇，金银流入；进口花外汇，金银流出。所以国际贸易在他们眼里是一场**零和博弈**（Zero-Sum Game）——你赚的就是我亏的。

> 零和游戏：蛋糕大小不变，你多切一块我就少一块。

政策上怎么操作？四个字：**奖出限入**。用贸易垄断、高关税、配额堵住进口，用出口补贴鼓励出口。这套政策组合叫**贸易保护主义**（Protectionism）。

重商主义分早晚两期。早期盯着单笔贸易顺差，晚期眼光长一点——追求长期总体的顺差。但核心逻辑没变：贸易是战争，顺差是胜利。

---

## 三、重农学派：第一次说不

到了十七世纪下半叶，法国冒出一批唱反调的人——**重农学派**（Physiocrats）。

带头人是魁奈（Francois Quesney）。他的核心主张很简单：**自由经济，包括自由贸易**。财富不在金银里，在土地和农业里。你限制贸易干嘛？让粮食自由出口不好吗？

重农学派对贸易本身并不特别重视——他们重视的是农业。但他们第一次喊出了"自由贸易"的口号，为重商主义到古典贸易理论的过渡打开了缺口。

---

## 四、绝对优势：亚当·斯密的正和游戏

真正的革命来自**亚当·斯密**（Adam Smith），1776年《国富论》。

斯密做了两件事：
1. **重新定义财富**：一国的财富不在金银，在**生产能力**。
2. **重新定义贸易**：贸易不是零和游戏，是**正和博弈**（Positive-Sum Game）。

### 分工与专业化 → 绝对优势

生产能力从哪来？**分工**（Division of Labor）和**专业化**（Specialization）。把生产拆成不同任务，每人（每国）专做自己最擅长的那一块——效率就上去了。

"擅长"怎么衡量？用**劳动生产率**（Labor Productivity）：生产一单位产品需要多少劳动。你生产布只要 100 小时，我要 110 小时——你在布上就有**绝对优势**（Absolute Advantage）。

斯密的逻辑链条：
```
分工 → 专业化 → 劳动生产率提高 → 绝对优势 → 国际贸易 → 双方都获益
```

### 英葡模型

斯密用英国和葡萄牙举例。分工前，各自生产一单位产品需要的劳动时间：

| | 布 | 酒 |
|---|---|---|
| 英国 | 100小时 | 120小时 |
| 葡萄牙 | 110小时 | 80小时 |

英国在布上有绝对优势（100 < 110），葡萄牙在酒上有绝对优势（80 < 120）。分工后英国专产布（220小时全产布 = 2.2单位），葡萄牙专产酒（190小时全产酒 = 2.375单位）。然后交换——两国都比自己什么都生产时获得了更多产品。

### 绝对优势的漏洞

但斯密的理论有个致命缺陷。问一个问题：如果葡萄牙在布和酒上**都比英国强**呢？按斯密的逻辑，葡萄牙有绝对优势就全部自己生产，英国什么都别生产——两国之间没有贸易。但现实不是这样的。这个漏洞，等李嘉图来补。

## 五、比较优势：李嘉图的"反直觉"洞察

**大卫·李嘉图**（David Ricardo），1817年《政治经济学和赋税原理》，提出了**比较优势**（Comparative Advantage）。

### 核心洞察

比较优势的实质就是八个字：**"两优择其重，两劣取其轻"**。

葡萄牙在布和酒上都有绝对优势（90<100, 80<120），但比较一下：
- 布：葡萄牙 90 / 英国 100 = 0.9——葡萄牙强 10%
- 酒：葡萄牙 80 / 英国 120 = 0.67——葡萄牙强 33%

葡萄牙在酒上的优势**更大**，英国在布上的劣势**更小**。所以：
- 葡萄牙专产酒（优势更大的），把布的产能给英国
- 英国专产布（劣势更小的），把酒的产能给葡萄牙

分工结果：

| | 布 | 酒 |
|---|---|---|
| 英国 | 220（2.2单位）| 0 |
| 葡萄牙 | 0 | 170（2.125单位）|

贸易条件：1W = 1C。两国总产量都增加了——这就是**贸易利益**（Gains from Trade）：生产率提高、消费增加、劳动节约。

### 革命性结论

> 即使一国在所有产品上都比另一国效率更低，两国之间仍存在互惠贸易的基础。

这话反直觉。直觉上你觉得"我什么都比不过人家，还做什么贸易？"李嘉图告诉你：不对，你虽然不是最牛的，但你做某些事**相对不那么差**——专做这些事，拿去向别人换你相对更不擅长的东西，你就赚了。

萨缪尔森（诺贝尔奖 1970）的评价一针见血：**"比较优势理论是经济学中为数不多的正确而不显而易见的观念之一。"**

## 六、多产品扩展：谁生产什么？

现实世界有 N 种产品，不止布和酒。李嘉图的框架可以推广。

**做法**：
1. 把每种产品的两国劳动投入比算出来：$a_{Li}^*/a_{Li}$
2. 按比值从高到低排序
3. 再算出两国工资率之比：$W/W^*$

**决策规则**：
- 若 $a_{Li}^*/a_{Li} > W/W^*$ → 本国生产出口
- 若 $a_{Li}^*/a_{Li} < W/W^*$ → 外国生产出口

工资率比 $W/W^*$ 就是**分割线**——线上的产品本国做，线下的产品外国做。$W/W^*$ 越低，本国生产的产品范围越广（因为本国劳动力更便宜）。

## 末、概念对照表

| 概念对 | 区别 |
|--------|------|
| **重商主义 vs 自由贸易** | 重商主义视贸易为零和游戏，主张保护；自由贸易视贸易为正和博弈，主张开放 |
| **绝对优势 vs 比较优势** | 绝对优势比"谁成本更低"（绝对值），比较优势比"谁相对更擅长"（相对比率）。绝对优势无法解释全部贸易，比较优势可以 |
| **零和博弈 vs 正和博弈** | 零和：一方得益=另一方损失（重商主义）；正和：双方均可获益（斯密/李嘉图） |
| **重农学派 vs 重商主义** | 重商主义：财富=金银，主张干预贸易；重农学派：财富=农业，主张自由经济 |
| **早期重商主义 vs 晚期重商主义** | 早期盯单笔顺差，晚期追求长期总体顺差 |

## 末+1、最重要的故事 / 考试最易考点

**核心因果链（绝对优势 → 比较优势）**：

```
斯密：分工+专业化 → 劳动生产率差异 → 绝对优势 → 贸易互利（正和博弈）
                                ↓ 但无法解释绝对落后国家如何参与贸易
李嘉图：劳动生产率相对差异 → 比较优势 → "两优择其重，两劣取其轻" → 即使全劣仍可贸易
```

**英葡数值题**（考试大概率考）：
- 分工前：英国布 100h/酒 120h，葡萄牙布 90h/酒 80h
- 分工后：英国专产布 2.2单位，葡萄牙专产酒 2.125单位
- 贸易条件：1W:1C
- 结论：即便葡萄牙在两种产品上都有绝对优势，比较优势仍保证两国均获利
