# 重农学派 (Physiocrats)

## 📌 定义
重农学派是十七世纪下半期兴起于法国的经济学派，反对重商主义的国家干预政策，主张经济自由和自由贸易，尤其重视农业的经济地位。

## 🧠 机制

### 代表人物
- **Francois Quesney（魁奈）**（1694-1774）：重农学派核心人物

### 核心主张
- **自由经济**：反对重商主义对贸易的干预
- **自由贸易**：尤其主张谷物的自由出口
- **农业本位**：认为农业是财富的根源，而非贸易顺差

### 与重商主义的对立
| | 重商主义 | 重农学派 |
|---|---|---|
| 财富来源 | 贵金属/贸易顺差 | 农业生产 |
| 贸易政策 | 国家干预/保护 | 自由贸易 |
| 经济观 | 零和游戏 | 自由放任 |

## 📖 相关术语

## 📄 来源原文
> 从十七世纪下半期开始，首先在法国出现了反对重商主义政策，主张经济自由和重视农业的思想，从而逐渐形成了重农学派。核心思想是主张自由经济，包括自由贸易。重农学派从"自由经济"的基本理念和法国农民的实际利益出发，反对重商主义对贸易进行干预的政策，提出了自由贸易的口号，尤其主张谷物的自由出口。
