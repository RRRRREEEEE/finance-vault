# 贸易保护主义 (Protectionism)

## 📌 定义
贸易保护主义是一国通过政策手段限制进口、鼓励出口以保护本国产业和维持贸易顺差的贸易政策立场。重商主义是保护主义的早期代表。

## 🧠 机制

### 主要工具
| 工具 | 说明 |
|------|------|
| 贸易垄断 | 国家控制特定商品的进出口权 |
| 高关税 | 对进口商品征收高额税，提高其价格 |
| 配额 | 限制进口商品的数量 |
| 出口补贴 | 对本国出口商品给予财政补贴 |

### 核心逻辑
奖出限入——鼓励出口换取金银流入，限制进口防止金银流出。
