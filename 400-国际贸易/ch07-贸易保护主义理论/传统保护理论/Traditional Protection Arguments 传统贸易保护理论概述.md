# 传统贸易保护理论概述 (Traditional Arguments for Protection)

## 📌 定义
传统贸易保护理论是区别于新贸易理论时代"战略性贸易政策"的早期保护主义学说体系，包含六大核心论点：幼稚产业保护、保护就业、改善贸易条件、改善国际收支、保护公平竞争、非经济目标。

## 🔗 前置知识
[[Infant Industry Argument 幼稚产业保护论]]、[[Tariff 关税]]

## 🧠 机制

### 六大传统论点

| 论点         | 核心逻辑                               | 局限性                         |
| ---------- | ---------------------------------- | --------------------------- |
| **幼稚产业保护** | 新兴工业需临时保护至具有国际竞争力                  | 需正确选择保护对象、期限                |
| **保护就业**   | 关税可增加特定行业就业                        | 减少进口 → 可能影响出口行业就业；总体失业不一定下降 |
| **改善贸易条件** | 大国限制进口 → $P_m$下降 → $T = P_x/P_m$改善 | 仅适用于大国                      |
| **改善国际收支** | 减少进口 → 改善贸易差额                      | 假定出口不变（不现实）；忽视经济政治连锁反应      |
| **保护公平竞争** | 抵消外国倾销和补贴等不公平行为                    | 可能被滥用（扩大到不对等开放市场）           |
| **非经济目标**  | 民族自尊、国家安全、社会公平                     | 经济学无法量化                     |
