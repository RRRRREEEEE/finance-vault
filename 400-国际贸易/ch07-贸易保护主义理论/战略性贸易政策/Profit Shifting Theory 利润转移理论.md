# 利润转移理论 (Profit Shifting Theory)

## 📌 定义
利润转移理论是战略性贸易政策的核心机制，包含三个方向：用关税抽取外国垄断利润（战略进口）、用进口保护促进出口（克鲁格曼）、用出口补贴转移利润（战略出口）。三者的共同逻辑是：通过政府干预将寡头垄断市场的超额利润从外国企业**转移**到本国企业。

## 🧠 机制

### 三条转移路径

| 路径 | 手段 | 机制 |
|------|------|------|
| **战略进口政策** | 关税 | 对外国垄断厂商征关税 → 抽取其垄断利润 → 利润从外国转移到本国政府 |
| **进口保护促出口** | 保护国内市场 | 保护本国厂商的国内市场 → 扩大规模降低成本 → 提高在国外市场竞争力（克鲁格曼） |
| **战略出口政策** | 出口补贴 | 在寡头市场补贴本国企业 → 提高市场占有率 → 获得超额利润（波音vs空客） |

### 波音vs空客博弈模型

**情况一：势均力敌**
- 初始：双方都生产 → 各亏损5 | 一方生产一方不生产 → 生产者赚100
- 需政府干预打破僵局
- 政府补贴25给波音 → 空客退出 → 波音赚125 → 净利润100 > 补贴25 ✅

**情况二：某公司有先天优势（如空客技术领先）**
- 初始：空客生产成本优势 → 即使波音也生产，空客赚5、波音亏20
- 无补贴时波音退出市场 → 空客独占赚125
- 政府补贴25给波音 → 波音生产赚5 → 空客利润从125降到5
- 波音净利5，但补贴花了25 → 净亏损20 ❌ 补贴得不偿失！

### 核心教训
> 战略性贸易政策的成功必须以**利润转移部分超过补贴额或关税保护成本**为条件。
