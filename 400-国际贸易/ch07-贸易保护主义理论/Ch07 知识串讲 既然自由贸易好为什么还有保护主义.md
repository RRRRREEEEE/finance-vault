# 第7章 知识串讲：既然自由贸易好，为什么还有保护主义？

## 一、一句话回答
经济理论有三种回答：传统理论说"国家需要时间成长"、战略贸易说"政府能帮企业在博弈中赢"、政治经济学说"少数获益者买通了政策"。

---

## 二、传统保护理论：扶上马，送一程

前六章我们花了大量篇幅证明自由贸易好——比较优势、H-O、规模经济，全都指向同一个结论：开放比封闭强。那为什么从汉密尔顿到特朗普，保护主义从未消失？

**幼稚产业保护论**是最古老也最站得住脚的回答。逻辑很简单：你让一个刚学走路的孩子去跟成年人赛跑，他肯定输。所以新兴工业需要关税这根"拐杖"扶一段时间。

汉密尔顿1791年就在推这个——美国刚建国，英国工业碾压全球，如果不保护，美国永远只能当农业国。《麦金利关税法》把关税提到49.5%，保护了美国工业半个多世纪。结果呢？1900年美国工业总产值占全球30%，世界第一。你说保护主义有没有用？

但德国人李斯特把这个理论推到了哲学高度。他的名言："结果子的树比果实本身价值更大。"自由贸易让你买到便宜货（果实），但保护贸易让你自己种出果树（**生产力**）。你选哪个？

李斯特的五阶段论很有操作性：不是永远保护，只在**农工业时期**保护。对象也很挑剔——只有"刚刚开始发展且有强有力外国竞争者"的幼稚工业才需要保护，农业不需要，没竞争者的也不需要。保护时间以**30年**为限。这是个有退出机制的保护主义。

那怎么判断一个产业值不值得保护？三个标准层层递进：
- **穆勒标准**：保护后能不能实现规模经济？（最松，只看潜力）
- **巴斯塔布尔标准**：未来利润贴现值能不能超过保护成本？（加入经济账）
- **肯普标准**：除了自己能赚钱，能不能带飞其他产业？（要求最高——必须产生外部经济）

至于其他传统论点——保护就业（当心得不偿失，进口行业的就业增加可能抵消出口行业的就业减少）、改善贸易条件（仅大国有效）、改善国际收支（假定出口不变不现实）、公平竞争（容易滥用）——都有道理但都有局限。

**《芯片与科学法》** 是传统保护理论的当代回响。527亿美元补贴半导体产业，同时要求接受资金的企业10年内不得在中国扩产。这跟汉密尔顿的逻辑一脉相承——国家安全+产业保护。只不过曼哈顿变成了硅谷。

## 三、战略性贸易政策：博弈论进入贸易理论

20世纪80年代，贸易理论发生了范式转变。Brander和Spencer站出来说：传统自由贸易理论的前提是**完全竞争**，但现实中很多产业是**寡头垄断**——只有两三家企业在全球竞争。在这种市场结构下，政府干预可能**提高**而不是降低本国福利。

这就是著名的**波音vs空客博弈**：

没有补贴时，市场可能陷入囚徒困境——两家都生产都亏损，但都不敢退出怕对方独占。政府给波音补贴25 → 波音生产有利可图 → 空客退出 → 波音赚125 → 净利润100远超补贴25。政府干预"解锁"了僵局。

但情况二打脸更快：空客有技术优势时，补贴波音只是把空客的利润从125减到5，波音自己才赚5，补贴花了25 → 净亏20。这说明什么？**战略性贸易政策的信息门槛极高**——政府必须精确知道每家企业的成本结构，否则补贴可能打水漂。

**利润转移理论**的三条路径各有侧重：战略进口政策（用关税抽外国垄断者的租）、进口保护促进出口（克鲁格曼——先在国内养大再出去打）、战略出口政策（发补贴帮企业在国际市场抢份额）。

**外部经济理论**补了一个关键视角：有些产业（比如半导体、航空航天）的价值远超过该产业本身的利润——它们的技术会外溢到整个经济。但企业自己做投资决策时不会考虑这些外溢效应 → 市场投资不足 → 政府补贴来填这个"外部性缺口"。

但所有战略贸易政策都必须满足苛刻条件：额外收益要超过补贴成本、能迫使对手削减产能、别国不会报复……这些条件在现实中很难同时满足。

## 四、政治经济学：为什么坏政策能活下来？

如果战略性贸易政策的条件那么难满足，为什么保护主义仍然盛行？政治经济学给出了最诚实的答案：**保护主义不是经济逻辑的产物，是政治过程的产物。**

Rodrik框架把贸易政策拆成供需两面：需求方是利益集团（组织游说），供给方是政策制定者（权衡选民福利与政治捐款）。三个模型解释了这个供需匹配：

- **中点选民模型**：政治家迎合中间选民的偏好。如果中间选民是受进口冲击的制造业工人 → 保护主义当选。
- **集体行动理论**（Olson）：小群体容易组织（利益集中），大群体存在搭便车（利益分散）。关税让厂商赚几亿 → 他们愿意花几百万游说。让两亿消费者每人多花几块钱 → 没人会为这几块钱上街。
- **竞选贡献模型**：特殊利益集团给政客捐款，政客在选票和钞票之间权衡。

**美国蔗糖政策**是教科书级的案例：
- 国内蔗糖价 = 国际价的2倍以上
- 蔗糖业仅占农业产值1%、农场0.5%、劳动力0.04%
- 每年蔗糖业多挣10亿 → 消费者多付20亿
- 如果自由贸易，蔗糖业只需3000人（现在6万人）
- 每个消费者每年多付不到$10 → 没人抗议；每个蔗糖农场每年多赚几百万 → 全力游说

这就是保护主义的政治经济学本质——**集中受益、分散受损**的利益结构让"坏"政策在民主制度下也能持续近百年。

## 五、概念对照表

| 概念对 | 区别 |
|--------|------|
| 穆勒标准 vs 巴斯塔布尔标准 | 穆勒只看能否实现规模经济；巴斯塔布尔要求保护成本<未来利润现值 |
| 巴斯塔布尔标准 vs 肯普标准 | 巴斯塔布尔看产业自身利润；肯普要求产业产生外部经济效应 |
| 汉密尔顿 vs 李斯特 | 汉密尔顿最早提出+美国实践；李斯特系统理论化+生产力理论+五阶段论 |
| 战略进口政策 vs 战略出口政策 | 进口：用关税抽租；出口：用补贴抢份额 |
| 中点选民模型 vs 集体行动理论 | 中点选民看选民偏好分布；集体行动看组织成本差异 |
| 传统保护理论 vs 战略性贸易政策 | 传统：完全竞争前提下的临时保护；战略：不完全竞争下政府干预可提福利 |

## 六、最重要的故事 / 考试最易考点

**必考1：幼稚产业保护的三大选择标准。** 穆勒（规模经济）、巴斯塔布尔（成本收益净现值）、肯普（外部经济）。能写出每个标准的判断逻辑。

**必考2：波音空客博弈矩阵。** 能画出两种情形（势均力敌 vs 一方有优势）的博弈矩阵，分析补贴是否值得。结论：补贴成功条件苛刻——利润转移>补贴成本。

**必考3：保护主义的政治经济学三模型 + 蔗糖案例。** 中点选民、集体行动（搭便车）、竞选贡献。能解释美国蔗糖政策为什么持续近百年——集中受益分散受损。
