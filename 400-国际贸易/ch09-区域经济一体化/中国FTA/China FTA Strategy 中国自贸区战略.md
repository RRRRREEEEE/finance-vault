# 中国自贸区战略 (China's FTA Strategy)

## 📌 定义
中国1990年代才开始参与区域一体化实践，2001年后加速推进。目前已签署**21个**自贸协定，正谈判十余个。战略方向：以周边为基础，形成面向全球的**高标准自贸区网络**。二十大提出"扩大面向全球的高标准自由贸易区网络"。

## 🔗 前置知识
[[RCEP 区域全面经济伙伴关系协定]]、[[TPP and CPTPP 跨太平洋伙伴关系协定]]

## 🧠 机制

### 主要里程碑

| 阶段 | 事件 |
|------|------|
| 1991 | 加入APEC（论坛性合作） |
| 2000 | 首次提出中国-东盟FTA构想 |
| 2002 | 签署中国-东盟全面经济合作框架协议 → 2010年全面建成 |
| 2015 | 中国-东盟FTA升级议定书（首个升级协议） |
| 2015 | 中韩FTA、中澳FTA签署 |
| 2022 | RCEP生效（中日首次FTA） |
| 进行中 | 中日韩FTA谈判、CPTPP加入研究 |

### 中国-东盟FTA（第一个正式区域一体化组织）
- 2002启动→2004"早期收获计划"（500种农产品先降税）
- 2005年7000个税目产品开始降税至20%
- 2007年服务贸易协定 → 2009年投资协议
- 2010年与老成员全面建成，**2015年**与新成员建成

### 中韩FTA（"含金量最高"）
- 覆盖货物贸易、服务贸易、投资共**17个领域**
- 包含电子商务、竞争政策、政府采购、环境等"21世纪议题"
- 货物自由化比例超"税目90%、贸易额85%"
- 预计中国GDP增1-2%，韩国增2-3% → **13.5亿人口、GDP 11万亿**的共同市场

### 未来方向
- 依托"**一带一路**"拓展FTA空间
- 利用亚投行、金砖银行、中国-东盟FTA三大平台
- 推进**制度型开放**（规则、规制、管理、标准）
- 向CPTPP标准靠拢
