# 区域一体化效应 (Effects of Regional Integration)

## 📌 定义
区域经济一体化的效应分为静态效应与动态效应。静态效应是资源配置的一次性改变（贸易创造/转移），动态效应是长期经济增长机制的变化。两者共同决定一体化的最终福利影响。

## 🔗 前置知识
[[Trade Creation and Trade Diversion 贸易创造与贸易转移]]

## 🧠 机制

### 静态效应（除贸易创造/转移外）
- **节约管理成本**：取消海关、边检等行政成本
- **改善集体贸易条件**：联盟整体议价能力增强
- **更强谈判能力**：作为一个集团在国际谈判中更有分量

### 动态效应

| 效应 | 机制 |
|------|------|
| ✅ **促进竞争** | 市场扩大 → 更多竞争者 → 效率提升 |
| ✅ **规模经济** | 面向更大市场生产 → 单位成本下降 |
| ✅ **投资增加** | 成员国厂商市场扩大增投 + 非成员国绕壁垒增投(跳板投资) |
| ✅ **要素流动性增强** | 资本、劳动力在区域内的配置效率提升 |

### 负面影响
- 拉大成员间经济发展差距
- 对外排他性（可能引发贸易战）

## 📄 来源原文
> 其他静态效应包括节约管理成本、改善集体贸易条件、更强谈判能力。动态效应包括促进竞争提高劳动生产率、实现规模经济、投资增加、要素流动性增强。负面影响为拉大成员间差距和对外排他性。
