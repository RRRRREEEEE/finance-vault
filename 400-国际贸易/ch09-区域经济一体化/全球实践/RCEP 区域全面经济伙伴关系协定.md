# RCEP 区域全面经济伙伴关系协定 (Regional Comprehensive Economic Partnership)

## 📌 定义
RCEP是2012年由东盟主导、2020年11月签署、**2022年1月1日生效**的全球**体量最大**的自由贸易协定。15个成员（东盟10国+中日韩澳新），覆盖**22.7亿人口**、GDP **26万亿美元**、均占全球约30%。印度于2019年退出。

> 💬 通俗理解：RCEP是亚洲版的"大自贸区"——体量全球最大、标准比CPTPP低但更务实、照顾了柬埔寨老挝等最不发达国家。对中国来说，首次与日本建立自贸关系是最大突破。

## 🧠 机制

### RCEP vs CPTPP 对比

| 维度 | RCEP | CPTPP |
|------|------|-------|
| GDP总和 | CPTPP的**2倍多** | — |
| 人均GDP | $11,528 | $21,225 |
| 货物贸易占比 | 出口31%/进口26% | 出口15%/进口15% |
| 服务投资开放 | **正面清单+负面清单**结合 | 全负面清单 |
| 知识产权 | 保护范围较窄 | 保护范围更广、执法更严 |
| 电子商务 | 数字流动自由度较低 | 跨境数据流动更开放 |
| 政府采购 | 无开放清单 | 有开放清单 |
| 国企/劳工/环境 | 无 | 有专门章节 |
| 争端解决 | 约束较弱，政策空间更大 | 约束更强 |
| 发展中国家待遇 | 较多特殊和差别待遇 | 标准统一、要求严格 |

### RCEP的突破
- 整合升级了东盟与各国的"10+1"规则
- **首次建立了中日、日韩自贸关系**（此前中日韩之间无FTA）
- 贸易便利化规则达到CPTPP水平
- 纳入柬埔寨、老挝、缅甸等最不发达国家 → **包容性**

### 对中国意义
1. 稳固周边：强化与东盟、日韩经贸联系
2. **中日首次FTA**：释放最大互补潜力
3. 对接高标准规则，为加入CPTPP铺路
