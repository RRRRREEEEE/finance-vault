# 欧盟 (European Union, EU)

## 📌 定义
欧盟是全球一体化程度最高的区域组织，从1951年6国煤钢共同体起步，历经7次扩大至27国，2020年英国脱欧。1992年单一欧洲法案实现四大自由流通，1994年《马约》建立经济货币联盟，1999年启动欧元（2002年成为欧元区唯一合法货币，18国）。

## 🧠 机制

### 一体化历程

| 时间 | 里程碑 |
|------|--------|
| 1951 | 法德意比荷卢6国成立煤钢共同体 |
| 1957 | 《罗马条约》→ 欧洲共同体（共同市场+共同对外关税+要素自由流动） |
| 1973-2013 | 七次扩大至28国 |
| 1992 | 单一欧洲法案：取消所有边境管制、产品标准互认、开放公共采购、资本自由流动 |
| 1994 | 《马斯特里赫特条约》→ 欧盟（经济货币联盟） |
| 1999 | 启动欧元 → 2002年成为唯一合法货币（18个欧元区国家） |
| 2020 | 英国正式脱欧 → 27国 |

## 📄 来源原文
> 1951年6国煤钢共同体→1957年罗马条约欧洲共同体→1992年单一欧洲法案→1994年马约→欧元。英国2020年脱欧，现有27个成员国。
