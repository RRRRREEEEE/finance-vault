# 北美自贸协定与美墨加协定 (NAFTA & USMCA)

## 📌 定义
NAFTA（1994-2020）是美加墨三国组成的自由贸易区。2020年被**USMCA**（美墨加协定）取代——保留NAFTA大部分内容，但收紧原产地规则、新增劳工价值含量要求、引入16年有效期+每6年重谈的条款。

## 🔗 前置知识
[[Free Trade Zone 自由贸易区]]

## 🧠 机制

### NAFTA（1994年生效）
- 取消商品和服务跨境流动壁垒
- 知识产权保护 + 环境标准
- 取消大部分FDI限制

| 收益 | 成本 |
|------|------|
| 扩大区域基地 | 美国就业流失到墨西哥 |
| 美加企业生产成本下降 | 墨西哥企业面临美加高效竞争 |
| 墨西哥获投资和就业 | 环境退化、国家主权流失 |
| 消费者低价 | — |

### USMCA关键改动（2020年）
| 条款 | NAFTA | USMCA |
|------|-------|-------|
| 汽车原产地比例 | 62.5% | **75%**（零部件不低于75%北美产才能零关税） |
| 劳工价值含量 | 无 | 40-45%零部件须由时薪≥**$16**工人生产（美加$20+，墨$7） |
| 汽车进口配额 | 无 | 加墨乘用车对美出口≤**260万辆**/年，超额加征25%关税 |
| 有效期 | 无期限 | **16年**，每6年重谈一次 |
| 日落条款 | 无 | 任何一方可退出 |

## 📄 来源原文
> NAFTA于1994年正式运转。USMCA 2020年替代NAFTA，汽车原产地比例上调至75%，40%-45%零部件须由时薪不低于16美元的工人生产。新协议有效期16年，每6年重谈。
