# 第9章 知识串讲：区域一体化的理论逻辑与全球版图

## 一、一句话回答
区域一体化是在多边贸易谈判越来越难的背景下，各国绕开WTO在区域内或双边框架下实现自由贸易的"B计划"——有贸易创造也有贸易转移，欧亚北美三大板块格局已定。

---

## 二、一体化的梯子：六个台阶

区域一体化不是"要不要"的二元选择，而是一个**连续光谱**，从最低到最高有六级台阶：

1. **优惠贸易安排**：只给你特别关税优惠（英联邦特惠制），最松散的
2. **自由贸易区**：内部零关税、各自对外独立（中国-东盟、NAFTA）
3. **关税同盟**：内部零关税 + **统一对外关税**（安第斯共同体5-20% CET）
4. **共同市场**：关税同盟 + **资本和劳动力自由流动**（南方共同市场）
5. **经济同盟**：共同市场 + **统一经济政策**（财政、货币、汇率；欧盟）
6. **完全经济一体化**：超国家机构管一切（理论上存在，现实中尚无）

每一级的核心增量：
- FTA → 关税同盟：加了CET（共同对外关税），没有CET的FTA需要原产地规则防止第三国借道
- 关税同盟 → 共同市场：加了要素流动
- 共同市场 → 经济同盟：加了政策协调，让渡了宏观经济主权

你记不住六个就记三档：FTA（只管商品）< 共同市场（管商品+要素）< 经济同盟（什么都管）。

---

## 三、贸易创造 vs 贸易转移：一体化是好事还是坏事？

Viner在1950年给出了至今仍是金标准的分析框架。区域一体化不是必然好也不是必然坏——取决于**贸易创造和贸易转移哪个更大**。

- **贸易创造**：德国汽车本来从法国高价买 → 取消关税 → 改从意大利低价买 → 福利提升 ✅
- **贸易转移**：德国汽车本来从日本低价买 → 建立关税同盟、日本被CET挡住 → 改从意大利高价买 → 福利下降 ❌

道理很简单：一体化在内部创造了一个"自由贸易岛"，但如果这个岛把更便宜的外部供应商挡在外面了，那岛内福利可能反而下降。

其他静态效应也值得记：**节约管理成本**（边境不用查了）、**改善集体贸易条件**（抱团议价）、**更强谈判能力**（一个人说话没人听，一群人说话有人听）。

动态效应更像"增长红利"：市场竞争加剧 → 效率提升；市场扩大 → 规模经济；投资增加（内部厂商扩大投资 + 外部厂商"跳板"投资绕壁垒）。

但别忘了负面影响：**拉大成员间差距**（富的更富穷的更穷）和**对外排他性**（你搞小圈子，别人也搞小圈子 → 世界被切割成贸易集团）。

## 四、四次浪潮：从政治抱团到经济抱团

二战后的一体化演进反映了国际格局的变迁：

- **第一次**（1950s-70s）：政治安全逻辑主导。法德煤钢共同体 → 欧共体，本质是"把战争资源绑在一起，就打不起来了"。
- **第二次**（1970s-80s初）：经济利益开始抬头，但政治安全仍在。欧共体扩大，澳新FTA。
- **第三次**（1980s末-2000s）：转折点——GATT/WTO多边谈判越来越难（145个成员怎么谈），各国不耐烦了。转向区域/双边"小圈子"快谈。欧盟升级为经济货币联盟，NAFTA诞生。
- **第四次**（21世纪）：爆炸式增长。2000年前累计才82个RTA，2001-2021年新增271个。特征：不再限于邻国（智利跟中国签FTA）、新议题（数字、环境、劳工）、双边为主、亚洲猛追。

现在每个WTO成员至少参与一个RTA。欧盟45个、智利30个、新加坡27个。中国16个，日本18个，韩国20个。

## 五、三大板块的核心玩家

### 欧洲：欧盟
一体化程度最深。从6国煤钢到27国经济货币联盟，欧元18国。1992年单一法案实现了真正的四大自由。英国脱欧是个注脚——说明一体化太深也有代价（主权让渡）。

### 北美：NAFTA → USMCA
1994年NAFTA是"发达+发展中"的经典实验。好处是墨西哥获投资就业、美加企业降成本，代价是美国工人丢饭碗、墨西哥环境恶化。2020年USMCA更"硬"了——汽车原产地62.5%→75%，还得$16时薪工人生产40-45%的零部件，实质是**用贸易规则抬高墨西哥劳动力成本，保护美国工人**。

### 亚洲：RCEP + CPTPP双轨
RCEP是全球体量最大的FTA——22.7亿人、26万亿GDP。关键是**中日首次建立FTA关系**，释放两国产业互补潜力。但标准比CPTPP低——正面清单+负面清单混合、无国企章节、数字贸易自由度不如CPTPP。

CPTPP是标准的天花板——"三零"、负面清单全开放、边境内措施、数字贸易。但少了美国，GDP占比从40%跌到13%。中国想加入，但国企和知识产权条款是硬骨头。

### IPEF：美国的新布局
2022年启动，14国、无中国。最大的特点是没有关税减让——不是传统FTA，而是规则制定权争夺。用劳工、环境、数字标准重塑中国周边的贸易环境，减少对华供应链依赖。

## 六、中国的FTA版图

中国1990年代才开始玩区域一体化，但2001年后加速惊人。21个已签FTA + 十余个在谈。路线很清楚：以周边为基础（东盟是第一个、RCEP是集大成），面向全球（中韩/中澳/未来中欧）。

东盟-中国FTA是最有战略意义的——中国参与的第一个正式区域一体化组织，2002启动 → 2010建成 → 2015升级。中韩FTA是含金量最高的——17个领域全覆盖、含21世纪议题、货物自由化超90%。

二十大定调："扩大面向全球的高标准自由贸易区网络。"翻译成大白话：一边继续用RCEP稳住亚洲后院，一边研究加入CPTPP提高标准，同时利用一带一路拓展FTA空间。

## 七、概念对照表

| 概念对 | 区别 |
|--------|------|
| FTA vs 关税同盟 | FTA各自对外、需原产地规则；关税同盟统一对外关税 |
| 关税同盟 vs 共同市场 | 共同市场多了一个要素自由流动 |
| 贸易创造 vs 贸易转移 | 创造：低成本成员国→高成本本国(好)；转移：高成本成员国→低成本非成员国(坏) |
| USMCA vs NAFTA | USMCA原产地比例提高、加劳工工资门槛、有期限条款 |
| RCEP vs CPTPP | RCEP体量大标准较低更包容；CPTPP标准高但少美国 |
| APEC vs IPEF | APEC自愿协商含中美；IPEF美国主导排除中国无关税减让 |

## 八、最重要的故事 / 考试最易考点

**必考1：一体化六个层次。** 能画出从PTA到完全经济一体化的阶梯，说出每一级在上一级基础上加了什么。

**必考2：贸易创造 vs 贸易转移。** Viner分析框架——能举例说明什么情况福利提升、什么情况下降。

**必考3：RCEP vs CPTPP对比。** 体量/标准/开放模式/成员差异。特别是中日首次FTA、印度退出、中国加入CPTPP的挑战。
