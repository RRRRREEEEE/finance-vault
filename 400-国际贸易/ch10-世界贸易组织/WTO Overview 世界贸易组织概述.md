# 世界贸易组织概述 (WTO Overview)

## 📌 定义
世界贸易组织（WTO）于**1995年1月1日**正式成立，取代GATT成为全球唯一的多边贸易体制管理机构。WTO有**164个成员**，继承了GATT的合理内核，但具有法人资格、永久性机构、更广泛的管辖范围和更强的争端解决效力。

> 💬 通俗理解：GATT是临时搭的帐篷（没有法人资格、只管货物贸易），WTO是永久性的大楼（有法人、管货物+服务+知识产权+投资、有强制性的争端解决）。

## 🧠 机制

### WTO vs GATT

| 维度 | WTO | GATT |
|------|-----|------|
| 法律人格 | **有**法人资格 | 无 |
| 组织机构 | **永久性**机构，设有秘书处 | 无组织基础 |
| 协议永久性 | 承诺完整、永久 | 临时性协议 |
| 管辖范围 | 货物 + **服务 + 知识产权 + 投资** | 仅货物贸易 |
| 协议性质 | 几乎全是**多边**协议（全体必须接受） | 包含许多诸边协议 |
| 执行效力 | 争端解决**效率高、效力强** | 低效、较弱 |

### 宗旨与目标
- 提高生活水平 + **可持续发展**（新增）
- 扩大服务的生产与贸易（新增）
- 通过谈判推动自由化 → 建立公平争端解决机制

### 五大基本原则

| 原则 | 核心内容 |
|------|----------|
| **非歧视** | 最惠国待遇（对所有成员一视同仁）+ 国民待遇（外国企业与本国企业同等待遇） |
| **自由贸易** | 互惠互利为基础 → 削减关税和非关税壁垒 |
| **公平竞争** | 禁止扭曲市场竞争的措施，纠正不公平贸易行为 |
| **透明度** | 贸易措施必须公布，不公布不得实施；须通知WTO |
| **差别待遇** | 以过渡期方式体现对发展中国家的特殊照顾 |

### 六大功能
1. 监督促进各项协议落实
2. 继续推动多边贸易谈判
3. 协调裁决成员间贸易纠纷
4. 监督检查成员国贸易政策
5. 协调全球经济政策（与IMF、世行合作）
6. 向发展中国家提供技术援助和培训
