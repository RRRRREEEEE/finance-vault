# 世贸组织改革危机 (WTO Reform Crisis)

## 📌 定义
WTO当前面临前所未有的危机：单边主义和保护主义盛行，权威性和有效性受挑战，**上诉机构自2019年正式停摆**，争端解决机制面临瘫痪。围绕特殊与差别待遇、协商一致原则、规则更新等问题，发达国家与发展中国家分歧激烈。

> 💬 通俗理解：WTO像一台老旧的电脑——硬件的争端解决被美国用否决权"拔了电源"，软件的上诉机构只剩一个法官没法开庭；发达国家想改系统（取消发展中国家特权），发展中国家说"你先把我的权益保障好"。

## 🧠 机制

### 四大争议焦点

#### 1. 上诉机构停摆（最严重危机）
- 上诉机构常设7名法官，最少需3人才能审理
- **2017年起美国持续阻止法官任命**
- 2019年12月仅剩1名法官 → **正式停摆**
- 2020年11月最后一位法官（赵宏）离任 → **完全停摆**
- 2025年1月美国**第83次**否决启动遴选程序
- 欧盟和中国牵头建立《多方临时上诉仲裁安排》（MPIA）作为替代，但参加方有限

#### 2. 特殊与差别待遇（S&D）争议

| 立场 | 观点 |
|------|------|
| **美欧日** | 一些"自称发展中国家"的先进经济体享受与小国同等的优惠不公平；应重新分类 |
| **中国+发展中国家** | 应保障发展中国家正当权益；WTO应继续支持发展中国家的特殊与差别待遇 |

#### 3. 协商一致原则
- **欧美日**：协商一致导致多哈回合停滞，应改革
- **中国**：协商一致是WTO"成员驱动型"核心要义；放弃等于放弃平衡发达国家与发展中国家权利义务的基石

#### 4. 规则更新与新议题
- 各成员就数字经济、电子商务等新议题达成共识
- 2018年美日欧联合声明：加速电子商务和数字贸易谈判
- 国企改革、补贴等议题仍存争议

### 新议题清单
| 议题 | 内容 |
|------|------|
| 贸易与环境 | 环境产品自由化、多边环境协议与WTO的关系 |
| 贸易与竞争政策 | 将竞争政策纳入多边体制 |
| 贸易与投资 | 建立多边投资框架 |
| 贸易与劳工标准 | 美欧推动：对违背核心劳工标准的产品采取贸易限制 |
| 电子商务 | 数字贸易规则、关税、隐私保护 |
