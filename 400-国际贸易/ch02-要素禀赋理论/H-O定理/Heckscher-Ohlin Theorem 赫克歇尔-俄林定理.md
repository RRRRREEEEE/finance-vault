# 赫克歇尔-俄林定理 (Heckscher-Ohlin Theorem)

## 📌 定义
H-O 定理是要素禀赋理论的核心命题：一国应出口密集使用其**相对充裕**生产要素的产品，进口密集使用其**相对稀缺**生产要素的产品。劳动相对丰富的国家出口劳动密集型产品，资本相对丰富的国家出口资本密集型产品。

> 💬 通俗理解：有什么就用什么。劳动力多→做劳动密集型产品出口，资本多→做资本密集型产品出口。

## 🧠 机制

### 理论逻辑

```
两种要素：资本(K)、劳动(L)
要素禀赋：一国K充裕、一国L充裕
    ↓
相对成本差异：充裕要素便宜 → 密集使用该要素的产品成本低
    ↓
相对价格差异：同种产品两国价格不同
    ↓
相互贸易：出口本国便宜的产品，进口对方便宜的产品
```

### General Equilibrium Framework
在两国、两产品、两要素的新古典框架中，给定相同的技术、相同的偏好、完全竞争等假设，要素禀赋差异是贸易发生的唯一原因。

### 局部均衡示例：大米贸易
- 中国（劳动丰裕）→ 大米（劳动密集型）价格低 → **出口**
- 美国（资本丰裕）→ 大米价格高 → **进口**
- 两国价格在**国际市场价格**（均衡价格）上达成交易
