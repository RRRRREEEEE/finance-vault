# 荷兰病 (Dutch Disease)

## 📌 定义
荷兰病是指一国自然资源部门的繁荣反而导致其他出口部门（尤其是制造业）萎缩的现象。得名于 20 世纪 60 年代荷兰发现天然气后制造业衰退的经历。

> 💬 通俗理解：发现石油/天然气/矿产好像是好事，但可能害了制造业——一个部门的繁荣挤出另一个部门。

## 🔗 前置知识
[[Magnification Effect 放大效应]]、[[Stolper-Samuelson Theorem 斯托尔珀-萨缪尔森定理]]

## 🧠 机制

### 传导链条
```
发现自然资源（天然气）
  → 大量开发出口 → 外汇大量流入
  → 本币升值 → 制造业产品在国际市场上变贵
  → 制造业出口竞争力下降 → 制造业萎缩
```

### 与 S-S 定理的关联
- 资源部门的扩张提高了相关要素的报酬（S-S 效应）
- 但通过**放大效应**，对其他贸易部门的负面影响超过了产品价格变化的幅度
- 这是 S-S 定理和放大效应在现实中的典型表现

## 📄 来源原文
> 荷兰越是增加天然气的开发，其生产出口产品的制造业就越受到抑制。
