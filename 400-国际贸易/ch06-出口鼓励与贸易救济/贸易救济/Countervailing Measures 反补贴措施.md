# 反补贴措施 (Countervailing Measures)

## 📌 定义
反补贴是指进口国主管机构依法对接受补贴的进口产品进行调查，并通过征收**反补贴税**或**价格承诺**等方式，抵消由禁止性补贴和部分可诉性补贴造成不利影响的贸易救济措施。

> 💬 通俗理解：出口国政府给本国企业发补贴 → 企业成本低、出口价低 → 进口国同行被"不公平竞争" → 进口国征反补贴税把钱"补回来"。

## 🧠 机制

### 反补贴与反倾销的关键区别
- 反倾销针对**企业**的倾销行为
- 反补贴针对**政府或公共机构**的补贴行为

### 外部基准问题
- 根据《中国加入WTO议定书》第15条(b)：对中国进行反补贴调查时，如使用中国数据存在"特殊困难"，可使用**第三方数据（外部基准）**
- 条款纪律性不强，易被滥用（如用其他国家银行贷款利率作为比较基准）
- 外部基准的使用期限未作限定 → 是我企业被裁定**高额反补贴税率**的重要原因

### 中国反补贴第一案
- 2004年加拿大对中国烧烤架发起反补贴调查（首例对华反补贴案）
- 初裁16%临时反补贴税 → **终裁终止调查**，退还已征临时关税 → 中方首战胜利

### 反补贴与市场经济地位
- 美国及欧盟以往认为反补贴不适用于非市场经济国家
- 近年政策转向：美国国会提案允许反补贴法适用于非市场经济国家
- 欧盟开始收集研究中国企业获政府补贴情况
