# 反倾销 (Anti-Dumping)

## 📌 定义
反倾销是指对外国商品在本国市场上倾销造成的损害所采取的救济措施。按照WTO《反倾销协定》，成员方采取反倾销措施需满足三个条件：存在倾销、造成实质损害、倾销与损害间存在因果关系。措施以**反倾销税**或**价格承诺**形式执行。

> 💬 通俗理解：外国企业在本国以低于"正常价"的价格卖东西，损害了本国同行 → 政府征收反倾销税，把这部分"不公平价差"补回来。

## 🧠 机制

### 三大实施条件

| 条件 | 内容 |
|------|------|
| **存在倾销** | 出口价格 < 正常价值（出口国国内售价/第三国售价/结构价格） |
| **存在损害** | 对国内产业造成实质损害/实质损害威胁/实质阻碍 |
| **因果关系** | 倾销与损害之间存在因果联系（排除其他因素） |

### 确定产业损害时的审查要素
1. 倾销进口产品**数量**是否有大量增加
2. 倾销产品对国内生产商的**冲击程度**（销售额、利润、市场份额）
3. 倾销产品是否**压低**了国内同类产品价格

### 措施形式
- **反倾销税**：征收期限不超过5年，经复审可延长
- **价格承诺**：出口商自愿提价，进口国不征反倾销税

### 反倾销复审制度
| 复审类型 | 目的 |
|----------|------|
| 日落复审（期终复审） | 决定措施是否到期终止 |
| 全面期中复审 | 审查措施是否继续必要 |
| 部分期中复审 | 调整税率 |
| 新出口商复审 | 为新出口商确定单独税率 |
| 反规避复审 | 防止企业规避反倾销措施 |

### 反倾销 vs 市场经济地位
- **非市场经济国家**的出口产品 → 进口国可用**第三国（替代国）**的价格数据测算正常价值
- 替代国往往成本远高于被调查国 → 倾销极易成立 → 倾销幅度被高估
- 美国对中国反倾销税平均税率**98%**，对市场经济国家仅**37%**
- 2018年以来美国18项涉华裁决中14项税率在**100%以上**

### 案例：中美彩电案（2003-2004）
- 中国彩电对美出口从2000年209,887套激增至2002年2,656,456套（增长**1166%**）
- 美国用**印度**作为替代国（中国被视为非市场经济）
- 终裁：厦华5.22%、长虹26.37%、TCL 21.25%、康佳9.69%

### 案例：中欧光伏争端（2012-2013）
- 欧盟对中国光伏产品发起反倾销调查，涉及200多亿美元对欧出口
- 最终达成**价格承诺协议**：94家中国企业以约€0.56/W最低售价出口
- 配额分配方案"631"：60%按历史份额 + 30%给抗辩企业 + 10%扶持出口小企业

## 📄 来源原文
> 按照WTO《反倾销协定》规定，成员方采取反倾销措施需具备三个条件：确定某一进口产品存在倾销；由此对该进口国内已建立的相关产业造成实质损害；倾销与损害之间存在因果关系。反倾销措施以征收反倾销税、价格承诺等形式执行。
