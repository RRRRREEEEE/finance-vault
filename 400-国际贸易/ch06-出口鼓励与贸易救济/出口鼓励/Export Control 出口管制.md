# 出口管制 (Export Control)

## 📌 定义
出口管制是国家对某些商品的出口实施限制或禁止的措施。虽名为"出口鼓励"一章，但并非所有出口都被鼓励——战略物资、国内紧缺品、文物等反而要限制出口。

## 🔗 前置知识
[[Import Quotas 进口配额]]、[[Non-Tariff Barriers 非关税壁垒]]

## 🧠 机制

### 管制方式
国家专营或垄断、出口税、出口配额、出口禁运、出口许可证

### 目标商品
| 类别 | 管制原因 |
|------|----------|
| 战略物资、尖端技术及资料 | 国家安全 |
| 国内紧缺物资 | 保障国内供应 |
| 历史文物和艺术珍品 | 文化遗产保护 |
| 需自愿限制出口的商品 | 国际贸易协议（如VER） |
| 受经济制裁的商品 | 外交政策工具 |

## 📄 来源原文
> 出口管制方面的措施：国家专营或垄断；出口税；出口配额；出口禁运；出口许可证。目标商品包括战略物资、尖端技术及其资料；国内紧缺物资；历史文物和艺术珍品等。
