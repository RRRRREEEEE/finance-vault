# 第6章 知识串讲：政府如何帮本国企业出口，又如何防御别国不公平贸易？

## 一、一句话回答
政府用补贴/信贷/特区帮企业出口，用反倾销/反补贴/保障措施防御别国的不公平（甚至公平）贸易——出口端千方百计推，进口端千方百计挡。

---

## 二、出口鼓励：左手给钱，右手给政策

这一章前半部分讲的是"怎么帮本国企业多出口"。

**出口补贴**是核心工具。WTO把补贴分成红灯/黄灯/绿灯——红灯（出口补贴+进口替代补贴）完全禁止；黄灯（国内支持补贴，有专向性）可以用但可能被反补贴；绿灯（研发/环保/落后地区）条款已失效。美国自己2000-2015年就补贴企业680亿美元，波音一家拿了145亿。

补贴的经济效应和关税正好相反——但结果都是$-(b+d)$的净福利损失。区别在哪？关税是消费者受损、生产者+政府受益；出口补贴是消费者+纳税人双输、只有出口厂商赚。为什么会这样？因为补贴让出口比内销更有利可图，厂商把产品全拉去出口了，国内供给不足、价格上升、消费者遭殃。政府还得为补贴买单。

除了直接补贴，还有：**出口信贷**（政府担保的优惠贷款，降低融资成本）、**外汇倾销**（本币贬值让出口变便宜）、**出口退税**（退还出口前缴的税，WTO认为是合法的）、**出口管制**（不是所有的都鼓励——战略物资、文物、紧缺品要限制）。

## 三、经济特区：国境内的"小外国"

经济特区的逻辑很简单：在国内划一块地方，海关不管、关税不征，让外资进来加工再出口。全球1200多个自贸区中，发展中国家占了65%以上。

自贸区有四个级别：最低级的准自贸区只是物理隔离加特殊监管；最高级的是自由港——如香港、迪拜——完全无关税。中国的六个海关特殊监管区是"境内关内"，还不是真正意义的自贸区。

**中国的FTZ战略**从2013年上海开始，到目前22个。两大制度创新：一是**负面清单管理**（外商投资禁入的才写上去，其他全放开），清单从190条砍到27条，制造业条目清零；二是**国际贸易单一窗口**（海关/税务/外汇一个窗口全搞定）。

**海南自贸港**是压轴大戏。2025年底全岛封关——一线（对境外）放开，货物自由进出不申报；二线（对内地）管住，进内地要补税；岛内完全自由。三张零关税清单、15%所得税、全国最短的负面清单（27项）。海南将成为中国的"香港"。

## 四、贸易救济三件套：反倾销、反补贴、保障措施

这一章的后半部分从"进攻"转向"防御"——当别国的产品涌进来损害了国内产业，怎么办？

### 反倾销：最常用的武器

**倾销**就是同一产品在国外卖得比国内便宜——价格歧视。有三种类型：偶然性的（清仓甩卖）、掠夺性的（先低价杀对手再提价赚垄断利润）、持续性的（长期在两个市场实行价格歧视）。

持续性倾销能存在需要三个条件：企业有垄断力、国内外需求弹性不同、出口国贸易壁垒够高防止回流。

WTO对反倾销规定了三个条件：存在倾销、造成实质损害、两者有因果关系。措施可以是反倾销税（最多5年+可延长）或价格承诺（出口商自愿提价）。

**案件规模惊人**：2008-2019年全球反倾销2462起，占贸易救济案件的80.8%。中国是最大目标——遭受1150起，其中美国发起177起。

### 市场经济地位：制度性歧视

这里有个关键的"潜规则"：如果出口国被认定为**非市场经济国家**，进口国就可以用一个成本更高的"替代国"来算倾销幅度。结果？美国对中国的反倾销税平均98%，而对市场经济国家才37%。中国入世15年后按约定应获市场经济待遇，但美欧至今不认。

中美彩电案是典型案例：中国彩电对美出口3年增1166%，美国用**印度**做替代国算成本（印度彩电比中国贵20-40%），结果倾销成立。

中欧光伏案更震撼：涉及200多亿美元出口、上千家企业、40多万人就业。最终通过**价格承诺**和解：94家企业以€0.56/W的最低价出口，配额按"631"方案分配。

### 反补贴：打政府的武器

这里针对的不是企业行为，而是**政府补贴**。关键陷阱是"外部基准"——对中国做反补贴调查时，可以用第三国的贷款利率做基准。中国2004年遭遇第一起反补贴调查（加拿大烧烤架案），最终反补贴调查被终止，中方首战告捷。

### 保障措施：对"好人"也能用的紧急刹车

反倾销打"坏人"、反补贴打"被帮凶的"，保障措施最特殊——打的是"好人"。哪怕进口完全没有不正当行为，只要数量激增对国内产业造成严重损害，就可以临时限制进口。这是WTO给各国的"安全阀"。

保障措施必须非歧视（对所有来源国一视同仁），但**特保措施**可以针对单一国家——中国入世时被迫接受的这个条款，已于2013年到期。

## 五、美国贸易法的四把刀

美国除了WTO框架下的救济措施，还有自己的国内法武器：

- **337调查**：专利警察——只要侵犯美国知识产权，直接禁止进入美国市场，不需要证明产业损害
- **201条款**：美国版保障措施——进口激增伤产业，总统可出手。光伏201案首年征30%关税
- **301条款**：贸易法的瑞士军刀——"你不公平我就制裁"。特朗普对华发起301调查→500亿+2000亿美元商品加征关税→中美贸易战
- **232调查**：国家安全牌——钢铁25%、铝10%关税。9个WTO成员起诉美国。又要对汽车下手

注意这四个工具的共同特征：**单边主义**——不需要WTO授权，美国自己说了算。

## 六、概念对照表

| 概念对 | 区别 |
|--------|------|
| 反倾销 vs 反补贴 | 反倾销打企业(倾销)，反补贴打政府(补贴) |
| 反倾销/反补贴 vs 保障措施 | 前者针对不公平贸易，后者针对公平贸易下的进口激增 |
| 保障措施 vs 特保措施 | 保障措施非歧视(对所有国家)，特保针对单一国家(歧视性) |
| 禁止性补贴 vs 可诉性补贴 | 红灯完全禁(与出口挂钩)，黄灯可实施但可被反补贴 |
| 价格承诺 vs 反倾销税 | 价格承诺是出口商自愿提价(无税)，反倾销税是强制征税 |
| 境内关内 vs 境内关外 | 境内关内海关仍管(中国现有关区)，境内关外海关撤离(国际自贸区) |
| 337 vs 301 | 337打知识产权侵权(禁入产品)，301打贸易行为不公(制裁国家) |

## 七、最重要的故事 / 考试最易考点

**必考1：出口补贴的经济效应分析。** 画出小国出口补贴的S-D图，标出$a,b,c,d$四个区域，说明消费者$-(a+b)$、生产者$+(a+b+c)$、政府$-(b+c+d)$、净福利$-(b+d)$。关键结论：出口国必定受损。

**必考2：反倾销的三个条件。** 倾销存在（出口价<正常价值）+实质损害+因果关系。能写出倾销幅度公式。

**必考3：贸易救济三件套对比表。** 反倾销(打企业倾销)、反补贴(打政府补贴)、保障措施(打进口激增，非歧视)。措施形式、针对对象的区别。

**次重点：中国自贸试验区的两项制度创新**——负面清单管理（190→27条）+ 国际贸易单一窗口。
