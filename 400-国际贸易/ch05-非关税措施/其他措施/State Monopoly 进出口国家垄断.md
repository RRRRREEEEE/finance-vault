# 进出口国家垄断 (State Monopoly / State Trading)

## 📌 定义
进出口国家垄断（国营贸易）是指在对外贸易中，某些商品的进出口由**国家直接经营**或把专营权授予某个垄断组织。通常涉及四类商品：烟酒（财政收入）、农产品（非传统安全）、石油（经济命脉）、武器（世界和平）。

## 🔗 前置知识
[[Non-Tariff Barriers 非关税壁垒]]

## 🧠 机制

| 垄断商品类别 | 垄断理由 |
|-------------|----------|
| **烟酒** | 财政收入 |
| **农产品** | 非传统安全（粮食安全） |
| **石油** | 国家经济命脉 |
| **武器** | 世界和平与国家安全 |

## 📄 来源原文
> 进出口的国家垄断：又称为国营贸易，是指在对外贸易中，某些商品的进出口由国家直接经营或把某些商品的进出口专营权给予某个垄断组织。
