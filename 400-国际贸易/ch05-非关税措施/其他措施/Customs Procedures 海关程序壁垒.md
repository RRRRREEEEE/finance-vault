# 海关程序壁垒 (Customs Procedures as Trade Barriers)

## 📌 定义
海关程序壁垒是指通过设置苛刻、繁琐的海关程序和特定要求来限制进口的非关税措施。看似中性的行政程序，实际操作中成为贸易障碍。

## 🔗 前置知识
[[Non-Tariff Barriers 非关税壁垒]]

## 🧠 机制

### 四种主要方式

| 方式 | 做法 |
|------|------|
| **申报表格与单证要求苛刻** | 要求大量复杂单据，增加进口商行政成本 |
| **通过货物归类提高税率** | 故意将进口商品归入高税率税目 |
| **通过海关估价提高关税** | 高估进口商品价值以增加关税 |
| **通过进口货物查验限制** | 频繁、严格、耗时的货物查验 |

### 案例：普瓦蒂埃之战
法国为限制日本录像机进口，将全部进口录像机的海关检验集中在偏远的普瓦蒂埃（Poitiers）小镇，且仅配备少量检验人员，人为制造通关瓶颈。
