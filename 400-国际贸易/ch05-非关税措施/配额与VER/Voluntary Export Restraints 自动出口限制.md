# 自动出口限制 (Voluntary Export Restraints, VER)

## 📌 定义
自动出口限制是指出口国在进口国的**要求或压力下**，"自愿"规定一定时期内某些商品向该进口国的出口量，在配额内自行控制出口，超过配额即禁止出口。VER 属于关贸总协定的**灰色区域措施**（Gray Area Measures），名义上自愿、实质上是被迫的数量限制。

> 💬 通俗理解：你邻居嫌你家太吵，暗示你自己控制音量——否则他会采取更严厉的手段。你"自愿"降低了音量。VER 就是这样：出口国在进口国威胁下"自愿"限制出口。

## 🧠 机制

### VER 与进口配额的差异
- 进口配额由**进口国**管理 → 配额租 $c$ 归属于进口国（政府或进口商）
- VER 由**出口国**管理（出口国分配出口许可证）→ 配额租 $c$ 归属于**出口国** → 进口国净损失更大（$b+d+c$）

### 案例一：多边纤维协定（MFA）
- 1974年签订，通过出口国"自动"约束纺织品出口来实现进口国数量限制
- 全球纺织品贸易长期游离于自由贸易体制之外
- 乌拉圭回合签订《纺织品与服装协定》（ATC），设十年过渡期：
  - 1995：16%产品一体化 → 1998：17% → 2002：18% → 2005：剩余49%全部一体化
- 实际上欧美将70%配额保留到最后，且多为敏感紧俏类别
- 2005年取消限制后：中国出口到欧洲的紧身衣增长2000%、针织衫增长1000%

### 案例二：日本汽车对美VER
- 1981-83：日本对美汽车出口限额168万辆/年；1985年后230万辆/年
- 后果：美国汽车平均价格上升43%（1979-1981）
- 日本进口车价格从$5,150升至$8,050（1980-1985），其中：
  - $1,100：日本生产商获得的配额租
  - $1,650：品质升级（更重更宽、马力更大）
  - $150：自由贸易下应有的涨价
