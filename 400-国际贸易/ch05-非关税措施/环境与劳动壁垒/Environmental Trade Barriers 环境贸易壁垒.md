# 环境贸易壁垒 (Environmental Trade Barriers, ETBs)

## 📌 定义
环境贸易壁垒（绿色贸易壁垒）是以保护自然资源、生态环境和人类健康为名，通过制定一系列复杂苛刻的环保制度和标准，对来自其他国家和地区的产品及服务设置障碍，限制进口的新型非关税壁垒。始于20世纪80年代，凭借**合法性、合理性、隐蔽性**迅速成为各国保护国内市场的重要手段。

## 🧠 机制

### WTO 法律基础
WTO《技术性贸易壁垒协议》（TBT）明确指出：不应阻止任何国家采取必要的措施以保护其居民、动植物及环境的基本安全利益。这一条款为绿色壁垒提供了合法性。

### 五种主要形式

| 形式 | 内容 |
|------|------|
| **多边环境协议** | 如《蒙特利尔议定书》关于消耗臭氧层物质的规定 |
| **环境技术法规与标准** | 如 ISO14000 环境管理体系 |
| **产品检疫检验制度** | 以环保为由的产品检验要求 |
| **绿色标志** | 环保认证标签 |
| **绿色包装与标签** | "4R"原则：Reduce(减量)、Refill(再填充)、Recycle(可循环)、Recovery(可回收) |

### ISO14000
为促进全球环境质量改善而制定的环境管理体系国际标准，内容包括：环境管理体系、环境审核、环境标志、环境行为评价、生命周期评估、术语和定义、产品标准中的环境指标。

### 趋势
- 2008-2021年全球以保护人类、动植物生命健康安全与保护环境为目的的贸易通报累计**16875例**，呈逐年增加态势
- 欧美发达国家环保标准趋向协调一致、相互承认
