# 劳动壁垒 (Labor Barriers / Blue Trade Barriers)

## 📌 定义
劳动壁垒（蓝色贸易壁垒）是指以劳动者**劳动环境和生存权利**为借口采取的贸易保护措施。核心争议在于：是否应将劳工标准与国际贸易挂钩。

> 💬 通俗理解：发达国家说"你们发展中国家工人工资低、劳动条件差，这是不公平竞争"。发展中国家反驳说"你们是用劳工标准当幌子，实际是保护主义"。这个争论至今没有定论。

## 🧠 机制

### 六种表现形式

| 形式 | 做法 |
|------|------|
| 征收附加税 | 对违反国际公认劳工标准的国家的产品征收附加税 |
| 限制/禁止出口 | 限制或禁止严重违反基本劳工标准的产品出口 |
| 贸易制裁 | 以劳工标准为由实施贸易制裁 |
| 工厂审核（客户验厂） | 跨国公司对供应商工厂的劳工条件审查 |
| 社会责任工厂认证 | 第三方认证机构对工厂劳工条件的认证 |
| 社会责任产品标志 | 在产品上标注符合劳工标准的标志 |

### 南北争议

| 立场 | 观点 |
|------|------|
| **发达国家** | 发展中国家较低的劳工标准使发达国家在成本价格和贸易竞争中处于不利地位，应将劳工标准纳入双边和多边贸易体系 |
| **发展中国家** | 担心劳工标准与国际贸易挂钩或滥用可能导致非关税壁垒，削弱本国企业国际竞争力 |
