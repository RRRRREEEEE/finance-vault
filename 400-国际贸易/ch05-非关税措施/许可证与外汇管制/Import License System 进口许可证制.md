# 进口许可证制 (Import License System)

## 📌 定义
进口许可证制是指商品的进口必须得到国家有关机构的批准、领取许可证后方可进口，否则一律不准进口的行政管理措施。按与配额的关系分为有定额的进口许可证和无定额的进口许可证。

## 🔗 前置知识
[[Import Quotas 进口配额]]、[[Non-Tariff Barriers 非关税壁垒]]

## 🧠 机制

### 按与配额的关系分类

| 类型 | 规则 |
|------|------|
| **有定额的进口许可证** | 在配额限度内发放许可证 |
| **无定额的进口许可证** | 不与配额挂钩，单独审批 |

### 中国实践
- 2002年1月1日起取消粮食、羊毛、棉花、化肥等产品的配额许可证
- 2007年取消汽车等产品的配额许可证管理

## 📄 来源原文
> 商品的进口必须得到配额许可证管理，此后又分期分批取国家有关机构的批准，领取许可证后方可进口，否则一律不准进口。
