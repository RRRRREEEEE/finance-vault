# 关税 (Tariff / Customs Duties)

## 📌 定义
关税是一国政府通过海关对经过本国关境的进出口商品征收的赋税。关税是贸易壁垒的主要形式之一（另一类为非关税壁垒）。关境一般等于国境，但设有自由贸易区时关境<国境，关税同盟时关境>国境。

## 🧠 机制

### 关税分类体系

**1. 按征收目的**
| 类型 | 目的 |
|------|------|
| **财政关税**（Revenue Tariff）| 增加本国财政收入 |
| **保护关税**（Protective Tariff）| 保护本国国内产业 |

**2. 按商品流向**
| 类型 | 说明 |
|------|------|
| **进口税**（Import Duties）| 含普通税和最惠国税（MFN）|
| **出口税**（Export Duties）| 对输往国外的本国商品征收 |
| **过境税**（Transit Duties）| 对通过关境的外国商品征收 |

**3. 按征收方法**
| 类型 | 英文 | 计征标准 |
|------|------|----------|
| **从量税** | Specific Duties | 按重量、体积、数量等计量单位 |
| **从价税** | Ad Valorem Duties | 按商品价格的百分比 |
| **复合税** | Mixed/Compound Duties | 同时使用从量税和从价税 |
| **选择税** | Alternative Duty | 规定两种税率，由海关选择 |

**4. 按差别待遇**
| 类型 | 说明 |
|------|------|
| **差价税**（Variable Levy）| 按国内价与进口价的差额征收 |
| **特惠关税**（Preferential Duties）| 对特定国家/地区给特别优惠（如洛美协定）|
| **普惠制**（GSP）| 发达国家对发展中国家的普遍关税减免 |
| **进口附加税**（Import Surtax）| 含反倾销税、反补贴税，WTO 允许的临时措施 |

### 当前贸易环境
- 全球贸易保护主义 2008 年后显著上升
- 美国 2008-2016 采取 600+ 项保护措施，位居全球之首
- 2025 年 4 月美国"对等关税"波及近 60 国，中国商品叠加税率高达 54%
