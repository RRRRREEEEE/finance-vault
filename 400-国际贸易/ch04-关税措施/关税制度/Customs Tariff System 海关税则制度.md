# 海关税则制度 (Customs Tariff System)

## 📌 定义
海关税则是国家对进出口商品计征关税的规章和对各类商品分类及税率安排的一览表。海关税则制度分为单式/复式税则（按税率栏数）和自主/协定税则（按制定方式）。

## 🧠 机制

### 按税率栏数

| 类型 | 说明 | 特点 |
|------|------|------|
| **单式税则**（Single Tariff）| 每种商品只有一个税率，不论产地 | 无歧视 |
| **复式税则**（Complex Tariff）| 一个税目设两个或以上税率，对不同国家采用不同税率 | 歧视性，世界上大多数国家采用 |

### 按制定方式

| 类型 | 说明 |
|------|------|
| **自主税则**（Autonomous Tariff）| 国家独立自主制定，分自主单一税则和自主多重税则 |
| **协定税则**（Conventional Tariff）| 通过与其他国家订立贸易条约或协定确定 |
| **混合税则** | 同时采用自主和协定两种方式 |

### 中国关税减让
- 2002 年起逐年调低进口关税，总水平从 15.3% → 9.8%
- 农产品平均税率：18.8% → 15.2%
- 工业品平均税率：14.7% → 8.9%
- 入世降税承诺已**全部履行完毕**
- 2006 年汽车整车进口关税从 70-80% 降至 25%

## 📄 来源原文
> 单式税则对每一种应税商品不论产于何地，每个税则项号下都只规定一个税率。复式税则一个税目设有两个或以上的税率，以便对来自不同国家的进口商品采用不同的税率。
