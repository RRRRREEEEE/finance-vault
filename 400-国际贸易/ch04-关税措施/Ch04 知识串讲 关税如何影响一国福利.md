# 第4章 知识串讲：关税如何影响一国福利？

## 一、一句话回答
关税保护了国内生产者，但以**更大的消费者损失**为代价——小国必然受损，大国可能通过改善贸易条件而获益。

---

## 二、关税是什么

关税就是海关对进出口商品征的税。听起来简单，但分类非常细。

**按目的分**：财政关税（为了收钱）vs 保护关税（为了保护国内产业）。

**按流向分**：进口税（最常见）、出口税、过境税。进口税里又分普通税和最惠国税（MFN）——MFN 是 WTO 非歧视原则的核心，你对一个国家征的税不能比对其他国家的更高。

**按征收方法分**：从量税（按重量/体积）、从价税（按价格的%）、复合税（两者结合）、选择税（海关选高的那个）。

**按差别待遇分**：差价税（把进口价拉到和国内价一样）、特惠关税（洛美协定——欧盟对非加太国家免关税免限量）、普惠制 GSP（发达国家对发展中国家普遍减免，三原则：普遍、非歧视、非互惠）、进口附加税（反倾销+反补贴，WTO允许的临时措施）。

---

## 三、小国 vs 大国：关税的福利效应完全不同

这是本章最核心的知识点。

**小国模型**：你是价格接受者，征关税 → 国内价格从 $P_E$ 涨到 $P_E+t$ → 消费者买少了，国内低效生产者进入 → 政府收到关税 $c$，但消费者损失了 $a+b+c+d$。净福利 $= -(b+d)$ ——**小国征关税一定吃亏**。

其中 $b$ 是生产扭曲（低效国内生产替代了高效进口），$d$ 是消费扭曲（消费者被迫放弃购买）。两者合起来叫**无谓损失**（Deadweight Loss）。

**大国模型**：你进口量大，能压价。征关税 → 外国为维持出口降价（从 $P_E$ 降到 $P_T'$）→ 你获得**贸易条件改善** $e$。净福利 $= e-(b+d)$ ——可能为正。

这就是为什么美国敢打关税战——它是大国，关税可能让它净赚。

---

## 四、最优关税：大国的甜蜜点

既然大国征关税可能赚钱，那征多少最赚？

$$t^* = \frac{1}{e_x}$$

最优关税 = 外国供给弹性的倒数。对方越离不开你的市场（弹性越小），你越能征高税。

在零关税和禁止性关税之间，存在一个**最优关税率**使本国福利最大化。超过这个点，效率损失超过贸易条件改善，福利开始下降。**小国最优关税为零**——因为小国不能压价，$e=0$。

---

## 五、有效保护率：名义税率骗了你

你以为对汽车征 80% 的关税就是保护了 80%？错了。

**有效保护率（ERP）**看的是关税让这个行业的**附加值增加了多少**。

$$ERP = \frac{V' - V}{V}$$

数字最直观。汽车 $150，发动机 $50，配件 $50，自由贸易下附加值 $V=50$。
- 只对汽车征 80% → $V'=170$ → $ERP=240\%$！
- 对发动机也征 50% → $V'=145$ → $ERP=190\%$

结论：**对中间品征税会削弱对最终产品的有效保护**。

这直接导向**关税升级**（Tariff Escalation）：各国普遍对原材料征最低税、半加工品中等税、制成品最高税。这保护了发达国家的加工业，但阻碍了原材料出口国发展自己的加工业——这是全球贸易体系的一个结构性不公平。

## 末、概念对照表

| 概念对                | 区别                                         |     |
| ------------------ | ------------------------------------------ | --- |
| **小国 vs 大国关税效应**   | 小国净损失 $-(b+d)$（必然受损）；大国 $e-(b+d)$（可能获益）    |     |
| **名义保护率 vs 有效保护率** | 名义看最终产品关税；有效看附加值的实际增幅，考虑中间品关税              |     |
| **生产扭曲 vs 消费扭曲**   | $b$：低效国内生产替代高效进口；$d$：消费者被迫放弃消费             |     |
| **MFN vs GSP**     | MFN：非歧视正常税率（WTO 核心原则）；GSP：发达国家对发展中国家的单方面优惠 |     |
| **从量税 vs 从价税**     | 从量：按数量/重量征固定额；从价：按价格征百分比                   |     |

## 末+1、最重要的故事 / 考试最易考点

**关税福利效应的推导链条**：

```
征关税 → 国内价格↑
  → 消费效应：消费者剩余减少 -(a+b+c+d)
  → 生产效应：生产者剩余增加 +a
  → 收入效应：政府关税收入 +c(+e 大国)
  → 小国净福利：-(b+d) ← 必然为负
  → 大国净福利：e-(b+d) ← 取决于贸易条件改善 vs 效率损失
```

**ERP 计算题套路**：给最终产品国际价、中间品价、最终产品关税率、中间品关税率 → 算 $V$ → 算 $V'$ → 算 $ERP$ → 比较与名义税率的差异。
