# 出口导向战略 (Export Promotion Strategy)

## 📌 定义
出口导向战略（外向型发展战略）以**国际市场**为重点，通过政府支持具有潜在比较优势的制造业部门，扩大出口来带动本国工业化和经济发展。与进口替代截然相反——不是关起门来自己造，而是打开门去国际市场竞争。

> 💬 通俗理解：进口替代是"自己练好了再出门"，出口导向是"边打边练"——把产品直接卖到国际市场竞争，在竞争中变强。

## 🔗 前置知识
[[Import Substitution Industrialization 进口替代战略]]

## 🧠 机制

### 政府支持措施
- 产业政策扶持
- 补贴贷款
- 研发补贴
- "奖出限入"贸易政策（关税、进口配额、出口补贴并用）

### 实施效果

| 指标 | 结果 |
|------|------|
| **增长率** | 1960s中期至今年均**8-9%** |
| **出口** | 贸易量大幅增加，制成品出口占比上升 |
| **贸易开放度** | 贸易/GDP比率显著提高 |
| **劳动生产率** | 低效率企业退出 + 存活企业生产率提升 → 整体生产率提高 |

但需注意：贸易自由化往往与其他改革同时进行，**难以单独分离**贸易政策的效果（部分国家自由化后经济增长并未起飞）。

### 东亚奇迹 (East Asian Miracle)
- **亚洲四小龙**：香港、台湾、韩国、新加坡 — 1960s起飞
- 1970-80s跟进：马来西亚、泰国、印度尼西亚、中国
- 1980s末大多数发展中国家认识到**幼稚产业保护并不成功**，开始转向贸易自由化

### 韩国案例（1955-1968）

| 时期 | 出口扩张贡献 | 进口替代贡献 |
|------|------------|------------|
| 1955-60 | 12.9% | 10.2% |
| 1960-63 | 6.3% | -6.9% |
| 1963-66 | **31.4%** | 8.9% |
| 1966-68 | 21.3% | -6.6% |

> 韩国经济增长从1963年起由出口扩张主导，进口替代贡献逐渐为负。
