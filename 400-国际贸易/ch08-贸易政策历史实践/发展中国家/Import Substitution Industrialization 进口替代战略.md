# 进口替代战略 (Import Substitution Industrialization, IS)

## 📌 定义
进口替代战略（内向型发展战略）由阿根廷经济学家**普雷维什**（Paul Prebisch）提出，通过建立和发展本国制造业来替代制成品的进口，以高关税和贸易壁垒保护国内幼稚工业，从而加快工业化和经济发展。

> 💬 通俗理解：与其花钱进口外国货，不如自己造。关起门来用高关税把外国产品挡在外面，让本国企业在本国市场里练手成长。

## 🔗 前置知识
[[Infant Industry Argument 幼稚产业保护论]]

## 🧠 机制

### 做法
- 以高关税和进口限制保护国内制造业
- 政府主导工业化进程
- 面向**国内市场**（内向型）

### 效果
- 拉美国家（墨西哥、巴西）在1950s-1960s制造业有所发展
- 但印度的数据显示**不太奏效**

### 四大弊端

| 弊端 | 表现 |
|------|------|
| **保护对象选错** | 应该保护的对象是能自力更生的产业；但既得利益使保护一旦开始就难以撤除 |
| **低效率企业存活** | 过高保护率让效率低下的企业也能生存（有时在被保护的还是"错误"行业） |
| **国内市场狭小** | 难以实现规模经济 |
| **保护成本过高** | 高关税提高企业进口投入品成本，消费者支付高价；涉及复杂耗时的行政管制 |
